## 学习建议

### 优先级排序（由高到低）

| 优先级 | 类别 | 说明 |
|--------|------|------|
| ⭐⭐⭐⭐⭐ | Go语言 | 后端主力语言，语法+并发+GC必须精通 |
| ⭐⭐⭐⭐⭐ | 数据库 | MySQL索引/事务/锁/MVCC是面试重灾区 |
| ⭐⭐⭐⭐⭐ | 计算机网络 | TCP/HTTP/HTTPS几乎每场必问 |
| ⭐⭐⭐⭐⭐ | 操作系统 | 进程/线程/内存/IO是底层基础 |
| ⭐⭐⭐⭐⭐ | 数据结构与算法 | 手撕代码的基础 |
| ⭐⭐⭐⭐ | 缓存 | Redis是后端标配，面试高频 |
| ⭐⭐⭐⭐ | 消息队列 | Kafka/RabbitMQ中大厂必问 |
| ⭐⭐⭐⭐ | 分布式系统 | 中大厂面试核心考察点 |
| ⭐⭐⭐⭐ | 设计模式 | 体现编码素养和设计能力 |
| ⭐⭐⭐ | 系统架构 | 高级岗位/架构师必备 |
| ⭐⭐⭐ | 鉴权与安全 | 项目实战常考 |
| ⭐⭐⭐ | API设计 | 体现工程规范和实战经验 |
| ⭐⭐⭐ | 代码整洁与规范 | 体现编码素养 |
| ⭐⭐ | Python语言 | 脚本/数据/AI场景，按需学习 |
| ⭐⭐ | 后端工程实践 | 综合实战能力 |
