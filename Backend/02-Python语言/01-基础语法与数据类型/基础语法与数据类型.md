## 1. Python有哪些基本数据类型？它们是如何分类的？

**回答：**

Python 的基本数据类型按可变性分为两大类：

```
┌──────────────────────────────────────────────────┐
│             Python 数据类型分类                     │
├────────────────────┬─────────────────────────────┤
│    不可变类型        │       可变类型               │
├────────────────────┼─────────────────────────────┤
│  int   - 整数       │  list     - 列表            │
│  float - 浮点数     │  dict     - 字典             │
│  bool  - 布尔值     │  set      - 集合             │
│  str   - 字符串     │  bytearray - 字节数组        │
│  tuple - 元组       │                             │
│  frozenset - 冻结集合│                             │
│  bytes - 字节串     │                             │
│  NoneType           │                             │
└────────────────────┴─────────────────────────────┘
```

```python
# 类型检查方式
type(42)          # <class 'int'>
isinstance(42, int)  # True（推荐，支持继承检查）

# Python中一切皆对象
x = 42
print(id(x))     # 对象的内存地址
print(type(x))   # 对象的类型

# 小整数池：CPython 缓存 [-5, 256] 的整数
a = 256
b = 256
print(a is b)  # True（同一对象）

a = 257
b = 257
print(a is b)  # False（不同对象，交互模式下）
```

面试要点：`is` 比较的是对象身份（id），`==` 比较的是对象值。

## 2. list和tuple的区别是什么？各自的底层实现原理？

**回答：**

| 特性 | list | tuple |
|------|------|-------|
| 可变性 | 可变 | 不可变 |
| 底层结构 | 动态数组（指针数组+过度分配） | 固定大小数组 |
| 内存占用 | 较大（预留空间） | 较小 |
| 创建速度 | 较慢 | 更快（可被缓存复用） |
| 可哈希 | ❌ | ✅（元素都可哈希时） |
| 增删操作 | 支持 | 不支持 |

```python
import sys

# 内存差异
lst = [1, 2, 3, 4, 5]
tup = (1, 2, 3, 4, 5)
print(sys.getsizeof(lst))  # 104（含预留空间）
print(sys.getsizeof(tup))  # 80

# list 的动态扩容机制
# 当 append 超出容量时，按 0, 4, 8, 16, 24, 32, 40, 52, 64... 增长
lst = []
for i in range(10):
    lst.append(i)
    print(f"len={len(lst)}, size={sys.getsizeof(lst)}")
```

```
┌───────────────── list 底层结构 ─────────────────┐
│                                                 │
│  PyListObject:                                  │
│  ┌──────────┐     ┌───┬───┬───┬───┬───┬───┐    │
│  │ ob_size=3 │────→│ * │ * │ * │   │   │   │    │
│  │allocated=6│     └─┬─┴─┬─┴─┬─┴───┴───┴───┘    │
│  └──────────┘       ↓   ↓   ↓   (预留空间)       │
│                    obj  obj  obj                  │
└──────────────────────────────────────────────────┘
```

**使用建议**：数据不需要修改时优先用 tuple，性能更好且语义更明确。

## 3. dict的底层实现原理？哈希冲突如何解决？

**回答：**

Python dict 从 3.6 开始采用**紧凑字典（compact dict）**实现，由两个数组组成：

```
┌──────────── Python 3.6+ dict 结构 ────────────┐
│                                                │
│  indices (哈希索引表，稀疏):                     │
│  ┌────┬────┬────┬────┬────┬────┬────┬────┐     │
│  │ -1 │  1 │ -1 │  0 │ -1 │  2 │ -1 │ -1 │     │
│  └────┴────┴────┴────┴────┴────┴────┴────┘     │
│                                                │
│  entries (紧凑存储，按插入顺序):                  │
│  ┌──────────────────────────────────────┐      │
│  │ [hash, key, value]  # index 0       │      │
│  │ [hash, key, value]  # index 1       │      │
│  │ [hash, key, value]  # index 2       │      │
│  └──────────────────────────────────────┘      │
└────────────────────────────────────────────────┘
```

**哈希冲突解决**：使用**开放寻址法**（open addressing），具体是二次探测的变种。发生冲突时按特定公式寻找下一个空槽位。

```python
# key 必须是可哈希的
# 自定义对象做 key 需要实现 __hash__ 和 __eq__
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __hash__(self):
        return hash((self.x, self.y))

    def __eq__(self, other):
        return self.x == other.x and self.y == other.y
```

**重要特性**：
- Python 3.7+ 字典保证**插入顺序**
- 负载因子超过 2/3 时自动扩容
- 查找/插入/删除平均 O(1)

## 4. set的实现原理和常用操作？

**回答：**

`set` 底层也是哈希表，类似于只有 key 没有 value 的 dict。

```python
# 集合运算
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

a | b    # 并集 {1,2,3,4,5,6,7,8}
a & b    # 交集 {4,5}
a - b    # 差集 {1,2,3}
a ^ b    # 对称差集 {1,2,3,6,7,8}

a.issubset(b)     # a 是否是 b 的子集
a.issuperset(b)   # a 是否是 b 的超集
a.isdisjoint(b)   # a 和 b 是否没有交集

# 集合推导式
even_set = {x for x in range(20) if x % 2 == 0}
```

```
┌────────────── 集合操作时间复杂度 ──────────────┐
│  操作            │  平均     │  最坏          │
├─────────────────┼──────────┼───────────────┤
│  x in s          │  O(1)    │  O(n)         │
│  s.add(x)        │  O(1)    │  O(n)         │
│  s.remove(x)     │  O(1)    │  O(n)         │
│  s | t (并集)    │  O(n+m)  │  O(n+m)       │
│  s & t (交集)    │  O(min)  │  O(n*m)       │
│  s - t (差集)    │  O(n)    │  O(n)         │
└─────────────────┴──────────┴───────────────┘
```

**frozenset**：不可变集合，可以做 dict 的 key 或 set 的元素。

## 5. Python中的浅拷贝和深拷贝有什么区别？

**回答：**

```python
import copy

original = [[1, 2], [3, 4], [5, 6]]

# 浅拷贝：只拷贝外层容器，内部元素仍是引用
shallow = copy.copy(original)    # 或 original[:]、list(original)
shallow[0].append(99)
print(original[0])  # [1, 2, 99]  ← 被影响了！

# 深拷贝：递归拷贝所有层级的对象
deep = copy.deepcopy(original)
deep[0].append(88)
print(original[0])  # [1, 2, 99]  ← 不受影响
```

```
┌────────────── 浅拷贝 vs 深拷贝 ──────────────┐
│                                              │
│  浅拷贝 (shallow copy):                       │
│  original ──→ [ref1, ref2, ref3]             │
│  shallow  ──→ [ref1, ref2, ref3]  (共享)      │
│                                              │
│  深拷贝 (deep copy):                          │
│  original ──→ [ref1, ref2, ref3]             │
│  deep     ──→ [new1, new2, new3]  (独立)      │
└──────────────────────────────────────────────┘
```

**浅拷贝的方式**：
- `copy.copy(obj)`
- 切片 `lst[:]`
- 构造函数 `list(lst)`、`dict(d)`
- `dict.copy()`、`set.copy()`

**深拷贝注意**：`copy.deepcopy` 会处理循环引用，内部维护一个 memo 字典避免无限递归。

## 6. 列表推导式、字典推导式、集合推导式的用法？

**回答：**

```python
# 列表推导式
squares = [x**2 for x in range(10)]
evens = [x for x in range(20) if x % 2 == 0]

# 嵌套推导
matrix = [[1,2,3], [4,5,6], [7,8,9]]
flat = [x for row in matrix for x in row]  # [1,2,3,4,5,6,7,8,9]

# 字典推导式
word_len = {word: len(word) for word in ["hello", "world", "python"]}
# 反转字典
inv = {v: k for k, v in original_dict.items()}

# 集合推导式
unique_lengths = {len(word) for word in ["hello", "hi", "world"]}

# 生成器表达式（惰性求值，省内存）
total = sum(x**2 for x in range(1000000))
```

**性能对比**：列表推导式通常比等价的 for 循环快 20-30%，因为在 C 层面优化了循环。

**注意**：推导式不宜过于复杂，超过两层嵌套建议改用普通 for 循环以保持可读性。

## 7. Python中的解包(unpacking)操作有哪些？

**回答：**

```python
# 基本解包
a, b, c = [1, 2, 3]
x, y = y, x  # 交换变量

# 星号解包（扩展解包）
first, *rest = [1, 2, 3, 4, 5]    # first=1, rest=[2,3,4,5]
first, *mid, last = [1,2,3,4,5]   # first=1, mid=[2,3,4], last=5
*head, tail = [1, 2, 3, 4, 5]     # head=[1,2,3,4], tail=5

# 嵌套解包
(a, b), (c, d) = (1, 2), (3, 4)

# 函数调用时解包
def add(a, b, c):
    return a + b + c

args = [1, 2, 3]
add(*args)      # 位置参数解包

kwargs = {'a': 1, 'b': 2, 'c': 3}
add(**kwargs)   # 关键字参数解包

# 合并字典（3.5+）
merged = {**dict1, **dict2}

# 合并列表
combined = [*list1, *list2]

# for 循环解包
pairs = [(1, 'a'), (2, 'b'), (3, 'c')]
for num, letter in pairs:
    print(num, letter)
```

## 8. Python中字符串的常用操作和编码问题？

**回答：**

```python
# 字符串是不可变序列
s = "Hello, Python!"

# 常用方法
s.split(', ')       # ['Hello', 'Python!']
', '.join(['a','b']) # 'a, b'
s.strip()           # 去除首尾空白
s.replace('o', '0') # 'Hell0, Pyth0n!'
s.startswith('He')  # True
s.find('Python')    # 7（未找到返回-1）
s.index('Python')   # 7（未找到抛出ValueError）

# f-string 格式化（推荐，3.6+）
name = "World"
f"Hello, {name}!"
f"{3.14159:.2f}"        # '3.14'
f"{'hello':>10}"        # '     hello'
f"{1000000:_}"          # '1_000_000'

# 编码与解码
text = "你好Python"
encoded = text.encode('utf-8')    # bytes: b'\xe4\xbd\xa0\xe5\xa5\xbd...'
decoded = encoded.decode('utf-8') # str: '你好Python'
```

```
┌───────────────── 编码关系 ─────────────────┐
│                                            │
│  str (Unicode)  ──encode()──→  bytes       │
│  str (Unicode)  ←──decode()──  bytes       │
│                                            │
│  Python 3 中 str 默认是 Unicode            │
│  文件I/O时注意指定 encoding='utf-8'        │
└────────────────────────────────────────────┘
```

## 9. Python中 is 和 == 的区别？True/False的本质？

**回答：**

```python
# is 比较对象身份（id），== 比较对象值
a = [1, 2, 3]
b = [1, 2, 3]
a == b  # True（值相等）
a is b  # False（不同对象）

# None 应该用 is 比较
if x is None:     # ✅ 正确
    pass
if x == None:     # ❌ 不推荐（可能被 __eq__ 覆盖）
    pass

# 小整数池和字符串驻留
a = 256
b = 256
a is b  # True（小整数缓存 [-5, 256]）

a = "hello"
b = "hello"
a is b  # True（字符串驻留，仅限简单字符串）

# bool 是 int 的子类
isinstance(True, int)   # True
True == 1               # True
False == 0              # True
True + True             # 2
```

**Python中的假值（falsy）**：
`None`、`False`、`0`、`0.0`、`""`、`[]`、`()`、`{}`、`set()`、`frozenset()`，以及自定义类中 `__bool__()` 返回 `False` 或 `__len__()` 返回 `0` 的对象。

## 10. 基础语法与数据类型面试速答？

**回答：**

```
Q: Python是编译型还是解释型语言？
A: Python是解释型语言，源码先编译为字节码(.pyc)，再由虚拟机(PVM)解释执行。

Q: Python 2和3的主要区别？
A: print成为函数、默认Unicode、整数除法返回float、range返回迭代器、移除了经典类。

Q: 什么是鸭子类型(duck typing)？
A: 不检查对象类型，而是检查对象是否有所需的方法/属性。"如果走路像鸭子、叫声像鸭子，那它就是鸭子。"

Q: list的append和extend区别？
A: append将参数作为单个元素添加，extend将可迭代对象逐个展开添加。

Q: dict获取值的get()和[]区别？
A: []找不到key抛KeyError，get()找不到返回默认值(默认None)。

Q: 如何判断一个对象的类型？
A: isinstance(obj, Type) 推荐（支持继承链），type(obj) is Type 精确检查。

Q: Python中的None是什么？
A: None是NoneType的唯一实例，表示空值。是单例对象，应使用is None判断。

Q: 什么是字符串驻留(string interning)？
A: CPython对简单字符串自动缓存，相同内容共享同一对象，节省内存。

Q: 列表推导式和map/filter比哪个更好？
A: 列表推导式更Pythonic、可读性更好，性能基本相当。

Q: Python中如何实现枚举类型？
A: 使用标准库enum模块：class Color(Enum): RED = 1; GREEN = 2; BLUE = 3
```
