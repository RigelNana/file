## 1. os和sys模块的核心功能和区别？

**回答：**

`os` 模块提供与操作系统交互的接口，`sys` 模块提供与 Python 解释器相关的功能。

```python
import os
import sys

# ===== os 模块 =====
# 文件和目录操作
os.getcwd()                    # 当前工作目录
os.listdir('.')                # 列出目录内容
os.makedirs('a/b/c', exist_ok=True)  # 递归创建目录
os.rename('old.txt', 'new.txt')
os.remove('file.txt')         # 删除文件
os.rmdir('empty_dir')         # 删除空目录

# 路径操作（推荐用 pathlib 替代）
os.path.join('dir', 'file.txt')
os.path.exists('/path/to/file')
os.path.isfile('x'), os.path.isdir('x')
os.path.basename('/a/b/c.txt')   # 'c.txt'
os.path.dirname('/a/b/c.txt')    # '/a/b'
os.path.splitext('file.txt')     # ('file', '.txt')

# 环境变量
os.environ['HOME']
os.environ.get('API_KEY', 'default')

# 进程操作
os.getpid()                    # 当前进程ID
os.cpu_count()                 # CPU核心数

# ===== sys 模块 =====
sys.argv                      # 命令行参数列表
sys.path                      # 模块搜索路径
sys.version                   # Python版本信息
sys.platform                  # 操作系统平台
sys.stdin, sys.stdout, sys.stderr  # 标准I/O
sys.exit(0)                   # 退出程序
sys.getsizeof(obj)            # 对象内存大小
sys.getrecursionlimit()       # 递归深度限制
sys.modules                   # 已导入模块的缓存
sys.executable                # Python解释器路径
```

## 2. pathlib模块相比os.path有什么优势？

**回答：**

`pathlib`（Python 3.4+）提供面向对象的路径操作，比 `os.path` 更直观易用。

```python
from pathlib import Path

# 创建路径
p = Path('/usr/local/bin')
p = Path.home() / 'documents' / 'file.txt'  # / 运算符拼接
p = Path.cwd()                               # 当前目录

# 路径信息
p = Path('/home/user/data.csv')
p.name         # 'data.csv'
p.stem         # 'data'
p.suffix       # '.csv'
p.parent       # Path('/home/user')
p.parts        # ('/', 'home', 'user', 'data.csv')
p.is_absolute()

# 文件操作
p.exists()
p.is_file()
p.is_dir()

# 读写文件（简洁！）
content = Path('file.txt').read_text(encoding='utf-8')
Path('output.txt').write_text("hello", encoding='utf-8')
data = Path('image.png').read_bytes()

# 目录操作
Path('a/b/c').mkdir(parents=True, exist_ok=True)
list(Path('.').iterdir())         # 列出目录
list(Path('.').glob('*.py'))      # 匹配模式
list(Path('.').rglob('*.py'))     # 递归匹配

# 路径解析
p.resolve()             # 绝对路径
p.relative_to('/home')  # 相对路径
p.with_name('new.csv')  # 替换文件名
p.with_suffix('.json')  # 替换扩展名

# 对比 os.path
# os.path: os.path.join(os.path.dirname(path), 'new.txt')
# pathlib: path.parent / 'new.txt'
```

## 3. json模块的使用和自定义序列化？

**回答：**

```python
import json
from datetime import datetime
from decimal import Decimal

# 基本操作
data = {"name": "Alice", "age": 30, "scores": [90, 85, 92]}

# 序列化
json_str = json.dumps(data, ensure_ascii=False, indent=2)
# 写文件
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

# 反序列化
obj = json.loads(json_str)
with open('data.json', 'r', encoding='utf-8') as f:
    obj = json.load(f)

# 自定义序列化（处理不支持的类型）
class CustomEncoder(json.JSONEncoder):
    def default(self, obj):
        if isinstance(obj, datetime):
            return obj.isoformat()
        if isinstance(obj, Decimal):
            return str(obj)
        if isinstance(obj, set):
            return list(obj)
        if hasattr(obj, '__dict__'):
            return obj.__dict__
        return super().default(obj)

data = {"time": datetime.now(), "price": Decimal("19.99")}
json.dumps(data, cls=CustomEncoder)

# 自定义反序列化
def custom_decoder(dct):
    if 'time' in dct:
        dct['time'] = datetime.fromisoformat(dct['time'])
    return dct

json.loads(json_str, object_hook=custom_decoder)
```

## 4. collections模块中的高效数据结构？

**回答：**

```python
from collections import (
    defaultdict, Counter, OrderedDict,
    deque, namedtuple, ChainMap
)

# 1. defaultdict - 带默认值的字典
word_groups = defaultdict(list)
for word in ["apple", "banana", "avocado", "blueberry"]:
    word_groups[word[0]].append(word)
# {'a': ['apple', 'avocado'], 'b': ['banana', 'blueberry']}

graph = defaultdict(set)  # 邻接表
graph['A'].add('B')
graph['A'].add('C')

# 2. Counter - 计数器
text = "hello world hello python hello"
c = Counter(text.split())       # Counter({'hello': 3, 'world': 1, 'python': 1})
c.most_common(2)                # [('hello', 3), ('world', 1)]
c['hello']                      # 3
c.update(["hello", "java"])     # 增量更新
Counter("aab") + Counter("bcc") # Counter({'b': 2, 'a': 2, 'c': 2})

# 3. deque - 双端队列
dq = deque([1, 2, 3], maxlen=5)
dq.appendleft(0)    # O(1)
dq.popleft()         # O(1)
dq.rotate(1)         # [3, 1, 2]（右旋）
dq.rotate(-1)        # [1, 2, 3]（左旋）

# 4. namedtuple - 命名元组
Point = namedtuple('Point', ['x', 'y'])
p = Point(1, 2)
print(p.x, p.y)     # 可通过名称访问
x, y = p             # 支持解包
p._asdict()          # {'x': 1, 'y': 2}
p._replace(x=10)     # Point(x=10, y=2)

# 5. ChainMap - 字典链
defaults = {'color': 'red', 'size': 'medium'}
user_prefs = {'color': 'blue'}
config = ChainMap(user_prefs, defaults)
config['color']  # 'blue'（优先查找第一个字典）
config['size']   # 'medium'（fallback到defaults）
```

# 基本类型提示

    return f"Hello, {name}"

def process(items: list[int]) -> dict[str, int]:
    return {str(i): i for i in items}

# Optional = Union[X, None]

    return cache.get(key)

    ...

# Callable 可调用类型
def apply(func: Callable[[int, int], int], a: int, b: int) -> int:
    return func(a, b)

# TypeVar 泛型
T = TypeVar('T')
def first(items: list[T]) -> T:
    return items[0]

# Generic 泛型类
class Stack(Generic[T]):
    def __init__(self) -> None:
        self._items: list[T] = []

    def push(self, item: T) -> None:
        self._items.append(item)

        return self._items.pop()

class Drawable(Protocol):

def render(obj: Drawable) -> None:

def set_mode(mode: Literal['read', 'write', 'append']) -> None:
    ...

# TypeAlias 类型别名
JSON: TypeAlias = dict[str, Any]

## 6. dataclasses模块的使用和高级功能？

**回答：**

`dataclasses`（Python 3.7+）自动生成 `__init__`、`__repr__`、`__eq__` 等方法。

```python
from dataclasses import dataclass, field, asdict, astuple
from typing import ClassVar

@dataclass
class User:
    name: str
    age: int
    email: str = ""  # 有默认值的字段必须在后面

    # 可变默认值使用 field(default_factory=...)
    tags: list[str] = field(default_factory=list)

    # 排除特定字段
    _cache: dict = field(default_factory=dict, repr=False, compare=False)

    # 类变量（不参与 __init__）
    species: ClassVar[str] = "Human"

    # 初始化后处理
    def __post_init__(self):
        if self.age < 0:
            raise ValueError("年龄不能为负")

# 使用
user = User("Alice", 30, "alice@example.com")
print(user)  # User(name='Alice', age=30, email='alice@example.com', tags=[])

# 转换
asdict(user)   # {'name': 'Alice', 'age': 30, ...}
astuple(user)  # ('Alice', 30, 'alice@example.com', [])

# 不可变数据类
@dataclass(frozen=True)
class Point:
    x: float
    y: float
    # 实例不可修改，可做 dict key

# 排序支持
@dataclass(order=True)
class Student:
    sort_index: float = field(init=False, repr=False)
    name: str
    grade: float

    def __post_init__(self):
        self.sort_index = self.grade

# 继承
@dataclass
class Admin(User):
    role: str = "admin"

# slots 优化（Python 3.10+）
@dataclass(slots=True)
class Efficient:
    x: int
    y: int
```

## 8. re正则表达式模块的核心用法？

**回答：**

```python
import re

text = "联系方式: alice@example.com, bob@test.org, 电话: 138-1234-5678"

# 常用函数
re.search(r'\d+', text)           # 搜索第一个匹配
re.match(r'联系', text)            # 从开头匹配
re.findall(r'[\w.]+@[\w.]+', text) # 所有匹配的列表
re.finditer(r'\d+', text)         # 所有匹配的迭代器
re.sub(r'\d', '*', text)          # 替换
re.split(r'[,，]', text)          # 分割

# 编译正则（重复使用时提高性能）
pattern = re.compile(r'''
    (?P<user>[\w.]+)    # 用户名
    @                    # @符号
    (?P<domain>[\w.]+)   # 域名
''', re.VERBOSE)

for match in pattern.finditer(text):
    print(match.group('user'), match.group('domain'))

# 常用模式
re.compile(r'\d+')            # 数字
re.compile(r'\b\w+\b')        # 单词
re.compile(r'^.+$', re.MULTILINE)  # 每行
re.compile(r'(?i)hello')      # 忽略大小写

# 分组
m = re.match(r'(\d{4})-(\d{2})-(\d{2})', '2024-01-15')
m.group(0)   # '2024-01-15'（完整匹配）
m.group(1)   # '2024'
m.groups()   # ('2024', '01', '15')

# 前瞻/后顾
re.findall(r'\w+(?=@)', text)        # 前瞻: @前的单词
re.findall(r'(?<=@)[\w.]+', text)    # 后顾: @后的域名
```

## 10. 标准库与常用模块面试速答？

**回答：**

```
Q: os.path和pathlib应该选哪个？
A: 新代码推荐pathlib，面向对象API更直观。pathlib是os.path的现代替代。

Q: json.dumps和json.dump的区别？
A: dumps返回JSON字符串，dump直接写入文件对象。loads/load同理。

Q: defaultdict和普通dict的区别？
A: defaultdict访问不存在的key时自动用工厂函数创建默认值，不会抛KeyError。

Q: namedtuple和dataclass选哪个？
A: 简单不可变数据用namedtuple，需要默认值、方法、可变性用dataclass。

Q: logging为什么不用print？
A: logging支持级别过滤、多输出目标、格式化、线程安全，生产环境必须用logging。

Q: 如何处理Python中的时区？
A: Python 3.9+用zoneinfo模块，之前用第三方pytz。始终用aware datetime（带时区信息）。

Q: Counter的most_common()时间复杂度？
A: O(n log k)，n是元素数，k是返回的前k个。底层使用堆排序。

Q: deque和list的区别？
A: deque左端操作O(1)，list左端操作O(n)。deque支持maxlen自动丢弃旧元素。

Q: typing模块的类型提示会影响运行时吗？
A: 不会。类型提示仅供静态检查工具(mypy)和IDE使用，运行时被忽略。

Q: re.match和re.search的区别？
A: match只匹配字符串开头，search搜索整个字符串中第一个匹配。
```
