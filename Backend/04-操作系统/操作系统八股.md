# 操作系统八股文

---

## 一、进程与线程

### 1. 进程和线程的区别？

**答：**

| 特性 | 进程 | 线程 |
|------|------|------|
| 定义 | 资源分配的基本单位 | CPU 调度的基本单位 |
| 内存 | 独立地址空间 | 共享进程地址空间 |
| 开销 | 创建/切换开销大 | 创建/切换开销小 |
| 通信 | IPC（管道/共享内存/消息队列） | 共享内存直接通信 |
| 安全 | 互不影响 | 一个崩溃影响全部 |

### 2. 进程间通信方式？

**答：**

| 方式 | 特点 | 适用场景 |
|------|------|---------|
| 管道 | 半双工，父子进程 | 简单数据传递 |
| 命名管道 | 无亲缘关系可用 | 不相关进程通信 |
| 消息队列 | 有格式的消息 | 异步通信 |
| 共享内存 | 最快，需同步 | 大数据量交换 |
| 信号量 | 同步/互斥 | 控制并发访问 |
| Socket | 跨网络 | 网络通信 |

### 3. 协程和线程的区别？

**答：**

| 特性 | 线程 | 协程 |
|------|------|------|
| 调度 | OS 内核调度 | 用户态调度 |
| 切换 | 需内核态切换，开销大 | 用户态切换，极轻量 |
| 栈大小 | 默认 1-8 MB | 几 KB（Go goroutine 2KB） |
| 并发数 | 数千级 | 数十万级 |

## 二、CPU 调度

### 4. 常见的 CPU 调度算法？

**答：**

| 算法 | 特点 | 适用 |
|------|------|------|
| FCFS | 先来先服务 | 简单但可能饥饿 |

## 三、内存管理

### 5. 虚拟内存的原理？

**答：**

每个进程有独立虚拟地址空间，通过页表映射到物理内存。访问未映射的页触发缺页中断，从磁盘加载到内存。实现了内存隔离和超过物理内存的使用。

### 6. 页面置换算法？

**答：**

| 算法 | 原理 | 特点 |
|------|------|------|
| FIFO | 淘汰最早进入的 | 简单，Belady异常 |
| LRU | 淘汰最久未使用的 | 效果好，实现复杂 |
| LFU | 淘汰使用次数最少的 | 考虑频率 |
| Clock | 近似LRU，循环扫描 | Linux 使用 |

## 四、锁与同步

### 7. 死锁的条件和解决？

**答：**

四个必要条件：互斥、占有等待、不可抢占、循环等待。打破任一条件可预防。检测方法：资源分配图检测环。解决：超时释放、按序加锁、银行家算法。

### 8. 互斥锁、读写锁、自旋锁的区别？

**答：**

| 锁 | 特点 | 适用 |
|----|------|------|
| 互斥锁 | 同时只一个线程持有 | 通用互斥 |
| 读写锁 | 读共享写独占 | 读多写少 |
| 自旋锁 | 忙等待不让出 CPU | 临界区极短 |

## 五、文件系统与 IO

### 9. IO 多路复用 select/poll/epoll？

**答：**

| 特性 | select | poll | epoll |
|------|--------|------|-------|
| 数据结构 | 位图 | 链表 | 红黑树+链表 |
| 最大连接 | 1024 | 无限制 | 无限制 |
| 遍历方式 | 线性扫描 | 线性扫描 | 回调通知 |
| 复杂度 | O(n) | O(n) | O(1) |
| 拷贝 | 每次全量拷贝 | 每次全量拷贝 | mmap 共享 |

### 10. 零拷贝技术？

**答：**

传统 IO：磁盘→内核缓冲→用户缓冲→Socket缓冲→网卡（4次拷贝）。mmap：减少一次拷贝。sendfile：数据不经过用户态，2次拷贝。适用于大文件传输（Nginx、Kafka）。

### 11. 同步/异步、阻塞/非阻塞的区别？

**答：**

| 模型 | 说明 |
|------|------|
| 同步阻塞 | 调用后等待完成才返回 |
| 同步非阻塞 | 立即返回但需轮询 |
| 异步 | 调用后立即返回，完成后回调通知 |

## 六、其他

### 12. 用户态和内核态？

**答：**

用户态权限受限不能直接操作硬件，内核态拥有全部权限。通过系统调用从用户态切换到内核态（软中断）。上下文切换需保存/恢复寄存器。

### 13. 中断和异常的区别？

**答：**

中断：外部设备产生（硬件中断），异步。异常：CPU 执行指令时产生（除零、缺页），同步。系统调用：用户主动触发的软中断。

| 工具 | 用途 |
|------|------|
| top/htop | CPU/内存总览 |
| vmstat | 系统性能统计 |
| iostat | 磁盘 IO |

| strace | 系统调用追踪 |
| perf | CPU 性能分析 |
| pprof | Go 程序性能分析 |
