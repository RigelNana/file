# 栈与队列

---

## 1. 栈和队列的基本区别是什么？

**回答：**

```
栈（LIFO）：           队列（FIFO）：
┌─────┐               ┌───→ 出队
│ top │ ← push/pop    │
├─────┤            入队 → [A][B][C][D] → 出队
│  C  │
├─────┤            双端队列（Deque）：
│  B  │            ←─ [A][B][C][D] ─→
├─────┤              两端都可入/出
│  A  │
└─────┘
```

| 特性 | 栈 | 队列 |
|------|-----|------|
| 原则 | LIFO（后进先出）| FIFO（先进先出）|
| 核心操作 | push / pop / peek | enqueue / dequeue / peek |
| 典型应用 | 函数调用栈、表达式求值、DFS | BFS、消息队列、任务调度 |
| 实现方式 | 数组或链表 | 数组（循环）或链表 |

## 2. 如何用栈实现队列？如何用队列实现栈？

**回答：**

**两个栈实现队列：**

```go
type MyQueue struct {
    inStack  []int  // 入队栈
    outStack []int  // 出队栈
}

func (q *MyQueue) Push(x int) {
    q.inStack = append(q.inStack, x)
}

func (q *MyQueue) Pop() int {
    q.peek() // 确保 outStack 有元素
    n := len(q.outStack)
    val := q.outStack[n-1]
    q.outStack = q.outStack[:n-1]
    return val
}

func (q *MyQueue) peek() {
    if len(q.outStack) == 0 {
        // 把 inStack 全部倒入 outStack
        for len(q.inStack) > 0 {
            n := len(q.inStack)
            q.outStack = append(q.outStack, q.inStack[n-1])
            q.inStack = q.inStack[:n-1]
        }
    }
}
// 均摊复杂度: Push O(1), Pop O(1)
```

**两个队列实现栈：**

```python
from collections import deque

class MyStack:
    def __init__(self):
        self.q1 = deque()
        self.q2 = deque()

    def push(self, x: int) -> None:
        self.q2.append(x)
        while self.q1:
            self.q2.append(self.q1.popleft())
        self.q1, self.q2 = self.q2, self.q1

    def pop(self) -> int:
        return self.q1.popleft()
    # Push O(n), Pop O(1)
```

## 3. 什么是单调栈？有什么应用？

**回答：**

单调栈保持栈内元素**单调递增或递减**。用于高效解决「下一个更大/更小元素」问题。

```go
// 下一个更大元素 — O(n)
func nextGreaterElement(nums []int) []int {
    n := len(nums)
    result := make([]int, n)
    for i := range result {
        result[i] = -1
    }
    stack := []int{} // 存下标

    for i := 0; i < n; i++ {
        for len(stack) > 0 && nums[i] > nums[stack[len(stack)-1]] {
            top := stack[len(stack)-1]
            stack = stack[:len(stack)-1]
            result[top] = nums[i]
        }
        stack = append(stack, i)
    }
    return result
}
```

```
示例：nums = [2, 1, 4, 3]
处理过程：
i=0: stack=[0(2)]
i=1: 1<2, stack=[0(2), 1(1)]
i=2: 4>1, result[1]=4; 4>2, result[0]=4; stack=[2(4)]
i=3: 3<4, stack=[2(4), 3(3)]
结果: [4, 4, -1, -1]
```

**常见应用：**
- 每日温度（下一个更高温度的天数）
- 直方图最大矩形面积
- 接雨水
- 股票价格跨度

## 4. 什么是单调队列？滑动窗口最大值如何求解？

**回答：**

单调队列（双端队列维护）：队首到队尾**单调递减**，队首始终是窗口最大值。

```go
// 滑动窗口最大值 — O(n)
func maxSlidingWindow(nums []int, k int) []int {
    deque := []int{}   // 存下标，对应值单调递减
    result := []int{}

    for i := 0; i < len(nums); i++ {
        // 移除过期元素
        if len(deque) > 0 && deque[0] <= i-k {
            deque = deque[1:]
        }
        // 维护单调性：移除所有比当前小的
        for len(deque) > 0 && nums[deque[len(deque)-1]] <= nums[i] {
            deque = deque[:len(deque)-1]
        }
        deque = append(deque, i)
        // 窗口形成后记录结果
        if i >= k-1 {
            result = append(result, nums[deque[0]])
        }
    }
    return result
}
```

```
示例：nums=[1,3,-1,-3,5,3,6,7], k=3

窗口             deque(值)   最大值
[1  3  -1]       [3, -1]      3
[3  -1  -3]      [3, -1, -3]  3
[-1  -3  5]      [5]          5
[-3  5  3]       [5, 3]       5
[5  3  6]        [6]          6
[3  6  7]        [7]          7
```

## 5. 如何用栈解决括号匹配和表达式求值问题？

**回答：**

**括号匹配：**

```go
func isValid(s string) bool {
    stack := []byte{}
    pairs := map[byte]byte{')': '(', ']': '[', '}': '{'}

    for i := 0; i < len(s); i++ {
        if s[i] == '(' || s[i] == '[' || s[i] == '{' {
            stack = append(stack, s[i])
        } else {
            if len(stack) == 0 || stack[len(stack)-1] != pairs[s[i]] {
                return false
            }
            stack = stack[:len(stack)-1]
        }
    }
    return len(stack) == 0
}
```

**中缀表达式求值（双栈法）：**

```
规则：
1. 数字直接入数栈
2. 左括号入符号栈
3. 右括号：弹运算符计算直到左括号
4. 运算符：弹出优先级 >= 当前的运算符先计算，再入栈

示例: 3 + 2 * 4
数栈: [3] → [3,2] → [3,2,4] → [3,8] → [11]
符号: [+] → [+,*] → [+] →    → []
```

**逆波兰表达式（后缀表达式）求值：**

```go
func evalRPN(tokens []string) int {
    stack := []int{}
    for _, token := range tokens {
        switch token {
        case "+", "-", "*", "/":
            b, a := stack[len(stack)-1], stack[len(stack)-2]
            stack = stack[:len(stack)-2]
            switch token {
            case "+": stack = append(stack, a+b)
            case "-": stack = append(stack, a-b)
            case "*": stack = append(stack, a*b)
            case "/": stack = append(stack, a/b)
            }
        default:
            num, _ := strconv.Atoi(token)
            stack = append(stack, num)
        }
    }
    return stack[0]
}
```

## 6. 如何用循环数组实现队列？

**回答：**

```go
type CircularQueue struct {
    data []int
    head int
    tail int
    size int
    cap  int
}

func NewCircularQueue(k int) *CircularQueue {
    return &CircularQueue{
        data: make([]int, k),
        head: 0,
        tail: 0,
        size: 0,
        cap:  k,
    }
}

func (q *CircularQueue) Enqueue(val int) bool {
    if q.size == q.cap {
        return false // 队满
    }
    q.data[q.tail] = val
    q.tail = (q.tail + 1) % q.cap
    q.size++
    return true
}

func (q *CircularQueue) Dequeue() (int, bool) {
    if q.size == 0 {
        return 0, false // 队空
    }
    val := q.data[q.head]
    q.head = (q.head + 1) % q.cap
    q.size--
    return val, true
}
```

```
循环队列状态：
Cap=5, 初始:
[_][_][_][_][_]
 ↑head
 ↑tail

Enqueue(1,2,3):
[1][2][3][_][_]
 ↑head     ↑tail

Dequeue() → 1:
[_][2][3][_][_]
    ↑head  ↑tail

Enqueue(4,5):
[_][2][3][4][5]
    ↑head      ↑tail=0 (回绕)
```

**关键技巧：** 用取模运算 `% cap` 实现循环，用 `size` 变量区分空和满。

## 7. 优先队列（堆）的底层实现原理？

**回答：**

优先队列通常用**二叉堆**实现，底层是数组：

```
最小堆：           数组存储：
      1            [1, 3, 2, 7, 5, 4, 6]
     / \            0  1  2  3  4  5  6
    3   2
   / \ / \         父节点: (i-1)/2
  7  5 4  6        左孩子: 2*i+1
                   右孩子: 2*i+2
```

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 插入 | O(log n) | 尾部加入，上浮 |
| 取最值 | O(1) | 堆顶 |
| 删除最值 | O(log n) | 堆顶换尾部，下沉 |
| 建堆 | O(n) | 自底向上下沉 |

```go
// 上浮操作
func (h *MinHeap) siftUp(i int) {
    for i > 0 {
        parent := (i - 1) / 2
        if h.data[i] < h.data[parent] {
            h.data[i], h.data[parent] = h.data[parent], h.data[i]
            i = parent
        } else {
            break
        }
    }
}

// 下沉操作
func (h *MinHeap) siftDown(i int) {
    n := len(h.data)
    for {
        smallest := i
        left, right := 2*i+1, 2*i+2
        if left < n && h.data[left] < h.data[smallest] {
            smallest = left
        }
        if right < n && h.data[right] < h.data[smallest] {
            smallest = right
        }
        if smallest == i {
            break
        }
        h.data[i], h.data[smallest] = h.data[smallest], h.data[i]
        i = smallest
    }
}
```

## 8. 什么是 LRU 缓存？如何实现？

**回答：**

LRU（Least Recently Used）：淘汰最近最少使用的数据。

```
结构：哈希表 + 双向链表
┌──────────────┐
│ HashMap      │     双向链表（最近访问在头部）
│ key → node   │     head ↔ [A] ↔ [B] ↔ [C] ↔ tail
└──────────────┘              ↑ 最新      最旧 ↑
                           GET/PUT → 移到头部
                           容量满 → 淘汰尾部
```

```go
type LRUCache struct {
    capacity int
    cache    map[int]*Node
    head     *Node // 哨兵头
    tail     *Node // 哨兵尾
}

type Node struct {
    key, val   int
    prev, next *Node
}

func Constructor(capacity int) LRUCache {
    head := &Node{}
    tail := &Node{}
    head.next = tail
    tail.prev = head
    return LRUCache{capacity: capacity, cache: make(map[int]*Node), head: head, tail: tail}
}

func (l *LRUCache) Get(key int) int {
    if node, ok := l.cache[key]; ok {
        l.moveToHead(node)
        return node.val
    }
    return -1
}

func (l *LRUCache) Put(key, value int) {
    if node, ok := l.cache[key]; ok {
        node.val = value
        l.moveToHead(node)
        return
    }
    node := &Node{key: key, val: value}
    l.cache[key] = node
    l.addToHead(node)
    if len(l.cache) > l.capacity {
        tail := l.removeTail()
        delete(l.cache, tail.key)
    }
}

func (l *LRUCache) moveToHead(node *Node) {
    l.removeNode(node)
    l.addToHead(node)
}
```

**变体：** LFU（最不经常使用）需要额外维护频率计数。

## 9. 栈在函数调用和递归中的作用？

**回答：**

```
函数调用栈（Call Stack）：
┌─────────────────────┐
│ factorial(1)        │  ← 栈顶（当前执行）
│ - n=1, return 1     │
├─────────────────────┤
│ factorial(2)        │
│ - n=2, 等待子调用   │
├─────────────────────┤
│ factorial(3)        │
│ - n=3, 等待子调用   │
├─────────────────────┤
│ main()              │  ← 栈底
└─────────────────────┘

每个栈帧包含：
- 返回地址
- 局部变量
- 参数
- 前一个栈帧指针（FP/BP）
```

**递归转迭代（手动栈模拟）：**

```go
// 递归版本
func inorder(root *TreeNode) []int {
    if root == nil { return nil }
    result := inorder(root.Left)
    result = append(result, root.Val)
    result = append(result, inorder(root.Right)...)
    return result
}

// 迭代版本（手动栈）
func inorderIterative(root *TreeNode) []int {
    var result []int
    var stack []*TreeNode
    curr := root
    for curr != nil || len(stack) > 0 {
        for curr != nil {
            stack = append(stack, curr)
            curr = curr.Left
        }
        curr = stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = append(result, curr.Val)
        curr = curr.Right
    }
    return result
}
```

**栈溢出风险：** 递归深度过大（如链式结构 10 万层），Go 默认协程栈 1MB 可增长到 1GB，Python 默认递归限制 1000 层。

## 10. 栈与队列面试速答？

**回答：**

```
Q: 栈的 LIFO 含义？
A: Last In First Out，后进先出

Q: 队列的 FIFO 含义？
A: First In First Out，先进先出

Q: 判断循环队列为空的条件？
A: head == tail（或 size == 0）

Q: 最小栈怎么实现？O(1) 获取最小值？
A: 辅助栈同步记录当前最小值，每次 push 时与辅助栈顶比较

Q: 什么时候用栈？
A: 需要后进先出的场景：撤销操作、DFS、括号匹配、表达式求值

Q: 什么时候用队列？
A: 需要先进先出的场景：BFS、任务调度、消息缓冲、打印队列

Q: deque 是什么？
A: 双端队列（Double-Ended Queue），两端都可以入队和出队

Q: 优先队列和普通队列的区别？
A: 优先队列按优先级出队，不是按入队顺序

Q: Go 中如何使用堆？
A: 实现 heap.Interface（Len/Less/Swap/Push/Pop），用 container/heap 包

Q: 递归都可以转成迭代吗？
A: 是的，任何递归都可以用显式栈模拟，但有些场景代码会复杂很多
```
