## 1. 动态规划的核心思想和要素？

**回答：**

```
动态规划三要素：
┌─────────────────────────────────────────┐
│ 1. 最优子结构：问题的最优解包含子问题最优解 │
│ 2. 重叠子问题：子问题被反复计算            │
│ 3. 无后效性：状态确定后不受后续决策影响     │
└─────────────────────────────────────────┘

解题步骤：
1. 定义状态：dp[i] 表示什么
2. 状态转移方程：dp[i] = f(dp[i-1], ...)
3. 初始条件：dp[0] = ?
4. 遍历顺序：确保计算 dp[i] 时所需状态已计算
5. 返回结果：dp[n] 或 max(dp)
```

**斐波那契（入门示例）：**

```go
// 暴力递归 O(2^n)
func fib(n int) int {
    if n <= 1 { return n }
    return fib(n-1) + fib(n-2)
}

// 记忆化递归（自顶向下）O(n)
func fib(n int, memo map[int]int) int {
    if n <= 1 { return n }
    if v, ok := memo[n]; ok { return v }
    memo[n] = fib(n-1, memo) + fib(n-2, memo)
    return memo[n]
}

// DP（自底向上）O(n)，空间 O(1)
func fib(n int) int {
    if n <= 1 { return n }
    a, b := 0, 1
    for i := 2; i <= n; i++ {
        a, b = b, a+b
    }
    return b
}
```

## 2. 背包问题的分类和解法？

**回答：**

**0-1 背包（每件物品用一次）：**

```go
// dp[j] = 用容量 j 能取到的最大价值
func knapsack01(weights, values []int, capacity int) int {
    dp := make([]int, capacity+1)
    for i := 0; i < len(weights); i++ {
        for j := capacity; j >= weights[i]; j-- { // 倒序！
            dp[j] = max(dp[j], dp[j-weights[i]]+values[i])
        }
    }
    return dp[capacity]
}
```

**完全背包（每件物品无限次）：**

```go
func knapsackComplete(weights, values []int, capacity int) int {
    dp := make([]int, capacity+1)
    for i := 0; i < len(weights); i++ {
        for j := weights[i]; j <= capacity; j++ { // 正序！
            dp[j] = max(dp[j], dp[j-weights[i]]+values[i])
        }
    }
    return dp[capacity]
}
```

**对比：**

| 类型 | 每件物品 | 内层遍历 | 应用 |
|------|---------|---------|------|
| 0-1 背包 | 只用一次 | 倒序 | 选课、投资 |
| 完全背包 | 无限次 | 正序 | 零钱兑换 |
| 多重背包 | 有限次 | 二进制拆分 | 购物 |
| 分组背包 | 每组选一个 | 先遍历组 | 互斥选择 |

## 3. 最长公共子序列（LCS）和最长递增子序列（LIS）？

**回答：**

**LCS — O(mn)：**

```go
func longestCommonSubsequence(text1, text2 string) int {
    m, n := len(text1), len(text2)
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }
    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if text1[i-1] == text2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}
```

**LIS — O(n log n)（二分优化）：**

```go
func lengthOfLIS(nums []int) int {
    // tails[i] = 长度为 i+1 的递增子序列的最小末尾
    tails := []int{}
    for _, num := range nums {
        pos := sort.SearchInts(tails, num)
        if pos == len(tails) {
            tails = append(tails, num)
        } else {
            tails[pos] = num
        }
    }
    return len(tails)
}
```

```
示例：nums = [10, 9, 2, 5, 3, 7, 101, 18]

tails 变化过程：
[10] → [9] → [2] → [2,5] → [2,3] → [2,3,7] → [2,3,7,101] → [2,3,7,18]
LIS 长度 = 4
```

## 4. 编辑距离问题？

**回答：**

给两个字符串，求最少操作次数（插入/删除/替换）将 word1 变成 word2。

```go
func minDistance(word1, word2 string) int {
    m, n := len(word1), len(word2)
    dp := make([][]int, m+1) // dp[i][j] = word1前i个 → word2前j个的最少操作
    for i := range dp {
        dp[i] = make([]int, n+1)
        dp[i][0] = i  // 删除 i 个字符
    }
    for j := 0; j <= n; j++ {
        dp[0][j] = j  // 插入 j 个字符
    }

    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if word1[i-1] == word2[j-1] {
                dp[i][j] = dp[i-1][j-1]  // 相同，不操作
            } else {
                dp[i][j] = min(
                    dp[i-1][j]+1,    // 删除
                    dp[i][j-1]+1,    // 插入
                    dp[i-1][j-1]+1,  // 替换
                )
            }
        }
    }
    return dp[m][n]
}
```

```
     ""  r  o  s
""    0  1  2  3
h     1  1  2  3
o     2  2  1  2
r     3  2  2  2
s     4  3  3  2
e     5  4  4  3

"horse" → "ros" 最少 3 步
```

**应用：** 拼写检查、DNA 序列比对、diff 算法。

## 5. 区间 DP 和状态压缩 DP？

**回答：**

**区间 DP：**

```
dp[i][j] = 区间 [i,j] 的最优值
遍历顺序：先枚举长度，再枚举起点

应用：矩阵链乘法、石子合并、戳气球
```

**戳气球：**

```go
func maxCoins(nums []int) int {
    n := len(nums)
    // 两端添加虚拟气球值为1
    val := make([]int, n+2)
    val[0], val[n+1] = 1, 1
    copy(val[1:], nums)

    dp := make([][]int, n+2)
    for i := range dp {
        dp[i] = make([]int, n+2)
    }

    // 枚举区间长度
    for length := 1; length <= n; length++ {
        for i := 1; i+length-1 <= n; i++ {
            j := i + length - 1
            for k := i; k <= j; k++ {
                // k 是最后一个戳破的
                coins := val[i-1]*val[k]*val[j+1] + dp[i][k-1] + dp[k+1][j]
                dp[i][j] = max(dp[i][j], coins)
            }
        }
    }
    return dp[1][n]
}
```

**状态压缩 DP：** 用二进制表示集合状态。

```go
// 旅行商问题 TSP — O(2^n × n²)
// dp[mask][i] = 访问了 mask 集合中的城市，当前在 i 的最短路
func tsp(dist [][]int) int {
    n := len(dist)
    dp := make([][]int, 1<<n)
    for i := range dp {
        dp[i] = make([]int, n)
        for j := range dp[i] {
            dp[i][j] = math.MaxInt64
        }
    }
    dp[1][0] = 0 // 从城市0出发

    for mask := 1; mask < 1<<n; mask++ {
        for i := 0; i < n; i++ {
            if dp[mask][i] == math.MaxInt64 { continue }
            for j := 0; j < n; j++ {
                if mask>>j&1 == 1 { continue }
                next := mask | (1 << j)
                dp[next][j] = min(dp[next][j], dp[mask][i]+dist[i][j])
            }
        }
    }
    // 回到起点
    ans := math.MaxInt64
    full := (1 << n) - 1
    for i := 0; i < n; i++ {
        ans = min(ans, dp[full][i]+dist[i][0])
    }
    return ans
}
```

## 6. 股票买卖问题的状态机解法？

**回答：**

```
状态机：
            buy          sell
  ┌──→ [持有] ──────→ [不持有] ──┐
  │      ↑ rest          rest ↓  │
  │      └───┘           └───┘  │
  └─────────────────────────────┘

dp[i][k][0] = 第i天，最多k次交易，不持有股票的最大利润
dp[i][k][1] = 第i天，最多k次交易，持有股票的最大利润
```

**通用模板：**

```go
// k 次交易限制
func maxProfit(k int, prices []int) int {
    n := len(prices)
    if n == 0 { return 0 }
    
    dp := make([][][2]int, n)
    for i := range dp {
        dp[i] = make([][2]int, k+1)
    }

    for i := 0; i < n; i++ {
        for j := k; j >= 1; j-- {
            if i == 0 {
                dp[i][j][0] = 0
                dp[i][j][1] = -prices[0]
                continue
            }
            dp[i][j][0] = max(dp[i-1][j][0], dp[i-1][j][1]+prices[i])
            dp[i][j][1] = max(dp[i-1][j][1], dp[i-1][j-1][0]-prices[i])
        }
    }
    return dp[n-1][k][0]
}
```

**特殊情况：**

| 变体 | 条件 | 简化 |
|------|------|------|
| 买卖1次 | k=1 | 记录最低价，更新最大利润 |
| 买卖无限次 | k=∞ | 贪心：只要涨就买卖 |
| 含冷冻期 | 卖出后等1天 | 增加冷冻状态 |
| 含手续费 | 每次交易扣费 | 卖出时减手续费 |

## 7. 打家劫舍问题系列？

**回答：**

**基础版（不能偷相邻）：**

```go
func rob(nums []int) int {
    n := len(nums)
    if n == 1 { return nums[0] }
    // dp[i] = 偷到第i家的最大金额
    dp0, dp1 := nums[0], max(nums[0], nums[1])
    for i := 2; i < n; i++ {
        dp0, dp1 = dp1, max(dp1, dp0+nums[i])
    }
    return dp1
}
```

**环形版（首尾相邻）：**

```go
func robCircle(nums []int) int {
    n := len(nums)
    if n == 1 { return nums[0] }
    // 偷 [0, n-2] 或 [1, n-1]，取最大值
    return max(robRange(nums, 0, n-2), robRange(nums, 1, n-1))
}
```

**树形版（二叉树，不能偷相连节点）：**

```go
func robTree(root *TreeNode) int {
    rob, notRob := dfs(root)
    return max(rob, notRob)
}

func dfs(node *TreeNode) (rob, notRob int) {
    if node == nil { return 0, 0 }
    leftRob, leftNot := dfs(node.Left)
    rightRob, rightNot := dfs(node.Right)
    rob = node.Val + leftNot + rightNot
    notRob = max(leftRob, leftNot) + max(rightRob, rightNot)
    return
}
```

## 8. 路径问题（网格 DP）？

**回答：**

**不同路径：**

```go
func uniquePaths(m, n int) int {
    dp := make([][]int, m)
    for i := range dp {
        dp[i] = make([]int, n)
        dp[i][0] = 1
    }
    for j := 0; j < n; j++ {
        dp[0][j] = 1
    }
    for i := 1; i < m; i++ {
        for j := 1; j < n; j++ {
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
        }
    }
    return dp[m-1][n-1]
}
```

**最小路径和：**

```go
func minPathSum(grid [][]int) int {
    m, n := len(grid), len(grid[0])
    for i := 1; i < m; i++ { grid[i][0] += grid[i-1][0] }
    for j := 1; j < n; j++ { grid[0][j] += grid[0][j-1] }
    for i := 1; i < m; i++ {
        for j := 1; j < n; j++ {
            grid[i][j] += min(grid[i-1][j], grid[i][j-1])
        }
    }
    return grid[m-1][n-1]
}
```

**三角形最小路径和（自底向上）：**

```go
func minimumTotal(triangle [][]int) int {
    n := len(triangle)
    dp := make([]int, n)
    copy(dp, triangle[n-1])
    for i := n - 2; i >= 0; i-- {
        for j := 0; j <= i; j++ {
            dp[j] = triangle[i][j] + min(dp[j], dp[j+1])
        }
    }
    return dp[0]
}
```

## 9. 子串和子序列类 DP 总结？

**回答：**

| 问题 | 转移方程 | 时间 |
|------|---------|------|
| 最长公共子序列 | 相等: dp[i-1][j-1]+1; 不等: max(dp[i-1][j], dp[i][j-1]) | O(mn) |
| 最长公共子串 | 相等: dp[i-1][j-1]+1; 不等: 0 | O(mn) |
| 最长回文子序列 | s[i]==s[j]: dp[i+1][j-1]+2; 不等: max(dp[i+1][j], dp[i][j-1]) | O(n²) |
| 最长回文子串 | dp[i][j] = s[i]==s[j] && dp[i+1][j-1] | O(n²) |
| 编辑距离 | 三种操作取最小 | O(mn) |
| 最长递增子序列 | O(n²) 朴素 / O(n log n) 贪心+二分 | — |

**最长回文子串（扩展中心法 O(n²)）：**

```go
func longestPalindrome(s string) string {
    start, maxLen := 0, 1
    expand := func(l, r int) {
        for l >= 0 && r < len(s) && s[l] == s[r] {
            if r-l+1 > maxLen {
                start, maxLen = l, r-l+1
            }
            l--; r++
        }
    }
    for i := 0; i < len(s); i++ {
        expand(i, i)   // 奇数长度
        expand(i, i+1) // 偶数长度
    }
    return s[start : start+maxLen]
}
```

## 10. 动态规划面试速答？

**回答：**

```
Q: DP 和贪心的区别？
A: DP 考虑所有子问题取最优，贪心只做当前最优选择不回头

Q: DP 和分治的区别？
A: DP 子问题重叠（需记忆化），分治子问题独立

Q: 自顶向下和自底向上哪个好？
A: 自底向上（递推）通常更快（无递归开销），自顶向下更直观

Q: 0-1 背包为什么要倒序遍历？
A: 防止同一物品被多次使用（完全背包才正序）

Q: 状态压缩 DP 的空间复杂度？
A: O(2^n)，n 一般不超过 20

Q: 如何优化 DP 的空间？
A: 滚动数组（只保留前一行/两个变量），状态只依赖有限个前序状态时

Q: 最长递增子序列的最优时间？
A: O(n log n)，贪心+二分

Q: 编辑距离的实际应用？
A: 拼写纠错、DNA比对、diff工具、模糊搜索

Q: 什么时候 DP 不适用？
A: 无最优子结构、无重叠子问题、状态空间太大

Q: 面试中 DP 题的解题思路？
A: 先想暴力递归 → 画递归树找重叠 → 加 memo → 转递推 → 空间优化
```
