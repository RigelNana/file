## 1. 二叉树有哪些遍历方式？

**回答：**

```
示例二叉树：
        1
       / \
      2   3
     / \   \
    4   5   6

前序（根左右）：1, 2, 4, 5, 3, 6
中序（左根右）：4, 2, 5, 1, 3, 6
后序（左右根）：4, 5, 2, 6, 3, 1
层序（BFS）：   1, 2, 3, 4, 5, 6
```

**递归实现：**

```go
func preorder(root *TreeNode) {
    if root == nil { return }
    visit(root)           // 前序：先访问根
    preorder(root.Left)
    preorder(root.Right)
}

func inorder(root *TreeNode) {
    if root == nil { return }
    inorder(root.Left)
    visit(root)           // 中序：中间访问根
    inorder(root.Right)
}
```

**迭代实现（中序，用栈）：**

```go
func inorderIterative(root *TreeNode) []int {
    var result []int
    var stack []*TreeNode
    curr := root
    for curr != nil || len(stack) > 0 {
        for curr != nil {
            stack = append(stack, curr)
            curr = curr.Left
        }
        curr = stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = append(result, curr.Val)
        curr = curr.Right
    }
    return result
}
```

**层序遍历（BFS，用队列）：**

```go
func levelOrder(root *TreeNode) [][]int {
    if root == nil { return nil }
    var result [][]int
    queue := []*TreeNode{root}
    for len(queue) > 0 {
        size := len(queue)
        level := make([]int, 0, size)
        for i := 0; i < size; i++ {
            node := queue[0]
            queue = queue[1:]
            level = append(level, node.Val)
            if node.Left != nil { queue = append(queue, node.Left) }
            if node.Right != nil { queue = append(queue, node.Right) }
        }
        result = append(result, level)
    }
    return result
}
```

## 2. 二叉搜索树（BST）的特点和操作？

**回答：**

```
BST 性质：左 < 根 < 右（中序遍历有序）

       8
      / \
     3   10
    / \    \
   1   6    14
      / \   /
     4   7 13

查找 6：8→3→6 ✓ (O(log n))
```

| 操作 | 平均 | 最坏（退化链表）|
|------|------|--------------|
| 查找 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

**删除节点的三种情况：**

```go
func deleteNode(root *TreeNode, key int) *TreeNode {
    if root == nil { return nil }
    if key < root.Val {
        root.Left = deleteNode(root.Left, key)
    } else if key > root.Val {
        root.Right = deleteNode(root.Right, key)
    } else {
        // 情况1：叶子节点，直接删
        if root.Left == nil && root.Right == nil {
            return nil
        }
        // 情况2：只有一个子节点，用子节点替代
        if root.Left == nil { return root.Right }
        if root.Right == nil { return root.Left }
        // 情况3：两个子节点，用后继（右子树最小）替代
        successor := findMin(root.Right)
        root.Val = successor.Val
        root.Right = deleteNode(root.Right, successor.Val)
    }
    return root
}
```

## 3. 平衡二叉树有哪些？AVL 树和红黑树的区别？

**回答：**

```
AVL 树：严格平衡
- 任意节点左右子树高度差 ≤ 1
- 通过旋转（LL/RR/LR/RL）维护平衡

红黑树：近似平衡
- 5条性质保证最长路径 ≤ 2倍最短
- 通过变色 + 旋转维护
```

| 特性 | AVL 树 | 红黑树 |
|------|--------|--------|
| 平衡度 | 严格（高度差 ≤ 1）| 宽松（最长 ≤ 2×最短）|
| 查找 | 更快（更矮）| 略慢 |
| 插入/删除 | 慢（旋转多）| 快（旋转最多3次）|
| 适用 | 查找密集 | 增删密集 |
| 应用 | 数据库索引 | TreeMap、CFS调度器、epoll |

**红黑树 5 条性质：**
1. 节点非红即黑
2. 根节点是黑色
3. 叶子节点（NIL）是黑色
4. 红色节点的子节点必须是黑色（不能连续红）
5. 从任一节点到叶子的路径上黑色节点数相同

## 4. B 树和 B+ 树的区别？为什么数据库用 B+ 树？

**回答：**

```
B 树（3阶）：
          [10, 20]
         /    |    \
    [5,8]  [15,18]  [25,30]
    数据在所有节点

B+ 树（3阶）：
          [10, 20]           ← 索引节点（只存 key）
         /    |    \
    [5,8] → [10,15] → [20,25,30]  ← 叶子节点（存数据+链表）
```

| 特性 | B 树 | B+ 树 |
|------|------|-------|
| 数据位置 | 所有节点 | 仅叶子节点 |
| 叶子链表 | 无 | 有（顺序访问快）|
| 查询性能 | 不稳定 | 稳定（总到叶子）|
| 非叶节点大小 | 大（含数据）| 小（只有 key）|
| 扇出（分支数）| 较小 | 更大 |
| 范围查询 | 需要中序遍历 | 链表直接扫描 |

**MySQL InnoDB 用 B+ 树的原因：**
- 非叶节点小 → 一页能放更多 key → 树更矮 → 磁盘IO更少
- 叶子链表 → 范围查询极快（SELECT ... WHERE id BETWEEN 1 AND 100）
- 查询稳定 → 所有查询都到叶子，性能可预测

**一般3-4层B+树可存储千万级数据（假设4KB页/16字节key）**

## 5. 堆（Heap）的原理和应用？

**回答：**

```
最小堆：父节点 ≤ 子节点
      1
     / \
    3   2
   / \ / \
  7  5 4  6

数组表示：[1, 3, 2, 7, 5, 4, 6]
父节点：(i-1)/2
左孩子：2*i+1
右孩子：2*i+2
```

**Go 标准库使用：**

```go
// 实现 heap.Interface
type IntHeap []int

func (h IntHeap) Len() int           { return len(h) }
func (h IntHeap) Less(i, j int) bool { return h[i] < h[j] }
func (h IntHeap) Swap(i, j int)      { h[i], h[j] = h[j], h[i] }
func (h *IntHeap) Push(x any)        { *h = append(*h, x.(int)) }
func (h *IntHeap) Pop() any {
    old := *h
    n := len(old)
    x := old[n-1]
    *h = old[:n-1]
    return x
}

// 使用
h := &IntHeap{5, 3, 7, 1}
heap.Init(h)           // 建堆 O(n)
heap.Push(h, 2)        // 插入 O(log n)
min := heap.Pop(h)     // 弹出最小值 O(log n)
```

**Top K 问题：**
- 前 K 大：维护大小为 K 的**最小堆**
- 前 K 小：维护大小为 K 的**最大堆**
- 时间复杂度 O(n log K)，空间 O(K)

## 6. 字典树（Trie）的原理和应用？

**回答：**

```
存储单词：app, apple, api, bee

        root
       /    \
      a      b
      |      |
      p      e
     / \     |
    p   i    e
    |        ✓(bee)
    l
    |
    e
    ✓(apple)

每个节点有：
- children map[byte]*Node（或 [26]*Node）
- isEnd bool（标记是否是完整单词）
```

```go
type Trie struct {
    children [26]*Trie
    isEnd    bool
}

func (t *Trie) Insert(word string) {
    node := t
    for _, c := range word {
        idx := c - 'a'
        if node.children[idx] == nil {
            node.children[idx] = &Trie{}
        }
        node = node.children[idx]
    }
    node.isEnd = true
}

func (t *Trie) Search(word string) bool {
    node := t.searchPrefix(word)
    return node != nil && node.isEnd
}

func (t *Trie) StartsWith(prefix string) bool {
    return t.searchPrefix(prefix) != nil
}

func (t *Trie) searchPrefix(prefix string) *Trie {
    node := t
    for _, c := range prefix {
        idx := c - 'a'
        if node.children[idx] == nil {
            return nil
        }
        node = node.children[idx]
    }
    return node
}
```

**应用场景：**
- 搜索引擎自动补全
- 拼写检查
- IP 路由表（最长前缀匹配）
- 字符串统计与排序

## 7. 二叉树的常见面试题解法？

**回答：**

**最大深度：**

```go
func maxDepth(root *TreeNode) int {
    if root == nil { return 0 }
    return max(maxDepth(root.Left), maxDepth(root.Right)) + 1
}
```

**判断是否对称：**

```go
func isSymmetric(root *TreeNode) bool {
    return isMirror(root, root)
}

func isMirror(a, b *TreeNode) bool {
    if a == nil && b == nil { return true }
    if a == nil || b == nil { return false }
    return a.Val == b.Val &&
        isMirror(a.Left, b.Right) &&
        isMirror(a.Right, b.Left)
}
```

**最近公共祖先（LCA）：**

```go
func lowestCommonAncestor(root, p, q *TreeNode) *TreeNode {
    if root == nil || root == p || root == q {
        return root
    }
    left := lowestCommonAncestor(root.Left, p, q)
    right := lowestCommonAncestor(root.Right, p, q)
    if left != nil && right != nil {
        return root  // p q 分别在左右子树
    }
    if left != nil { return left }
    return right
}
```

**从前序+中序重建二叉树：**

```go
func buildTree(preorder, inorder []int) *TreeNode {
    if len(preorder) == 0 { return nil }
    root := &TreeNode{Val: preorder[0]}
    mid := indexOf(inorder, preorder[0])
    root.Left = buildTree(preorder[1:mid+1], inorder[:mid])
    root.Right = buildTree(preorder[mid+1:], inorder[mid+1:])
    return root
}
```

## 8. 什么是线段树和树状数组？

**回答：**

**线段树：** 用于区间查询和区间修改，O(log n)。

```
数组 [1, 3, 5, 7, 9, 11]

线段树（区间和）：
            [0,5]=36
           /        \
      [0,2]=9      [3,5]=27
      /    \       /     \
   [0,1]=4 [2]=5 [3,4]=16 [5]=11
   /   \         /    \
 [0]=1 [1]=3  [3]=7  [4]=9
```

**树状数组（Binary Indexed Tree / Fenwick Tree）：**

```
更轻量的区间求和方案：
- 单点修改 O(log n)
- 前缀和查询 O(log n)
- 利用 lowbit(x) = x & (-x)

// 更新
func update(tree []int, i, delta, n int) {
    for ; i <= n; i += i & (-i) {
        tree[i] += delta
    }
}

// 前缀和查询
func query(tree []int, i int) int {
    sum := 0
    for ; i > 0; i -= i & (-i) {
        sum += tree[i]
    }
    return sum
}
```

| 特性 | 线段树 | 树状数组 |
|------|--------|---------|
| 区间查询 | ✅ 灵活（和/最值等）| ✅ 主要求和 |
| 区间修改 | ✅ 懒标记 | 差分数组 |
| 空间 | 4n | n |
| 实现 | 复杂 | 简单 |

## 9. 二叉树的序列化和反序列化？

**回答：**

```go
// 前序序列化（用 "null" 表示空节点）
func serialize(root *TreeNode) string {
    if root == nil {
        return "null"
    }
    return fmt.Sprintf("%d,%s,%s",
        root.Val,
        serialize(root.Left),
        serialize(root.Right))
}

// 反序列化
func deserialize(data string) *TreeNode {
    vals := strings.Split(data, ",")
    idx := 0
    var build func() *TreeNode
    build = func() *TreeNode {
        if idx >= len(vals) || vals[idx] == "null" {
            idx++
            return nil
        }
        val, _ := strconv.Atoi(vals[idx])
        idx++
        node := &TreeNode{Val: val}
        node.Left = build()
        node.Right = build()
        return node
    }
    return build()
}
```

       1
      / \
     2   3
        / \
       4   5

序列化："1,2,null,null,3,4,null,null,5,null,null"

也可以用层序（BFS）序列化：
"1,2,3,null,null,4,5,null,null,null,null"

## 10. 树与二叉树面试速答？

**回答：**

```
Q: 满二叉树和完全二叉树的区别？
A: 满二叉树每层都满；完全二叉树除最后一层外都满，最后一层从左到右连续

Q: n 个节点的完全二叉树高度？
A: ⌊log₂n⌋

Q: 二叉树前中后序遍历的时间复杂度？
A: 都是 O(n)，每个节点访问一次

Q: BST 中序遍历的结果？
A: 升序排列

Q: 红黑树最多旋转几次？
A: 插入最多2次，删除最多3次

Q: B+ 树的阶是什么意思？
A: 每个节点最多有 m 个子节点（m 阶）

Q: 堆排序是稳定排序吗？
A: 不是

Q: 为什么前缀树用空间换时间？
A: 每个节点最多26个子指针，空间大但查找快 O(L)，L 是字符串长度

Q: 如何判断一棵树是否是 BST？
A: 中序遍历严格递增，或递归传递上下界

Q: 哈夫曼树用于什么？
A: 数据压缩（哈夫曼编码），频率高的字符用短编码
```
