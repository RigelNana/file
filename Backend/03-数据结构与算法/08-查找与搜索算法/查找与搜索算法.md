## 1. 二分查找的原理和常见变体？

**回答：**

**标准二分查找：**

```go
func binarySearch(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := left + (right-left)/2  // 防溢出
        if nums[mid] == target {
            return mid
        } else if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return -1
}
```

**常见变体：**

```go
// 查找第一个等于 target 的位置（左边界）
func searchFirst(nums []int, target int) int {
    left, right := 0, len(nums)-1
    result := -1
    for left <= right {
        mid := left + (right-left)/2
        if nums[mid] >= target {
            if nums[mid] == target {
                result = mid
            }
            right = mid - 1
        } else {
            left = mid + 1
        }
    }
    return result
}

// 查找最后一个等于 target 的位置（右边界）
func searchLast(nums []int, target int) int {
    left, right := 0, len(nums)-1
    result := -1
    for left <= right {
        mid := left + (right-left)/2
        if nums[mid] <= target {
            if nums[mid] == target {
                result = mid
            }
            left = mid + 1
        } else {
            right = mid - 1
        }
    }
    return result
}

// 查找第一个大于等于 target 的位置
func lowerBound(nums []int, target int) int {
    left, right := 0, len(nums)
    for left < right {
        mid := left + (right-left)/2
        if nums[mid] < target {
            left = mid + 1
        } else {
            right = mid
        }
    }
    return left
}
```

## 2. 二分查找的应用场景有哪些？

**回答：**

**1. 旋转排序数组搜索：**

```go
func searchRotated(nums []int, target int) int {
    left, right := 0, len(nums)-1
    for left <= right {
        mid := left + (right-left)/2
        if nums[mid] == target {
            return mid
        }
        if nums[left] <= nums[mid] { // 左半有序
            if nums[left] <= target && target < nums[mid] {
                right = mid - 1
            } else {
                left = mid + 1
            }
        } else { // 右半有序
            if nums[mid] < target && target <= nums[right] {
                left = mid + 1
            } else {
                right = mid - 1
            }
        }
    }
    return -1
}
```

**2. 二分答案（猜答案+验证）：**

```go
// 例：分割数组的最大值最小化
// 二分答案范围 [max(nums), sum(nums)]
func splitArray(nums []int, k int) int {
    left, right := maxOf(nums), sumOf(nums)
    for left < right {
        mid := left + (right-left)/2
        if canSplit(nums, k, mid) {
            right = mid
        } else {
            left = mid + 1
        }
    }
    return left
}
```

**3. 浮点数二分（求平方根等）：**

```go
func mySqrt(x float64) float64 {
    left, right := 0.0, x
    for right-left > 1e-7 {
        mid := (left + right) / 2
        if mid*mid > x {
            right = mid
        } else {
            left = mid
        }
    }
    return left
}
```

## 3. 回溯算法的基本模板？

**回答：**

```go
// 回溯模板
func backtrack(路径, 选择列表) {
    if 满足结束条件 {
        结果集 = append(结果集, 路径)
        return
    }
    for 选择 in 选择列表 {
        做选择        // 将选择加入路径
        backtrack(路径, 选择列表)
        撤销选择      // 将选择从路径移除
    }
}
```

**全排列：**

```go
func permute(nums []int) [][]int {
    var result [][]int
    var path []int
    used := make([]bool, len(nums))

    var backtrack func()
    backtrack = func() {
        if len(path) == len(nums) {
            temp := make([]int, len(path))
            copy(temp, path)
            result = append(result, temp)
            return
        }
        for i := 0; i < len(nums); i++ {
            if used[i] { continue }
            used[i] = true
            path = append(path, nums[i])
            backtrack()
            path = path[:len(path)-1]
            used[i] = false
        }
    }
    backtrack()
    return result
}
```

**子集：**

```go
func subsets(nums []int) [][]int {
    var result [][]int
    var path []int

    var backtrack func(start int)
    backtrack = func(start int) {
        temp := make([]int, len(path))
        copy(temp, path)
        result = append(result, temp)

        for i := start; i < len(nums); i++ {
            path = append(path, nums[i])
            backtrack(i + 1)
            path = path[:len(path)-1]
        }
    }
    backtrack(0)
    return result
}
```

## 4. 如何在矩阵中进行搜索？

**回答：**

**搜索二维矩阵（每行递增，下行首元素 > 上行末）— O(log(mn))：**

```go
func searchMatrix(matrix [][]int, target int) bool {
    m, n := len(matrix), len(matrix[0])
    left, right := 0, m*n-1
    for left <= right {
        mid := left + (right-left)/2
        val := matrix[mid/n][mid%n]
        if val == target { return true }
        if val < target { left = mid + 1 } else { right = mid - 1 }
    }
    return false
}
```

**搜索二维矩阵II（每行每列递增）— O(m+n)：**

```go
func searchMatrixII(matrix [][]int, target int) bool {
    m, n := len(matrix), len(matrix[0])
    row, col := 0, n-1  // 从右上角开始
    for row < m && col >= 0 {
        if matrix[row][col] == target {
            return true
        } else if matrix[row][col] > target {
            col--  // 排除当前列
        } else {
            row++  // 排除当前行
        }
    }
    return false
}
```

## 5. 什么是分治算法？经典应用有哪些？

**回答：**

```
分治思想：
┌──────────────────────────────────┐
│ 原问题                           │
│   ├─→ 分解为子问题               │
│   │     ├─→ 子问题1 → 解1        │
│   │     └─→ 子问题2 → 解2        │
│   └─→ 合并解1和解2 → 原问题的解  │
└──────────────────────────────────┘
```

**经典应用：**

| 问题 | 分治策略 | 时间 |
|------|---------|------|
| 归并排序 | 分两半，排序，合并 | O(n log n) |
| 快速排序 | 分区，递归 | O(n log n) |
| 最大子数组和 | 分两半+跨中间 | O(n log n) |
| 求逆序对 | 归并排序中计数 | O(n log n) |
| 最近点对 | 分两半+跨分界线 | O(n log n) |
| Strassen矩阵乘法 | 分块乘法 | O(n^2.81) |
| 大整数乘法 | Karatsuba | O(n^1.585) |

**求逆序对（归并排序变体）：**

```go
func reversePairs(nums []int) int {
    count := 0
    mergeSort(nums, 0, len(nums)-1, &count)
    return count
}

func mergeSort(nums []int, left, right int, count *int) {
    if left >= right { return }
    mid := (left + right) / 2
    mergeSort(nums, left, mid, count)
    mergeSort(nums, mid+1, right, count)
    merge(nums, left, mid, right, count)
}

func merge(nums []int, left, mid, right int, count *int) {
    temp := make([]int, right-left+1)
    i, j, k := left, mid+1, 0
    for i <= mid && j <= right {
        if nums[i] <= nums[j] {
            temp[k] = nums[i]; i++
        } else {
            *count += mid - i + 1  // 关键：统计逆序对
            temp[k] = nums[j]; j++
        }
        k++
    }
    for i <= mid { temp[k] = nums[i]; i++; k++ }
    for j <= right { temp[k] = nums[j]; j++; k++ }
    copy(nums[left:], temp)
}
```

## 6. KMP 算法的原理？

**回答：**

KMP 解决字符串匹配问题，O(n+m) 时间。

```
暴力匹配 O(nm) vs KMP O(n+m)

KMP 核心：利用已匹配部分的信息，避免回退主串指针

next 数组（前缀函数）：
模式串: A B A B C
next:  [0,0,1,2,0]

含义：next[i] = pattern[0..i] 的最长相等前后缀长度
ABAB → 前缀"AB" = 后缀"AB"，next[3]=2
```

```go
func kmpSearch(text, pattern string) int {
    n, m := len(text), len(pattern)
    if m == 0 { return 0 }

    // 构建 next 数组
    next := make([]int, m)
    j := 0
    for i := 1; i < m; i++ {
        for j > 0 && pattern[i] != pattern[j] {
            j = next[j-1]
        }
        if pattern[i] == pattern[j] {
            j++
        }
        next[i] = j
    }

    // 匹配
    j = 0
    for i := 0; i < n; i++ {
        for j > 0 && text[i] != pattern[j] {
            j = next[j-1]  // 不回退 i，只回退 j
        }
        if text[i] == pattern[j] {
            j++
        }
        if j == m {
            return i - m + 1  // 匹配成功
        }
    }
    return -1
}
```

## 7. 贪心算法的原理和经典问题？

**回答：**

**贪心策略：** 每步选择局部最优，期望达到全局最优。需要证明贪心选择性质。

**区间调度（最多不重叠区间）：**

```go
func intervalSchedule(intervals [][]int) int {
    // 按结束时间排序
    sort.Slice(intervals, func(i, j int) bool {
        return intervals[i][1] < intervals[j][1]
    })
    count := 1
    end := intervals[0][1]
    for i := 1; i < len(intervals); i++ {
        if intervals[i][0] >= end {
            count++
            end = intervals[i][1]
        }
    }
    return count
}
```

**跳跃游戏：**

```go
func canJump(nums []int) bool {
    maxReach := 0
    for i := 0; i < len(nums); i++ {
        if i > maxReach { return false }
        if i+nums[i] > maxReach {
            maxReach = i + nums[i]
        }
    }
    return true
}
```

**常见贪心问题：**

| 问题 | 贪心策略 |
|------|---------|
| 零钱兑换（特定面值）| 从大到小选面值 |
| 区间调度 | 按结束时间排序 |
| 加油站 | 累计净油量 |
| 分配糖果 | 两次遍历 |
| 任务调度器 | 最高频任务优先 |

## 8. 双指针和滑动窗口的核心思想？

**回答：**

**对撞指针 — 三数之和：**

```go
func threeSum(nums []int) [][]int {
    sort.Ints(nums)
    var result [][]int
    for i := 0; i < len(nums)-2; i++ {
        if i > 0 && nums[i] == nums[i-1] { continue } // 去重
        left, right := i+1, len(nums)-1
        for left < right {
            sum := nums[i] + nums[left] + nums[right]
            if sum == 0 {
                result = append(result, []int{nums[i], nums[left], nums[right]})
                for left < right && nums[left] == nums[left+1] { left++ }
                for left < right && nums[right] == nums[right-1] { right-- }
                left++; right--
            } else if sum < 0 {
                left++
            } else {
                right--
            }
        }
    }
    return result
}
```

**滑动窗口模板：**

```go
func slidingWindow(s string) int {
    window := map[byte]int{}
    left := 0
    result := 0

    for right := 0; right < len(s); right++ {
        // 1. 扩大窗口
        c := s[right]
        window[c]++

        // 2. 条件满足时收缩窗口
        for /* 需要收缩 */ {
            d := s[left]
            window[d]--
            left++
        }

        // 3. 更新结果
        result = max(result, right-left+1)
    }
    return result
}
```

**无重复字符的最长子串：**

```go
func lengthOfLongestSubstring(s string) int {
    window := map[byte]int{}
    left, result := 0, 0
    for right := 0; right < len(s); right++ {
        window[s[right]]++
        for window[s[right]] > 1 {
            window[s[left]]--
            left++
        }
        result = max(result, right-left+1)
    }
    return result
}
```

## 9. 位运算的常见技巧？

**回答：**

| 操作 | 表达式 | 用途 |
|------|--------|------|
| 判断奇偶 | `n & 1` | 0偶1奇 |
| 除以2 | `n >> 1` | 等价于 n/2 |
| 乘以2 | `n << 1` | 等价于 n*2 |
| 取最低位1 | `n & (-n)` | lowbit |
| 消除最低位1 | `n & (n-1)` | 统计1的个数 |
| 两数交换 | `a ^= b; b ^= a; a ^= b` | 不用临时变量 |
| 判断2的幂 | `n > 0 && n & (n-1) == 0` | 只有一个1 |

**统计二进制中 1 的个数：**

```go
func hammingWeight(n uint32) int {
    count := 0
    for n > 0 {
        n &= n - 1  // 消除最低位的 1
        count++
    }
    return count
}
```

**只出现一次的数字：**

```go
// 其他数字都出现两次
func singleNumber(nums []int) int {
    result := 0
    for _, n := range nums {
        result ^= n  // a ^ a = 0, a ^ 0 = a
    }
    return result
}
```

## 10. 查找与搜索面试速答？

**回答：**

```
Q: 二分查找的前提条件？
A: 数组有序（或满足单调性）

Q: 二分查找的时间复杂度？
A: O(log n)

Q: 回溯和 DFS 的区别？
A: 回溯是 DFS + 撤销选择，本质相同，回溯强调"试探-撤销"

Q: 贪心算法一定能得到最优解吗？
A: 不一定，需要证明贪心选择性质和最优子结构

Q: KMP 的时间复杂度？
A: O(n+m)，n 主串长度，m 模式串长度

Q: 什么问题适合用二分答案？
A: 答案具有单调性，可以用 O(n) 验证某个答案是否可行

Q: 滑动窗口适用于什么问题？
A: 连续子数组/子串的最值/计数问题

Q: 位运算 n & (n-1) 的作用？
A: 消除 n 的最低位的 1

Q: 分治和 DP 的区别？
A: 分治子问题互不重叠，DP 子问题重叠（需要记忆化）

Q: 如何判断一个数是否是2的幂？
A: n > 0 && n & (n-1) == 0
```
