## 学习建议

### 优先级排序（由高到低）

| 优先级 | 类别 | 说明 |
|--------|------|------|
| ⭐⭐⭐⭐⭐ | Linux基础 | DevOps的根基，必须熟练掌握 |
| ⭐⭐⭐⭐⭐ | Docker容器 | 容器化是现代DevOps的核心 |
| ⭐⭐⭐⭐⭐ | Kubernetes | 容器编排事实标准 |
| ⭐⭐⭐⭐⭐ | CI/CD | DevOps工程师的核心工作 |
| ⭐⭐⭐⭐ | 计算机网络 | 排障和架构设计的基础 |
| ⭐⭐⭐⭐ | Git版本控制 | 日常工作必备 |
| ⭐⭐⭐⭐ | Shell脚本 | 自动化的基础工具 |
| ⭐⭐⭐⭐ | 监控与日志 | 保障系统可观测性 |
| ⭐⭐⭐ | Nginx与负载均衡 | Web服务必备知识 |
| ⭐⭐⭐ | Ansible配置管理 | 配置管理主流工具 |
| ⭐⭐⭐ | Terraform IaC | 基础设施即代码主流工具 |
| ⭐⭐⭐ | 云计算 | 云原生时代必备 |
| ⭐⭐ | 数据库基础 | 了解即可，不需要DBA级别 |
| ⭐⭐ | 安全DevSecOps | 安全左移趋势 |
| ⭐⭐ | 微服务架构 | 理解架构设计即可 |
