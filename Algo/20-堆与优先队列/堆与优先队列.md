# 堆与优先队列

> 堆（二叉堆）是一种能 O(log n) 插入/删除、O(1) 查询最值的数据结构，是贪心和图算法的核心工具。

---

## 1. 堆的基础 ⭐

### 理解与可视化

```
最小堆: 父节点 ≤ 子节点

  完全二叉树:
          1
        /   \
       3     2
      / \   /
     7   4  5

  数组表示: [1, 3, 2, 7, 4, 5]
  父节点: (i-1)//2    子节点: 2i+1, 2i+2

  核心操作:
    push (上浮): 插入末尾, 与父比较, 小则交换, 直到合适位置
    pop  (下沉): 取走堆顶, 末尾补上, 与较小子比较, 大则交换

  Python heapq 是最小堆
  要用最大堆 → 取负
```

### 代码模板

```python
import heapq

# 基本操作
heap = []
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappush(heap, 2)
top = heapq.heappop(heap)     # 1 (最小值)
peek = heap[0]                 # 2 (查看堆顶)

# 最大堆 → 取负
max_heap = []
heapq.heappush(max_heap, -5)
heapq.heappush(max_heap, -3)
largest = -heapq.heappop(max_heap)  # 5

# 从列表建堆 O(n)
nums = [5, 3, 1, 4, 2]
heapq.heapify(nums)  # 原地建堆

# 前 k 大/小
heapq.nlargest(3, nums)   # 前3大
heapq.nsmallest(3, nums)  # 前3小
```

## 2. Top-K 问题 ⭐

### 理解与可视化

```
第 K 大元素:
  方法1: 排序 O(n log n)
  方法2: 最小堆 O(n log k) ── 维护大小为 k 的最小堆
  方法3: 快速选择 O(n) 平均

  最小堆法:
    维护 k 个最大元素的最小堆
    堆顶就是第 k 大

    [3, 2, 1, 5, 6, 4], k = 2
    遍历: 堆变化:
      3 → [3]
      2 → [2, 3]           堆满k=2
      1 → [2, 3]           1 < 堆顶2, 跳过
      5 → [3, 5]           pop 2, push 5
      6 → [5, 6]           pop 3, push 6
      4 → [5, 6]           4 < 堆顶5, 跳过
    答案: 堆顶 = 5
```

### 代码模板

```python
import heapq

# 第 k 大元素 (最小堆法)
def find_kth_largest(nums, k):
    heap = []
    for x in nums:
        heapq.heappush(heap, x)
        if len(heap) > k:
            heapq.heappop(heap)
    return heap[0]

# 前 K 个高频元素
from collections import Counter
def top_k_frequent(nums, k):
    count = Counter(nums)
    return heapq.nlargest(k, count.keys(), key=count.get)

# 数据流中的中位数（对顶堆）
class MedianFinder:
    def __init__(self):
        self.small = []   # 最大堆（存负数）── 较小的一半
        self.large = []   # 最小堆 ── 较大的一半

    def addNum(self, num):
        heapq.heappush(self.small, -num)
        heapq.heappush(self.large, -heapq.heappop(self.small))
        if len(self.large) > len(self.small):
            heapq.heappush(self.small, -heapq.heappop(self.large))

    def findMedian(self):
        if len(self.small) > len(self.large):
            return -self.small[0]
        return (-self.small[0] + self.large[0]) / 2
```

```python
# 查找和最小的 K 对数字
def k_smallest_pairs(nums1, nums2, k):
    if not nums1 or not nums2:
        return []
    heap = [(nums1[i] + nums2[0], i, 0) for i in range(min(k, len(nums1)))]
    heapq.heapify(heap)
    result = []
    while heap and len(result) < k:
        _, i, j = heapq.heappop(heap)
        result.append([nums1[i], nums2[j]])
        if j + 1 < len(nums2):
            heapq.heappush(heap, (nums1[i] + nums2[j+1], i, j + 1))
    return result
```
