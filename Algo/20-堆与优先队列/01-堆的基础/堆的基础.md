# 堆的基础

> 堆是能 O(log n) 插入/删除、O(1) 查询最值的数据结构，Python 中用 heapq 实现最小堆。

---

## 1. 堆的原理

```
堆 = 完全二叉树 + 堆序性质

最小堆: 父 ≤ 子 → 堆顶是最小值
最大堆: 父 ≥ 子 → 堆顶是最大值

完全二叉树 → 用数组表示:
  父节点: (i-1) // 2
  左子:   2*i + 1
  右子:   2*i + 2

  例: [1, 3, 2, 7, 4, 5]

          1          │ index: 0
        /   \        │
       3     2       │ index: 1, 2
      / \   /        │
     7   4  5        │ index: 3, 4, 5

核心操作:
  push (上浮 sift up):
    新元素加到末尾
    与父节点比较, 比父小就交换
    直到合适位置
    O(log n)

  pop (下沉 sift down):
    取走堆顶 (最小值)
    末尾元素放到堆顶
    与较小的子节点比较, 比子大就交换
    直到合适位置
    O(log n)

  heapify (建堆):
    从最后一个非叶节点开始, 依次下沉
    O(n) (不是 O(n log n)!)
```

---

## 2. Python heapq 用法

```python
import heapq

# === 基本操作 ===

# 创建空堆
heap = []

# push: 插入元素
heapq.heappush(heap, 3)
heapq.heappush(heap, 1)
heapq.heappush(heap, 2)
# heap = [1, 3, 2]

# pop: 弹出最小值
top = heapq.heappop(heap)       # 1
# heap = [2, 3]

# peek: 查看堆顶 (不弹出)
peek = heap[0]                   # 2

# pushpop: push 然后 pop (比分开做更快)
val = heapq.heappushpop(heap, 0)  # push 0, pop 0 → val=0

# replace: pop 然后 push (比分开做更快)
val = heapq.heapreplace(heap, 5)  # pop 2, push 5 → val=2

# === 建堆 ===
nums = [5, 3, 1, 4, 2]
heapq.heapify(nums)              # 原地建堆, O(n)
# nums = [1, 2, 5, 4, 3] (最小堆)

# === 前 K 大/小 ===
heapq.nlargest(3, nums)          # [5, 4, 3] 前3大
heapq.nsmallest(3, nums)         # [1, 2, 3] 前3小

# ⚠️ nlargest/nsmallest:
#    k 小时用堆 O(n log k)
#    k 大时不如排序 O(n log n)
```

## 3. 最大堆技巧

```python
import heapq

# Python heapq 只有最小堆
# 最大堆 → 存负数!

max_heap = []
heapq.heappush(max_heap, -5)
heapq.heappush(max_heap, -3)
heapq.heappush(max_heap, -8)

largest = -heapq.heappop(max_heap)  # 8
# max_heap = [-5, -3]

# ⚠️ 存负数:
#    push 时取负: heappush(h, -x)
#    pop 时取负:  -heappop(h) 得到原值
#    peek:       -h[0]
```

## 4. 元组比较与自定义排序

```python
import heapq

# 元组: 按第一个元素排序, 相同则比较第二个
heap = []
heapq.heappush(heap, (2, 'b'))
heapq.heappush(heap, (1, 'a'))
heapq.heappush(heap, (2, 'a'))

heapq.heappop(heap)  # (1, 'a')
heapq.heappop(heap)  # (2, 'a')
heapq.heappop(heap)  # (2, 'b')

# ⚠️ 如果元素不可比较, 加 index 打破平局:
#    heappush(heap, (priority, index, item))
#    index 单调递增, 保证不会比较 item

counter = 0
heapq.heappush(heap, (priority, counter, item))
counter += 1

# ⚠️ 常见错误:
#    heappush(heap, (1, some_object))
#    如果优先级相同, 会比较 some_object
#    如果 some_object 不支持 < → TypeError
```

## 5. 懒删除技巧

```python
import heapq

# 堆不支持高效删除任意元素
# 懒删除: 标记删除, 弹出时检查

class LazyHeap:
    def __init__(self):
        self.heap = []
        self.deleted = set()
    
    def push(self, val):
        heapq.heappush(self.heap, val)
    
    def remove(self, val):
        self.deleted.add(val)          # ⚠️ 只标记, 不真删
    
    def pop(self):
        while self.heap:
            val = heapq.heappop(self.heap)
            if val not in self.deleted:
                return val             # ⚠️ 有效元素
            self.deleted.discard(val)   # ⚠️ 已标记的, 丢弃
        return None
    
    def peek(self):
        while self.heap and self.heap[0] in self.deleted:
            self.deleted.discard(heapq.heappop(self.heap))
        return self.heap[0] if self.heap else None

# ⚠️ 懒删除适用于:
#    需要从堆中删除特定元素, 但不知道它在哪
#    例: 滑动窗口中维护最值, 移除过期元素
#
# ⚠️ 如果有重复值, 用 (val, index) 作为标识更安全
```

## 本节要点速查

```
✅ 最小堆: 父≤子, 堆顶最小, push/pop O(log n)
✅ heapify O(n), nlargest/nsmallest 小k用
✅ 最大堆: 存负数 → push(-x), pop取负
✅ 元组排序: 第一个相同比第二个, 加index防比较对象
✅ 懒删除: 标记删除, 弹出时跳过已删元素
✅ heap[0] 查看堆顶, 不弹出
```
