# 多路归并

> 用最小堆高效合并多个有序序列，是优先队列在归并类问题中的核心应用。

---

## 1. 核心思想

```
多路归并: 合并 K 个有序序列

  暴力: 全部放一起排序 O(N log N), N = 总元素数
  堆法: 维护 K 个指针, 每次取最小 O(N log K)

  可视化:
    序列1: [1, 4, 7]
    序列2: [2, 5, 8]  
    序列3: [3, 6, 9]

    堆 (维护每个序列的当前最小):
    初始: [(1,0), (2,1), (3,2)]  → 弹出 1, 加入 4
          [(2,1), (3,2), (4,0)]  → 弹出 2, 加入 5
          [(3,2), (4,0), (5,1)]  → 弹出 3, 加入 6
          ...

    结果: [1, 2, 3, 4, 5, 6, 7, 8, 9]

  堆中始终只有 K 个元素
  每次 pop + push = O(log K)
  总共 N 个元素 → O(N log K)

  ⚠️ 当 K << N 时, O(N log K) << O(N log N)
```

## 2. LeetCode 23: 合并K个升序链表

```python
import heapq

class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

def mergeKLists(lists):
    heap = []
    
    # 初始化: 每个链表的头节点入堆
    for i, node in enumerate(lists):
        if node:
            heapq.heappush(heap, (node.val, i, node))
            # ⚠️ (val, i, node): i 用于打破 val 相同时的比较
    
    dummy = ListNode(0)
    curr = dummy
    
    while heap:
        val, i, node = heapq.heappop(heap)
        curr.next = node
        curr = curr.next
        
        if node.next:
            heapq.heappush(heap, (node.next.val, i, node.next))
            # ⚠️ 弹出后把同一链表的下一个节点入堆
    
    return dummy.next

# ⚠️ 为什么需要 i (链表编号)?
#    堆比较元组: 先比 val, 相同比 i
#    如果没有 i, 会比较 ListNode 对象 → TypeError
#
# ⚠️ 时间: O(N log K), N = 总节点数, K = 链表数
#    空间: O(K) 堆大小
```

## 3. LeetCode 373: 查找和最小的K对数字

```
给定两个升序数组 nums1, nums2
找和最小的 k 对 (nums1[i], nums2[j])

思路: 类似多路归并
  把 nums1 的每个元素看作一条"序列的起点"
  nums1[i] + nums2[0] ≤ nums1[i] + nums2[1] ≤ ...
  K 路序列, 归并取前 K 小

  优化: 不需要把所有 nums1 都入堆
        先入前 min(k, len(nums1)) 个
```

```python
import heapq

def kSmallestPairs(nums1, nums2, k):
    if not nums1 or not nums2:
        return []
    
    heap = []
    # 初始化: nums1 的前 k 个元素与 nums2[0] 配对
    for i in range(min(k, len(nums1))):
        heapq.heappush(heap, (nums1[i] + nums2[0], i, 0))
        # ⚠️ (和, nums1下标, nums2下标)
    
    result = []
    while heap and len(result) < k:
        _, i, j = heapq.heappop(heap)
        result.append([nums1[i], nums2[j]])
        
        if j + 1 < len(nums2):
            heapq.heappush(heap, (nums1[i] + nums2[j+1], i, j+1))
            # ⚠️ 同一个 nums1[i], 配 nums2 的下一个
    
    return result

# ⚠️ 模式: 每弹出 (i,j), 推入 (i, j+1)
#    j 在 nums2 中前进, i 固定
#    这样不会遗漏, 也不会重复
#
# ⚠️ 时间: O(k log k), 堆大小 ≤ k
```

## 4. LeetCode 378: 有序矩阵中第K小的元素

```
n×n 矩阵, 每行每列都递增
找第 K 小的元素

方法1: 多路归并 (每行是一条有序序列)
方法2: 二分查找 (值域二分)
```

```python
import heapq

# 方法1: 多路归并
def kthSmallest(matrix, k):
    n = len(matrix)
    heap = []
    
    # 每行第一个元素入堆
    for i in range(min(k, n)):
        heapq.heappush(heap, (matrix[i][0], i, 0))
    
    for _ in range(k):
        val, i, j = heapq.heappop(heap)
        if j + 1 < n:
            heapq.heappush(heap, (matrix[i][j+1], i, j+1))
    
    return val                         # ⚠️ 第 k 次弹出的就是第 k 小

# 方法2: 二分查找
def kthSmallest_bs(matrix, k):
    n = len(matrix)
    lo, hi = matrix[0][0], matrix[n-1][n-1]
    
    while lo < hi:
        mid = (lo + hi) // 2
        count = 0
        j = n - 1
        for i in range(n):
            while j >= 0 and matrix[i][j] > mid:
                j -= 1
            count += j + 1             # ⚠️ 第i行有 j+1 个 ≤ mid
        
        if count < k:
            lo = mid + 1
        else:
            hi = mid
    
    return lo

# ⚠️ 方法1: O(k log n) 
# ⚠️ 方法2: O(n log(max-min))
#    k 小时方法1好, k 大时方法2好
```

## 5. 多路归并模式总结

```
模式:
  1. 初始化: 每条序列的第一个元素入堆
  2. 循环: 弹出最小, 从同一序列加入下一个
  3. 直到处理完所有元素或达到 K 个

  ⚠️ 堆中存 (值, 序列标识, 序列内位置)
  ⚠️ 弹出 (i, j) → 推入 (i, j+1)

适用场景:
  · 合并K个有序链表/数组
  · K对最小和
  · 有序矩阵第K小
  · 外部排序 (磁盘分块排序后合并)
```

```
✅ 多路归并: K条有序序列, 堆维护K个指针
✅ 弹出(i,j) → 推入(i,j+1), 不遗漏不重复
✅ 堆存(值, 序列号, 位置), 序列号打破平局
✅ 合并K链表: O(N log K)
✅ K对最小和: 初始化前k个, j方向前进
✅ 有序矩阵: 多路归并 O(k log n) 或二分
```
