# 并查集

> 并查集（Union-Find）高效处理「动态连通性」问题：合并集合、查询是否连通。

---

## 1. 并查集基础 ⭐⭐

### 理解与可视化

```
并查集: 通过 parent 数组维护"集合归属"

  初始: 每个元素是自己的父节点
    parent: [0, 1, 2, 3, 4]

  union(0, 1): 让 1 的根指向 0 的根
    parent: [0, 0, 2, 3, 4]
         0
         |
         1

  union(2, 3):
    parent: [0, 0, 2, 2, 4]

  union(1, 3): find(1)=0, find(3)=2, 让 2 指向 0
    parent: [0, 0, 0, 2, 4]
         0
        / \
       1   2
           |
           3

  两个优化:
    1. 路径压缩: find 时直接指向根
    2. 按秩合并: 矮树挂到高树上

  复杂度: 均摊 O(α(n)) ≈ O(1)
```

### 代码模板

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
        self.count = n  # 连通分量数

    def find(self, x):
        while self.parent[x] != x:
            self.parent[x] = self.parent[self.parent[x]]  # 路径压缩
            x = self.parent[x]
        return x

    def union(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return False  # 已连通
        # 按秩合并
        if self.rank[rx] < self.rank[ry]:
            rx, ry = ry, rx
        self.parent[ry] = rx
        if self.rank[rx] == self.rank[ry]:
            self.rank[rx] += 1
        self.count -= 1
        return True

    def connected(self, x, y):
        return self.find(x) == self.find(y)
```

## 2. 经典应用

### 代码模板

```python
# 冗余连接（找到添加后导致成环的那条边）
def find_redundant_connection(edges):
    n = len(edges)
    uf = UnionFind(n + 1)
    for u, v in edges:
        if not uf.union(u, v):
            return [u, v]

# 岛屿数量（并查集解法）
def num_islands_uf(grid):
    if not grid:
        return 0
    m, n = len(grid), len(grid[0])
    uf = UnionFind(m * n)
    water = 0
    for i in range(m):
        for j in range(n):
            if grid[i][j] == '0':
                water += 1
                continue
            for di, dj in [(1, 0), (0, 1)]:
                ni, nj = i + di, j + dj
                if 0 <= ni < m and 0 <= nj < n and grid[ni][nj] == '1':
                    uf.union(i * n + j, ni * n + nj)
    return uf.count - water

# 账户合并
from collections import defaultdict
def accounts_merge(accounts):
    uf = UnionFind(len(accounts))
    email_to_id = {}
    for i, account in enumerate(accounts):
        for email in account[1:]:
            if email in email_to_id:
                uf.union(i, email_to_id[email])
            email_to_id[email] = i

    # 归类
    groups = defaultdict(set)
    for email, i in email_to_id.items():
        groups[uf.find(i)].add(email)

    return [[accounts[i][0]] + sorted(emails)
            for i, emails in groups.items()]
```

## 3. 带权并查集

### 理解与可视化

```
带权并查集: 边上带权值, 表示两个元素之间的关系

  例: 除法求值 a/b=2, b/c=3 → a/c=?
  将 a→b 权重设为 2, b→c 权重设为 3
  查询 a→c: 路径上权重之积 = 2×3 = 6
```

### 代码模板

```python
# 带权并查集（除法求值）
class WeightedUnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.weight = [1.0] * n  # weight[i] = i 到 parent[i] 的权重

    def find(self, x):
        if self.parent[x] != x:
            root = self.find(self.parent[x])
            self.weight[x] *= self.weight[self.parent[x]]
            self.parent[x] = root
        return self.parent[x]

    def union(self, x, y, w):
        """x / y = w"""
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return
        self.parent[rx] = ry
        self.weight[rx] = w * self.weight[y] / self.weight[x]

    def query(self, x, y):
        if self.find(x) != self.find(y):
            return -1.0
        return self.weight[x] / self.weight[y]
```
