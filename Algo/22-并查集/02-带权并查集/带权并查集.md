# 带权并查集

> 在并查集的边上附加权值，表示元素之间的定量关系（如比值、大小差等）。

---

## 1. 核心思想

```
带权并查集: parent 边上带权重

  普通并查集: 只知道 a 和 b 连通
  带权并查集: 知道 a/b = 2, b/c = 3 → a/c = 6

  权值含义: weight[x] = x 到 parent[x] 的权重
  
  例: a/b=2, b/c=3
    a ──2→ b ──3→ c (c是根)
    weight[a] = 2 (a = 2 × parent[a] = 2 × b)
    weight[b] = 3 (b = 3 × parent[b] = 3 × c)

    a/c = weight[a] × weight[b] = 6
    (沿路径把权值乘起来)

  查询 a/b:
    find(a) → 压缩路径, weight[a] 变成 a 到根的权
    find(b) → 压缩路径, weight[b] 变成 b 到根的权
    a/b = weight[a] / weight[b]
```

---

## 2. 代码模板

```python
class WeightedUnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.weight = [1.0] * n        # ⚠️ weight[x] = x / parent[x]
        self.rank = [0] * n
    
    def find(self, x):
        if self.parent[x] != x:
            root = self.find(self.parent[x])
            self.weight[x] *= self.weight[self.parent[x]]
            # ⚠️ 路径压缩时累乘权值
            # x/parent[x] × parent[x]/root = x/root
            self.parent[x] = root
        return self.parent[x]
    
    def union(self, x, y, w):
        """合并, 表示 x/y = w"""
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return                     # 已连通
        
        # 算 rx 到 ry 的权值
        # x/y = w, x/rx = weight[x], y/ry = weight[y]
        # rx/ry = (x/y) × (y/ry) / (x/rx) = w × weight[y] / weight[x]
        
        if self.rank[rx] < self.rank[ry]:
            self.parent[rx] = ry
            self.weight[rx] = w * self.weight[y] / self.weight[x]
        else:
            self.parent[ry] = rx
            self.weight[ry] = self.weight[x] / (w * self.weight[y])
            if self.rank[rx] == self.rank[ry]:
                self.rank[rx] += 1
    
    def query(self, x, y):
        """查询 x/y 的值, 不连通返回 -1"""
        if self.find(x) != self.find(y):
            return -1.0
        return self.weight[x] / self.weight[y]
        # ⚠️ find 后 weight[x]=x/root, weight[y]=y/root
        #    x/y = (x/root) / (y/root)

# ⚠️ 路径压缩时权值累乘:
#    find 递归压缩, weight[x] 从"x到父"变成"x到根"
#    必须先递归 find(parent), 再乘上 parent 的 weight
#
# ⚠️ 合并推导:
#    union(x, y, w): 表示 x/y = w
#    需要把 rx 和 ry 连起来
#    rx/ry = (x/rx)^(-1) × (x/y) × (y/ry)
#          = weight[x]^(-1) × w × weight[y]
```

## 3. LeetCode 399: 除法求值

```
给定方程 equations 和对应的值 values
equations[i] = [Ai, Bi], values[i] = Ai/Bi
给定查询 queries[i] = [Ci, Di]
求 Ci/Di 的值, 不存在返回 -1.0
```

```python
def calcEquation(equations, values, queries):
    # 变量名 → 编号
    var_id = {}
    idx = 0
    for a, b in equations:
        if a not in var_id:
            var_id[a] = idx
            idx += 1
        if b not in var_id:
            var_id[b] = idx
            idx += 1
    
    uf = WeightedUnionFind(idx)
    
    for (a, b), v in zip(equations, values):
        uf.union(var_id[a], var_id[b], v)
    
    result = []
    for c, d in queries:
        if c not in var_id or d not in var_id:
            result.append(-1.0)
        else:
            result.append(uf.query(var_id[c], var_id[d]))
    
    return result

# ⚠️ 也可以用图 BFS/DFS 解:
#    a/b=2 → 边 a→b 权2, b→a 权0.5
#    查 c/d: 从 c 到 d 的路径, 权值乘积
#    图解法代码更直觉, 但带权并查集更高效
```

## 4. 带权并查集的其他应用

```
应用场景:
  1. 除法求值: weight = 比值
  2. 食物链/朋友敌人: weight = 关系类别
  3. 相对位置: weight = 坐标差

  食物链 (经典):
    三种动物 A, B, C: A吃B, B吃C, C吃A
    关系用 0, 1, 2 表示:
      0: 同类
      1: x 吃 parent[x]
      2: parent[x] 吃 x (即 x 被吃)
    合并时关系取模: (rel_x - rel_y + w) % 3

  ⚠️ 带权并查集本质:
     记录元素间的定量关系
     find 时沿路径累积 (加/乘/取模)
     query 时用累积值推导
```

## 本节要点速查

```
✅ weight[x] = x到parent的权 (比值/差值)
✅ find压缩时累乘: weight[x] *= weight[parent]
✅ union推导: rx到ry的权 = w × weight[y] / weight[x]
✅ query: x/y = weight[x] / weight[y] (find后)
✅ 除法求值: 变量编号 + 带权合并 + 查询
✅ 也可用图BFS解, 带权并查集更高效
```
