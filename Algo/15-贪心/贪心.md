# 贪心

> 贪心的核心：每一步都做局部最优选择，最终得到全局最优。贪心没有固定模板，关键在于证明贪心策略的正确性。

---

## 1. 贪心基础思想

### 理解与可视化

```
贪心 vs 动态规划:

  DP：考虑所有子问题，自底向上构建最优解
  贪心：每步选最优，不回头

何时可以用贪心？
  1. 贪心选择性质: 局部最优能导出全局最优
  2. 最优子结构: 子问题的最优解包含在全局最优解中

常见贪心策略:
  ┌─────────────────────────────────────┐
  │ 排序贪心: 按某个维度排序后贪心选择     │
  │ 区间贪心: 按端点排序处理区间问题       │
  │ 字典序贪心: 逐位确定最优字符           │
  │ 反悔贪心: 允许撤销之前的选择           │
  └─────────────────────────────────────┘
```

---

## 2. 排序贪心

### 理解与可视化

```
排序贪心: 先按某个属性排序，然后贪心处理

例: 分发饼干
  胃口: [1, 2, 3]    饼干: [1, 1]
  排序后从小到大匹配，满足尽量多的孩子

例: 无重叠区间（最少删除几个区间使不重叠）
  按右端点排序 → 贪心选择结束最早的区间

例: 合并区间
  按左端点排序 → 逐个合并有交集的区间
```

### 代码模板

```python
# 分发饼干
def find_content_children(g, s):
    g.sort()  # 胃口
    s.sort()  # 饼干
    child = cookie = 0
    while child < len(g) and cookie < len(s):
        if s[cookie] >= g[child]:
            child += 1
        cookie += 1
    return child

# 合并区间
def merge_intervals(intervals):
    intervals.sort()
    merged = [intervals[0]]
    for start, end in intervals[1:]:
        if start <= merged[-1][1]:
            merged[-1][1] = max(merged[-1][1], end)
        else:
            merged.append([start, end])
    return merged

# 按身高重建队列
def reconstruct_queue(people):
    people.sort(key=lambda x: (-x[0], x[1]))
    result = []
    for p in people:
        result.insert(p[1], p)
    return result
```

## 3. 区间贪心 ⭐

### 理解与可视化

```
区间问题五大类型:

  ┌─ 最多不相交区间（按右端点排序，贪心选结束最早的）
  │
  ├─ 区间分组（按左端点排序，用堆维护各组的右端点）
  │
  ├─ 最少点覆盖所有区间（按右端点排序，选右端点）
  │
  ├─ 最少区间覆盖目标（按左端点排序，贪心选覆盖最远的）
  │
  └─ 合并区间（按左端点排序，合并重叠的）

最多不相交区间的可视化:
  排序后:
  ──[  ]
  ─────[    ]
  ──────[ ]
  ──────────[   ]
  ─────────────[  ]

  按右端点排序 → 贪心选结束最早 → 不冲突的就选
```

### 代码模板

```python
# 最多不相交区间 / 无重叠区间
def erase_overlap_intervals(intervals):
    intervals.sort(key=lambda x: x[1])  # 按右端点排序
    count = 0
    end = -float('inf')
    for s, e in intervals:
        if s >= end:      # 不重叠，选择这个区间
            end = e
            count += 1
    return len(intervals) - count  # 需要删除的个数

# 用最少箭引爆气球
def find_min_arrow_shots(points):
    points.sort(key=lambda x: x[1])
    arrows = 1
    end = points[0][1]
    for s, e in points[1:]:
        if s > end:
            arrows += 1
            end = e
    return arrows

# 区间分组（会议室 II）
import heapq
def min_meeting_rooms(intervals):
    intervals.sort()
    heap = []  # 存各会议室的结束时间
    for s, e in intervals:
        if heap and heap[0] <= s:
            heapq.heapreplace(heap, e)
        else:
            heapq.heappush(heap, e)
    return len(heap)

# 最少区间覆盖（视频拼接）
def video_stitching(clips, time):
    clips.sort()
    res = 0
    cur_end = 0
    farthest = 0
    i = 0
    while cur_end < time:
        while i < len(clips) and clips[i][0] <= cur_end:
            farthest = max(farthest, clips[i][1])
            i += 1
        if farthest == cur_end:
            return -1
        cur_end = farthest
        res += 1
    return res
```

## 4. 字典序贪心

# 移掉 K 位数字，使剩余数字最小
def remove_k_digits(num, k):
    stack = []
    for d in num:
        while k and stack and stack[-1] > d:
            stack.pop()
            k -= 1
        stack.append(d)
    stack = stack[:len(stack) - k]  # 还需要删的直接从尾部删
    return ''.join(stack).lstrip('0') or '0'

# 拼接最大数（从两个数组中选 k 个数组成最大数）
# 思路: 枚举从 nums1 选 i 个 + 从 nums2 选 k-i 个 → 单调栈选最大子序列 → 合并
```

## 5. 数学贪心

### 代码模板

# 分割整数（使乘积最大）
def integer_break(n):
    if n <= 3:
        return n - 1
    q, r = divmod(n, 3)
    if r == 0:
        return 3 ** q
    elif r == 1:
        return 3 ** (q - 1) * 4
    else:
        return 3 ** q * 2

# 加油站（环形路线）
def can_complete_circuit(gas, cost):
    if sum(gas) < sum(cost):
        return -1
    start = 0
    tank = 0
    for i in range(len(gas)):
        tank += gas[i] - cost[i]
        if tank < 0:
            start = i + 1
            tank = 0
    return start
```

## 6. 反悔贪心

### 理解与可视化

```
反悔贪心: 当前贪心选择可能不是最优
→ 保留"反悔"的机会，后续发现更优就撤销之前的选择

经典: 买卖股票的最佳时机（不限次数，等价于反悔贪心）
  用堆维护之前的决策，当发现更好选择时"反悔"

例: 课程表 III（最多能选几门课？）
  按结束时间排序
  用最大堆维护已选课程的时长
  如果当前课程放不下，但比堆顶短 → 反悔（替换堆顶）
```

### 代码模板

```python
import heapq

# 课程表 III（最多能选几门课）
def schedule_course(courses):
    courses.sort(key=lambda x: x[1])  # 按截止时间排序
    heap = []  # 最大堆（存负的持续时间）
    total = 0
    for duration, deadline in courses:
        if total + duration <= deadline:
            total += duration
            heapq.heappush(heap, -duration)
        elif heap and -heap[0] > duration:
            # 反悔: 替换掉之前耗时最长的课程
            total += duration + heapq.heapreplace(heap, -duration)
    return len(heap)
```
