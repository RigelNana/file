# 最小生成树

> 最小生成树（MST）：连接所有节点且总边权最小的树（无环连通子图，恰好 n-1 条边）。

---

## 1. Kruskal 算法 ⭐

### 理解与可视化

```
Kruskal: 贪心 + 并查集

思路: 将所有边按权重排序，从小到大依次加入
      如果加入后不成环（用并查集判断），就选这条边
      选够 n-1 条边后停止

  例:
    边: (0,1,1) (0,2,4) (1,2,2) (1,3,3) (2,3,5)

    排序后:
      (0,1,1) → 0和1 不连通，选！
      (1,2,2) → 1和2 不连通，选！
      (1,3,3) → 1和3 不连通，选！
      (0,2,4) → 0和2 已连通，跳过
      (2,3,5) → 2和3 已连通，跳过

    MST 总权重 = 1 + 2 + 3 = 6

  时间: O(E log E)
```

### 代码模板

```python
# 并查集
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n

    def find(self, x):
        while self.parent[x] != x:
            self.parent[x] = self.parent[self.parent[x]]
            x = self.parent[x]
        return x

    def union(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return False
        if self.rank[rx] < self.rank[ry]:
            rx, ry = ry, rx
        self.parent[ry] = rx
        if self.rank[rx] == self.rank[ry]:
            self.rank[rx] += 1
        return True

# Kruskal 算法
def kruskal(n, edges):
    """edges: [(u, v, w), ...]"""
    edges.sort(key=lambda x: x[2])
    uf = UnionFind(n)
    mst_weight = 0
    mst_edges = 0
    for u, v, w in edges:
        if uf.union(u, v):
            mst_weight += w
            mst_edges += 1
            if mst_edges == n - 1:
                break
    return mst_weight if mst_edges == n - 1 else -1  # -1 = 图不连通
```

## 2. Prim 算法 ⭐

### 理解与可视化

```
Prim: 从任一节点出发，不断选择连接已选集合和未选集合的最小边

  类似 Dijkstra 的贪心策略

  过程:
    已选: {0}
    候选: 0 的所有边
    选最小边 (0,1,1) → 已选: {0, 1}
    加入 1 的边到候选
    选最小边 (1,2,2) → 已选: {0, 1, 2}
    选最小边 (1,3,3) → 已选: {0, 1, 2, 3}

  时间: O(E log V) (堆优化)
  适合稠密图（邻接矩阵时 O(V²)）
```

### 代码模板

```python
import heapq
from collections import defaultdict

# Prim 算法 (堆优化)
def prim(n, edges):
    graph = defaultdict(list)
    for u, v, w in edges:
        graph[u].append((w, v))
        graph[v].append((w, u))

    visited = [False] * n
    heap = [(0, 0)]  # (weight, node)
    mst_weight = 0
    count = 0
    while heap and count < n:
        w, u = heapq.heappop(heap)
        if visited[u]:
            continue
        visited[u] = True
        mst_weight += w
        count += 1
        for nw, v in graph[u]:
            if not visited[v]:
                heapq.heappush(heap, (nw, v))
    return mst_weight if count == n else -1
```

## 3. 算法对比

```
┌────────────────────────────────────────────────────┐
│ 算法     │ 策略          │ 时间          │ 适用     │
├────────────────────────────────────────────────────┤
│ Kruskal  │ 全局选最小边   │ O(E log E)   │ 稀疏图   │
│ Prim     │ 局部扩展最近点 │ O(E log V)   │ 稠密图   │
└────────────────────────────────────────────────────┘

Kruskal 实现更简单, 面试推荐优先使用
```
