# 最小生成树总结

> Kruskal 与 Prim 对比、常见变体与应用场景速查。

---

## 1. MST 核心性质

```
最小生成树 (MST):
  1. 恰好 n-1 条边 (n 个节点)
  2. 连通且无环 (是一棵树)
  3. 总边权最小

切割性质 (Cut Property):
  对图的任意切割, 跨越切割的最小权边一定在 MST 中
  → 这是 Kruskal 和 Prim 贪心正确性的基础

环性质 (Cycle Property):
  对图中任意环, 环上的最大权边不在 MST 中
  (除非有多条相同最大权的边)

唯一性:
  如果所有边权不同 → MST 唯一
  如果有相同边权 → 可能有多个 MST, 但总权重相同
```

```

│ 面试默认                   │ Kruskal      │

```

## 3. MST 变体

### 最大生成树

```python
# 边权取负 → Kruskal
def max_spanning_tree(n, edges):
    edges.sort(key=lambda x: -x[2])    # ⚠️ 按权重从大到小
    uf = UnionFind(n)
    total = 0
    for u, v, w in edges:
        if uf.union(u, v):
            total += w
    return total
```

### 次小生成树

```
先求 MST, 然后:
  枚举非MST边 (u,v,w), 加入后形成环
  删除环上最大的MST边 (非当前边)
  新权重 = MST权重 - 删的边 + 加的边
  取所有替换方案中的最小值
```

### 虚拟节点技巧

```
LeetCode 1168: 水资源分配优化
  n 个房子, 每个房子可以自建水井 (costs[i])
  也可以从相邻房子接水 (pipes[i] = [h1, h2, cost])
  求最小费用让所有房子通水

  技巧: 添加虚拟节点 0 (水源)
        自建水井 = 虚拟节点0 到 房子i 的边, 权重=costs[i]
        接水管 = 房子之间的边

  转化为 n+1 个节点的 MST 问题!
```

## 4. 并查集进阶

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
        self.size = [1] * n            # ⚠️ 每个连通分量的大小
        self.components = n
    
    def find(self, x):
        while self.parent[x] != x:
            self.parent[x] = self.parent[self.parent[x]]
            x = self.parent[x]
        return x
    
    def union(self, x, y):
        rx, ry = self.find(x), self.find(y)
        if rx == ry:
            return False
        if self.rank[rx] < self.rank[ry]:
            rx, ry = ry, rx
        self.parent[ry] = rx
        self.size[rx] += self.size[ry]  # ⚠️ 合并大小
        if self.rank[rx] == self.rank[ry]:
            self.rank[rx] += 1
        self.components -= 1
        return True
    
    def get_size(self, x):
        return self.size[self.find(x)]
    
    def is_connected(self, x, y):
        return self.find(x) == self.find(y)
```

## 5. 常见错误

```
❌ Prim 堆中存累积距离 → 应该存边权
❌ Kruskal 忘排序 → 随机选边不是贪心
❌ 无向图只加单向边 → 漏连接
❌ 并查集忘合并 → find 结果不一致
❌ 不判断连通性 → 图不连通时返回错误结果
❌ 节点 1-indexed 用 0-indexed 数组 → 越界
```

## 本节要点速查

```
✅ MST: n-1条边, 连通无环, 总权最小
✅ Kruskal: 边排序+并查集, O(E log E), 面试首选
✅ Prim: 局部扩展+堆, O(E log V), 稠密图
✅ 切割性质: 最小跨切割边一定在MST
✅ 最大生成树: 边权从大到小排序
✅ 虚拟节点: 把"自建"转化为"连边到虚拟源"
✅ 边权全不同 → MST唯一
```
