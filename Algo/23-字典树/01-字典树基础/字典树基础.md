# 字典树基础

> Trie（前缀树）高效存储和查询字符串集合，支持前缀搜索，是字符串匹配的核心数据结构。

---

## 1. 核心概念

```
Trie (字典树 / 前缀树):
  多叉树, 每条边代表一个字符
  从根到某个节点的路径 = 一个前缀
  标记结束的节点 = 一个完整单词

  存储: ["apple", "app", "bat"]

       root
      /    \
     a      b
     |      |
     p      a
     |      |
     p*     t*
     |
     l
     |
     e*

  * 表示 is_end = True (完整单词)

  查找 "app":  root→a→p→p → is_end=True ✓
  查找 "ap":   root→a→p   → is_end=False ✗ (不是完整单词)
  前缀 "ap":   root→a→p   → 存在 ✓
  查找 "abc":  root→a→b   → 不存在 ✗

  时间: insert/search/startsWith 都是 O(L), L = 字符串长度
  空间: O(总字符数 × 字符集大小)
```

## 2. LeetCode 208: 实现 Trie

```python
class TrieNode:
    def __init__(self):
        self.children = {}             # ⚠️ 字符 → 子节点
        self.is_end = False            # ⚠️ 是否是完整单词

class Trie:
    def __init__(self):
        self.root = TrieNode()
    
    def insert(self, word):
        node = self.root
        for ch in word:
            if ch not in node.children:
                node.children[ch] = TrieNode()
            node = node.children[ch]
        node.is_end = True             # ⚠️ 标记结束
    
    def search(self, word):
        node = self._find(word)
        return node is not None and node.is_end
        # ⚠️ 必须 is_end=True 才算找到
    
    def startsWith(self, prefix):
        return self._find(prefix) is not None
        # ⚠️ 只要路径存在就行, 不需要 is_end
    
    def _find(self, s):
        """返回 s 对应的节点, 不存在返回 None"""
        node = self.root
        for ch in s:
            if ch not in node.children:
                return None
            node = node.children[ch]
        return node

# ⚠️ children 用 dict:
#    灵活, 支持任意字符
#    也可以用 [None]*26 (纯小写字母时更快)
#
# ⚠️ search vs startsWith:
#    search: 完整匹配, 需要 is_end=True
#    startsWith: 前缀匹配, 只要路径存在
```

## 3. 数组实现 (更快)

```python
class Trie:
    def __init__(self):
        self.children = [[0] * 26 for _ in range(100001)]  # ⚠️ 预分配
        self.is_end = [False] * 100001
        self.cnt = 0                   # ⚠️ 已分配节点数
    
    def insert(self, word):
        node = 0                       # 根节点编号 0
        for ch in word:
            c = ord(ch) - ord('a')
            if self.children[node][c] == 0:
                self.cnt += 1
                self.children[node][c] = self.cnt
            node = self.children[node][c]
        self.is_end[node] = True

# ⚠️ 数组实现:
#    children[node][c] = 子节点编号
#    0 表示不存在
#    比 dict 快, 但需要预估最大节点数
```

## 4. LeetCode 211: 添加与搜索单词

```
支持 . 通配符的搜索
. 可以匹配任意一个字符

方法: Trie + DFS (遇到 . 时遍历所有子节点)
```

```python
class WordDictionary:
    def __init__(self):
        self.root = TrieNode()
    
    def addWord(self, word):
        node = self.root
        for ch in word:
            if ch not in node.children:
                node.children[ch] = TrieNode()
            node = node.children[ch]
        node.is_end = True
    
    def search(self, word):
        def dfs(node, i):
            if i == len(word):
                return node.is_end
            
            ch = word[i]
            if ch == '.':
                for child in node.children.values():
                    if dfs(child, i + 1):  # ⚠️ 遍历所有子节点
                        return True
                return False
            else:
                if ch not in node.children:
                    return False
                return dfs(node.children[ch], i + 1)
        
        return dfs(self.root, 0)

# ⚠️ . 通配符:
#    遇到 . → 尝试所有子节点, 任一成功即可
#    最坏 O(26^L), 但实际很快 (树很稀疏)
```

## 5. Trie 上的 DFS 统计

```python
class Trie:
    def __init__(self):
        self.root = TrieNode()
    
    def insert(self, word):
        node = self.root
        for ch in word:
            if ch not in node.children:
                node.children[ch] = TrieNode()
            node = node.children[ch]
        node.is_end = True
    
    def count_prefix(self, prefix):
        """统计以 prefix 为前缀的单词数"""
        node = self.root
        for ch in prefix:
            if ch not in node.children:
                return 0
            node = node.children[ch]
        
        # DFS 统计子树中 is_end 的数量
        count = 0
        stack = [node]
        while stack:
            n = stack.pop()
            if n.is_end:
                count += 1
            for child in n.children.values():
                stack.append(child)
        return count

# ⚠️ 也可以在 TrieNode 中加 prefix_count 字段
#    insert 时每经过一个节点 prefix_count += 1
#    这样 count_prefix 就是 O(L) 而不需要 DFS
```

