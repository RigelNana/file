## 1. LeetCode 212: 单词搜索 II (Trie + 回溯)

```
在 m×n 网格中搜索单词列表 words
每个单词由相邻单元格的字母组成 (上下左右)
每个单元格在一个单词中最多使用一次

暴力: 对每个单词做回溯 → O(words × m×n × 4^L) 太慢
优化: 把所有 words 建 Trie, 在网格上 DFS 时沿 Trie 走
     一次 DFS 同时搜索所有单词!
```

```python
class TrieNode:
    def __init__(self):
        self.children = {}
        self.word = None               # ⚠️ 存完整单词 (而非 is_end)

def findWords(board, words):
    # 建 Trie
    root = TrieNode()
    for word in words:
        node = root
        for ch in word:
            if ch not in node.children:
                node.children[ch] = TrieNode()
            node = node.children[ch]
        node.word = word               # ⚠️ 叶子存完整单词
    
    m, n = len(board), len(board[0])
    result = []
    
    def dfs(i, j, node):
        ch = board[i][j]
        if ch not in node.children:
            return
        
        next_node = node.children[ch]
        
        if next_node.word:             # ⚠️ 找到一个单词
            result.append(next_node.word)
            next_node.word = None      # ⚠️ 去重: 防止重复找到
        
        board[i][j] = '#'             # ⚠️ 标记已访问
        
        for di, dj in [(-1,0),(1,0),(0,-1),(0,1)]:
            ni, nj = i + di, j + dj
            if 0 <= ni < m and 0 <= nj < n and board[ni][nj] != '#':
                dfs(ni, nj, next_node)
        
        board[i][j] = ch              # ⚠️ 恢复
        
        # 剪枝: 如果子节点为空, 删除 (减少后续搜索)
        if not next_node.children:
            del node.children[ch]
    
    for i in range(m):
        for j in range(n):
            dfs(i, j, root)
    
    return result

# ⚠️ 关键优化:
#   1. Trie 中存 word 而非 is_end → 直接获取找到的单词
#   2. 找到后 word=None → 去重
#   3. 子节点为空时删除 → 减少搜索空间
#   4. board[i][j]='#' 就地标记 → 不需要 visited 集合
#
# ⚠️ 时间: O(m × n × 4^L), 但 Trie 剪枝大幅减少搜索
```

## 2. 0-1 Trie (最大异或值)

```
0-1 Trie: 每条边是 0 或 1 (二进制位)
用于快速查找最大异或值

原理:
  把数字的二进制表示插入 Trie (从高位到低位)
  查询与某个数 x 异或最大的数:
    从高位到低位, 每一位尽量走"相反的分支"
    (异或为1才贡献, 所以尽量走不同的)

  例: nums = [3, 10, 5, 25, 2, 8]
  
  25 = 11001
  10 = 01010
  异或 = 10011 = 28 (最大)

  查 5(00101) 的最大异或:
    位4: 0, 想走1 → 有(25以1开头) → 走1
    位3: 0, 想走1 → 有(25: 11...) → 走1
    位2: 1, 想走0 → 有(25: 110..) → 走0
    位1: 0, 想走1 → 没有 → 走0
    位0: 1, 想走0 → 有(25: 11001→0? 不对)
  实际匹配到 25, 异或 = 28
```

### LeetCode 421: 数组中两个数的最大异或值

```python
class TrieNode:
    def __init__(self):
        self.children = [None, None]   # ⚠️ 0 和 1 两个分支

def findMaximumXOR(nums):
    # 建 0-1 Trie
    root = TrieNode()
    MAX_BIT = max(nums).bit_length()   # ⚠️ 最大位数
    
    def insert(num):
        node = root
        for i in range(MAX_BIT - 1, -1, -1):   # ⚠️ 从高位到低位
            bit = (num >> i) & 1
            if node.children[bit] is None:
                node.children[bit] = TrieNode()
            node = node.children[bit]
    
    def query(num):
        """查找与 num 异或最大的值"""
        node = root
        xor_val = 0
        for i in range(MAX_BIT - 1, -1, -1):
            bit = (num >> i) & 1
            want = 1 - bit             # ⚠️ 想走相反的
            if node.children[want] is not None:
                xor_val |= (1 << i)    # ⚠️ 这一位异或为 1
                node = node.children[want]
            else:
                node = node.children[bit]  # ⚠️ 只能走相同的
        return xor_val
    
    max_xor = 0
    for num in nums:
        insert(num)
        max_xor = max(max_xor, query(num))
    
    return max_xor

# ⚠️ 从高位到低位: 高位的 1 贡献更大
#    所以优先让高位异或为 1
#
# ⚠️ 边插入边查询:
#    每个 num 先查 (和已插入的数异或)
#    再插入 (供后续数查询)
#    不会遗漏: 异或对称, a^b = b^a
#
# ⚠️ 时间: O(n × MAX_BIT), 空间: O(n × MAX_BIT)
```

## 3. 回文对 (Trie 应用)

### LeetCode 336

```
给定字符串数组 words
找所有 (i,j) 使得 words[i]+words[j] 是回文

方法: 把每个单词反转后插入 Trie
     对每个单词在 Trie 上搜索:
       中间或末尾有回文片段时可以配对

⚠️ 这道题也可以用哈希表解:
   对每个单词, 枚举所有划分
   如果一半是回文且另一半的反转在列表中 → 配对
```

