# 邻项消除

> 邻项消除是栈的经典应用：相邻相同元素碰到就消除，类似"消消乐"。

---

## 1. 核心思想

```
遍历元素, 和栈顶比较:
  - 相同 → 弹出栈顶 (消除)
  - 不同 → 压入栈

像"消消乐"一样, 相邻相同就消除
消除后新的栈顶可能和下一个元素又相同, 形成连锁消除

示例 s = "abbaca":
  a → 栈: [a]
  b → b ≠ a → 栈: [a, b]
  b → b = b → pop → 栈: [a]    ← 消除!
  a → a = a → pop → 栈: []     ← 连锁消除!
  c → 栈: [c]
  a → a ≠ c → 栈: [c, a]
  结果: "ca"
```

## 2. 删除所有相邻重复项

### LeetCode 1047

```python
def removeDuplicates(s):
    stack = []
    for c in s:
        if stack and stack[-1] == c:
            stack.pop()             # ⚠️ 相同就消除
        else:
            stack.append(c)
    return "".join(stack)

# ⚠️ 为什么用栈而不是双指针?
#    消除后可能产生新的相邻对
#    "abba" → 消 bb → "aa" → 消 aa → ""
#    栈自动处理这种连锁反应
#
# ⚠️ 时间 O(n), 空间 O(n)
```

## 3. 删除所有相邻重复项 II

### LeetCode 1209

```
连续 k 个相同字符才删除

s = "deeedbbcccbdaa", k = 3
  d    → [(d,1)]
  e    → [(d,1), (e,1)]
  e    → [(d,1), (e,2)]
  e    → [(d,1), (e,3)] → k=3, pop → [(d,1)]  ← 消除!
  d    → [(d,2)]         ← 和栈顶合并
  b    → [(d,2), (b,1)]
  b    → [(d,2), (b,2)]
  c    → [(d,2), (b,2), (c,1)]
  c    → [(d,2), (b,2), (c,2)]
  c    → [(d,2), (b,2), (c,3)] → k=3, pop → [(d,2), (b,2)]
  b    → [(d,2), (b,3)] → k=3, pop → [(d,2)]  ← 连锁!
  d    → [(d,3)] → k=3, pop → []               ← 再连锁!
  a    → [(a,1)]
  a    → [(a,2)]
  结果: "aa"
```

```python
def removeDuplicates(s, k):
    stack = []                      # ⚠️ 存 [字符, 计数]
    for c in s:
        if stack and stack[-1][0] == c:
            stack[-1][1] += 1
            if stack[-1][1] == k:
                stack.pop()         # ⚠️ 达到 k 个, 消除
        else:
            stack.append([c, 1])
    
    return "".join(c * n for c, n in stack)

# ⚠️ 栈中存 (char, count) 对:
#    不用每个字符单独入栈
#    合并计数, 达到 k 就消除
#
# ⚠️ 消除后不需要额外检查:
#    下一轮 for 会自动和新栈顶比较
#    如果新栈顶和下一个字符相同, 自然会继续合并
```

## 4. 消除游戏变体

### 相邻不同消除

```
某些题的规则是"相邻不同才消除" (如 "ab" → 消除)
这时候需要匹配具体规则
```

```python
# 棒球比赛 LC682: 操作型栈
def calPoints(operations):
    stack = []
    for op in operations:
        if op == '+':
            stack.append(stack[-1] + stack[-2])  # ⚠️ 前两个之和
        elif op == 'D':
            stack.append(stack[-1] * 2)           # ⚠️ 前一个翻倍
        elif op == 'C':
            stack.pop()                           # ⚠️ 消除前一个
        else:
            stack.append(int(op))
    return sum(stack)
```

### 字符串中的退格

```
LeetCode 844: 比较含退格的字符串
# 表示退格 (删除前一个字符)

s = "ab#c"  → "ac" (b被#删除)
t = "ad#c"  → "ac"
```

```python
def backspaceCompare(s, t):
    def process(string):
        stack = []
        for c in string:
            if c == '#':
                if stack:
                    stack.pop()       # ⚠️ 退格: 删除前一个
                # ⚠️ 栈空时 # 无效, 不报错
            else:
                stack.append(c)
        return stack
    
    return process(s) == process(t)

# ⚠️ O(1) 空间做法: 从后往前双指针
#    遇到 # 计数, 遇到字母消耗计数或比较
#    但栈的做法更直观
```

## 5. 邻项消除的识别特征

```
什么时候想到邻项消除?

✅ "相邻相同/匹配则删除"
✅ "删除后可能产生新的相邻对"
✅ "连续 k 个相同则消除"
✅ "退格/撤销操作"

模式总结:
┌────────────────┬────────────────────────┐
│ 类型           │ 栈存什么               │
├────────────────┼────────────────────────┤
│ 相邻两个消除   │ char                   │
│ 连续k个消除    │ (char, count)          │
│ 退格/撤销      │ char (遇#就pop)        │
│ 操作型(棒球)   │ number                 │
└────────────────┴────────────────────────┘
```
