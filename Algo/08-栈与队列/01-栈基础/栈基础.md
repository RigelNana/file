# 栈基础

> 栈是后进先出（LIFO）的线性结构，是算法中最常用的数据结构之一。

---

## 1. 栈的概念与操作

```
栈 (Stack): 后进先出 (LIFO - Last In First Out)

想象一叠盘子: 只能从顶部放入和取出

push(1):  │ 1 │    push(2):  │ 2 │    push(3):  │ 3 │
          └───┘              │ 1 │              │ 2 │
                             └───┘              │ 1 │
                                                └───┘

pop():    │ 2 │  返回 3     pop():    │ 1 │  返回 2
          │ 1 │                       └───┘
          └───┘

操作与复杂度:
  push(x)   - 压入栈顶       O(1)
  pop()     - 弹出栈顶并返回  O(1)
  peek/top  - 查看栈顶(不弹出) O(1)
  isEmpty   - 是否为空        O(1)
  size      - 栈中元素个数    O(1)
```

## 2. Python 中的栈

```python
# ⚠️ Python 中用 list 作为栈
stack = []

# push
stack.append(1)
stack.append(2)
stack.append(3)          # stack = [1, 2, 3]

# peek
top = stack[-1]          # 3, ⚠️ 不弹出

# pop
val = stack.pop()        # 3, stack = [1, 2]

# isEmpty
is_empty = len(stack) == 0   # 或 not stack

# ⚠️ 易错: 空栈 pop 会报 IndexError
#    操作前先检查: if stack: stack.pop()
#
# ⚠️ 不要用 stack[0] 做队列操作
#    list.pop(0) 是 O(n), 用 deque 代替
```

## 3. 有效的括号

### LeetCode 20

```
给定字符串只包含 '(', ')', '{', '}', '[', ']'
判断是否有效: 每个左括号都有对应右括号, 顺序正确

思路: 栈
  - 遇到左括号: 压栈
  - 遇到右括号: 检查栈顶是否匹配
  - 最终栈空 = 有效

执行过程 s = "{[()]}":
  { → 栈: [{]
  [ → 栈: [{, []
  ( → 栈: [{, [, (]
  ) → ) 匹配 ( ✓ → pop → 栈: [{, []
  ] → ] 匹配 [ ✓ → pop → 栈: [{]
  } → } 匹配 { ✓ → pop → 栈: []
  栈空 → 有效 ✓
```

```python
def isValid(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}
    
    for c in s:
        if c in pairs:                      # 右括号
            if not stack or stack[-1] != pairs[c]:
                return False                # ⚠️ 栈空或不匹配
            stack.pop()
        else:                               # 左括号
            stack.append(c)
    
    return not stack                        # ⚠️ 栈必须为空

# ⚠️ 三种 False 情况:
#    1. 遇到右括号但栈空 → 多了右括号
#    2. 栈顶不匹配 → 类型不对
#    3. 遍历完栈不空 → 多了左括号
#
# ⚠️ 用字典 right→left 比 left→right 方便:
#    遇到字符, 直接查字典判断"是左还是右"
```

## 4. 最小栈

### LeetCode 155

```
设计一个栈, 支持 O(1) 获取最小值

思路: 用辅助栈存每一步的最小值
  主栈: [3, 5, 2, 1]
  最小栈: [3, 3, 2, 1]  ← 每一步的最小值

  push(7):
    主栈: [3, 5, 2, 1, 7]
    最小: [3, 3, 2, 1, 1]  ← min(7, 1) = 1

  pop():
    主栈: [3, 5, 2, 1]
    最小: [3, 3, 2, 1]     ← 同步弹出
```

```python
class MinStack:
    def __init__(self):
        self.stack = []
        self.min_stack = []         # ⚠️ 辅助栈, 同步操作
    
    def push(self, val):
        self.stack.append(val)
        # ⚠️ 取 min(val, 当前最小值)
        cur_min = min(val, self.min_stack[-1] if self.min_stack else float('inf'))
        self.min_stack.append(cur_min)
    
    def pop(self):
        self.stack.pop()
        self.min_stack.pop()        # ⚠️ 同步弹出
    
    def top(self):
        return self.stack[-1]
    
    def getMin(self):
        return self.min_stack[-1]   # ⚠️ O(1) 获取最小值

# ⚠️ 两个栈始终同步 push/pop
#    如果不同步, 弹出后最小值可能失效
#
# ⚠️ 空间优化: 只在新最小值时压入辅助栈
#    但实现更复杂, 面试中同步版足够
```

## 5. 用栈实现队列

### LeetCode 232

```
用两个栈实现队列的 push, pop, peek, empty

思路: 两个栈 in_stack 和 out_stack
  push: 放入 in_stack
  pop/peek: 从 out_stack 取
            如果 out_stack 空, 把 in_stack 全部倒入

过程:
  push(1), push(2), push(3):
    in:  [1, 2, 3]
    out: []

  pop():  out 空 → 倒入
    in:  []
    out: [3, 2, 1]   ← in 的元素倒序进入 out
    pop → 返回 1 (队首)
```

```python
class MyQueue:
    def __init__(self):
        self.in_stack = []
        self.out_stack = []
    
    def push(self, x):
        self.in_stack.append(x)
    
    def pop(self):
        self._move()
        return self.out_stack.pop()
    
    def peek(self):
        self._move()
        return self.out_stack[-1]
    
    def empty(self):
        return not self.in_stack and not self.out_stack
    
    def _move(self):
        if not self.out_stack:          # ⚠️ 只有 out 空时才倒入!
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())
    
    # ⚠️ 关键: 只有 out_stack 空时才从 in_stack 倒入
    #    如果每次都倒, 顺序会乱
    #    均摊时间复杂度: 每个元素被 push/pop 两次 → O(1)
```
