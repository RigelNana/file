## 1. 每日温度（下一个更大元素）

### LeetCode 739

```
给定温度数组, 对每天求"再等几天能遇到更高温度"
temperatures = [73,74,75,71,69,72,76,73]
答案:           [1, 1, 4, 2, 1, 1, 0, 0]

思路: 单调递减栈 (栈存索引)
  遍历到 t, 弹出所有栈中比 t 小的温度
  被弹出的元素, 它的答案就是 i - j

执行过程:
  i=0  73  栈: [0]              ← 73 进栈
  i=1  74  74>73 → pop(0)      答案[0]=1-0=1, 栈: [1]
  i=2  75  75>74 → pop(1)      答案[1]=2-1=1, 栈: [2]
  i=3  71  71<75                栈: [2, 3]
  i=4  69  69<71                栈: [2, 3, 4]
  i=5  72  72>69 → pop(4)      答案[4]=5-4=1
           72>71 → pop(3)      答案[3]=5-3=2, 栈: [2, 5]
  i=6  76  76>72 → pop(5)      答案[5]=6-5=1
           76>75 → pop(2)      答案[2]=6-2=4, 栈: [6]
  i=7  73  73<76                栈: [6, 7]
  剩余栈中元素答案 = 0 (默认值)
```

```python
def dailyTemperatures(temperatures):
    n = len(temperatures)
    ans = [0] * n
    stack = []                    # ⚠️ 存索引, 不是值
    
    for i, t in enumerate(temperatures):
        while stack and temperatures[stack[-1]] < t:
            j = stack.pop()
            ans[j] = i - j       # ⚠️ 距离 = 当前下标 - 弹出下标
        stack.append(i)
    
    return ans

# ⚠️ 栈里存索引: 既能通过 temperatures[stack[-1]] 获取值
#    又能计算距离 i - j
#
# ⚠️ 单调性: 栈从底到顶递减 (如果更大就弹出)
#    详见单调栈专题 (21-单调栈与单调队列)
#
# ⚠️ 时间复杂度 O(n): 每个元素最多入栈和出栈各一次
```

## 2. 字符串解码

### LeetCode 394

```
s = "3[a2[c]]"  →  "accaccacc"
s = "2[abc]3[cd]ef"  →  "abcabccdcdcdef"

思路: 栈存 (之前的字符串, 重复次数)
  遇到数字: 累积数字
  遇到 '[': 把(当前串, 数字)压栈, 重置
  遇到 ']': 弹出(前串, 次数), 当前串 = 前串 + 当前串*次数
  遇到字母: 追加到当前串

执行 "3[a2[c]]":
  3     → num=3
  [     → push("", 3), cur_str="", num=0
  a     → cur_str="a"
  2     → num=2
  [     → push("a", 2), cur_str="", num=0
  c     → cur_str="c"
  ]     → pop("a", 2) → cur_str = "a" + "c"*2 = "acc"
  ]     → pop("", 3) → cur_str = "" + "acc"*3 = "accaccacc"
```

```python
def decodeString(s):
    stack = []
    cur_str = ""
    cur_num = 0
    
    for c in s:
        if c.isdigit():
            cur_num = cur_num * 10 + int(c)   # ⚠️ 多位数字
        elif c == '[':
            stack.append((cur_str, cur_num))   # ⚠️ 保存现场
            cur_str = ""
            cur_num = 0
        elif c == ']':
            prev_str, num = stack.pop()
            cur_str = prev_str + cur_str * num # ⚠️ 恢复现场+重复
        else:
            cur_str += c                       # 字母追加
    
    return cur_str

# ⚠️ 压栈的是"之前的字符串"和"当前重复次数"
#    不是当前正在构建的字符串
#
# ⚠️ 多位数字: "100[a]" → num = 1*10+0 = 10, 10*10+0 = 100
#    不能只取一个字符
```

## 3. 验证栈序列

### LeetCode 946

```
给定 pushed 和 popped 两个序列, 判断 popped 是否是合法的弹出顺序

pushed = [1,2,3,4,5], popped = [4,5,3,2,1]

模拟:
  push 1 → [1]        popped[0]=4 ≠ 1
  push 2 → [1,2]      popped[0]=4 ≠ 2
  push 3 → [1,2,3]    popped[0]=4 ≠ 3
  push 4 → [1,2,3,4]  popped[0]=4 = 4 → pop → [1,2,3], j=1
                       popped[1]=5 ≠ 3
  push 5 → [1,2,3,5]  popped[1]=5 = 5 → pop → [1,2,3], j=2
                       popped[2]=3 = 3 → pop → [1,2], j=3
                       popped[3]=2 = 2 → pop → [1], j=4
                       popped[4]=1 = 1 → pop → [], j=5
  全部匹配 → True
```

```python
def validateStackSequences(pushed, popped):
    stack = []
    j = 0                            # ⚠️ popped 的指针
    
    for x in pushed:
        stack.append(x)
        while stack and stack[-1] == popped[j]:
            stack.pop()
            j += 1                   # ⚠️ 每次匹配后 j 前进
    
    return j == len(popped)          # ⚠️ 全部匹配完

# ⚠️ 贪心思想: 每次能 pop 就 pop
#    如果某个时刻无法匹配, 后续也不可能匹配
#
# ⚠️ 也可以 return not stack (栈空 = 全匹配)
```

## 4. 去除重复字母

### LeetCode 316

```
给定字符串, 去除重复字母使每个字母只出现一次
结果必须是所有可能结果中字典序最小的

s = "bcabc" → "abc"
s = "cbacdcbc" → "acdb"

思路: 单调栈 + 贪心
  维护递增栈, 如果当前字符更小且栈顶后面还会出现, 就弹出栈顶
```

```python
def removeDuplicateLetters(s):
    last = {c: i for i, c in enumerate(s)}  # ⚠️ 每个字符最后出现位置
    stack = []
    in_stack = set()                          # ⚠️ 已在栈中的字符
    
    for i, c in enumerate(s):
        if c in in_stack:
            continue                          # ⚠️ 已在栈中, 跳过
        
        # 栈顶比当前大 且 栈顶后面还会出现 → 弹出
        while stack and stack[-1] > c and last[stack[-1]] > i:
            in_stack.remove(stack.pop())
        
        stack.append(c)
        in_stack.add(c)
    
    return "".join(stack)

# ⚠️ 三个条件同时满足才弹出:
#    1. stack 非空
#    2. stack[-1] > c (栈顶比当前字典序大)
#    3. last[stack[-1]] > i (栈顶字符后面还会出现)
#
# ⚠️ in_stack 集合: 避免重复加入
#    已经在栈中的字符直接跳过
```
