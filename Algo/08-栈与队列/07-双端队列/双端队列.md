# 双端队列

> 双端队列（Deque）两端都能入出，是滑动窗口最值问题的核心数据结构。

---

## 1. 双端队列的概念

```
双端队列 (Double-Ended Queue, Deque):
  两端都可以入队和出队

  ┌───────────────────────┐
  │  appendleft  append   │
  │     ←   [1][2][3]  →  │
  │  popleft      pop     │
  └───────────────────────┘

  四种操作, 全部 O(1):
    append(x)       右端入
    appendleft(x)   左端入
    pop()           右端出
    popleft()       左端出
```

---

## 2. Python 中的 deque

```python
from collections import deque

dq = deque()
dq.append(1)         # 右端入: [1]
dq.append(2)         # 右端入: [1, 2]
dq.appendleft(0)     # 左端入: [0, 1, 2]

dq.pop()             # 右端出: 2, 变为 [0, 1]
dq.popleft()         # 左端出: 0, 变为 [1]

# ⚠️ deque 也支持:
dq[0]                # 查看左端 O(1)
dq[-1]               # 查看右端 O(1)
len(dq)              # 长度 O(1)

# ⚠️ deque 的随机访问 dq[i] 是 O(n), 不是 O(1)!
#    如果需要频繁随机访问, 用 list
#    deque 的优势是两端操作 O(1)

# 限长 deque
dq = deque(maxlen=3)
dq.append(1)   # [1]
dq.append(2)   # [1, 2]
dq.append(3)   # [1, 2, 3]
dq.append(4)   # [2, 3, 4] ← 自动弹出左端最老的
```

## 3. 设计循环双端队列

### LeetCode 641

```python
class MyCircularDeque:
    def __init__(self, k):
        self.data = [0] * k
        self.head = 0
        self.count = 0
        self.capacity = k
    
    def insertFront(self, value):
        if self.isFull():
            return False
        self.head = (self.head - 1) % self.capacity  # ⚠️ 向左移
        self.data[self.head] = value
        self.count += 1
        return True
    
    def insertLast(self, value):
        if self.isFull():
            return False
        tail = (self.head + self.count) % self.capacity
        self.data[tail] = value
        self.count += 1
        return True
    
    def deleteFront(self):
        if self.isEmpty():
            return False
        self.head = (self.head + 1) % self.capacity  # ⚠️ 向右移
        self.count -= 1
        return True
    
    def deleteLast(self):
        if self.isEmpty():
            return False
        self.count -= 1
        return True
    
    def getFront(self):
        return -1 if self.isEmpty() else self.data[self.head]
    
    def getRear(self):
        if self.isEmpty():
            return -1
        return self.data[(self.head + self.count - 1) % self.capacity]
    
    def isEmpty(self):
        return self.count == 0
    
    def isFull(self):
        return self.count == self.capacity

# ⚠️ insertFront: head 向左移 (head - 1) % capacity
#    insertLast: tail 向右 (head + count) % capacity
#    都是 % capacity 实现循环
```

## 4. 滑动窗口最大值

### LeetCode 239 (Hard)

```
给定数组和窗口大小 k, 返回每个窗口的最大值
nums = [1,3,-1,-3,5,3,6,7], k = 3
输出: [3,3,5,5,6,7]

思路: 单调递减双端队列
  队列存索引, 满足 nums[队列元素] 递减
  入队时弹出所有比当前小的 (从右端弹)
  检查队首是否过期 (滑出窗口, 从左端弹)
  队首 = 当前窗口最大值

过程:
  i=0  1: dq=[0]              
  i=1  3: 3>1 pop(0), dq=[1]           
  i=2 -1: dq=[1,2]            ans=[3]  (nums[1]=3)
  i=3 -3: dq=[1,2,3]          ans=[3,3]
  i=4  5: 5>-3 pop(3), 5>-1 pop(2), 5>3 pop(1), dq=[4]
                               ans=[3,3,5]
  i=5  3: dq=[4,5]            ans=[3,3,5,5]
  i=6  6: 6>3 pop(5), 6>5 pop(4), dq=[6]
                               ans=[3,3,5,5,6]
  i=7  7: 7>6 pop(6), dq=[7]  ans=[3,3,5,5,6,7]
```

```python
from collections import deque

def maxSlidingWindow(nums, k):
    dq = deque()                      # ⚠️ 存索引, 不是值
    ans = []
    
    for i, x in enumerate(nums):
        # 1. 入队: 弹出所有比 x 小的 (从右端)
        while dq and nums[dq[-1]] <= x:
            dq.pop()                  # ⚠️ 右端弹出
        dq.append(i)
        
        # 2. 检查队首过期 (滑出窗口)
        if dq[0] <= i - k:
            dq.popleft()             # ⚠️ 左端弹出
        
        # 3. 窗口形成后记录答案
        if i >= k - 1:               # ⚠️ i >= k-1 时窗口大小达到 k
            ans.append(nums[dq[0]])  # ⚠️ 队首 = 当前窗口最大值
    
    return ans

# ⚠️ 队列维护的性质:
#    从左到右递减: nums[dq[0]] ≥ nums[dq[1]] ≥ ...
#    入队时弹出比自己小的: 因为它们不可能再成为最大值
#    "如果我(新来的)比你大, 你以后也不可能当最大, 淘汰!"
#
# ⚠️ <= 还是 <:
#    while nums[dq[-1]] <= x: 弹出等于的
#    这样队首严格是最大值
#    用 < 也行, 结果一样 (队中可能有重复值)
#
# ⚠️ 过期检查: dq[0] <= i - k
#    索引 dq[0] 对应的元素不在 [i-k+1, i] 窗口内了
#    等价: dq[0] < i - k + 1 即 dq[0] <= i - k
#
# ⚠️ 时间 O(n): 每个元素最多入队出队各一次
```

## 5. 双端队列应用场景

```
┌──────────────────┬────────────────────────────┐
│ 场景             │ 用法                       │
├──────────────────┼────────────────────────────┤
│ 滑动窗口最大值   │ 单调递减双端队列           │
│ 滑动窗口最小值   │ 单调递增双端队列           │
│ BFS 最短路       │ 普通队列 (deque左出右入)    │
│ 0-1 BFS         │ 权0左入, 权1右入            │
│ 固定大小缓冲     │ maxlen 限长 deque          │
└──────────────────┴────────────────────────────┘

⚠️ 单调队列 vs 单调栈:
   单调栈: 只右端操作 (append/pop)
   单调队列: 两端都操作 (左端弹过期, 右端弹小的)
   单调队列 = 单调栈 + 左端过期机制
```

## 本节要点速查

```
✅ deque: append/pop右端, appendleft/popleft左端, 全O(1)
✅ 滑动窗口最大值: 单调递减队列, 队首=最大值
✅ 入队: 弹出所有<=当前的 (从右端)
✅ 过期: dq[0] <= i-k 时左端弹出
✅ 窗口形成: i >= k-1 时开始记录答案
✅ 单调队列 = 单调栈 + 左端过期
```
