# 栈与队列

> 栈（后进先出）和队列（先进先出）是两种最基础的线性数据结构，在括号匹配、表达式求值、BFS 等场景中广泛使用。

---

## 1. 栈基础

### 理解与可视化

```
栈 (Stack): 后进先出 (LIFO - Last In First Out)

  想象一叠盘子: 只能从顶部放和取

  push(1): │ 1 │    push(2): │ 2 │    pop(): │ 1 │  返回 2
           └───┘             │ 1 │           └───┘
                             └───┘

  操作:
    push(x)  - 压入栈顶     O(1)
    pop()    - 弹出栈顶     O(1)
    peek()   - 查看栈顶     O(1)
    isEmpty  - 是否为空     O(1)

Python 中用 list 当做栈:
  stack = []
  stack.append(x)  # push
  stack.pop()      # pop
  stack[-1]        # peek
```

### 代码模板

```python
# Python 用 list 模拟栈
stack = []
stack.append(1)   # push
stack.append(2)
stack.append(3)
top = stack[-1]   # peek → 3
val = stack.pop() # pop → 3

# 有效的括号
def is_valid(s):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}
    for c in s:
        if c in pairs:
            if not stack or stack[-1] != pairs[c]:
                return False
            stack.pop()
        else:
            stack.append(c)
    return not stack
```

## 2. 栈进阶

### 代码模板

```python
# 每日温度（下一个更大元素 → 单调栈，见专题21）
def daily_temperatures(temperatures):
    n = len(temperatures)
    ans = [0] * n
    stack = []  # 存索引
    for i, t in enumerate(temperatures):
        while stack and temperatures[stack[-1]] < t:
            j = stack.pop()
            ans[j] = i - j
        stack.append(i)
    return ans

# 字符串解码 "3[a2[c]]" → "accaccacc"
def decode_string(s):
    stack = []
    cur_str = ""
    cur_num = 0
    for c in s:
        if c.isdigit():
            cur_num = cur_num * 10 + int(c)
        elif c == '[':
            stack.append((cur_str, cur_num))
            cur_str = ""
            cur_num = 0
        elif c == ']':
            prev_str, num = stack.pop()
            cur_str = prev_str + cur_str * num
        else:
            cur_str += c
    return cur_str
```

## 3. 邻项消除

### 理解与可视化

```
邻项消除: 用栈模拟，相邻相同则消除

例: 删除字符串中所有相邻重复项 "abbaca"
  遍历:
  a → 栈: [a]
  b → 栈: [a, b]
  b → b == 栈顶b → 消除 → 栈: [a]
  a → a == 栈顶a → 消除 → 栈: []
  c → 栈: [c]
  a → 栈: [c, a]
  结果: "ca"
```

### 代码模板

```python
# 删除所有相邻重复项
def remove_duplicates(s):
    stack = []
    for c in s:
        if stack and stack[-1] == c:
            stack.pop()
        else:
            stack.append(c)
    return "".join(stack)

# 删除所有相邻重复项 II（连续 k 个相同则删除）
def remove_duplicates_k(s, k):
    stack = []  # [(char, count)]
    for c in s:
        if stack and stack[-1][0] == c:
            stack[-1][1] += 1
            if stack[-1][1] == k:
                stack.pop()
        else:
            stack.append([c, 1])
    return "".join(c * n for c, n in stack)
```

## 4. 合法括号字符串 (RBS)

### 代码模板

```python
# 最长有效括号
def longest_valid_parentheses(s):
    stack = [-1]  # 存索引，初始放 -1 作为哨兵
    ans = 0
    for i, c in enumerate(s):
        if c == '(':
            stack.append(i)
        else:
            stack.pop()
            if not stack:
                stack.append(i)  # 当前右括号作为新的哨兵
            else:
                ans = max(ans, i - stack[-1])
    return ans

# 括号生成
def generate_parenthesis(n):
    res = []
    def dfs(path, left, right):
        if len(path) == 2 * n:
            res.append(path)
            return
        if left < n:
            dfs(path + '(', left + 1, right)
        if right < left:
            dfs(path + ')', left, right + 1)
    dfs("", 0, 0)
    return res
```

## 5. 表达式解析

### 理解与可视化

```
计算器: "3+2*2" = 7

思路: 用栈处理优先级
  遇到数字: 累积
  遇到 +/-: 把当前数压栈
  遇到 *//: 弹出栈顶，和当前数运算后压栈
  最后: 栈中所有数求和

  "3+2*2" 处理过程:
  3 → num=3
  + → 压入 +3，记住当前符号 +
  2 → num=2
  * → 压入 +2，记住当前符号 *
  2 → num=2
  结束 → 弹出 2，*2 = 4 压入... 
  实际: 栈 [3, 4]，和=7
```

### 代码模板

```python
# 基本计算器 II（+, -, *, /）
def calculate(s):
    stack = []
    num = 0
    sign = '+'
    for i, c in enumerate(s):
        if c.isdigit():
            num = num * 10 + int(c)
        if (c in "+-*/" or i == len(s) - 1):
            if sign == '+':
                stack.append(num)
            elif sign == '-':
                stack.append(-num)
            elif sign == '*':
                stack.append(stack.pop() * num)
            elif sign == '/':
                stack.append(int(stack.pop() / num))
            sign = c
            num = 0
    return sum(stack)

# 基本计算器（支持括号和 +, -）
def calculate_with_parens(s):
    stack = []
    num = 0
    sign = 1
    result = 0
    for c in s:
        if c.isdigit():
            num = num * 10 + int(c)
        elif c == '+':
            result += sign * num
            num = 0
            sign = 1
        elif c == '-':
            result += sign * num
            num = 0
            sign = -1
        elif c == '(':
            stack.append(result)
            stack.append(sign)
            result = 0
            sign = 1
        elif c == ')':
            result += sign * num
            num = 0
            result *= stack.pop()  # sign
            result += stack.pop()  # prev result
    result += sign * num
    return result
```

## 6. 队列基础

### 理解与可视化

```
队列 (Queue): 先进先出 (FIFO - First In First Out)

  想象排队: 先来的先服务

  入队 (enqueue):  → [1] [2] [3] →  出队 (dequeue)
  从右边进               从左边出

  Python 用 collections.deque:
  deque 两端操作都是 O(1)
  list 做队列的话，pop(0) 是 O(n)，不推荐
```

### 代码模板

```python
from collections import deque

# 队列基本操作
q = deque()
q.append(1)      # 入队（右端）
q.append(2)
q.append(3)
front = q[0]     # 查看队首 → 1
val = q.popleft() # 出队（左端） → 1

# 用栈实现队列
class MyQueue:
    def __init__(self):
        self.in_stack = []
        self.out_stack = []

    def push(self, x):
        self.in_stack.append(x)

    def pop(self):
        self._move()
        return self.out_stack.pop()

    def peek(self):
        self._move()
        return self.out_stack[-1]

    def empty(self):
        return not self.in_stack and not self.out_stack

    def _move(self):
        if not self.out_stack:
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())
```
