# 队列基础

> 队列是先进先出（FIFO）的线性结构，在 BFS、任务调度等场景中广泛使用。

---

## 1. 队列的概念与操作

```
队列 (Queue): 先进先出 (FIFO - First In First Out)

想象排队买东西: 先来的先买

  入队 (enqueue/push):  → [1] [2] [3] →  出队 (dequeue/pop)
  新元素从右边进                            老元素从左边出

操作与复杂度 (deque):
  append(x)    - 入队（右端）  O(1)
  popleft()    - 出队（左端）  O(1)
  q[0]         - 查看队首      O(1)
  len(q)       - 队列长度      O(1)

⚠️ Python 中用 collections.deque 作为队列
   不要用 list：list.pop(0) 是 O(n)!
```

---

## 2. Python 中的队列实现

```python
from collections import deque

# 基本操作
q = deque()
q.append(1)          # 入队: [1]
q.append(2)          # 入队: [1, 2]
q.append(3)          # 入队: [1, 2, 3]

front = q[0]         # 查看队首: 1 ⚠️ 不弹出
val = q.popleft()    # 出队: 1, 队列变为 [2, 3]

is_empty = len(q) == 0  # 或 not q

# ⚠️ deque vs list:
#    deque.popleft()  O(1)  ← 用这个
#    list.pop(0)      O(n)  ← 不要用
#
# ⚠️ deque 也支持右端操作:
#    q.appendleft(x)  在左端插入 O(1)
#    q.pop()          从右端弹出 O(1)
#    这就是双端队列 (下一节)
```

## 3. 用栈实现队列

### LeetCode 232

```
用两个栈实现队列: push, pop, peek, empty

核心思想: 两个栈 "翻转" 顺序
  in_stack:  接收 push 的元素
  out_stack: 提供 pop/peek 的元素
  当 out_stack 空时, 把 in_stack 全部倒入

过程:
  push(1), push(2), push(3):
    in:  [1, 2, 3]  (栈顶是3)
    out: []

  pop():
    out 空 → 倒入:
    in 弹3放out, 弹2放out, 弹1放out
    in:  []
    out: [3, 2, 1]  (栈顶是1 = 队首!)
    pop → 返回 1 ✓
```

```python
class MyQueue:
    def __init__(self):
        self.in_stack = []
        self.out_stack = []
    
    def push(self, x):
        self.in_stack.append(x)
    
    def pop(self):
        self._move()
        return self.out_stack.pop()
    
    def peek(self):
        self._move()
        return self.out_stack[-1]
    
    def empty(self):
        return not self.in_stack and not self.out_stack
    
    def _move(self):
        if not self.out_stack:           # ⚠️ 只有 out 空时才倒入
            while self.in_stack:
                self.out_stack.append(self.in_stack.pop())

# ⚠️ 关键: out_stack 非空时不倒入
#    否则顺序会乱
#    例: out=[3,2,1], in=[4,5]
#    如果把 4,5 也倒入 out → [3,2,1,5,4]
#    pop 得到 4, 但应该得到 2
#
# ⚠️ 均摊 O(1): 每个元素总共被 push/pop 两次
```

## 4. 用队列实现栈

### LeetCode 225

```
用队列实现栈

方法1: 两个队列
  push 时放入辅助队列, 再把主队列倒入辅助, 交换

方法2: 一个队列
  push(x) 后, 把前面的元素依次弹出再入队
  相当于把 x 移到队首
```

```python
from collections import deque

class MyStack:
    def __init__(self):
        self.q = deque()
    
    def push(self, x):
        self.q.append(x)
        # ⚠️ 把之前的元素全部重新入队, 让x到队首
        for _ in range(len(self.q) - 1):
            self.q.append(self.q.popleft())
    
    def pop(self):
        return self.q.popleft()
    
    def top(self):
        return self.q[0]
    
    def empty(self):
        return not self.q

# push O(n), pop O(1)
#
# ⚠️ push 后旋转的过程:
#    q = [1, 2, 3], push(4) → [1, 2, 3, 4]
#    旋转3次:
#    popleft 1, append 1 → [2, 3, 4, 1]
#    popleft 2, append 2 → [3, 4, 1, 2]
#    popleft 3, append 3 → [4, 1, 2, 3]
#    队首 4 = 栈顶 ✓
```

## 5. 设计循环队列

### LeetCode 622

```
用数组实现固定大小的循环队列

用 head 和 count 追踪头位置和元素个数
tail = (head + count) % capacity
```

```python
class MyCircularQueue:
    def __init__(self, k):
        self.data = [0] * k
        self.head = 0
        self.count = 0
        self.capacity = k
    
    def enQueue(self, value):
        if self.isFull():
            return False
        tail = (self.head + self.count) % self.capacity
        self.data[tail] = value
        self.count += 1
        return True
    
    def deQueue(self):
        if self.isEmpty():
            return False
        self.head = (self.head + 1) % self.capacity  # ⚠️ 取模实现循环
        self.count -= 1
        return True
    
    def Front(self):
        return -1 if self.isEmpty() else self.data[self.head]
    
    def Rear(self):
        if self.isEmpty():
            return -1
        tail = (self.head + self.count - 1) % self.capacity  # ⚠️ -1 是最后一个元素
        return self.data[tail]
    
    def isEmpty(self):
        return self.count == 0
    
    def isFull(self):
        return self.count == self.capacity

# ⚠️ % capacity 是循环的关键
#    到达数组末尾后回到开头
#
# ⚠️ 用 count 而非 head/tail 判断空满:
#    避免 head==tail 时无法区分空和满的问题
```
