# 合法括号字符串

> 括号问题是栈的经典应用场景，从基础匹配到最长有效括号、括号生成等。

---

## 1. 括号匹配的本质

```
合法括号字符串 (Regular Bracket Sequence, RBS):
  - 空串是合法的
  - 如果 A 合法, 则 (A) 合法
  - 如果 A、B 都合法, 则 AB 合法

等价条件:
  1. 左括号数 = 右括号数
  2. 任意前缀中, 左括号数 ≥ 右括号数

用"平衡值" balance 理解:
  ( → balance + 1
  ) → balance - 1
  合法 = 过程中 balance ≥ 0, 且最终 balance = 0

  "(()())":
  (  (  )  (  )  )
  1  2  1  2  1  0  ← 始终 ≥ 0, 最终 = 0 ✓

  "())":
  (  )  )
  1  0 -1  ← -1 < 0 ✗
```

## 2. 最长有效括号

### LeetCode 32

```
找字符串中最长的合法括号子串的长度

s = "(()"    → 2  (后两个)
s = ")()())" → 4  (中间四个)

方法1: 栈法 (存索引)
  栈初始放 -1 作为哨兵
  ( → 压入索引
  ) → 弹出栈顶
      栈空: 当前索引作为新哨兵
      栈非空: 长度 = i - stack[-1]

执行 ")()())":
  i=0  )  pop(-1)  栈空 → push(0) 哨兵  栈: [0]
  i=1  (  push(1)                        栈: [0, 1]
  i=2  )  pop(1)   栈非空 → len=2-0=2   栈: [0]     ans=2
  i=3  (  push(3)                        栈: [0, 3]
  i=4  )  pop(3)   栈非空 → len=4-0=4   栈: [0]     ans=4
  i=5  )  pop(0)   栈空 → push(5) 哨兵  栈: [5]
```

```python
def longestValidParentheses(s):
    stack = [-1]                    # ⚠️ 哨兵: 假设-1位置有个匹配不上的)
    ans = 0
    
    for i, c in enumerate(s):
        if c == '(':
            stack.append(i)
        else:
            stack.pop()
            if not stack:
                stack.append(i)     # ⚠️ 栈空: 当前)作为新的分隔点
            else:
                ans = max(ans, i - stack[-1])  # ⚠️ 栈顶是"最近的未匹配位置"
    
    return ans

# ⚠️ 栈中存的是"未匹配的括号索引":
#    栈顶 = 上一个未匹配的位置 = 当前有效段的左边界-1
#    长度 = 当前位置 - 栈顶
#
# ⚠️ 初始 -1 的含义: "整个字符串之前"的虚拟分隔点
#    如果第一个字符就是 (, 那么匹配后长度 = i - (-1) = i + 1
#
# ⚠️ 方法2: DP
#    dp[i] = 以 s[i] 结尾的最长有效括号长度
#    s[i]='(' → dp[i]=0
#    s[i]=')' 且 s[i-1]='(' → dp[i]=dp[i-2]+2
#    s[i]=')' 且 s[i-dp[i-1]-1]='(' → dp[i]=dp[i-1]+2+dp[i-dp[i-1]-2]
```

## 3. 括号生成

### LeetCode 22

```
生成 n 对括号的所有合法组合

n=3 → ["((()))","(()())","(())()","()(())","()()()"]

回溯: 每个位置选 ( 或 )
  约束:
    left < n → 可以放 (
    right < left → 可以放 )
```

```python
def generateParenthesis(n):
    res = []
    
    def dfs(path, left, right):
        if len(path) == 2 * n:
            res.append(path)
            return
        
        if left < n:                    # ⚠️ 左括号不超过 n 个
            dfs(path + '(', left + 1, right)
        if right < left:                # ⚠️ 右括号不超过左括号
            dfs(path + ')', left, right + 1)
    
    dfs("", 0, 0)
    return res

# ⚠️ 条件 right < left (不是 right < n):
#    保证任意前缀中 左数 ≥ 右数 = 合法
#
# ⚠️ 为什么不用回溯(撤销)?
#    path + '(' 创建了新字符串, 没有修改原字符串
#    如果用 list (path.append/pop) 就需要回溯
#
# ⚠️ 时间复杂度: 卡特兰数 C(2n,n)/(n+1) ≈ O(4^n/n^(3/2))
```

## 4. 使括号有效的最少添加

### LeetCode 921

```
给定括号字符串, 求最少添加几个括号使之有效

思路: 用 balance 计数
  未匹配的 ( → 最后需要 ) 来匹配
  未匹配的 ) → 需要 ( 来匹配

s = "())":
  ( → balance=1
  ) → balance=0
  ) → balance=-1 → 实际: need_left+=1, balance重置0

答案 = 未匹配的左括号 + 未匹配的右括号
```

```python
def minAddToMakeValid(s):
    balance = 0           # 未匹配的 (
    need_left = 0         # 需要添加的 (
    
    for c in s:
        if c == '(':
            balance += 1
        else:
            if balance > 0:
                balance -= 1        # ⚠️ 匹配成功
            else:
                need_left += 1      # ⚠️ 多余的), 需要一个(
    
    return balance + need_left      # ⚠️ 未匹配(需要) + 未匹配)需要(

# ⚠️ 不需要用栈, 计数器就够了
#    因为只有一种括号, 不需要匹配类型
#
# ⚠️ 如果有多种括号 ({[, 就需要栈
```

## 5. 删除无效的括号

### LeetCode 301 (Hard)

```
删除最少数量的无效括号, 返回所有可能结果

思路: BFS 逐层删除
  第0层: 原字符串
  第1层: 删除1个括号的所有结果
  第2层: 删除2个括号的所有结果
  找到第一个有合法结果的层即停止
```

```python
from collections import deque

def removeInvalidParentheses(s):
    def is_valid(string):
        balance = 0
        for c in string:
            if c == '(':
                balance += 1
            elif c == ')':
                balance -= 1
            if balance < 0:
                return False
        return balance == 0
    
    if is_valid(s):
        return [s]
    
    res = []
    visited = {s}
    queue = deque([s])
    found = False
    
    while queue:
        for _ in range(len(queue)):
            cur = queue.popleft()
            if is_valid(cur):
                res.append(cur)
                found = True                 # ⚠️ 标记找到
            if found:
                continue                     # ⚠️ 不再生成下一层
            
            for i in range(len(cur)):
                if cur[i] not in '()':
                    continue                 # ⚠️ 只删括号
                nxt = cur[:i] + cur[i+1:]
                if nxt not in visited:
                    visited.add(nxt)
                    queue.append(nxt)
        
        if found:
            break
    
    return res

# ⚠️ BFS 保证"删除最少":
#    第一层找到合法的就停止
#    同层全部检查完再停 (可能有多个答案)
#
# ⚠️ visited 去重: 同一个字符串不重复入队
```

```

✅ 最长有效: 栈存索引, 初始-1哨兵, 栈空时更新哨兵
```
