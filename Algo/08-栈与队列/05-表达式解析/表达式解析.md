# 表达式解析

> 用栈处理运算符优先级，是计算器类问题的通用解法。

---

## 1. 核心思想

```
计算器的关键:
  1. 数字可能是多位
  2. 运算符有优先级: * / 高于 + -
  3. 可能有括号改变优先级
  4. 可能有负数/正号

处理策略:
  用栈存待求和的"项"
  + 数字 → 压入 +num
  - 数字 → 压入 -num
  * 数字 → 弹出栈顶, 乘以num, 压回
  / 数字 → 弹出栈顶, 除以num, 压回
  最后栈中所有数求和

为什么这样做?
  +/- 优先级低, 不知道后面还有没有 */
  所以先存着, 遇到 */ 当场算掉
  最后 sum(stack) = 把所有 +/- 一起算
```

## 2. 基本计算器 II

### LeetCode 227

```
实现计算器: 只有 +, -, *, / 和空格, 无括号
s = "3+2*2" → 7
s = " 3/2 " → 1

执行过程 "3+2*2":
  初始 sign='+'
  3  → num=3
  +  → sign是'+' → push(3), sign='+', num=0
  2  → num=2
  *  → sign是'+' → push(2), sign='*', num=0
  2  → num=2
  结束 → sign是'*' → pop()=2, push(2*2=4)
  栈: [3, 4], sum = 7
```

```python
def calculate(s):
    stack = []
    num = 0
    sign = '+'                       # ⚠️ 初始 sign='+', 第一个数被正号处理
    
    for i, c in enumerate(s):
        if c.isdigit():
            num = num * 10 + int(c)  # ⚠️ 多位数字
        
        if c in "+-*/" or i == len(s) - 1:   # ⚠️ 是运算符 或 最后一个字符
            if sign == '+':
                stack.append(num)
            elif sign == '-':
                stack.append(-num)
            elif sign == '*':
                stack.append(stack.pop() * num)
            elif sign == '/':
                # ⚠️ Python 整除: 向零截断, 不是向下取整
                stack.append(int(stack.pop() / num))
            sign = c
            num = 0
    
    return sum(stack)

# ⚠️ sign 的含义: "num 前面的运算符"
#    遇到新运算符时, 处理的是上一个 sign 和已累积的 num
#
# ⚠️ i == len(s) - 1: 最后一个字符要触发处理
#    否则最后一个数不会被压入栈
#
# ⚠️ Python 除法: -7 / 2
#    // 结果 = -4 (向下取整)
#    int(/) 结果 = -3 (向零截断) ← 题目要求
#
# ⚠️ 跳过空格: 不需要特殊处理
#    空格既不是数字也不是运算符, 什么都不做
```

## 3. 基本计算器 (支持括号)

### LeetCode 224

```
实现计算器: +, -, (, ), 空格, 无 * /
s = "(1+(4+5+2)-3)+(6+8)" → 23

括号处理:
  遇到 ( → 保存当前结果和符号, 重新开始
  遇到 ) → 恢复之前的结果, 合并
```

```python
def calculate(s):
    stack = []
    num = 0
    sign = 1                         # ⚠️ 1 表示 +, -1 表示 -
    result = 0
    
    for c in s:
        if c.isdigit():
            num = num * 10 + int(c)
        elif c == '+':
            result += sign * num
            num = 0
            sign = 1
        elif c == '-':
            result += sign * num
            num = 0
            sign = -1
        elif c == '(':
            stack.append(result)     # ⚠️ 保存之前的结果
            stack.append(sign)       # ⚠️ 保存括号前的符号
            result = 0               # ⚠️ 括号内重新计算
            sign = 1
        elif c == ')':
            result += sign * num
            num = 0
            result *= stack.pop()    # ⚠️ 乘以括号前的符号
            result += stack.pop()    # ⚠️ 加上之前的结果
    
    result += sign * num             # ⚠️ 处理最后一个数
    return result

# ⚠️ 入栈顺序: 先 result, 后 sign
#    出栈顺序: 先 pop → sign, 再 pop → prev_result
#    result = result * sign + prev_result
#
# ⚠️ 这个版本没有 * /:
#    如果有, 用 LC227 的方法处理 * / 优先级
#
# ⚠️ num 重置时机: 每次遇到运算符或括号时
#    不要忘记最后的 result += sign * num
```

## 4. 逆波兰表达式求值

### LeetCode 150

```
逆波兰表达式 (后缀表达式):
  "3 4 + 2 *" = (3 + 4) * 2 = 14

  运算符在操作数之后
  不需要括号和优先级规则

计算方法:
  遇到数字 → 压栈
  遇到运算符 → 弹出两个数, 运算, 结果压栈

"2 1 + 3 *":
  2 → [2]
  1 → [2, 1]
  + → pop 1, pop 2, 2+1=3 → [3]
  3 → [3, 3]
  * → pop 3, pop 3, 3*3=9 → [9]
  结果: 9
```

```python
def evalRPN(tokens):
    stack = []
    for t in tokens:
        if t in "+-*/":
            b = stack.pop()          # ⚠️ 先弹出的是第二个操作数
            a = stack.pop()          # ⚠️ 后弹出的是第一个操作数
            if t == '+':   stack.append(a + b)
            elif t == '-': stack.append(a - b)
            elif t == '*': stack.append(a * b)
            elif t == '/': stack.append(int(a / b))  # ⚠️ 向零截断
        else:
            stack.append(int(t))
    
    return stack[0]

# ⚠️ a 和 b 的顺序:
#    b = pop() → 第二个操作数 (后面的)
#    a = pop() → 第一个操作数 (前面的)
#    运算: a op b
#    写反了减法和除法会出错: a-b ≠ b-a
#
# ⚠️ 判断是否为运算符: t in "+-*/"
#    注意: "-3" 是负数不是运算符
#    但 tokens 中运算符长度为1, 负数长度>1
#    所以 t in "+-*/" (且 len(t)==1) 即可
#    题目保证合法, 一般不用额外检查
```

## 5. 中缀→后缀转换 (调度场算法)

```
中缀: 3 + 4 * 2  (人类习惯)
后缀: 3 4 2 * +  (计算机友好)

调度场算法:
  数字 → 直接输出
  ( → 入栈
  ) → 弹到 ( 为止
  运算符 → 弹出所有优先级 ≥ 当前的, 再入栈

优先级: * / > + -
```

```python
def infix_to_postfix(tokens):
    output = []
    ops = []
    prec = {'+': 1, '-': 1, '*': 2, '/': 2}
    
    for t in tokens:
        if t.isdigit():
            output.append(t)
        elif t == '(':
            ops.append(t)
        elif t == ')':
            while ops[-1] != '(':
                output.append(ops.pop())
            ops.pop()                    # ⚠️ 弹出 (
        else:  # 运算符
            while ops and ops[-1] != '(' and prec.get(ops[-1], 0) >= prec[t]:
                output.append(ops.pop())  # ⚠️ 弹出优先级 >= 的
            ops.append(t)
    
    while ops:
        output.append(ops.pop())
    
    return output

# ⚠️ 这个不是常考题, 但理解后有助于计算器题的思路
```

## 本节要点速查

```
✅ 无括号: +/-压栈, *//当场算, 最后sum
✅ 有括号: 遇(保存(result,sign), 遇)恢复
✅ sign 意思: num 前面的运算符, 遇到新运算符时才处理
✅ 逆波兰: 数字入栈, 运算符弹两个算完压回
✅ 弹出顺序: b=pop, a=pop, 计算a op b
✅ Python除法: int(a/b) 向零截断
```
