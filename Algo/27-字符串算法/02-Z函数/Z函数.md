# Z函数

> Z函数 z[i] 表示从位置 i 开始与字符串前缀的最长匹配长度，O(n) 时间。

---

## 1. 核心思想

```
Z函数定义:
  z[i] = 从位置 i 开始, 与 s[0..] 前缀的最长公共长度

  s = "aabxaab"
  z = [7, 1, 0, 0, 3, 1, 0]

  z[0] = 7: 整个字符串是自己的前缀 (通常定义为 n)
  z[1] = 1: s[1..] = "abxaab", 与 "aabxaab" 前缀匹配 "a" (1位)
  z[4] = 3: s[4..] = "aab",    与 "aabxaab" 前缀匹配 "aab" (3位)

Z-box 优化:
  维护右端最远的匹配区间 [l, r)
  如果 i < r, 可以利用 z[i-l] 的信息跳过已知匹配

  ┌─────────────────────────────────────────┐
  │ i 在 [l, r) 内:                         │
  │   z[i] = min(r-i, z[i-l])   (初始值)    │
  │   然后从 z[i] 继续暴力扩展               │
  │                                         │
  │ i 不在 [l, r) 内:                       │
  │   z[i] = 0, 暴力扩展                    │
  └─────────────────────────────────────────┘
```

---

## 2. 代码模板

```python
def z_function(s):
    n = len(s)
    z = [0] * n
    z[0] = n                               # ⚠️ z[0] 定义为 n (或不使用)
    l = r = 0                              # ⚠️ Z-box 的左右边界
    
    for i in range(1, n):
        if i < r:
            z[i] = min(r - i, z[i - l])    # ⚠️ 利用已知信息
        
        while i + z[i] < n and s[z[i]] == s[i + z[i]]:
            z[i] += 1                      # ⚠️ 暴力扩展
        
        if i + z[i] > r:                   # ⚠️ 更新 Z-box
            l, r = i, i + z[i]
    
    return z

# ⚠️ 时间 O(n):
#    每个字符最多被暴力比较一次 (r 只增不减)
#    z[i] 的初始值来自 Z-box 内的已知信息
```

## 3. Z函数做字符串匹配

```
在 text 中找 pattern 的所有出现位置

方法: 构造 s = pattern + "$" + text
      计算 z_function(s)
      z[i] == len(pattern) 的位置 → 匹配

⚠️ 用 "$" (不在两个串中出现的字符) 分隔
   防止跨界匹配
```

```python
def z_search(text, pattern):
    s = pattern + "$" + text               # ⚠️ $ 分隔
    z = z_function(s)
    m = len(pattern)
    
    results = []
    for i in range(m + 1, len(s)):
        if z[i] == m:                      # ⚠️ 匹配长度 == pattern 长度
            results.append(i - m - 1)      # ⚠️ 在 text 中的位置
    
    return results
```

## 4. Z函数 vs KMP

```
┌──────────┬──────────────────────┬──────────────────────┐
│          │ KMP (前缀函数)       │ Z 函数               │
├──────────┼──────────────────────┼──────────────────────┤
│ 定义     │ next[i]=最长相等     │ z[i]=从i开始与前缀    │
│          │ 真前后缀长度         │ 的最长匹配长度        │
├──────────┼──────────────────────┼──────────────────────┤
│ 字符匹配 │ ✓                    │ ✓ (拼接后计算)       │
│ 周期检测 │ ✓ (n - next[n-1])   │ ✓                    │
│ 实现难度 │ 稍难 (回退逻辑)      │ 较直观               │
│ 时间     │ O(n+m)              │ O(n+m)               │
└──────────┴──────────────────────┴──────────────────────┘

⚠️ 两者可以互相转化, 面试选择熟悉的即可
   Z函数的优势: 直观, 好理解
   KMP的优势: 在线匹配 (逐字符输入)
```

## 5. Z函数应用: 最小周期

```python
def min_period(s):
    """求字符串 s 的最小周期"""
    z = z_function(s)
    n = len(s)
    for p in range(1, n + 1):
        if n % p == 0 and z[p] == n - p:   # ⚠️ z[p] = n-p 说明 s[p:] == s[:n-p]
            return p
    return n                               # ⚠️ 自身就是最小周期
```

## 6. Z函数应用: 每个前缀出现次数

```python
def prefix_count(s):
    """统计每个前缀在 s 中出现的次数"""
    z = z_function(s)
    n = len(s)
    cnt = [0] * (n + 1)
    
    for i in range(1, n):
        cnt[z[i]] += 1                     # ⚠️ z[i] 长度的前缀多出现1次
    
    # 后缀和: 长前缀出现包含短前缀
    for i in range(n - 1, 0, -1):
        cnt[i - 1] += cnt[i]
    
    # cnt[i] + 1: 加上 s 本身 (z[0]=n 的贡献)
    for i in range(n):
        cnt[i] += 1
    
    return cnt  # cnt[i] = 长度为 i+1 的前缀出现次数
```

## 本节要点速查

```
✅ z[i]: 从位置 i 开始与前缀的最长匹配长度
✅ Z-box [l,r): 维护最远已匹配区间, 跳过已知部分
✅ 字符串匹配: pattern+"$"+text, z[i]==m 处匹配
✅ 时间 O(n): r 只增不减, 暴力扩展总量 O(n)
✅ 最小周期: z[p]==n-p 且 n%p==0
✅ z[0] 通常定义为 n
```
