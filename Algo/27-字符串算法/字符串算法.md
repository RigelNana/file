# 字符串算法

> 字符串匹配和处理是面试高频题。掌握 KMP、Z 函数、Manacher 能高效解决子串匹配和回文问题。

## 1. KMP 算法 ⭐⭐

### 理解与可视化

```
KMP: 在文本 text 中查找模式串 pattern, O(n+m) 时间

核心: next 数组 (前缀函数)
  next[i] = pattern[0..i] 中最长的 相等前后缀 长度

  pattern = "ABCABD"
  index:     0 1 2 3 4 5
  next:     [0,0,0,1,2,0]

  解释:
    "A"      → 0  (没有真前后缀)
    "AB"     → 0
    "ABC"    → 0
    "ABCA"   → 1  ("A" = "A")
    "ABCAB"  → 2  ("AB" = "AB")
    "ABCABD" → 0

匹配过程:
  text:    A B C A B C A B D
  pattern: A B C A B D
                     ✗ 不匹配!
  
  传统做法: 回退到 text 的下一位重新匹配
  KMP: 利用 next 数组, pattern 跳到 next[4]=2 的位置继续
  
  text:    A B C A B C A B D
  pattern:       A B C A B D
                           ✓ 匹配!
```

### 代码模板

```python
# KMP 前缀函数
def build_next(pattern):
    m = len(pattern)
    nxt = [0] * m
    j = 0
    for i in range(1, m):
        while j > 0 and pattern[i] != pattern[j]:
            j = nxt[j - 1]
        if pattern[i] == pattern[j]:
            j += 1
        nxt[i] = j
    return nxt

# KMP 匹配
def kmp_search(text, pattern):
    n, m = len(text), len(pattern)
    if m == 0:
        return 0
    nxt = build_next(pattern)
    j = 0
    results = []
    for i in range(n):
        while j > 0 and text[i] != pattern[j]:
            j = nxt[j - 1]
        if text[i] == pattern[j]:
            j += 1
        if j == m:
            results.append(i - m + 1)
            j = nxt[j - 1]
    return results  # 所有匹配的起始位置
```

## 2. Z 函数

### 理解与可视化

```
Z 函数: z[i] = 从位置 i 开始,与整个字符串的前缀最长匹配长度

  s = "aabxaab"
  z = [_, 1, 0, 0, 3, 1, 0]
       (z[0] 不定义, 通常设为 n 或 0)

  z[1]=1: s[1..] = "abxaab", 与 s 前缀 "a" 匹配1位
  z[4]=3: s[4..] = "aab",    与 s 前缀 "aab" 匹配3位

  应用: 字符串匹配
    构造 pattern + "$" + text, 计算 Z 函数
    z[i] == len(pattern) 的位置就是匹配位置
```

### 代码模板

```python
# Z 函数
def z_function(s):
    n = len(s)
    z = [0] * n
    z[0] = n
    l = r = 0
    for i in range(1, n):
        if i < r:
            z[i] = min(r - i, z[i - l])
        while i + z[i] < n and s[z[i]] == s[i + z[i]]:
            z[i] += 1
        if i + z[i] > r:
            l, r = i, i + z[i]
    return z

# 用 Z 函数做字符串匹配
def z_search(text, pattern):
    s = pattern + "$" + text
    z = z_function(s)
    m = len(pattern)
    return [i - m - 1 for i in range(m + 1, len(s)) if z[i] == m]
```

## 3. Manacher 算法 ⭐

### 理解与可视化

```
Manacher: O(n) 求所有回文子串 / 最长回文子串

  关键: 在字符间插入 '#', 统一奇偶

  s = "abba"
  转换: "#a#b#b#a#"

  p[i] = 以 i 为中心的最长回文半径

  利用已知回文的对称性加速:
    如果 i 在某个大回文内, 利用对称点的信息
```

### 代码模板

```python
# Manacher 算法
def manacher(s):
    # 预处理: 插入 #
    t = '#' + '#'.join(s) + '#'
    n = len(t)
    p = [0] * n  # p[i] = 回文半径
    c = r = 0    # 当前最远回文的中心和右边界
    for i in range(n):
        if i < r:
            p[i] = min(r - i, p[2 * c - i])
        # 暴力扩展
        while i - p[i] - 1 >= 0 and i + p[i] + 1 < n \
                and t[i - p[i] - 1] == t[i + p[i] + 1]:
            p[i] += 1
        # 更新最远回文
        if i + p[i] > r:
            c, r = i, i + p[i]
    return p

# 最长回文子串
def longest_palindrome(s):
    t = '#' + '#'.join(s) + '#'
    p = manacher(s)
    # 在 manacher 返回的 p 数组中找最大值
    # 但这里直接用完整逻辑:
    t = '#' + '#'.join(s) + '#'
    n = len(t)
    p = [0] * n
    c = r = 0
    max_len = 0
    center = 0
    for i in range(n):
        if i < r:
            p[i] = min(r - i, p[2 * c - i])
        while i - p[i] - 1 >= 0 and i + p[i] + 1 < n \
                and t[i - p[i] - 1] == t[i + p[i] + 1]:
            p[i] += 1
        if i + p[i] > r:
            c, r = i, i + p[i]
        if p[i] > max_len:
            max_len = p[i]
            center = i
    # 还原到原串
    start = (center - max_len) // 2
    return s[start:start + max_len]
```

## 4. 字符串哈希

### 理解与可视化

```
字符串哈希 (Rabin-Karp): 将字符串映射为数字，O(1) 比较子串

  hash(s[l..r]) = (s[l]*base^(r-l) + ... + s[r]) % mod

  预处理前缀哈希, 任意子串哈希 O(1) 计算
  用于: 快速子串比较, 重复子串检测
```

### 代码模板

```python
# 字符串哈希
class StringHash:
    def __init__(self, s, base=131, mod=10**18 + 9):
        n = len(s)
        self.mod = mod
        self.base = base
        self.h = [0] * (n + 1)
        self.pw = [1] * (n + 1)
        for i in range(n):
            self.h[i+1] = (self.h[i] * base + ord(s[i])) % mod
            self.pw[i+1] = self.pw[i] * base % mod

    def query(self, l, r):
        """返回 s[l..r] 的哈希值"""
        return (self.h[r+1] - self.h[l] * self.pw[r-l+1]) % self.mod
```

```

│ 算法            │ 用途              │ 时间          │
├──────────────────────────────────────────────────────┤
│ KMP             │ 单模式串匹配       │ O(n + m)     │
```
