# 字符串算法总结

> KMP、Z函数、Manacher、字符串哈希的对比、选型和综合应用。

---

## 1. 算法对比

```
┌───────────────┬──────────────────┬──────────┬────────────────────┐
│ 算法          │ 用途             │ 时间     │ 核心数据结构        │
├───────────────┼──────────────────┼──────────┼────────────────────┤
│ KMP           │ 单模式串匹配     │ O(n+m)   │ next 数组 (前缀函数)│
│ Z 函数        │ 前缀匹配/多用途  │ O(n)     │ z 数组              │
│ Manacher      │ 所有回文子串     │ O(n)     │ p 数组 (回文半径)   │
│ 字符串哈希    │ 子串比较/匹配    │ O(n) 均摊│ 前缀哈希+幂数组     │
│ Trie (字典树) │ 前缀查找/多模式  │ O(Σ|s|)  │ 树结构              │
│ AC 自动机     │ 多模式匹配       │ O(n+m+z) │ Trie + fail 指针    │
└───────────────┴──────────────────┴──────────┴────────────────────┘

选型指南:
  单模式匹配            → KMP 或 Z函数
  最长回文子串          → Manacher
  子串是否出现          → 字符串哈希 (O(1) 比较)
  多个模式同时匹配      → AC 自动机
  前缀查找/自动补全     → Trie
```

## 2. 问题到算法映射


## 3. 模板速查

```python
# ─── KMP next 数组 ───
def build_next(p):
    m = len(p)
    nxt = [0] * m
    j = 0
    for i in range(1, m):
        while j > 0 and p[i] != p[j]:
            j = nxt[j-1]
        if p[i] == p[j]:
            j += 1
        nxt[i] = j
    return nxt

# ─── Z 函数 ───
def z_function(s):
    n = len(s)
    z = [0] * n
    z[0] = n
    l = r = 0
    for i in range(1, n):
        if i < r:
            z[i] = min(r-i, z[i-l])
        while i + z[i] < n and s[z[i]] == s[i + z[i]]:
            z[i] += 1
        if i + z[i] > r:
            l, r = i, i + z[i]
    return z

# ─── Manacher ───
def manacher(s):
    t = '#' + '#'.join(s) + '#'
    n = len(t)
    p = [0] * n
    c = r = 0
    for i in range(n):
        if i < r:
            p[i] = min(r-i, p[2*c-i])
        while (i-p[i]-1 >= 0 and i+p[i]+1 < n 
               and t[i-p[i]-1] == t[i+p[i]+1]):
            p[i] += 1
        if i + p[i] > r:
            c, r = i, i + p[i]
    return p

# ─── 字符串哈希 ───
class StringHash:
    def __init__(self, s, base=131, mod=10**18+9):
        n = len(s)
        self.mod, self.base = mod, base
        self.h = [0]*(n+1)
        self.pw = [1]*(n+1)
        for i in range(n):
            self.h[i+1] = (self.h[i]*base + ord(s[i])) % mod
            self.pw[i+1] = self.pw[i]*base % mod
    def query(self, l, r):
        return (self.h[r+1] - self.h[l]*self.pw[r-l+1]) % self.mod
```

## 4. 常见易错点

```
⚠️ KMP:
  - j = nxt[j-1] 回退, 不是 j -= 1
  - nxt[0] 永远是 0
  - 匹配成功后 j = nxt[j-1] 继续找下一个

⚠️ Z函数:
  - z[0] = n 或不使用
  - 匹配时用分隔符 "$" 防止跨界

⚠️ Manacher:
  - 插入 '#' 后长度变为 2n+1
  - p[i] 就是原串中的回文长度
  - 原串起始 = (i - p[i]) // 2

⚠️ 字符串哈希:
  - 可能冲突! 关键比较要验证原串
  - h[r+1] - h[l]*pw[r-l+1], 注意下标
  - Python 自动处理负数取模, 其他语言要 +mod
```

## 5. 面试建议

```
面试中的字符串算法:
  1. 简单匹配题 → Python 内置 s.find() / s.index()
  2. 要求 O(n+m) → 写 KMP (最常考)
  3. 回文题 → 中心扩展 O(n²) 通常够用
     如果要 O(n) → Manacher
  4. 子串比较 → 字符串哈希

  优先级:
    KMP > Manacher > Z函数 > 字符串哈希 > AC自动机

  ⚠️ 不要在面试中硬写 AC 自动机, 除非明确要求
```
