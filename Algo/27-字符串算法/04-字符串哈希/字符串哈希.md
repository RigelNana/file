# 字符串哈希

> 将字符串映射为数值，支持 O(1) 子串比较，用于重复子串检测、滚动匹配等场景。

---

## 1. 核心思想

```
字符串哈希 (Rabin-Karp 思想):
  把字符串看成一个 base 进制的数

  hash("abc") = 'a'×base² + 'b'×base + 'c'  (mod p)

  预处理前缀哈希 h[i] 和幂数组 pw[i]:
    h[0] = 0
    h[i+1] = h[i] × base + s[i]   (mod p)
    pw[i] = base^i

  子串 s[l..r] 的哈希:
    hash(l, r) = h[r+1] - h[l] × pw[r-l+1]  (mod p)

  ⚠️ 两个不同字符串可能哈希相同 (冲突)
     减少冲突: 用大质数/双哈希
     关键判断时最好加字符串比较验证
```

---

## 2. 代码模板

```python
class StringHash:
    def __init__(self, s, base=131, mod=10**18 + 9):
        n = len(s)
        self.mod = mod
        self.base = base
        self.h = [0] * (n + 1)            # ⚠️ 前缀哈希, h[0]=0
        self.pw = [1] * (n + 1)            # ⚠️ 幂数组
        
        for i in range(n):
            self.h[i+1] = (self.h[i] * base + ord(s[i])) % mod
            self.pw[i+1] = self.pw[i] * base % mod
    
    def query(self, l, r):
        """返回 s[l..r] 的哈希值 (0-indexed, 闭区间)"""
        return (self.h[r+1] - self.h[l] * self.pw[r-l+1]) % self.mod
        # ⚠️ Python 取模自动处理负数
        # 其他语言要 (... % mod + mod) % mod

# 使用示例:
# sh = StringHash("abcabc")
# sh.query(0, 2) == sh.query(3, 5)  # "abc" == "abc" → True

# ⚠️ base 选质数 (131, 13331 常用)
# ⚠️ mod 选大质数 (10^18+9, 10^9+7)
# ⚠️ 冲突概率 ≈ n²/mod, 用双哈希可大幅降低
```

## 3. 双哈希 (降低冲突)

```python
class DoubleHash:
    def __init__(self, s):
        self.h1 = StringHash(s, base=131, mod=10**18 + 9)
        self.h2 = StringHash(s, base=13331, mod=10**18 + 7)
    
    def query(self, l, r):
        """返回 (hash1, hash2) 元组"""
        return (self.h1.query(l, r), self.h2.query(l, r))

# ⚠️ 两个不同的 base 和 mod
#    冲突概率 ≈ n² / (mod1 × mod2) → 极小
#    比较时: 两个哈希都相等才算匹配
```

## 4. LeetCode 1044: 最长重复子串

```
给定字符串 s, 找最长的出现至少两次的子串
返回任意一个即可

方法: 二分 + 字符串哈希
  二分答案长度 mid
  检查: 是否存在长度为 mid 的重复子串?
  用哈希 O(n) 检查: 滑窗计算所有长度 mid 子串的哈希, 看有无重复
```

```python
def longestDupSubstring(s):
    n = len(s)
    sh = StringHash(s)
    
    def check(length):
        """检查是否存在长度为 length 的重复子串"""
        seen = set()
        for i in range(n - length + 1):
            h = sh.query(i, i + length - 1)
            if h in seen:                  # ⚠️ 哈希冲突可能误判
                return i                   # 返回起始位置
            seen.add(h)
        return -1
    
    lo, hi = 0, n - 1
    result = ""
    
    while lo <= hi:
        mid = (lo + hi) // 2
        idx = check(mid)
        if idx != -1:
            result = s[idx : idx + mid]
            lo = mid + 1                   # ⚠️ 尝试更长
        else:
            hi = mid - 1
    
    return result

# ⚠️ 时间: O(n log n), 二分 O(log n) × 检查 O(n)
# ⚠️ 为了避免哈希冲突: 用双哈希, 或发现冲突时实际比较字符串
```

## 5. LeetCode 187: 重复的DNA序列

```
找所有长度为 10 且出现超过一次的 DNA 子串

简单方法: 直接切片放入 set
哈希方法: 滚动哈希 O(n)
```

```python
# 直接法 (Python 字符串哈希够快)
def findRepeatedDnaSequences(s):
    seen = set()
    result = set()
    for i in range(len(s) - 9):
        sub = s[i:i+10]
        if sub in seen:
            result.add(sub)
        seen.add(sub)
    return list(result)

# 位压缩法: A=0, C=1, G=2, T=3, 10个字符用20位
def findRepeatedDnaSequences_bit(s):
    mapping = {'A': 0, 'C': 1, 'G': 2, 'T': 3}
    mask = (1 << 20) - 1                   # ⚠️ 20位掩码
    h = 0
    seen = set()
    result = set()
    
    for i, c in enumerate(s):
        h = ((h << 2) | mapping[c]) & mask
        if i >= 9:                         # ⚠️ 够10个字符了
            if h in seen:
                result.add(s[i-9:i+1])
            seen.add(h)
    
    return list(result)
```

## 6. Rabin-Karp 匹配

```python
def rabin_karp(text, pattern):
    """Rabin-Karp 字符串匹配"""
    n, m = len(text), len(pattern)
    if m > n:
        return []
    
    base, mod = 131, 10**18 + 9
    
    # 计算 pattern 的哈希
    ph = 0
    for c in pattern:
        ph = (ph * base + ord(c)) % mod
    
    # 计算 text 的滚动哈希
    th = 0
    pw = pow(base, m - 1, mod)             # ⚠️ base^(m-1)
    result = []
    
    for i in range(n):
        th = (th * base + ord(text[i])) % mod
        
        if i >= m:
            th = (th - ord(text[i - m]) * pw * base) % mod
            # ⚠️ 滑出窗口的字符
        
        if i >= m - 1 and th == ph:        # ⚠️ 哈希匹配
            if text[i-m+1:i+1] == pattern: # ⚠️ 验证! 防冲突
                result.append(i - m + 1)
    
    return result
```

## 本节要点速查

```
✅ 前缀哈希: h[i+1] = h[i]*base + s[i]
✅ 子串哈希: h[r+1] - h[l]*pw[r-l+1]
✅ base=131, mod=10^18+9 常用
✅ 双哈希降低冲突: 两组不同的 base 和 mod
✅ 二分+哈希: O(n log n) 找最长重复子串
✅ 关键比较时验证实际字符串, 防止哈希冲突
```
