# 二叉树

> 二叉树是算法面试中最重要的数据结构之一。掌握二叉树的遍历和递归思维，是学好后续所有高级算法的基础。

---

## 1. 二叉树基础概念

### 理解与可视化

**什么是二叉树？**

一棵树中每个节点最多有两个孩子（左孩子、右孩子）。

```
二叉树:
         1          ← 根节点 (root)，深度=0
        / \
       2   3        ← 深度=1
      / \   \
     4   5   6      ← 深度=2（叶子节点：没有孩子的节点）

术语:
  节点 (node):    树中的元素
  根 (root):      最顶部的节点
  叶子 (leaf):    没有孩子的节点
  深度 (depth):   从根到该节点的边数
  高度 (height):  从该节点到最远叶子的边数
  层 (level):     depth + 1

常见二叉树类型:
  满二叉树: 每个节点要么有 0 个孩子，要么有 2 个孩子
  完全二叉树: 除最后一层外全满，最后一层从左往右连续
  二叉搜索树 (BST): 左子树 < 根 < 右子树
  平衡二叉树: 任意节点左右子树高度差 ≤ 1
```

### 代码模板

```python
# 二叉树节点定义
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```

## 2. 遍历二叉树 ⭐

### 理解与可视化

遍历就是按某种顺序访问树中所有节点。四种主要遍历方式：

```
         1
        / \
       2   3
      / \   \
     4   5   6

前序遍历 (Pre-order):   根→左→右  →  1, 2, 4, 5, 3, 6
  口诀: 先看自己，再看左边，最后看右边

中序遍历 (In-order):    左→根→右  →  4, 2, 5, 1, 3, 6
  口诀: 先看左边，再看自己，最后看右边
  ⭐ BST 的中序遍历是有序的！

后序遍历 (Post-order):  左→右→根  →  4, 5, 2, 6, 3, 1
  口诀: 先看左右孩子，最后看自己

层序遍历 (Level-order): 逐层从左到右  →  1, 2, 3, 4, 5, 6
  口诀: 一层一层，从左到右

可视化前序遍历:
         1 ①
        / \
    ②  2   3  ⑤
      / \   \
  ③ 4   5④  6 ⑥
```

### 代码模板

```python
# ========== 递归遍历（最直观）==========

def preorder(root):
    """前序: 根-左-右"""
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)

def inorder(root):
    """中序: 左-根-右"""
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)

def postorder(root):
    """后序: 左-右-根"""
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]


# ========== 迭代遍历（用栈模拟递归）==========

def preorder_iter(root):
    """前序迭代"""
    if not root:
        return []
    stack, res = [root], []
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.right:  # 先压右，后压左（栈后进先出）
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return res

def inorder_iter(root):
    """中序迭代"""
    stack, res = [], []
    cur = root
    while cur or stack:
        while cur:          # 一路向左
            stack.append(cur)
            cur = cur.left
        cur = stack.pop()   # 弹出最左
        res.append(cur.val)
        cur = cur.right     # 转向右子树
    return res

def postorder_iter(root):
    """后序迭代（前序变体：根右左 → 反转 → 左右根）"""
    if not root:
        return []
    stack, res = [root], []
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.left:
            stack.append(node.left)
        if node.right:
            stack.append(node.right)
    return res[::-1]


# ========== 层序遍历（BFS，用队列）==========

from collections import deque

def level_order(root):
    """层序遍历，返回二维列表"""
    if not root:
        return []
    queue = deque([root])
    res = []
    while queue:
        level = []
        for _ in range(len(queue)):  # 遍历当前层
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        res.append(level)
    return res
```

## 3. 自顶向下 DFS（先序遍历思维）

### 理解与可视化

**从根出发，把信息「带下去」给子节点。**

```
求二叉树的最大深度:

思路: 从根开始，每往下走一层，深度 +1，走到叶子时更新答案

         1         depth=1
        / \
       2   3       depth=2
      / \
     4   5         depth=3  ← 最大深度=3

自顶向下 = 先处理当前节点，再递归孩子
  ┌─────────────────────────┐
  │ def dfs(node, depth):    │
  │   处理当前节点            │  ← 先序位置
  │   dfs(left, depth+1)     │
  │   dfs(right, depth+1)    │
  └─────────────────────────┘
```

### 代码模板

```python
# 自顶向下求最大深度
def max_depth(root):
    ans = 0
    def dfs(node, depth):
        nonlocal ans
        if not node:
            return
        ans = max(ans, depth)  # 先处理
        dfs(node.left, depth + 1)
        dfs(node.right, depth + 1)
    dfs(root, 1)
    return ans

# 判断二叉树是否对称
def is_symmetric(root):
    def dfs(left, right):
        if not left and not right:
            return True
        if not left or not right:
            return False
        return (left.val == right.val and
                dfs(left.left, right.right) and
                dfs(left.right, right.left))
    return dfs(root.left, root.right) if root else True

# 二叉树的所有路径
def binary_tree_paths(root):
    res = []
    def dfs(node, path):
        if not node:
            return
        path.append(str(node.val))
        if not node.left and not node.right:
            res.append("->".join(path))
        else:
            dfs(node.left, path)
            dfs(node.right, path)
        path.pop()  # 回溯
    dfs(root, [])
    return res
```

## 4. 自底向上 DFS（后序遍历思维）

### 理解与可视化

**先递归处理左右子树，拿到结果后再决定自己的答案。信息「从下往上传」。**

```
求二叉树最大深度（自底向上版）:

         1         max(2,1)+1 = 3
        / \
       2   3       max(1,1)+1 = 2    0+1 = 1
      / \
     4   5         0+1 = 1           0+1 = 1

  叶子节点: 深度 = 1 (或0看定义)
  非叶节点: 深度 = max(左深度, 右深度) + 1

自底向上 = 先递归孩子，再处理自己
  ┌──────────────────────────────────┐
  │ def dfs(node):                    │
  │   left = dfs(node.left)           │
  │   right = dfs(node.right)         │
  │   return 处理(left, right)        │  ← 后序位置
  └──────────────────────────────────┘
```

### 代码模板

```python
# 自底向上求最大深度
def max_depth(root):
    if not root:
        return 0
    return max(max_depth(root.left), max_depth(root.right)) + 1

# 判断平衡二叉树
def is_balanced(root):
    def height(node):
        if not node:
            return 0
        left = height(node.left)
        if left == -1:
            return -1
        right = height(node.right)
        if right == -1:
            return -1
        if abs(left - right) > 1:
            return -1  # 不平衡
        return max(left, right) + 1
    return height(root) != -1

# 翻转二叉树
def invert_tree(root):
    if not root:
        return None
    root.left, root.right = invert_tree(root.right), invert_tree(root.left)
    return root

# 合并二叉树
def merge_trees(t1, t2):
    if not t1:
        return t2
    if not t2:
        return t1
    t1.val += t2.val
    t1.left = merge_trees(t1.left, t2.left)
    t1.right = merge_trees(t1.right, t2.right)
    return t1
```

## 5. 二叉树的直径

### 理解与可视化

```
直径: 树中任意两个节点之间最长路径的边数

         1
        / \
       2   3
      / \
     4   5

  最长路径: 4→2→1→3 或 5→2→1→3，长度=3
  
  关键思路: 对于每个节点，经过它的最长路径 = 左子树深度 + 右子树深度
  全局答案 = 所有节点中最大的 "左深度 + 右深度"
```

### 代码模板

```python
def diameter_of_binary_tree(root):
    ans = 0
    def depth(node):
        nonlocal ans
        if not node:
            return 0
        left = depth(node.left)
        right = depth(node.right)
        ans = max(ans, left + right)  # 更新直径
        return max(left, right) + 1   # 返回深度
    depth(root)
    return ans
```

## 6. 最近公共祖先 (LCA)

### 理解与可视化

```
找两个节点 p, q 的最近公共祖先:

         3
        / \
       5   1
      / \ / \
     6  2 0  8
       / \
      7   4

  LCA(5, 1) = 3    （分别在 3 的左右子树）
  LCA(5, 4) = 5    （4 在 5 的子树中，5 本身就是 LCA）

思路: 后序遍历，从下往上找
  - 如果当前节点是 p 或 q，返回当前节点
  - 递归左右子树
  - 如果 p 和 q 分别在左右子树中 → 当前节点就是 LCA
  - 如果都在同一侧 → LCA 在那一侧
```

### 代码模板

```python
def lowest_common_ancestor(root, p, q):
    if not root or root == p or root == q:
        return root
    left = lowest_common_ancestor(root.left, p, q)
    right = lowest_common_ancestor(root.right, p, q)
    if left and right:   # p 和 q 分别在左右子树
        return root
    return left or right  # 都在同一侧
```

## 7. 二叉搜索树 (BST) ⭐

### 理解与可视化

```
BST 性质: 对于任意节点，左子树所有值 < 节点值 < 右子树所有值

         8
        / \
       3   10
      / \    \
     1   6   14
        / \  /
       4  7 13

BST 的中序遍历是有序的: 1, 3, 4, 6, 7, 8, 10, 13, 14

查找 val=6:
  8 → 6<8 去左 → 3 → 6>3 去右 → 6 ✓ 找到！
  时间: O(h)，h=树高，平衡时 O(log n)
```

### 代码模板

```python
# BST 查找
def search_bst(root, val):
    if not root or root.val == val:
        return root
    if val < root.val:
        return search_bst(root.left, val)
    return search_bst(root.right, val)

# BST 插入
def insert_into_bst(root, val):
    if not root:
        return TreeNode(val)
    if val < root.val:
        root.left = insert_into_bst(root.left, val)
    else:
        root.right = insert_into_bst(root.right, val)
    return root

# BST 删除
def delete_node(root, key):
    if not root:
        return None
    if key < root.val:
        root.left = delete_node(root.left, key)
    elif key > root.val:
        root.right = delete_node(root.right, key)
    else:
        # 找到要删除的节点
        if not root.left:
            return root.right
        if not root.right:
            return root.left
        # 有两个孩子：用右子树最小值替代
        successor = root.right
        while successor.left:
            successor = successor.left
        root.val = successor.val
        root.right = delete_node(root.right, successor.val)
    return root

# 验证 BST
def is_valid_bst(root):
    def dfs(node, lo, hi):
        if not node:
            return True
        if node.val <= lo or node.val >= hi:
            return False
        return dfs(node.left, lo, node.val) and dfs(node.right, node.val, hi)
    return dfs(root, float('-inf'), float('inf'))

# BST 中第 K 小的元素（中序遍历）
def kth_smallest(root, k):
    stack = []
    cur = root
    while cur or stack:
        while cur:
            stack.append(cur)
            cur = cur.left
        cur = stack.pop()
        k -= 1
        if k == 0:
            return cur.val
        cur = cur.right
```

## 8. 二叉树 BFS

### 理解与可视化

```
BFS = 广度优先搜索 = 层序遍历

         1
        / \
       2   3
      / \   \
     4   5   6

  第 1 层: [1]
  第 2 层: [2, 3]
  第 3 层: [4, 5, 6]

BFS 用队列实现:
  队列: [1]           → 弹出 1，加入 2,3
  队列: [2, 3]        → 弹出 2，加入 4,5；弹出 3，加入 6
  队列: [4, 5, 6]     → 逐个弹出
  队列: []             → 结束
```

### 代码模板

```python
from collections import deque

# BFS 模板
def bfs(root):
    if not root:
        return []
    queue = deque([root])
    res = []
    while queue:
        level_size = len(queue)
        level = []
        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        res.append(level)
    return res

# 二叉树的最大深度（BFS）
def max_depth_bfs(root):
    if not root:
        return 0
    queue = deque([root])
    depth = 0
    while queue:
        depth += 1
        for _ in range(len(queue)):
            node = queue.popleft()
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
    return depth

# 二叉树的最小深度（BFS，找第一个叶子）
def min_depth(root):
    if not root:
        return 0
    queue = deque([root])
    depth = 0
    while queue:
        depth += 1
        for _ in range(len(queue)):
            node = queue.popleft()
            if not node.left and not node.right:
                return depth
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
    return depth
```

## 9. 构建二叉树

### 理解与可视化

```
从前序+中序构建二叉树:

  前序: [3, 9, 20, 15, 7]    根在最前面 → 根=3
  中序: [9, 3, 15, 20, 7]    根前面的是左子树，后面的是右子树

  中序中找到 3 的位置:
  [9] | 3 | [15, 20, 7]
  左树    右树

  递归处理左右子树:
  左子树: 前序[9]，中序[9] → 节点9
  右子树: 前序[20,15,7]，中序[15,20,7] → 节点20，左15，右7

  结果:
       3
      / \
     9  20
       / \
      15   7
```

### 代码模板

```python
# 从前序和中序构建二叉树
def build_tree(preorder, inorder):
    if not preorder:
        return None
    root = TreeNode(preorder[0])
    mid = inorder.index(preorder[0])
    root.left = build_tree(preorder[1:mid+1], inorder[:mid])
    root.right = build_tree(preorder[mid+1:], inorder[mid+1:])
    return root

# 从中序和后序构建二叉树
def build_tree2(inorder, postorder):
    if not postorder:
        return None
    root = TreeNode(postorder[-1])
    mid = inorder.index(postorder[-1])
    root.left = build_tree2(inorder[:mid], postorder[:mid])
    root.right = build_tree2(inorder[mid+1:], postorder[mid:-1])
    return root

# 序列化与反序列化（前序）
def serialize(root):
    if not root:
        return "null"
    return f"{root.val},{serialize(root.left)},{serialize(root.right)}"

def deserialize(data):
    vals = iter(data.split(","))
    def build():
        val = next(vals)
        if val == "null":
            return None
        node = TreeNode(int(val))
        node.left = build()
        node.right = build()
        return node
    return build()
```
