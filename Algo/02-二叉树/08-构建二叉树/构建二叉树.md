# 构建二叉树

> 从遍历序列重建二叉树是面试常考题。关键是理解前序/中序/后序序列中根的位置和子树的划分。

---

## 1. 前序 + 中序构建 ⭐⭐

### LeetCode 105

### 理解与可视化

```
前序: [3, 9, 20, 15, 7]    ← 根在最前面!
中序: [9, 3, 15, 20, 7]    ← 根的左边=左子树, 右边=右子树

  步骤:
    1. 前序第一个元素 = 根 → 根 = 3
    2. 在中序中找到 3 的位置(index=1)
       左: 中序[0:1] = [9]         → 左子树有 1 个节点
       右: 中序[2:5] = [15, 20, 7] → 右子树有 3 个节点
    3. 前序中:
       左子树: 前序[1:2] = [9]     (紧跟根后面, 数量=1)
       右子树: 前序[2:5] = [20,15,7]
    4. 递归构建左右子树

  结果:
       3
      / \
     9  20
       / \
      15   7

  ⚠️ 中序中根的位置决定了左右子树各有多少节点
  ⚠️ 前序中: [根, 左子树(mid个), 右子树(剩余)]
```

### 代码模板

```python
# ━━━━ 方法 1: 切片法（好理解但效率低）━━━━
def buildTree(preorder, inorder):
    if not preorder:
        return None
    
    root_val = preorder[0]            # 前序第一个 = 根
    root = TreeNode(root_val)
    
    mid = inorder.index(root_val)     # ⚠️ 在中序中找根的位置
    
    # 递归构建左右子树
    root.left = buildTree(preorder[1:mid+1], inorder[:mid])
    root.right = buildTree(preorder[mid+1:], inorder[mid+1:])
    
    return root

# ⚠️ 前序划分:
#   preorder[0]         = 根
#   preorder[1:mid+1]   = 左子树（mid 个节点）
#   preorder[mid+1:]    = 右子树
#
# ⚠️ 中序划分:
#   inorder[:mid]       = 左子树
#   inorder[mid+1:]     = 右子树（跳过 mid 处的根）

# ━━━━ 方法 2: 用哈希表 + 索引（高效）━━━━
def buildTree_v2(preorder, inorder):
    index_map = {val: i for i, val in enumerate(inorder)}  # ⚠️ 预建哈希表
    
    def build(pre_left, pre_right, in_left, in_right):
        if pre_left > pre_right:
            return None
        
        root_val = preorder[pre_left]
        root = TreeNode(root_val)
        
        in_root = index_map[root_val]        # O(1) 找到中序中的位置
        left_size = in_root - in_left        # 左子树节点数
        
        root.left = build(pre_left + 1, pre_left + left_size,
                         in_left, in_root - 1)
        root.right = build(pre_left + left_size + 1, pre_right,
                          in_root + 1, in_right)
        return root
    
    return build(0, len(preorder) - 1, 0, len(inorder) - 1)

# ⚠️ 方法 2 优势:
#   不用切片, 用索引范围表示子数组
#   inorder.index() 是 O(n), 用哈希表变 O(1)
#   总时间从 O(n²) 降到 O(n)
```

## 2. 中序 + 后序构建

### LeetCode 106

```
后序: [9, 15, 7, 20, 3]    ← 根在最后面!
中序: [9, 3, 15, 20, 7]    ← 同

  后序最后一个 = 根 → 根 = 3
  其余逻辑和前序+中序一样
```

```python
def buildTree_in_post(inorder, postorder):
    if not postorder:
        return None
    
    root_val = postorder[-1]          # ⚠️ 后序最后一个 = 根
    root = TreeNode(root_val)
    
    mid = inorder.index(root_val)
    
    root.left = buildTree_in_post(inorder[:mid], postorder[:mid])
    root.right = buildTree_in_post(inorder[mid+1:], postorder[mid:-1])
    
    return root

# ⚠️ 后序划分:
#   postorder[-1]      = 根
#   postorder[:mid]    = 左子树（mid 个节点）
#   postorder[mid:-1]  = 右子树（⚠️ 不含最后一个!）
#
# ⚠️ 和前序的区别:
#   前序: 根在最前 → preorder[0]
#   后序: 根在最后 → postorder[-1]
#   划分数量都由中序中根的位置决定
```

## 3. 前序 + 后序构建

### LeetCode 889

```
⚠️ 前序+后序不能唯一确定二叉树! (除非是满二叉树)
  因为没有中序, 无法区分左子树和右子树

  但是可以构建出其中一种合法的树:
  前序: [1, 2, 4, 5, 3, 6, 7]
  后序: [4, 5, 2, 6, 7, 3, 1]

  根 = 1 (前序第一个 = 后序最后一个)
  左子树根 = 2 (前序第二个)
  在后序中找到 2 → index = 2
  左子树有 index+1 = 3 个节点
```

```python
def constructFromPrePost(preorder, postorder):
    if not preorder:
        return None
    root = TreeNode(preorder[0])
    if len(preorder) == 1:
        return root
    
    # 前序第二个 = 左子树的根
    left_root_val = preorder[1]
    # 在后序中找到左子树根的位置
    mid = postorder.index(left_root_val)
    left_size = mid + 1               # ⚠️ 左子树大小 = mid + 1
    
    root.left = constructFromPrePost(preorder[1:1+left_size],
                                     postorder[:left_size])
    root.right = constructFromPrePost(preorder[1+left_size:],
                                      postorder[left_size:-1])
    return root
```

## 4. 最大二叉树

### LeetCode 654

```
题意: 数组中最大值作为根, 
     最大值左边的部分构建左子树, 
     最大值右边的部分构建右子树, 递归
```

```python
def constructMaximumBinaryTree(nums):
    if not nums:
        return None
    max_val = max(nums)
    max_idx = nums.index(max_val)
    root = TreeNode(max_val)
    root.left = constructMaximumBinaryTree(nums[:max_idx])
    root.right = constructMaximumBinaryTree(nums[max_idx+1:])
    return root

# ⚠️ 和从前序+中序构建类似
#   "最大值" 相当于 "根"
#   max_idx 相当于 "中序中根的位置"
```

## 5. 序列化与反序列化 ⭐

### LeetCode 297

```
序列化: 把树变成字符串
反序列化: 把字符串还原成树

方法: 前序遍历, 用 "null" 表示空节点
```

```python
class Codec:
    def serialize(self, root):
        if not root:
            return "null"
        # 前序: 根, 左, 右
        return (str(root.val) + "," + 
                self.serialize(root.left) + "," + 
                self.serialize(root.right))
    
    def deserialize(self, data):
        vals = iter(data.split(","))      # ⚠️ 用迭代器
        
        def build():
            val = next(vals)
            if val == "null":
                return None
            node = TreeNode(int(val))
            node.left = build()           # ⚠️ 递归顺序和序列化时一致(前序)
            node.right = build()
            return node
        
        return build()

# ⚠️ 用 iter() 创建迭代器:
#   每次 next(vals) 取下一个值
#   不需要传索引, 迭代器自动记住位置
#
# ⚠️ 反序列化的递归顺序必须和序列化一致!
#   序列化: 根 → 左 → 右（前序）
#   反序列化: 取根 → 递归左 → 递归右（前序）
#
# ⚠️ 也可以不用 null, 直接存前序 + 中序, 然后用 105 的方法构建
#   但 null 标记法更简单
```

## 6. 边界与易错总结

```
┌───────────────────────────────────────────────────────┐
│              构建二叉树易错点                            │
├───────────────────────────────────────────────────────┤
│ 1. 前序切片: preorder[1:mid+1] 不是 preorder[1:mid]   │
│    因为左子树有 mid 个节点, 从 index 1 取 mid 个       │
│                                                        │
│ 2. 后序切片: postorder[mid:-1] 不是 postorder[mid+1:]  │
│    因为后序最后一个是根, 要排除                          │
│                                                        │
│ 3. 前序+后序不唯一!                                    │
│    [1,2] 前序[1,2]后序[2,1] → 2 可以是左孩子或右孩子   │
│                                                        │
│ 4. index() 是 O(n) 操作                               │
│    如果追求效率, 预建哈希表                              │
│                                                        │
│ 5. 递归终点: if not preorder → 不是 if len()==1        │
│    空数组代表空子树, 应该返回 None                       │
└───────────────────────────────────────────────────────┘
```

## 本节要点速查

```
✅ 前序第一个 = 根; 后序最后一个 = 根
✅ 中序中根的位置 → 划分左右子树
✅ 左子树大小 = mid (中序中根的索引)
✅ 前序划分: [根][1:mid+1左][mid+1:右]
✅ 后序划分: [:mid左][mid:-1右][根]
✅ 前序+后序不唯一, 必须有中序才能唯一确定
✅ 序列化: 前序+null标记, 反序列化用 iter+next
✅ 效率: 用哈希表避免 index() O(n) 查找
```
