# 二叉搜索树 (BST) ⭐

> BST 利用有序性将操作优化到 O(h)。中序遍历有序是 BST 的核心性质，贯穿所有 BST 题目。

---

## 1. BST 基本性质

### 理解与可视化

```
BST 性质: 对于任意节点 node:
  - node 的左子树中所有节点的值 < node.val
  - node 的右子树中所有节点的值 > node.val

         8
        / \
       3   10
      / \    \
     1   6   14
        / \  /
       4  7 13

  ⚠️ 是"所有"左子树节点 < root, 不只是左孩子!
  错误理解: 只看左孩子 < 根 < 右孩子
  正确理解: 左子树的最大值 < 根 < 右子树的最小值

  ⭐ 中序遍历是有序的: 1, 3, 4, 6, 7, 8, 10, 13, 14
  这个性质是 BST 绝大多数题目的解题关键!

  ⚠️ BST 不允许重复值（标准定义）
     有些题目允许 <=, 但 LeetCode 标准 BST 不重复
```

## 2. BST 查找

### 代码模板

```python
# ━━━━ LeetCode 700: 二叉搜索树中的搜索 ━━━━
def searchBST(root, val):
    if not root:
        return None
    if val == root.val:
        return root
    if val < root.val:            # ⚠️ 小了去左, 大了去右
        return searchBST(root.left, val)
    return searchBST(root.right, val)

# 迭代版:
def searchBST_iter(root, val):
    while root:
        if val == root.val:
            return root
        elif val < root.val:
            root = root.left
        else:
            root = root.right
    return None

# ⚠️ 时间 O(h), 平衡时 h=log(n), 退化时 h=n
```

## 3. BST 插入

### 代码模板

```python
# ━━━━ LeetCode 701: BST 中的插入操作 ━━━━
def insertIntoBST(root, val):
    if not root:
        return TreeNode(val)          # ⚠️ 找到空位, 创建新节点
    if val < root.val:
        root.left = insertIntoBST(root.left, val)   # ⚠️ 接住返回值!
    else:
        root.right = insertIntoBST(root.right, val)
    return root

# ⚠️ root.left = insertIntoBST(root.left, val)
#   不能只写 insertIntoBST(root.left, val)!
#   因为如果 root.left 是 None, 递归会创建新节点
#   但不赋值回 root.left 就丢失了新节点
#
# ⚠️ 插入位置永远是叶子节点!
#   BST 插入不会改变现有结构, 只是在某个空位放一个新叶子
```

```
可视化:
  插入 5 到:
       8              8
      / \            / \
     3   10    →    3   10
    / \            / \
   1   6          1   6
                     /
                    5     ← 新叶子

  查找路径: 8→3→6→左=空 → 插入
```

## 4. BST 删除 ⭐

### 代码模板

```python
# ━━━━ LeetCode 450: 删除 BST 中的节点 ━━━━
def deleteNode(root, key):
    if not root:
        return None
    
    if key < root.val:
        root.left = deleteNode(root.left, key)
    elif key > root.val:
        root.right = deleteNode(root.right, key)
    else:
        # 找到要删除的节点, 分三种情况:
        
        # 情况 1: 没有左孩子 → 用右孩子替代
        if not root.left:
            return root.right         # ⚠️ 包括叶子节点(left=right=None)
        
        # 情况 2: 没有右孩子 → 用左孩子替代
        if not root.right:
            return root.left
        
        # 情况 3: 有两个孩子 → 用右子树最小值(后继)替代
        successor = root.right
        while successor.left:         # ⚠️ 找右子树的最左节点(最小值)
            successor = successor.left
        root.val = successor.val      # ⚠️ 值替换, 不是节点交换
        root.right = deleteNode(root.right, successor.val)  # 删除后继节点
    
    return root

# ⚠️ 三种情况的理解:
#   叶子: left=right=None → if not root.left → return None ✓
#   只有右: return root.right (右子树顶上来)
#   只有左: return root.left (左子树顶上来)
#   两孩子: 找右子树最小值(后继) → 值替换 → 递归删除后继
#
# ⚠️ 为什么用后继(右子树最小)?
#   后继 > 左子树所有值, < 右子树其他值
#   用它替代删除节点, 仍满足 BST 性质
#
# ⚠️ 也可以用前驱(左子树最大):
#   predecessor = root.left
#   while predecessor.right:
#       predecessor = predecessor.right
```

```
可视化: 删除节点 3

  情况 1 (叶子):      情况 2 (一个孩子):     情况 3 (两个孩子):
    5                     5                      5
   / \                   / \                    / \
  3   8   →  5          3   8   →  5           3   8
             / \       /         / \          / \
            8          2        2   8        2   4
                                              \
                                               4
  删除3→返回None       删除3→返回2              找后继4,
                                               3→4, 删原4

  结果:               结果:                   结果:
    5                   5                       5
     \                 / \                     / \
      8               2   8                   4   8
                                             /
                                            2
```

## 5. 验证 BST ⭐

### LeetCode 98

```python
def isValidBST(root):
    def dfs(node, lo, hi):
        if not node:
            return True
        if node.val <= lo or node.val >= hi:  # ⚠️ 严格 < 和 >
            return False
        return (dfs(node.left, lo, node.val) and      # 左子树上界=当前值
                dfs(node.right, node.val, hi))        # 右子树下界=当前值
    return dfs(root, float('-inf'), float('inf'))

# ⚠️ node.val <= lo 而不是 < lo
#   BST 不允许等于! 相等也是无效的
#
# ⚠️ 不能只比较父子关系!
#     10
#    /  \
#   5    15
#       /  \
#      6    20
#   6 < 15 ✓ 但 6 < 10 ✗ → 无效!
#   必须用上下界传递

# ━━━━ 方法 2: 中序遍历判断递增 ━━━━
def isValidBST_inorder(root):
    prev = float('-inf')
    def dfs(node):
        nonlocal prev
        if not node:
            return True
        if not dfs(node.left):        # 先左
            return False
        if node.val <= prev:          # ⚠️ 中序位置: 检查是否严格递增
            return False
        prev = node.val
        return dfs(node.right)        # 后右
    return dfs(root)

# ⚠️ 方法 2 利用了 "BST 中序遍历有序" 的性质
#   只要中序遍历时每个值 > 前一个值, 就是有效 BST
```

## 6. 第 K 小的元素

### LeetCode 230

```python
def kthSmallest(root, k):
    # 中序遍历, 第 k 个就是答案
    stack = []
    cur = root
    while cur or stack:
        while cur:
            stack.append(cur)
            cur = cur.left
        cur = stack.pop()
        k -= 1
        if k == 0:                   # ⚠️ 第 k 个弹出的就是第 k 小
            return cur.val
        cur = cur.right

# ⚠️ 中序遍历迭代版, 弹出第 k 个就停
# ⚠️ k 是 1-indexed: 第 1 小 = 最小值
# ⚠️ 时间 O(h+k), 不需要遍历整棵树
```

## 7. 有序数组转 BST

### LeetCode 108

```python
def sortedArrayToBST(nums):
    if not nums:
        return None
    mid = len(nums) // 2              # ⚠️ 取中间作为根 → 自动平衡
    root = TreeNode(nums[mid])
    root.left = sortedArrayToBST(nums[:mid])
    root.right = sortedArrayToBST(nums[mid+1:])
    return root

# ⚠️ 取中间保证左右子树节点数差 ≤ 1 → 平衡 BST
# ⚠️ 使用切片, 也可以用索引:
def sortedArrayToBST_v2(nums):
    def build(lo, hi):
        if lo > hi:
            return None
        mid = (lo + hi) // 2
        root = TreeNode(nums[mid])
        root.left = build(lo, mid - 1)     # ⚠️ mid-1, 不包括 mid
        root.right = build(mid + 1, hi)
        return root
    return build(0, len(nums) - 1)         # ⚠️ 闭区间 [0, n-1]
```
