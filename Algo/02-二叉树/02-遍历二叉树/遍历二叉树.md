# 遍历二叉树 ⭐

> 遍历是二叉树一切操作的基础。四种遍历方式（前序、中序、后序、层序）的递归和迭代写法都要掌握。

---

## 1. 四种遍历顺序

### 理解与可视化

```
         1
        / \
       2   3
      / \   \
     4   5   6

  ┌────────────┬────────┬──────────────────────┐
  │ 遍历方式    │ 顺序   │ 结果                  │
  ├────────────┼────────┼──────────────────────┤
  │ 前序 Pre    │ 根左右 │ 1, 2, 4, 5, 3, 6    │
  │ 中序 In     │ 左根右 │ 4, 2, 5, 1, 3, 6    │
  │ 后序 Post   │ 左右根 │ 4, 5, 2, 6, 3, 1    │
  │ 层序 Level  │ 逐层   │ [1], [2,3], [4,5,6] │
  └────────────┴────────┴──────────────────────┘

  记忆口诀:
    前序: "自己"先看, 再看左右
    中序: 先看左, 再看"自己", 最后看右
    后序: 先看左右, 最后看"自己"
    层序: 一层一层, 从左到右

  ⚠️ BST 的中序遍历是有序的! 这个性质非常重要!
```

```
  前序遍历执行过程:

         1 ①           ← 先访问根
        / \
    ②  2   3  ⑤       ← 先左后右
      / \   \
  ③ 4   5④  6 ⑥       ← 继续先左后右

  中序遍历执行过程:

         1 ④           ← 先走到最左, 然后才访问自己
        / \
    ②  2   3  ⑤       
      / \   \
  ① 4   5③  6 ⑥       ← 4 是最左, 第一个被访问
```

## 2. 递归遍历

### 代码模板

```python
# ━━━━ 前序遍历: 根 → 左 → 右 ━━━━
def preorder(root):
    if not root:
        return []                 # ⚠️ 空节点返回空列表
    return [root.val] + preorder(root.left) + preorder(root.right)

# ━━━━ 中序遍历: 左 → 根 → 右 ━━━━
def inorder(root):
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)

# ━━━━ 后序遍历: 左 → 右 → 根 ━━━━
def postorder(root):
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# ⚠️ 上面写法简洁但每次创建新列表, 效率不高
# ⚠️ 面试推荐用下面的 DFS + append 写法:

def preorder_v2(root):
    res = []
    def dfs(node):
        if not node:
            return
        res.append(node.val)      # 先序位置: 在递归前
        dfs(node.left)
        dfs(node.right)
    dfs(root)
    return res

def inorder_v2(root):
    res = []
    def dfs(node):
        if not node:
            return
        dfs(node.left)
        res.append(node.val)      # 中序位置: 在两个递归中间
        dfs(node.right)
    dfs(root)
    return res

def postorder_v2(root):
    res = []
    def dfs(node):
        if not node:
            return
        dfs(node.left)
        dfs(node.right)
        res.append(node.val)      # 后序位置: 在递归后
    dfs(root)
    return res

# ⚠️ 三种遍历的唯一区别就是 res.append 放在哪!
#    先序: 递归前
#    中序: 两次递归中间
#    后序: 递归后
```

## 3. 迭代遍历（用栈模拟递归）⭐

### 理解与可视化

```
递归本质是用函数调用栈, 迭代就是自己用栈模拟

前序迭代:
  核心: 弹出时访问, 先压右再压左（栈是后进先出, 先压右则左先弹出）

  栈: [1]
  弹1, 访问1, 压右3压左2 → 栈: [3, 2]
  弹2, 访问2, 压右5压左4 → 栈: [3, 5, 4]
  弹4, 访问4, 无孩子     → 栈: [3, 5]
  弹5, 访问5, 无孩子     → 栈: [3]
  弹3, 访问3, 压右6      → 栈: [6]
  弹6, 访问6             → 栈: []
  结果: [1, 2, 4, 5, 3, 6] ✓
```

### 代码模板

```python
# ━━━━ 前序迭代 ━━━━
def preorder_iter(root):
    if not root:
        return []
    stack = [root]
    res = []
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.right:            # ⚠️ 先压右! 栈后进先出, 要左先弹
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return res

# ⚠️ 前序迭代是最简单的, 直接弹出就访问

# ━━━━ 中序迭代 ⭐⭐ ━━━━
def inorder_iter(root):
    stack = []
    res = []
    cur = root
    while cur or stack:           # ⚠️ cur 或 stack 任一非空就继续
        while cur:                # 一路向左, 全部压栈
            stack.append(cur)
            cur = cur.left
        cur = stack.pop()         # 弹出最左的
        res.append(cur.val)       # 访问它
        cur = cur.right           # 转向右子树
    return res

# ⚠️ 中序迭代的核心:
#   1. 一路向左压栈 (while cur: stack.append(cur); cur=cur.left)
#   2. 弹出并访问
#   3. 转向右子树 (cur = cur.right)
#   4. 对右子树重复 1-3
#
# ⚠️ while cur or stack:
#   cur: 还有未入栈的节点
#   stack: 还有未访问的节点
#   两者都空才停止

# 中序迭代详细执行过程:
#          1
#         / \
#        2   3
#       / \
#      4   5
#
# cur=1, stack=[]
# 一路向左: stack=[1,2,4], cur=None
# 弹4: res=[4], cur=4.right=None
# 弹2: res=[4,2], cur=2.right=5
# 一路向左: stack=[1,5], cur=None
# 弹5: res=[4,2,5], cur=5.right=None
# 弹1: res=[4,2,5,1], cur=1.right=3
# 一路向左: stack=[3], cur=None
# 弹3: res=[4,2,5,1,3], cur=3.right=None
# stack=[], cur=None → 结束

# ━━━━ 后序迭代（技巧: 前序变体反转）━━━━
def postorder_iter(root):
    if not root:
        return []
    stack = [root]
    res = []
    while stack:
        node = stack.pop()
        res.append(node.val)
        if node.left:             # ⚠️ 注意: 先压左再压右（和前序相反!）
            stack.append(node.left)
        if node.right:
            stack.append(node.right)
    return res[::-1]              # ⚠️ 反转! 根右左 → 左右根 = 后序

# ⚠️ 后序 = reverse(根右左)
#   前序: 根左右  → 先压右后压左
#   变体: 根右左  → 先压左后压右 ← 就是这个
#   反转: 左右根  = 后序 ✓
```

## 4. 层序遍历（BFS）

### 代码模板

```python
from collections import deque

# ━━━━ 层序遍历（返回二维列表）━━━━
def level_order(root):
    if not root:
        return []
    queue = deque([root])
    res = []
    while queue:
        level = []
        for _ in range(len(queue)):  # ⚠️ 必须用 range(len(queue))!
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        res.append(level)
    return res

# ⚠️ range(len(queue)): 先记录本层有多少节点, 再处理这么多
#   不能用 while queue, 否则无法区分层!
#   for 循环中 queue 会变长(加入下一层), 但 range 已经固定了次数

# ━━━━ 锯齿形层序遍历 (LC 103) ━━━━
def zigzag_level_order(root):
    if not root:
        return []
    queue = deque([root])
    res = []
    left_to_right = True
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        if not left_to_right:
            level.reverse()           # ⚠️ 偶数层反转
        res.append(level)
        left_to_right = not left_to_right
    return res

# ━━━━ 右视图 (LC 199) ━━━━
def right_side_view(root):
    if not root:
        return []
    queue = deque([root])
    res = []
    while queue:
        for i in range(len(queue)):
            node = queue.popleft()
            if i == len(queue):       # 不对, 应该用下面的方法
                pass
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        res.append(node.val)          # ⚠️ 每层最后一个就是右视图的值!
    return res
```

## 5. Morris 遍历（进阶, O(1) 空间）

```
核心思想: 利用叶子节点的空指针, 把树临时改成线索二叉树
         不需要栈, 空间 O(1), 时间 O(n)

面试中不常考, 但知道概念即可
```

```python
# Morris 中序遍历
def inorder_morris(root):
    res = []
    cur = root
    while cur:
        if not cur.left:
            res.append(cur.val)
            cur = cur.right
        else:
            # 找左子树的最右节点（前驱）
            pred = cur.left
            while pred.right and pred.right != cur:
                pred = pred.right
            
            if not pred.right:
                pred.right = cur      # 建立线索
                cur = cur.left
            else:
                pred.right = None     # 恢复树结构
                res.append(cur.val)
                cur = cur.right
    return res
```

## 本节要点速查

```
✅ 前序(根左右) 中序(左根右) 后序(左右根) 层序(逐层)
✅ 递归三种遍历: 唯一区别是 append 放在 dfs(left)/dfs(right) 的哪个位置
✅ 前序迭代: 弹出即访问, 先压右后压左
✅ 中序迭代: 一路向左压栈 → 弹出访问 → 转右 (while cur or stack)
✅ 后序迭代: 根右左(先压左后压右) → 反转 → 左右根
✅ 层序: queue + for range(len(queue)) 分层处理
✅ BST 中序遍历 = 有序序列 (超重要性质!)
✅ while cur or stack 的含义: cur有未入栈的, stack有未弹出的
```
