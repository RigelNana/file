## 1. 根号分解 (Sqrt Decomposition)

### 理解与可视化

```
根号分解: 将长度 n 的数组分成 √n 块, 每块 √n 个元素

  ┌────┐ ┌────┐ ┌────┐ ┌────┐
  │ 块1 │ │ 块2 │ │ 块3 │ │ 块4 │  (n=16, 每块4个)
  └────┘ └────┘ └────┘ └────┘

  区间查询 [l, r]:
    完整块: O(1) 使用预处理值
    两端散块: O(√n) 暴力

  总时间: O(√n) 查询, O(√n) 修改

  适用场景:
    当线段树太复杂或不好维护时
    离线查询 (莫队)
```

### 代码模板

```python
import math

class SqrtDecomposition:
    def __init__(self, nums):
        self.nums = nums[:]
        self.n = len(nums)
        self.block = int(math.sqrt(self.n)) + 1
        self.blocks = (self.n + self.block - 1) // self.block
        self.block_sum = [0] * self.blocks
        for i in range(self.n):
            self.block_sum[i // self.block] += nums[i]

    def update(self, i, val):
        self.block_sum[i // self.block] += val - self.nums[i]
        self.nums[i] = val

    def query(self, l, r):
        """查询 sum[l..r]"""
        res = 0
        bl, br = l // self.block, r // self.block
        if bl == br:
            for i in range(l, r + 1):
                res += self.nums[i]
        else:
            # 左边散块
            for i in range(l, (bl + 1) * self.block):
                res += self.nums[i]
            # 中间完整块
            for b in range(bl + 1, br):
                res += self.block_sum[b]
            # 右边散块
            for i in range(br * self.block, r + 1):
                res += self.nums[i]
        return res
```

## 2. 莫队算法 ⭐

### 理解与可视化

```
莫队算法: 离线处理区间查询, O(n√n) 总时间

  核心思想:
    将所有查询 [l, r] 离线排序
    通过 "两个指针" 在当前区间基础上扩展/收缩
    排序方式使得指针总移动量最小

  排序方式 (奇偶优化):
    按 l 所在的块分组
    同一块内按 r 排序（奇数块升序, 偶数块降序）

  适用: 无修改的区间统计问题
    - 区间内不同元素个数
    - 区间内出现次数统计
```

### 代码模板

```python
import math
from collections import defaultdict

def mo_algorithm(nums, queries):
    """
    queries: [(l, r, idx), ...] 0-indexed
    返回每个查询的答案
    """
    n = len(nums)
    block = max(1, int(math.sqrt(n)))

    # 排序: 按块号, 同块按 r (奇偶优化)
    def sort_key(q):
        b = q[0] // block
        return (b, q[1] if b % 2 == 0 else -q[1])

    queries_sorted = sorted(queries, key=sort_key)
    ans = [0] * len(queries)
    count = defaultdict(int)
    distinct = 0  # 当前区间不同元素个数
    cur_l = 0
    cur_r = -1

    def add(idx):
        nonlocal distinct
        count[nums[idx]] += 1
        if count[nums[idx]] == 1:
            distinct += 1

    def remove(idx):
        nonlocal distinct
        count[nums[idx]] -= 1
        if count[nums[idx]] == 0:
            distinct -= 1

    for l, r, qi in queries_sorted:
        while cur_r < r:
            cur_r += 1
            add(cur_r)
        while cur_l > l:
            cur_l -= 1
            add(cur_l)
        while cur_r > r:
            remove(cur_r)
            cur_r -= 1
        while cur_l < l:
            remove(cur_l)
            cur_l += 1
        ans[qi] = distinct

    return ans
```

## 3. 离线算法

### 理解与可视化

```
离线 vs 在线:
  在线: 必须按顺序处理查询, 一个一个回答
  离线: 可以先读取所有查询, 任意顺序处理

常见离线技巧:
  1. 排序后处理 (按右端点/值排序)
  2. 莫队
  3. CDQ 分治
  4. 离线 + 树状数组

例: 求区间内 ≤ k 的元素个数
  → 按 k 排序所有查询
  → 从小到大添加元素到树状数组
  → 添加完 ≤ k 的元素后立即回答该查询
```

### 代码模板

```python
# 离线 + 排序 + 树状数组
# 例: 每个查询 [l, r, k], 求区间内 ≤ k 的元素个数
def offline_count(nums, queries):
    n = len(nums)
    # 将元素和查询按值排序
    indexed_nums = sorted(enumerate(nums), key=lambda x: x[1])
    sorted_queries = sorted(enumerate(queries), key=lambda x: x[1][2])  # 按 k 排序
    
    bit = BIT(n)
    ans = [0] * len(queries)
    j = 0
    for qi, (l, r, k) in sorted_queries:
        while j < n and indexed_nums[j][1] <= k:
            bit.update(indexed_nums[j][0] + 1, 1)
            j += 1
        ans[qi] = bit.range_query(l + 1, r + 1)
    return ans
```

## 4. CDQ 分治

### 理解与可视化

```
CDQ 分治: 处理三维偏序等复杂计数问题

  思路:
    将问题分成左右两半
    递归处理左半和右半
    计算左半对右半的贡献（跨区间贡献）

  三维偏序 (a, b, c):
    第一维: 排序
    第二维: CDQ 分治时归并排序
    第三维: 树状数组
```

## 5. 综合问题技巧

### 代码模板

```python
# 扫描线 + 线段树（矩形面积并）
def rectangle_area(rectangles):
    """计算多个矩形的面积并"""
    events = []
    ys = set()
    for x1, y1, x2, y2 in rectangles:
        events.append((x1, y1, y2, 1))   # 左边界, 开始
        events.append((x2, y1, y2, -1))  # 右边界, 结束
        ys.add(y1)
        ys.add(y2)

    ys = sorted(ys)
    y_map = {y: i for i, y in enumerate(ys)}
    n = len(ys) - 1

    events.sort()
    count = [0] * (n + 1)  # 每段被覆盖次数

    area = 0
    prev_x = events[0][0]
    for x, y1, y2, typ in events:
        # 累加上一段的面积
        covered = sum(ys[i+1] - ys[i] for i in range(n) if count[i] > 0)
        area += covered * (x - prev_x)
        prev_x = x
        # 更新覆盖
        for i in range(y_map[y1], y_map[y2]):
            count[i] += typ

    return area % (10**9 + 7)
```
