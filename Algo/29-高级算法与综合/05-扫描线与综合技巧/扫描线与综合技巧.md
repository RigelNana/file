# 扫描线与综合技巧

> 扫描线将二维问题降维为一维动态维护，是矩形面积并、天际线问题等的核心思想；本节还涵盖常见高级综合技巧。

---

## 1. 扫描线核心思想

```
扫描线: 用一条竖直线从左到右扫描平面

  ┌────┐
  │ R1 │     ┌──────┐
  │    │     │  R2  │
  └────┘     │      │
             └──────┘

扫描线 ──→
  x = x1: 遇到 R1 左边界 → 在 [y1, y2] 区间 +1
  x = x2: 遇到 R1 右边界 → 在 [y1, y2] 区间 -1
  x = x3: 遇到 R2 左边界 → 在 [y3, y4] 区间 +1
  x = x4: 遇到 R2 右边界 → 在 [y3, y4] 区间 -1

  每个 x 坐标处:
    被覆盖的 y 区间总长度 × (当前 x - 上一个 x) = 面积增量

事件驱动:
  ┌──────────────────────────────────────────────┐
  │ event = (x, y_low, y_high, type)             │
  │ type = +1 (左边界, 进入) / -1 (右边界, 离开) │
  └──────────────────────────────────────────────┘
  按 x 排序所有事件, 逐个处理
```

## 2. 矩形面积并

### LeetCode 850 - 矩形面积 II

```
题意: 给定若干矩形 [x1, y1, x2, y2], 求面积并 (重叠部分只算一次)

分析:
  1. 坐标离散化 (y 坐标可能很大)
  2. 扫描线 + 事件排序
  3. 用计数数组 count[i] 表示第 i 段 y 区间被覆盖的次数
```

```python
def rectangleArea(rectangles):
    MOD = 10 ** 9 + 7

    # 收集事件和 y 坐标
    events = []
    ys = set()
    for x1, y1, x2, y2 in rectangles:
        events.append((x1, y1, y2, 1))   # 左边界: +1
        events.append((x2, y1, y2, -1))  # 右边界: -1
        ys.add(y1)
        ys.add(y2)

    # ⚠️ y 坐标离散化
    ys = sorted(ys)
    y_index = {y: i for i, y in enumerate(ys)}
    n = len(ys) - 1  # ⚠️ n 段区间, 不是 n 个点

    # 每段区间的覆盖计数
    count = [0] * n  # count[i] 表示 [ys[i], ys[i+1]] 被覆盖的次数

    events.sort()  # ⚠️ 按 x 排序
    area = 0
    prev_x = events[0][0]

    for x, y1, y2, typ in events:
        # 累加上一段到当前 x 的面积
        covered = 0
        for i in range(n):
            if count[i] > 0:  # ⚠️ 被覆盖的区间段
                covered += ys[i + 1] - ys[i]
        area += covered * (x - prev_x)
        area %= MOD
        prev_x = x

        # 更新覆盖
        for i in range(y_index[y1], y_index[y2]):
            count[i] += typ  # ⚠️ +1 或 -1

    return area % MOD
```

### ⚠️ 关键细节

```
1. 离散化后是"区间段":
   ys = [1, 3, 5, 7]
   区间段: [1,3], [3,5], [5,7]  → n = len(ys) - 1 = 3
   ⚠️ count 数组大小是 n = len(ys) - 1, 不是 len(ys)

2. 事件排序:
   ⚠️ 必须按 x 排序
   相同 x 的事件顺序无所谓（+1 或 -1 先处理都对）

3. 面积计算:
   ⚠️ 在处理当前事件 x 时, 先算 prev_x 到 x 的面积
   再更新 count
   → 不要先更新再算, 否则第一个事件会出错

4. 覆盖判断:
   count[i] > 0 表示该段被至少一个矩形覆盖
   ⚠️ 不是 count[i] == 1, 可能重叠多次
```

## 3. 线段树优化扫描线

```
上面暴力 count 是 O(n) 更新和查询
用线段树优化: O(logn) 更新, O(1) 查询被覆盖总长度

线段树节点维护:
  cnt: 该区间被完整覆盖的次数
  len: 该区间内被覆盖的总长度

push_up 逻辑:
  if cnt > 0:
    len = ys[r+1] - ys[l]    # 整段被覆盖
  elif l == r:
    len = 0                    # 叶节点且 cnt=0
  else:
    len = left.len + right.len # 由子节点合并

⚠️ 特殊: 此线段树不需要 push_down!
   因为 cnt 只在完整覆盖时修改, 不会传递给子节点
```

```python
class ScanlineSegTree:
    def __init__(self, ys):
        self.ys = ys
        self.n = len(ys) - 1
        self.cnt = [0] * (4 * self.n)
        self.length = [0] * (4 * self.n)

    def update(self, node, nl, nr, ql, qr, val):
        """区间 [ql, qr) 的覆盖次数 += val"""
        if ql >= nr or qr <= nl:
            return
        if ql <= nl and nr <= qr:
            self.cnt[node] += val
        else:
            mid = (nl + nr) // 2
            self.update(2*node, nl, mid, ql, qr, val)
            self.update(2*node+1, mid, nr, ql, qr, val)
        # push_up
        if self.cnt[node] > 0:
            self.length[node] = self.ys[nr] - self.ys[nl]
        elif nl + 1 == nr:  # ⚠️ 叶节点
            self.length[node] = 0
        else:
            self.length[node] = self.length[2*node] + self.length[2*node+1]

    def get_covered(self):
        return self.length[1]  # 根节点的覆盖长度
```

## 4. 天际线问题

### LeetCode 218 - The Skyline Problem

```
题意: 给定建筑 [left, right, height]
     返回天际线的关键点

分析:
  扫描线 + 最大堆维护当前最高建筑

  事件:
    建筑左边界: (left, -height, right)   ← 加入堆
    建筑右边界: (right, 0, 0)            ← 标记结束

  用最大堆维护当前活跃建筑的高度
  每个 x 坐标处, 如果最大高度变化 → 产生关键点
```

```python
import heapq

def getSkyline(buildings):
    # 收集事件
    events = []
    for left, right, height in buildings:
        events.append((left, -height, right))  # ⚠️ 负高度: 进入事件
        events.append((right, 0, 0))           # 离开事件

    # ⚠️ 排序: 先按 x, 同 x 先处理进入(负高度更小)
    events.sort()

    # 最大堆: (-height, right_end)
    # ⚠️ Python 只有最小堆, 存负高度实现最大堆
    heap = [(0, float('inf'))]  # 地面 (高度0, 永不结束)
    result = []
    prev_max = 0

    for x, neg_h, r in events:
        if neg_h < 0:
            # 进入事件: 加入堆
            heapq.heappush(heap, (neg_h, r))
        
        # ⚠️ 清理过期建筑 (right_end <= x)
        while heap[0][1] <= x:
            heapq.heappop(heap)

        curr_max = -heap[0][0]
        if curr_max != prev_max:
            result.append([x, curr_max])
            prev_max = curr_max

    return result
```

### ⚠️ 易错点

```
1. 事件排序:
   按 (x, neg_h, r) 排序
   ⚠️ 同一个 x, 进入事件 (neg_h < 0) 排在前面
     → 负高度更小, 自然排在前面 ✓

2. 堆中初始元素:
   ⚠️ 放一个 (0, inf) 表示地面
   → 保证堆永远非空, 不用特判

3. 过期清理:
   ⚠️ heap[0][1] <= x, 用 <= 不是 <
   因为 right 是开区间（建筑在 right 处结束）

4. 关键点判断:
   ⚠️ curr_max != prev_max 时才产生关键点
   同一个 x 可能处理多个事件, 只在最后判断一次
```

## 5. 其他综合技巧

### 数据流中位数 — 对顶堆

```
LeetCode 295 - 数据流的中位数

思路: 用两个堆维护
  max_heap: 存较小的一半（最大堆）
  min_heap: 存较大的一半（最小堆）

  始终保持:
    len(max_heap) == len(min_heap)  或
    len(max_heap) == len(min_heap) + 1

  中位数 = max_heap[0] 或 (max_heap[0] + min_heap[0]) / 2
```

```python
class MedianFinder:
    def __init__(self):
        self.small = []  # 最大堆 (存负值)
        self.large = []  # 最小堆

    def addNum(self, num):
        # ⚠️ 先加入 small, 再把 small 的最大值移到 large
        heapq.heappush(self.small, -num)
        heapq.heappush(self.large, -heapq.heappop(self.small))
        # ⚠️ 保持 small 的长度 >= large 的长度
        if len(self.large) > len(self.small):
            heapq.heappush(self.small, -heapq.heappop(self.large))

    def findMedian(self):
        if len(self.small) > len(self.large):
            return -self.small[0]
        return (-self.small[0] + self.large[0]) / 2
```

### 前缀和 + 哈希表 — 子数组问题

```
经典模式: 求和为 k 的子数组个数 (LC 560)

  前缀和 prefix[j] - prefix[i] = k
  → 对每个 j, 找有多少个 i 使得 prefix[i] = prefix[j] - k
  → 用哈希表存已经见过的前缀和

⚠️ 初始化: prefix_count = {0: 1}
   表示空前缀的和为 0, 出现 1 次
```

```python
def subarraySum(nums, k):
    count = 0
    prefix = 0
    prefix_count = {0: 1}  # ⚠️ 空前缀
    for num in nums:
        prefix += num
        count += prefix_count.get(prefix - k, 0)
        prefix_count[prefix] = prefix_count.get(prefix, 0) + 1
    return count
```

## 本节要点速查

```
✅ 扫描线: 事件=(x, y_low, y_high, ±1), 按 x 排序
✅ 离散化: y 坐标去重排序, 建立映射, 段数 = 点数 - 1
✅ 面积计算: 先算面积增量, 再更新 count (顺序不能反)
✅ 覆盖判断: count[i] > 0 (不是 ==1)
✅ 天际线: 最大堆维护活跃建筑高度, 高度变化时产生关键点
✅ 堆清理: right_end <= x 的过期建筑要弹出
✅ 对顶堆: small(最大堆) + large(最小堆), 保持 |small| ≥ |large|
✅ 前缀和+哈希: prefix_count = {0: 1} 初始化别忘
```
