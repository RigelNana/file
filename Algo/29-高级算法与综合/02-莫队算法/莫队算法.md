# 莫队算法 (Mo's Algorithm)

> 离线处理无修改区间查询的利器，通过巧妙排序使双指针总移动量降至 O(n√n)。

---

## 1. 核心思想与可视化

```
问题: 给定数组, 有 q 个查询 [l, r], 求区间内不同元素个数

暴力: 每次 O(n) → 总 O(nq)
莫队: 离线排序 + 双指针增量维护 → 总 O((n+q)√n)

核心思想:
  1. 维护当前区间 [cur_l, cur_r] 的统计信息
  2. 通过 add(idx) / remove(idx) 增减一个元素来扩展/收缩区间
  3. 巧妙排序所有查询，使 cur_l 和 cur_r 的总移动量最小

排序策略:
  ┌────────────────────────────────────────────┐
  │ 将数组分成 √n 块                           │
  │ 第一关键字: l 所在的块编号                  │
  │ 第二关键字: r 值                            │
  │ 奇偶优化: 奇数块 r 升序, 偶数块 r 降序      │
  └────────────────────────────────────────────┘

可视化 — 指针移动轨迹:
  查询 (排序后): [1,3] → [1,7] → [2,5] → [4,8] → [5,9]
  
  cur_r 的移动:
                  ·····→→→→     ←←     →→→     →
  cur_l 的移动:
                  ·     ·  →     →→     →
  
  同一块内 l 变化小(≤√n), r 单调递增
  切换块时 l 跳跃(≤√n), r 可能回退

移动量分析:
  - l: 同块内移动 ≤ √n, 共 q 次查询 → O(q√n)
  - r: 同块内单调, 共 √n 个块, 每块 r 移动 ≤ n → O(n√n)
  - 总计: O((n+q)√n)
```

## 2. 标准模板：区间不同元素个数

```python
import math
from collections import defaultdict

def mo_algorithm(nums, queries):
    """
    nums: 原数组
    queries: [(l, r), ...] 0-indexed 闭区间
    返回: 每个查询的答案（区间内不同元素个数）
    """
    n = len(nums)
    q = len(queries)
    block = max(1, int(math.sqrt(n)))  # ⚠️ max(1,...) 防除零

    # ⚠️ 给每个查询附上原始索引，最后还原顺序
    indexed_queries = [(l, r, i) for i, (l, r) in enumerate(queries)]

    # 排序: 块号为第一关键字, r 为第二关键字 (奇偶优化)
    def sort_key(q):
        b = q[0] // block
        return (b, q[1] if b % 2 == 0 else -q[1])
    
    indexed_queries.sort(key=sort_key)

    ans = [0] * q
    count = defaultdict(int)  # 值 → 出现次数
    distinct = 0              # 当前区间不同元素个数

    cur_l = 0
    cur_r = -1  # ⚠️ 初始为空区间 [0, -1]

    def add(idx):
        nonlocal distinct
        count[nums[idx]] += 1
        if count[nums[idx]] == 1:  # ⚠️ 从 0 变 1: 新增一个不同元素
            distinct += 1

    def remove(idx):
        nonlocal distinct
        count[nums[idx]] -= 1
        if count[nums[idx]] == 0:  # ⚠️ 从 1 变 0: 减少一个不同元素
            distinct -= 1

    for l, r, qi in indexed_queries:
        # ⚠️ 扩展/收缩的顺序很重要！
        # 先扩展再收缩，保证区间始终有效
        while cur_r < r:    # 右扩展
            cur_r += 1
            add(cur_r)
        while cur_l > l:    # 左扩展
            cur_l -= 1
            add(cur_l)
        while cur_r > r:    # 右收缩
            remove(cur_r)
            cur_r -= 1
        while cur_l < l:    # 左收缩
            remove(cur_l)
            cur_l += 1

        ans[qi] = distinct

    return ans
```

### ⚠️ 关键细节

```
1. 指针移动顺序:
   ⚠️ 推荐: 先扩展(add) 再收缩(remove)
   如果先收缩可能导致区间变为空再变负 → 出错

   正确顺序:
     右扩展 → 左扩展 → 右收缩 → 左收缩
   或:
     右扩展 → 左收缩 → 右收缩 → 左扩展
   只要保证不出现非法区间

2. 初始状态:
   cur_l = 0, cur_r = -1
   ⚠️ 表示空区间，第一次循环会自然扩展

3. add/remove 中判断条件:
   add:    count[x] == 1 (加后==1，说明是新增)
   remove: count[x] == 0 (减后==0，说明是移除)
   ⚠️ 不要写反！

4. 奇偶优化:
   偶数块: r 升序 → 到达最大后
   奇数块: r 降序 → 直接回退
   减少块间切换时 r 的移动量，常数优化约 30%
```

## 3. 模板变体：区间内出现次数的平方和

```python
# 求 sum(count[x]^2) for all x in [l, r]
# 应用: 莫队常见变形，如 "小Z的袜子" (洛谷 P1494)

def mo_square_sum(nums, queries):
    n = len(nums)
    block = max(1, int(math.sqrt(n)))
    indexed_queries = [(l, r, i) for i, (l, r) in enumerate(queries)]
    indexed_queries.sort(key=lambda q: (q[0]//block, q[1] if (q[0]//block)%2==0 else -q[1]))

    ans = [0] * len(queries)
    count = defaultdict(int)
    square_sum = 0  # 维护 sum(count[x]^2)
    cur_l, cur_r = 0, -1

    def add(idx):
        nonlocal square_sum
        x = nums[idx]
        square_sum -= count[x] * count[x]     # ⚠️ 先减去旧的贡献
        count[x] += 1
        square_sum += count[x] * count[x]     # 再加上新的贡献

    def remove(idx):
        nonlocal square_sum
        x = nums[idx]
        square_sum -= count[x] * count[x]
        count[x] -= 1
        square_sum += count[x] * count[x]

    for l, r, qi in indexed_queries:
        while cur_r < r: cur_r += 1; add(cur_r)
        while cur_l > l: cur_l -= 1; add(cur_l)
        while cur_r > r: remove(cur_r); cur_r -= 1
        while cur_l < l: remove(cur_l); cur_l += 1
        ans[qi] = square_sum

    return ans
```

### ⚠️ 维护贡献的通用模式

```
add(idx):
  1. 减去旧贡献: total -= f(count[x])
  2. 更新计数: count[x] += 1
  3. 加上新贡献: total += f(count[x])

remove(idx):
  1. 减去旧贡献: total -= f(count[x])
  2. 更新计数: count[x] -= 1
  3. 加上新贡献: total += f(count[x])

⚠️ 不要试图只算差值 delta = f(new) - f(old)
   虽然等价，但 "先减后加" 模式更不容易出错
```

## 4. 例题详解

### LeetCode 2080 - 区间内查询数字的频率

```
题意: 实现 RangeFreqQuery 类:
  - 构造: 给定数组 arr
  - query(left, right, value): 返回 arr[left..right] 中 value 的出现次数

分析:
  方法 1 (在线): 哈希表 + 二分查找
    对每个值维护它出现的下标列表
    查询时二分找 [left, right] 范围内的个数
  
  方法 2 (离线): 莫队 (如果需要离线处理多种统计)

方法 1 更适合本题 (在线查询):
```

```python
from collections import defaultdict
from bisect import bisect_left, bisect_right

class RangeFreqQuery:
    def __init__(self, arr):
        self.pos = defaultdict(list)
        for i, v in enumerate(arr):
            self.pos[v].append(i)

    def query(self, left, right, value):
        indices = self.pos[value]
        # ⚠️ bisect_left 找 >= left 的第一个位置
        # ⚠️ bisect_right 找 > right 的第一个位置
        return bisect_right(indices, right) - bisect_left(indices, left)
```

### 洛谷 P1494 - 小Z的袜子

```
题意: 
  n 只袜子, 每只有颜色 c[i]
  q 次查询 [l, r], 求随机取两只袜子颜色相同的概率

分析:
  分子 = sum(C(count[x], 2)) = sum(count[x] * (count[x]-1) / 2)
       = (sum(count[x]^2) - (r-l+1)) / 2
  分母 = C(r-l+1, 2) = (r-l+1) * (r-l) / 2

  关键: 维护 sum(count[x]^2) → 标准莫队

伪代码:
  对每个查询:
    len = r - l + 1
    numerator = square_sum - len  (即 sum(cnt^2) - sum(cnt))
    denominator = len * (len - 1)
    answer = numerator / denominator (约分输出)
```

## 5. 带修莫队 (Mo's Algorithm with Modifications)

```
普通莫队只能处理无修改的查询
带修莫队: 支持单点修改 + 区间查询

核心变化:
  1. 多一维时间戳 t (第几次修改之前)
  2. 块大小改为 n^(2/3)
  3. 排序: (块号l, 块号r, t)
  4. 多一个指针 cur_t: 通过回退/前进修改来维护

复杂度: O(n^(5/3))

排序:
  def sort_key(q):
      bl = q.l // block
      br = q.r // block
      return (bl, br, q.t if br % 2 == 0 else -q.t)
  
  block = max(1, int(n ** (2/3)))  # ⚠️ 注意是 2/3 次方

⚠️ 带修莫队更复杂，竞赛中较少考，面试基本不考
```

## 本节要点速查

```
✅ 莫队 = 离线 + 分块排序 + 双指针增量维护
✅ 块大小: block = max(1, int(sqrt(n)))
✅ 排序: (l//block, r if 偶数块 else -r)  奇偶优化
✅ 初始状态: cur_l=0, cur_r=-1 (空区间)
✅ 指针移动: 先扩展再收缩，保证区间合法
✅ add/remove: 加后==1 为新增, 减后==0 为移除
✅ 维护贡献: 先减旧贡献，更新计数，再加新贡献
✅ 带修莫队: 多一维时间戳，块大小 n^(2/3)
✅ 总复杂度: O((n+q)√n)
```
