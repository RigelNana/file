# 同向双指针与原地修改

> 快慢指针同向移动：fast 遍历所有元素，slow 记录有效位置。用于原地删除、去重、分区。

---

## 1. 核心思想

```
fast (快指针): 遍历所有元素, 决定每个元素是否保留
slow (慢指针): 指向下一个要写入的位置

  [  已处理区域  | slow ... fast → 未处理 →  ]
  nums[0..slow-1] 是结果
  nums[slow..fast-1] 是被覆盖/跳过的

⚠️ slow 的含义: "下一个要写入的位置"
   最终 slow 就是结果长度
```

---

## 2. 移除元素

### LeetCode 27

### 理解与可视化

```
输入: nums = [3,2,2,3], val = 3
输出: 2, nums = [2,2,_,_]

  fast=0: nums[0]=3 == val → 跳过
  fast=1: nums[1]=2 != val → nums[0]=2, slow=1
  fast=2: nums[2]=2 != val → nums[1]=2, slow=2
  fast=3: nums[3]=3 == val → 跳过

  结果: slow=2, nums[:2] = [2,2]
```

```python
def removeElement(nums, val):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:         # ⚠️ 不等于 val 才保留
            nums[slow] = nums[fast]
            slow += 1
    return slow

# ⚠️ 返回 slow (结果长度), 不是 slow-1 (最后一个下标)
# ⚠️ nums[fast] != val 是保留条件, 不是删除条件
# ⚠️ 原地修改: 不需要额外数组
```

## 3. 删除有序数组中的重复项

### LeetCode 26

```
输入: nums = [1,1,2]
输出: 2, nums = [1,2,_]

排序数组, 只保留每个值的第一次
```

```python
def removeDuplicates(nums):
    if len(nums) <= 1:
        return len(nums)
    slow = 1                          # ⚠️ 第一个元素一定保留
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow - 1]:  # ⚠️ 和已保留的最后一个比
            nums[slow] = nums[fast]
            slow += 1
    return slow

# ⚠️ slow 从 1 开始 (第0个元素天然保留)
# ⚠️ 比较对象是 nums[slow-1] (已保留的最后一个)
#    不是 nums[fast-1] (上一个遍历到的)
#    但在本题中效果相同, 因为数组有序
```

### 变体: 最多保留 k 次

### LeetCode 80 (k=2)

```python
def removeDuplicates_k(nums, k=2):
    if len(nums) <= k:
        return len(nums)
    slow = k                          # ⚠️ 前 k 个一定保留
    for fast in range(k, len(nums)):
        if nums[fast] != nums[slow - k]:  # ⚠️ 和前面第 k 个比
            nums[slow] = nums[fast]
            slow += 1
    return slow

# ⚠️ 通用规则: 最多保留 k 次
#   和 nums[slow - k] 比较
#   如果相等 → slow前面已经有k个这样的值了 → 跳过
#   如果不等 → 保留
#
# ⚠️ k=1 时退化为 LC26 (每个值只保留1次)
#   k=2 时就是 LC80
```

## 4. 移动零

### LeetCode 283

```
输入: [0,1,0,3,12]
输出: [1,3,12,0,0]

保持非零元素的相对顺序, 零移到末尾
```

```python
def moveZeroes(nums):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow], nums[fast] = nums[fast], nums[slow]  # ⚠️ 交换
            slow += 1

# ⚠️ 这里用交换, 不是赋值!
#   赋值: nums[slow] = nums[fast] → 后面的零没了
#   交换: 非零移前面, 零移后面
#
# ⚠️ 也可以先赋值再补零:
#   slow = 0
#   for x in nums:
#       if x != 0: nums[slow] = x; slow += 1
#   for i in range(slow, len(nums)):
#       nums[i] = 0
```

## 5. 合并两个有序数组

### LeetCode 88

```
nums1 = [1,2,3,0,0,0], m=3
nums2 = [2,5,6], n=3
输出: [1,2,2,3,5,6]

⚠️ 从后往前填! 避免覆盖 nums1 中未处理的元素
```

```python
def merge(nums1, m, nums2, n):
    i, j, k = m - 1, n - 1, m + n - 1
    
    while i >= 0 and j >= 0:
        if nums1[i] >= nums2[j]:      # ⚠️ >= 保证稳定
            nums1[k] = nums1[i]
            i -= 1
        else:
            nums1[k] = nums2[j]
            j -= 1
        k -= 1
    
    # ⚠️ 只需处理 nums2 剩余
    while j >= 0:
        nums1[k] = nums2[j]
        j -= 1
        k -= 1

# ⚠️ 从后往前填 (k 从 m+n-1 开始):
#   从前往前填会覆盖 nums1 中还没处理的元素
#
# ⚠️ nums1 剩余不需要处理:
#   nums1 的元素已经在正确位置上了
# ⚠️ nums2 剩余需要处理:
#   nums2 的小元素要拷贝到 nums1 前面
```

## 6. 判断子序列

### LeetCode 392

```
s = "abc", t = "ahbgdc" → True (a...b...c 是子序列)

双指针: i 遍历 s, j 遍历 t
  s[i] == t[j] → i++, j++
  s[i] != t[j] → j++ (跳过 t 中的字符)
```

```python
def isSubsequence(s, t):
    i = j = 0
    while i < len(s) and j < len(t):
        if s[i] == t[j]:
            i += 1                    # ⚠️ 匹配了才移i
        j += 1                        # j 每次都移
    return i == len(s)                # ⚠️ i 到头了说明全匹配

# ⚠️ 不管是否匹配, j 都 +1
# ⚠️ 最终看 i == len(s), 不是 j == len(t)
#    i 到头 = s 中所有字符都找到了
```
