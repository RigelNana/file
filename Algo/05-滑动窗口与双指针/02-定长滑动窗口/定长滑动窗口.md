# 定长滑动窗口

> 窗口大小固定为 k，整体向右滑动。核心技巧：新窗口 = 旧窗口 + 右边新元素 - 左边旧元素。

---

## 1. 定长窗口模板

### 理解与可视化

```
数组: [2, 1, 5, 1, 3, 2], k=3

  [2, 1, 5] 1, 3, 2    sum=8
  2, [1, 5, 1] 3, 2    sum=8+1-2=7    加 nums[3], 减 nums[0]
  2, 1, [5, 1, 3] 2    sum=7+3-1=9    加 nums[4], 减 nums[1]
  2, 1, 5, [1, 3, 2]   sum=9+2-5=6    加 nums[5], 减 nums[2]

  窗口右端 i 从 k-1 开始滑到 n-1
  窗口左端 = i - k + 1

  ⚠️ 不需要每次 O(k) 重新计算窗口内的值
     每次只需 O(1) 增减 → 总时间 O(n)
```

### 代码模板

```python
# 写法一: 先初始化, 再滑动
def fixed_window_v1(nums, k):
    n = len(nums)
    window_sum = sum(nums[:k])        # ⚠️ 初始化前 k 个
    ans = window_sum
    for i in range(k, n):
        window_sum += nums[i] - nums[i - k]  # ⚠️ 加右减左
        ans = max(ans, window_sum)
    return ans

# 写法二: 统一循环 (推荐, 更通用)
def fixed_window_v2(nums, k):
    ans = 0
    window = 0
    for i, x in enumerate(nums):
        window += x                   # 1. 加入右端
        if i < k - 1:                 # 2. 窗口还没满
            continue                  # ⚠️ i < k-1, 不是 i < k
        ans = max(ans, window)        # 3. 更新答案
        window -= nums[i - k + 1]    # 4. 移出左端
    return ans

# ⚠️ 写法二的执行顺序:
#   i=0,1,...,k-2: 只加入, 不处理 (窗口还没满)
#   i=k-1: 第一个完整窗口, 更新答案, 移出 nums[0]
#   i=k:   更新答案, 移出 nums[1]
#   ...
#
# ⚠️ 移出元素下标: i - k + 1 (当前窗口的左端)
#    不是 i - k (那是窗口左端的前一个)
```

## 2. 子数组最大平均数

### LeetCode 643

```
输入: nums = [1,12,-5,-6,50,3], k = 4
输出: 12.75 → (12-5-6+50)/4 = 51/4

先求最大和, 最后除以 k
```

```python
def findMaxAverage(nums, k):
    window_sum = sum(nums[:k])
    max_sum = window_sum
    for i in range(k, len(nums)):
        window_sum += nums[i] - nums[i - k]
        max_sum = max(max_sum, window_sum)
    return max_sum / k                # ⚠️ 最后才除, 避免浮点误差

# ⚠️ 不要每步都除以 k 再比较
#    只需要比较 sum 的大小, 最后除一次
```

## 3. 找到字符串中所有字母异位词

### LeetCode 438

### 理解与可视化

```
输入: s = "cbaebabacd", p = "abc"
输出: [0, 6] → s[0:3]="cba", s[6:9]="bac" 是 "abc" 的异位词

异位词 = 字符频率完全相同
窗口大小 = len(p) = 3

方法: 维护窗口内字符频率, 和 p 的频率比较
  或者维护差值计数器, 差值全0时就是异位词
```

```python
from collections import Counter

def findAnagrams(s, p):
    k = len(p)
    if k > len(s):
        return []
    
    p_cnt = Counter(p)
    s_cnt = Counter(s[:k])            # ⚠️ 初始窗口
    res = []
    
    if s_cnt == p_cnt:
        res.append(0)
    
    for i in range(k, len(s)):
        # 加入右端
        s_cnt[s[i]] += 1
        # 移出左端
        left_char = s[i - k]
        s_cnt[left_char] -= 1
        if s_cnt[left_char] == 0:
            del s_cnt[left_char]      # ⚠️ 删除 0 值, 否则 == 比较失败
        
        if s_cnt == p_cnt:
            res.append(i - k + 1)     # ⚠️ 左端下标
    
    return res

# ⚠️ 必须 del 掉计数为 0 的键!
#   Counter({'a':1, 'b':0}) != Counter({'a':1})
#   如果不删, 永远不会 == p_cnt
#
# ⚠️ 左端下标 = i - k + 1, 不是 i - k
# ⚠️ 时间 O(n × 26) ≈ O(n), 比较 Counter 是 O(26)
```

### 优化: 用差值计数器

```python
def findAnagrams_v2(s, p):
    k = len(p)
    if k > len(s):
        return []
    
    diff = [0] * 26
    for c in p:
        diff[ord(c) - ord('a')] -= 1
    for c in s[:k]:
        diff[ord(c) - ord('a')] += 1
    
    res = []
    if all(d == 0 for d in diff):
        res.append(0)
    
    for i in range(k, len(s)):
        diff[ord(s[i]) - ord('a')] += 1
        diff[ord(s[i-k]) - ord('a')] -= 1
        if all(d == 0 for d in diff):
            res.append(i - k + 1)
    
    return res

# ⚠️ diff[c] = 窗口中字符c的个数 - p中字符c的个数
#    全0 → 完全匹配
# ⚠️ 可以进一步优化: 维护 diff 中非0值的个数
#    非0个数 == 0 时就是异位词 → O(1) 判断
```

## 4. 定长窗口中的最大值

### LeetCode 239 (滑动窗口最大值)

```
这道题用单调队列更优 (见单调栈与单调队列章节)
这里给出堆的思路作为定长窗口的应用

输入: nums = [1,3,-1,-3,5,3,6,7], k = 3
窗口:    [1,3,-1] → 3
        [3,-1,-3] → 3
        [-1,-3,5] → 5
        [-3,5,3]  → 5
        [5,3,6]   → 6
        [3,6,7]   → 7
```

```python
import heapq

def maxSlidingWindow(nums, k):
    # 最大堆: 存 (-值, 下标)
    heap = [(-nums[i], i) for i in range(k)]
    heapq.heapify(heap)
    res = [-heap[0][0]]
    
    for i in range(k, len(nums)):
        heapq.heappush(heap, (-nums[i], i))
        # ⚠️ 惰性删除: 堆顶不在窗口内就弹出
        while heap[0][1] <= i - k:
            heapq.heappop(heap)
        res.append(-heap[0][0])
    
    return res

# ⚠️ 惰性删除: 不从堆中主动删旧元素
#    只在取堆顶时检查是否过期
# ⚠️ 过期判断: 下标 <= i - k (不在窗口 [i-k+1, i] 内)
# ⚠️ 时间 O(n log n), 单调队列可以做到 O(n)
```

```

✅ 窗口没满时 continue (i < k-1)
✅ Counter 比较要 del 掉 0 值的键
✅ 惰性删除: 堆顶过期才弹出
✅ 先算和再除, 避免浮点误差
```
