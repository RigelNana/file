# 分组循环

> 分组循环：把数组中连续的、满足某个条件的元素划为一组，逐组处理。

---

## 1. 分组循环模板

### 理解与可视化

```
将数组按相邻相同值分组:
  [1,1,2,2,2,3,3] → [1,1] [2,2,2] [3,3]
  
  i=0: start=0, nums[0]=1=nums[0] → i=1, nums[1]=1=nums[0] → i=2
       处理 [0, 2) 即 [1,1], 长度=2
  
  i=2: start=2, nums[2]=2=nums[2] → i=3, i=4 → i=5
       处理 [2, 5) 即 [2,2,2], 长度=3
  
  i=5: start=5, nums[5]=3=nums[5] → i=6, i=7
       处理 [5, 7) 即 [3,3], 长度=2

⚠️ 结果区间是 [start, i), 左闭右开
⚠️ 内层 while 结束后 i 已经指向下一组的开头
⚠️ 外层不需要 i += 1 (i 在内层已经前进了)
```

### 代码模板

```python
def group_loop(nums):
    n = len(nums)
    i = 0
    while i < n:
        start = i
        while i < n and nums[i] == nums[start]:  # ⚠️ 同一组
            i += 1
        # 处理 nums[start:i], 长度 = i - start
        process(nums[start:i])

# ⚠️ 外层 while 不需要 i += 1
#    内层已经把 i 移到了下一组开头
#
# ⚠️ 内层条件可以是任意"属于同一组"的判断:
#    nums[i] == nums[start]       相同值
#    nums[i] >= nums[i-1]        递增段
#    nums[i] % 2 == nums[start] % 2  同奇偶
```

## 2. 最长连续递增序列

### LeetCode 674

```
输入: [1,3,5,4,7] → 最长连续递增: [1,3,5], 长度3

分组: 每个递增段是一组
  [1,3,5] | [4,7]
  长度 3     长度 2
  答案: 3
```

```python
def findLengthOfLCIS(nums):
    if not nums:
        return 0
    n = len(nums)
    ans = 0
    i = 0
    while i < n:
        start = i
        i += 1                        # ⚠️ 至少包含当前元素
        while i < n and nums[i] > nums[i-1]:  # ⚠️ 严格递增
            i += 1
        ans = max(ans, i - start)
    return ans

# ⚠️ 内层条件 nums[i] > nums[i-1], 不是 >= (严格递增)
# ⚠️ i += 1 写在内层 while 之前: 保证至少选一个元素
```

## 3. 交替子数组计数

### LeetCode 2765

```
输入: [0,1,0,1,0]
统计所有长度>=2的交替子数组个数

分组: [0,1,0,1,0] 是一整个交替组, 长度5
  长度为 L 的交替组, 贡献 (L-1) 个长度>=2的子数组?
  不对! 贡献的是 L-1 + L-2 + ... + 1 = L*(L-1)/2
  
  但更精确: 长度为 L 的交替段中
  以每个位置结尾的交替子数组个数 = 它在当前交替段中的位置
```

```python
def alternatingSubarray(nums):
    n = len(nums)
    ans = -1                          # ⚠️ 找不到返回 -1
    i = 0
    while i < n:
        start = i
        i += 1
        if i < n and nums[i] - nums[start] != 1:
            continue                  # ⚠️ 差值必须是 1 开头
        while i < n and nums[i] - nums[i-1] == (-1) ** (i - start):
            i += 1
        if i - start >= 2:
            ans = max(ans, i - start)
    return ans
```

## 4. 分组循环的变体

```
变体一: 按条件分段, 统计段数
  [1,0,1,1,0,1] → 统计全1段的个数
  i=0: [1] → 全1段 ✓
  i=1: [0] → skip
  i=2: [1,1] → 全1段 ✓
  i=4: [0] → skip
  i=5: [1] → 全1段 ✓
  答案: 3 段

变体二: 找最长/最短段
  直接在处理段时 ans = max(ans, i - start)

变体三: 按段处理后合并
  如摘水果、分组后计算连续段和

⚠️ 分组循环的优点:
  1. 不需要额外记录前一组的状态
  2. 每组的处理逻辑互相独立
  3. 代码结构清晰, 不容易出错
```

## 本节要点速查

```
✅ 双层 while: 外层遍历组, 内层遍历组内元素
✅ 外层不需要 i++, 内层已经移动 i
✅ 区间 [start, i) 左闭右开
✅ 内层条件可以灵活变化 (相同值/递增/交替等)
✅ 分组循环 = 枚举段, 每段独立处理
```
