# 滑动窗口与双指针

> 滑动窗口和双指针是面试中最高频的技巧之一。核心思想是用两个指针维护一个区间，通过移动指针避免暴力枚举。

---

## 1. 分组循环

### 理解与可视化

```
分组循环: 把数组按连续相同的特征分组

例: [1,1,2,2,2,3,3] → [1,1] [2,2,2] [3,3]

模板思路:
  i = 0
  while i < n:
      start = i
      while i < n and 满足条件:
          i += 1
      处理 [start, i) 这一组

  无需在循环外维护额外变量，每组处理互不干扰
```

### 代码模板

```python
# 分组循环通用模板
def group_loop(nums):
    n = len(nums)
    i = 0
    while i < n:
        start = i
        while i < n and nums[i] == nums[start]:  # 同一组
            i += 1
        # 处理 nums[start:i] 这一组，长度 = i - start
        print(f"组: {nums[start:i]}")
```

## 2. 定长滑动窗口

### 理解与可视化

```
定长窗口: 窗口大小固定为 k，整体滑动

数组: [2, 1, 5, 1, 3, 2]，k=3，求最大和

  窗口滑动过程:
  [2, 1, 5] 1, 3, 2    sum=8
  2, [1, 5, 1] 3, 2    sum=7    加入 1，移出 2
  2, 1, [5, 1, 3] 2    sum=9    加入 3，移出 1  ← 最大
  2, 1, 5, [1, 3, 2]   sum=6    加入 2，移出 5

关键: 不需要每次重新算窗口内的和
  新窗口和 = 旧窗口和 + 新加入元素 - 移出元素
  时间复杂度: O(n) 而非 O(n×k)

  ┌─────────────────────────────┐
  │   [  窗口 k 个元素  ]       │
  │    ←─── 一起向右滑动 ───→   │
  │   加入右边  │  移出左边      │
  └─────────────────────────────┘
```

### 代码模板

```python
# 定长窗口模板
def fixed_window(nums, k):
    n = len(nums)
    # 初始化第一个窗口
    window_sum = sum(nums[:k])
    ans = window_sum
    # 滑动
    for i in range(k, n):
        window_sum += nums[i] - nums[i - k]  # 加右移左
        ans = max(ans, window_sum)
    return ans

# 定长窗口（更通用的写法）
def fixed_window_v2(nums, k):
    ans = 0
    window = 0  # 窗口内的统计量
    for i, x in enumerate(nums):
        window += x           # 加入右端
        if i < k - 1:         # 窗口还没满
            continue
        ans = max(ans, window)
        window -= nums[i - k + 1]  # 移出左端
    return ans

# 字符串中字符种类数恰好为 k 的最长子串长度
def max_vowels(s, k):
    """大小为 k 的窗口中元音字母的最大数目"""
    vowels = set("aeiou")
    cnt = sum(1 for c in s[:k] if c in vowels)
    ans = cnt
    for i in range(k, len(s)):
        cnt += (s[i] in vowels) - (s[i - k] in vowels)
        ans = max(ans, cnt)
    return ans
```

## 3. 不定长滑动窗口：求最长/最大 ⭐

### 理解与可视化

```
不定长窗口: 窗口大小可变，根据条件伸缩

求最长无重复字符子串: "abcabcbb"

  左指针 left，右指针 right
  right 不断向右扩，如果窗口内有重复，left 向右缩

  a b c a b c b b
  l     r           "abc" 长度3
  l       r         a重复 → 移动l
    l     r         "bca" 长度3
    l       r       b重复 → 移动l
      l     r       "cab" 长度3
      l       r     c重复 → 移动l
        l     r     "abc" 长度3
        l       r   b重复 → 移动l
          l     r   b重复 → 移动l
            l   r   "cb" 长度2
            l     r b重复 → 移动l
              l   r "b" 长度1

  答案: 3

核心思想:
  ┌──────────────────────────────────┐
  │  right 不断向右扩展（扩大窗口）   │
  │  当不满足条件时:                  │
  │    left 向右收缩（缩小窗口）      │
  │  每次更新答案 = max(ans, r-l+1)  │
  └──────────────────────────────────┘
```

### 代码模板

```python
# 不定长窗口：求最长（⭐ 核心模板）
def longest_window(nums):
    left = 0
    ans = 0
    # window_state: 维护窗口内的状态（计数器、和等）
    for right in range(len(nums)):
        # 扩大窗口：更新状态
        更新window_state(nums[right])
        # 当窗口不满足条件时，收缩左端
        while 不满足条件:
            撤销window_state(nums[left])
            left += 1
        # 更新答案
        ans = max(ans, right - left + 1)
    return ans

# 最长无重复字符子串
def length_of_longest_substring(s):
    left = 0
    ans = 0
    cnt = {}  # 字符计数
    for right, c in enumerate(s):
        cnt[c] = cnt.get(c, 0) + 1
        while cnt[c] > 1:  # 有重复
            cnt[s[left]] -= 1
            left += 1
        ans = max(ans, right - left + 1)
    return ans

# 最长连续 1 的个数（最多翻转 k 个 0）
def longest_ones(nums, k):
    left = 0
    zeros = 0
    ans = 0
    for right in range(len(nums)):
        if nums[right] == 0:
            zeros += 1
        while zeros > k:
            if nums[left] == 0:
                zeros -= 1
            left += 1
        ans = max(ans, right - left + 1)
    return ans
```

## 4. 不定长滑动窗口：求最短/最小 ⭐

### 理解与可视化

```
和求最长相反: 满足条件时尝试收缩，找最短

求和 ≥ target 的最短子数组: nums=[2,3,1,2,4,3], target=7

  [2,3,1,2,4,3]
  [2,3,1,2]          sum=8≥7 ✓ 长度4, 收缩
   [3,1,2]           sum=6 <7, 继续扩
   [3,1,2,4]         sum=10≥7 ✓ 长度4, 收缩
    [1,2,4]          sum=7≥7 ✓ 长度3, 收缩
     [2,4]           sum=6 <7, 继续扩
     [2,4,3]         sum=9≥7 ✓ 长度3, 收缩
      [4,3]          sum=7≥7 ✓ 长度2 ← 最短!
       [3]           sum=3 <7

  答案: 2

区别:
  求最长: while 不满足条件时收缩
  求最短: while 满足条件时收缩（尽量缩短）
```

### 代码模板

```python
# 不定长窗口：求最短
def shortest_window(nums, target):
    left = 0
    ans = float('inf')
    window_sum = 0
    for right in range(len(nums)):
        window_sum += nums[right]
        while window_sum >= target:  # 满足条件时收缩
            ans = min(ans, right - left + 1)
            window_sum -= nums[left]
            left += 1
    return ans if ans != float('inf') else 0

# 长度最小的子数组
def min_sub_array_len(target, nums):
    left = 0
    ans = float('inf')
    s = 0
    for right in range(len(nums)):
        s += nums[right]
        while s >= target:
            ans = min(ans, right - left + 1)
            s -= nums[left]
            left += 1
    return ans if ans != float('inf') else 0

# 最小覆盖子串（⭐ 经典难题）
from collections import Counter

def min_window(s, t):
    need = Counter(t)
    missing = len(t)  # 还需要的字符数
    left = 0
    start, end = 0, float('inf')
    for right, c in enumerate(s):
        if need[c] > 0:
            missing -= 1
        need[c] -= 1
        while missing == 0:  # 满足条件，尝试收缩
            if right - left < end - start:
                start, end = left, right
            need[s[left]] += 1
            if need[s[left]] > 0:
                missing += 1
            left += 1
    return s[start:end+1] if end != float('inf') else ""
```

## 5. 不定长窗口：求子数组个数

### 理解与可视化

```
技巧: 恰好等于 k = 至多 k - 至多 (k-1)

因为直接求「恰好等于 k」很难用滑窗，但「至多 k」可以

至多 k 个不同字符的子串数:
  对于每个 right，满足条件的子串是 [left,right], [left+1,right], ..., [right,right]
  个数 = right - left + 1
```

### 代码模板

```python
# 至多 k 个不同字符的子串数
def at_most_k(s, k):
    left = 0
    cnt = {}
    ans = 0
    for right, c in enumerate(s):
        cnt[c] = cnt.get(c, 0) + 1
        while len(cnt) > k:
            cnt[s[left]] -= 1
            if cnt[s[left]] == 0:
                del cnt[s[left]]
            left += 1
        ans += right - left + 1
    return ans

# 恰好 k 个不同字符的子串数
def exactly_k(s, k):
    return at_most_k(s, k) - at_most_k(s, k - 1)
```

## 6. 相向双指针 ⭐

### 理解与可视化

```
两个指针从两端向中间移动

经典: 两数之和（排序后）
  排序数组: [2, 7, 11, 15]  target=9

  left=0, right=3:  2+15=17 > 9  → right--
  left=0, right=2:  2+11=13 > 9  → right--
  left=0, right=1:  2+7=9  ✓     → 找到！

  ┌──────────────────────────┐
  │ left →           ← right │
  │ 和太大: right--          │
  │ 和太小: left++           │
  │ 刚好: 返回               │
  └──────────────────────────┘
```

### 代码模板

```python
# 两数之和（排序数组）
def two_sum_sorted(nums, target):
    left, right = 0, len(nums) - 1
    while left < right:
        s = nums[left] + nums[right]
        if s == target:
            return [left, right]
        elif s < target:
            left += 1
        else:
            right -= 1
    return []

# 三数之和
def three_sum(nums):
    nums.sort()
    res = []
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue  # 去重
        left, right = i + 1, len(nums) - 1
        while left < right:
            s = nums[i] + nums[left] + nums[right]
            if s == 0:
                res.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left+1]:
                    left += 1
                while left < right and nums[right] == nums[right-1]:
                    right -= 1
                left += 1
                right -= 1
            elif s < 0:
                left += 1
            else:
                right -= 1
    return res

# 盛最多水的容器
def max_area(height):
    left, right = 0, len(height) - 1
    ans = 0
    while left < right:
        area = min(height[left], height[right]) * (right - left)
        ans = max(ans, area)
        if height[left] < height[right]:
            left += 1
        else:
            right -= 1
    return ans

# 接雨水（双指针法）
def trap(height):
    left, right = 0, len(height) - 1
    left_max = right_max = 0
    ans = 0
    while left < right:
        left_max = max(left_max, height[left])
        right_max = max(right_max, height[right])
        if left_max < right_max:
            ans += left_max - height[left]
            left += 1
        else:
            ans += right_max - height[right]
            right -= 1
    return ans
```

## 7. 同向双指针 & 原地修改

### 代码模板

```python
# 移除元素（原地修改）
def remove_element(nums, val):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow

# 删除有序数组中的重复项
def remove_duplicates(nums):
    if len(nums) <= 1:
        return len(nums)
    slow = 1
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow - 1]:
            nums[slow] = nums[fast]
            slow += 1
    return slow

# 移动零
def move_zeroes(nums):
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != 0:
            nums[slow], nums[fast] = nums[fast], nums[slow]
            slow += 1
```

## 8. 滑动窗口与双指针总结

```
选择哪种技巧？

  ┌────────────────────┬────────────────────────────┐
  │ 问题类型            │ 技巧                        │
  ├────────────────────┼────────────────────────────┤
  │ 连续子数组/子串     │ 滑动窗口                    │
  │ 固定大小的窗口      │ 定长滑动窗口                │
  │ 最长/最大子数组     │ 不定长窗口(不满足时收缩)     │
  │ 最短/最小子数组     │ 不定长窗口(满足时收缩)       │
  │ 子数组个数          │ at_most(k) - at_most(k-1)  │
  │ 有序数组两数之和    │ 相向双指针                  │
  │ 原地修改数组       │ 快慢指针                    │
  └────────────────────┴────────────────────────────┘
```
