# 算法学习路径（零基础完整规划）

> 本路径将所有专题按**由浅入深**的顺序分为 **6 个阶段**，每个阶段内部也标注了建议的先后顺序。
> 原则：**先掌握数据结构，再学算法；先学暴力，再学优化；先做基础，再做进阶。**

| 阶段一 | 2-3 题 | 以理解为主，不赶进度 |
| 阶段二 | 2-3 题 | 重复练习模板题 |
| 阶段三 | 1-2 题 | DP 需要大量思考，不贪多 |
| 阶段四 | 1-2 题 | 图论需要画图理解 |
| 阶段五 | 1-2 题 | 难度大，允许看题解 |
| 阶段六 | 1 题 | 耐心推导 |

## 学习原则

1. **先看视频/题解理解思路**，再自己写代码，最后对拍优化
2. **每个专题先做基础题**，确保理解模板后再做进阶
3. **标注 ⭐ 的专题优先学习**，标注"选做"和"竞赛向"的可以跳过
4. **做不出来很正常**，看题解后一定要自己重新写一遍
5. **定期复习**，每周末回顾本周做过的题，间隔 1-2 周再做一次
6. **阶段三和阶段四可以穿插进行**，避免长时间只做 DP 产生倦怠
