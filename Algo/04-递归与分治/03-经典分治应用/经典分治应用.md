## 1. 第 K 大元素（快速选择）

### LeetCode 215

### 理解与可视化

```
快速选择 (QuickSelect) = 快排的变体
  快排: 排完左右两边
  快选: 只排目标所在的一边 → 平均 O(n)

nums = [3,2,1,5,6,4], k=2 → 找第2大 = 5

  三路分区 (pivot=4):
    small=[3,2,1]  equal=[4]  big=[5,6]
    
    要第2大, big有2个(≥k) → 答案在 big 中
    → 递归 quick_select(big=[5,6], k=2)
    
    pivot=5:
    small=[]  equal=[5]  big=[6]
    big有1个(<k=2) → k -= 1 = 1, 再看 equal
    1 <= len(big)+len(equal) → 答案是 pivot=5 ✓
```

### 代码模板

```python
import random

def findKthLargest(nums, k):
    def quick_select(arr, k):
        pivot = random.choice(arr)    # ⚠️ 随机选 pivot
        big   = [x for x in arr if x > pivot]
        equal = [x for x in arr if x == pivot]
        small = [x for x in arr if x < pivot]
        
        if k <= len(big):             # ⚠️ 第k大在 big 中
            return quick_select(big, k)
        elif k <= len(big) + len(equal):  # 在 equal 中
            return pivot
        else:                         # 在 small 中
            return quick_select(small, k - len(big) - len(equal))
    
    return quick_select(nums, k)

# ⚠️ 这是找第 k 大, 不是第 k 小!
#   第 k 大: 先看 big, 再看 equal, 最后 small
#   第 k 小: 先看 small, 再看 equal, 最后 big
#
# ⚠️ 时间复杂度:
#   平均 O(n): 每次期望砍掉一半, n + n/2 + n/4 + ... = 2n
#   最坏 O(n²): 每次只砍掉一个元素 (随机化避免)
#
# ⚠️ k <= len(big) 不是 k < len(big)
#   因为 k 是 1-based, 第1大 = big[0]
```

## 2. 最大子数组和（分治法）

### LeetCode 53

```
Kadane 算法 O(n) 更优, 但分治法体现分治思想

思路: 将数组分为左半和右半
  最大子数组只可能在三个位置:
  1. 完全在左半
  2. 完全在右半
  3. 跨越中间

  ┌──────────┬──────────┐
  │  左半     │  右半     │
  │  left_max │ right_max│
  │     ├─ cross_max ─┤  │
  └──────────┴──────────┘

  cross_max = 从 mid 往左的最大和 + 从 mid+1 往右的最大和
```

```python
def maxSubArray(nums):
    def helper(lo, hi):
        if lo == hi:                  # ⚠️ 只剩一个元素
            return nums[lo]
        
        mid = (lo + hi) // 2
        left_max = helper(lo, mid)
        right_max = helper(mid + 1, hi)
        
        # 计算跨越中间的最大和
        left_sum = float('-inf')
        cur = 0
        for i in range(mid, lo - 1, -1):   # ⚠️ 从 mid 往左
            cur += nums[i]
            left_sum = max(left_sum, cur)
        
        right_sum = float('-inf')
        cur = 0
        for i in range(mid + 1, hi + 1):   # ⚠️ 从 mid+1 往右
            cur += nums[i]
            right_sum = max(right_sum, cur)
        
        cross_max = left_sum + right_sum
        return max(left_max, right_max, cross_max)
    
    return helper(0, len(nums) - 1)

# ⚠️ cross 部分必须包含 mid 和 mid+1
#    左半从 mid 开始往左扫 → 必须到达 mid
#    右半从 mid+1 开始往右扫 → 必须到达 mid+1
# ⚠️ 初始值用 float('-inf') 而非 0
#    因为所有元素可能都是负数
# ⚠️ 时间 O(n log n), 不如 Kadane 的 O(n)
```

## 3. 多数元素

### LeetCode 169

```
多数元素: 出现次数 > n/2 的元素

分治思路:
  左半的多数元素和右半的多数元素
  如果相同 → 就是答案
  如果不同 → 在整个范围内数谁出现更多

⚠️ 更优解法: Boyer-Moore 投票法 O(n), O(1)
```

```python
# 分治法 O(n log n)
def majorityElement(nums):
    def helper(lo, hi):
        if lo == hi:
            return nums[lo]
        mid = (lo + hi) // 2
        left = helper(lo, mid)
        right = helper(mid + 1, hi)
        if left == right:
            return left
        # 在 [lo, hi] 中分别数 left 和 right 的出现次数
        left_count = sum(1 for i in range(lo, hi + 1) if nums[i] == left)
        right_count = sum(1 for i in range(lo, hi + 1) if nums[i] == right)
        return left if left_count > right_count else right
    
    return helper(0, len(nums) - 1)

# Boyer-Moore 投票法 O(n) → 推荐
def majorityElement_vote(nums):
    candidate = count = 0
    for x in nums:
        if count == 0:
            candidate = x
        count += 1 if x == candidate else -1
    return candidate

# ⚠️ 投票法原理:
#   把多数元素看成 +1, 其他看成 -1
#   由于多数元素 > n/2, 总和一定 > 0
#   count=0 时重新选 candidate, 最后剩的就是答案
```

## 4. 搜索二维矩阵 II

### LeetCode 240

```
矩阵每行递增, 每列递增. 查找 target

[1,  4,  7, 11, 15]
[2,  5,  8, 12, 19]
[3,  6,  9, 16, 22]
[10, 13, 14, 17, 24]
[18, 21, 23, 26, 30]

分治: 从右上角开始
  如果 == target   → 找到!
  如果 > target    → 排除当前列 (下面更大)
  如果 < target    → 排除当前行 (左边更小)
  
  ⚠️ 每次排除一行或一列 → O(m + n)

查找 target=5:
  右上角 15 > 5 → 左移 → 11 > 5 → 左移 → 7 > 5 → 左移
  → 4 < 5 → 下移 → 5 == target ✓
```

```python
def searchMatrix(matrix, target):
    if not matrix or not matrix[0]:
        return False
    m, n = len(matrix), len(matrix[0])
    r, c = 0, n - 1                   # ⚠️ 从右上角开始
    
    while r < m and c >= 0:           # ⚠️ 边界: r<m 和 c>=0
        if matrix[r][c] == target:
            return True
        elif matrix[r][c] > target:
            c -= 1                    # ⚠️ 当前值大 → 左移
        else:
            r += 1                    # ⚠️ 当前值小 → 下移
    
    return False

# ⚠️ 也可以从左下角开始:
#   大了 → 上移 (r -= 1)
#   小了 → 右移 (c += 1)
# ⚠️ 不能从左上角或右下角开始:
#   左上角: 右移和下移都是变大, 无法排除
#   右下角: 左移和上移都是变小, 无法排除
# ⚠️ 时间 O(m+n), 空间 O(1)
```

## 5. 分治思想扩展

```
分治不仅用于数组, 还广泛应用于:

┌──────────────┬──────────────────────┬───────────┐
│ 应用         │ 分治点                │ 时间       │
├──────────────┼──────────────────────┼───────────┤
│ 归并排序     │ 数组中点              │ O(n log n)│
│ 快速排序     │ pivot                │ O(n log n)│
│ 快速选择     │ pivot (只递归一半)    │ O(n) avg  │
│ 二分查找     │ 比较中点值            │ O(log n)  │
│ 线段树       │ 区间中点              │ O(log n)  │
│ FFT快速傅里叶│ 奇偶分治              │ O(n log n)│
│ 最近点对     │ x坐标中位数           │ O(n log n)│
│ 矩阵乘法     │ 子矩阵 (Strassen)    │ O(n^2.81) │
└──────────────┴──────────────────────┴───────────┘

分治的关键:
  1. 子问题互相独立 (否则考虑 DP)
  2. 合并步骤不能太贵
  3. 分割要均匀 (否则退化)
```

## 本节要点速查

```
✅ 快速选择: 随机pivot, 只递归一半, 平均 O(n)
✅ 第k大: 先看big, 再equal, 最后small
✅ 跨越中间: 从mid往左+从mid+1往右, 初始值-inf
✅ 搜索矩阵: 右上角或左下角出发, O(m+n)
✅ 投票法: count=0重选, 多数元素一定最后胜出
✅ 分治关键: 子问题独立 + 合并不贵 + 分割均匀
```
