# 递归与分治

> 递归是函数调用自身的编程技巧。分治是「分而治之」，把大问题拆成小问题，各自解决后合并结果。

---

## 1. 递归基础

### 理解与可视化

```
递归 = 函数调用自己

每个递归都需要:
  1. 基本情况 (base case):    最小的问题，直接返回答案
  2. 递归步骤 (recursive case): 把问题缩小，调用自己

例: 计算 5! = 5 × 4 × 3 × 2 × 1

  factorial(5)
    → 5 × factorial(4)
        → 4 × factorial(3)
            → 3 × factorial(2)
                → 2 × factorial(1)
                    → 1  ← base case!
                ← 2 × 1 = 2
            ← 3 × 2 = 6
        ← 4 × 6 = 24
    ← 5 × 24 = 120

调用栈可视化:
  ┌─────────────────┐
  │ factorial(1) = 1│ ← 栈顶（最后调用，最先返回）
  ├─────────────────┤
  │ factorial(2)    │
  ├─────────────────┤
  │ factorial(3)    │
  ├─────────────────┤
  │ factorial(4)    │
  ├─────────────────┤
  │ factorial(5)    │ ← 栈底（最先调用，最后返回）
  └─────────────────┘
```

### 代码模板

```python
# 阶乘
def factorial(n):
    if n <= 1:      # base case
        return 1
    return n * factorial(n - 1)  # 递归

# 斐波那契（朴素递归，有重复计算）
def fib(n):
    if n <= 1:
        return n
    return fib(n - 1) + fib(n - 2)

# 斐波那契（带记忆化，避免重复计算）
from functools import cache

@cache
def fib_memo(n):
    if n <= 1:
        return n
    return fib_memo(n - 1) + fib_memo(n - 2)

# 幂运算 (快速幂)
def power(x, n):
    if n == 0:
        return 1
    if n < 0:
        return 1 / power(x, -n)
    half = power(x, n // 2)
    if n % 2 == 0:
        return half * half
    return half * half * x
```

## 2. 分治法

### 理解与可视化

```
分治 (Divide and Conquer):
  1. Divide:   把问题分成若干个子问题
  2. Conquer:  递归解决每个子问题
  3. Combine:  合并子问题的解

经典例子: 归并排序

  原始:    [38, 27, 43, 3, 9, 82, 10]

  分:      [38, 27, 43, 3]  |  [9, 82, 10]
           [38, 27] | [43, 3]  [9, 82] | [10]
           [38]|[27]  [43]|[3]  [9]|[82]  [10]

  治(排序): [27,38]  [3,43]   [9,82]   [10]

  合:       [3, 27, 38, 43]  |  [9, 10, 82]

  合:       [3, 9, 10, 27, 38, 43, 82]

分治模板:
  ┌──────────────────────────────┐
  │ def divide_conquer(problem): │
  │   if 足够小:                  │
  │       return 直接解决         │
  │   分割成子问题                │
  │   left = divide_conquer(左)  │
  │   right = divide_conquer(右) │
  │   return merge(left, right)  │
  └──────────────────────────────┘
```

### 代码模板

```python
# 归并排序
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    res = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            res.append(left[i])
            i += 1
        else:
            res.append(right[j])
            j += 1
    res.extend(left[i:])
    res.extend(right[j:])
    return res

# 快速排序
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    mid = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + mid + quick_sort(right)

# 求逆序对数量（归并排序变体）
def count_inversions(arr):
    if len(arr) <= 1:
        return arr, 0
    mid = len(arr) // 2
    left, cnt_l = count_inversions(arr[:mid])
    right, cnt_r = count_inversions(arr[mid:])
    merged, cnt_m = merge_count(left, right)
    return merged, cnt_l + cnt_r + cnt_m

def merge_count(left, right):
    res = []
    count = 0
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            res.append(left[i])
            i += 1
        else:
            res.append(right[j])
            count += len(left) - i  # 左边剩余的都比 right[j] 大
            j += 1
    res.extend(left[i:])
    res.extend(right[j:])
    return res, count
```

## 3. 经典分治应用

### 代码模板

```python
# 最大子数组和（分治法）
def max_subarray_dc(nums):
    def helper(lo, hi):
        if lo == hi:
            return nums[lo]
        mid = (lo + hi) // 2
        # 左半最大、右半最大
        left_max = helper(lo, mid)
        right_max = helper(mid + 1, hi)
        # 跨越中间的最大
        left_sum = cur = 0
        for i in range(mid, lo - 1, -1):
            cur += nums[i]
            left_sum = max(left_sum, cur)
        right_sum = cur = 0
        for i in range(mid + 1, hi + 1):
            cur += nums[i]
            right_sum = max(right_sum, cur)
        cross_max = left_sum + right_sum
        return max(left_max, right_max, cross_max)
    return helper(0, len(nums) - 1)

# 多数元素（投票法更优，此处展示分治思路）
def majority_element(nums):
    def helper(lo, hi):
        if lo == hi:
            return nums[lo]
        mid = (lo + hi) // 2
        left = helper(lo, mid)
        right = helper(mid + 1, hi)
        if left == right:
            return left
        left_count = sum(1 for i in range(lo, hi + 1) if nums[i] == left)
        right_count = sum(1 for i in range(lo, hi + 1) if nums[i] == right)
        return left if left_count > right_count else right
    return helper(0, len(nums) - 1)

# 第 K 大元素（快速选择，期望 O(n)）
import random

def find_kth_largest(nums, k):
    def quick_select(arr, k):
        pivot = random.choice(arr)
        big = [x for x in arr if x > pivot]
        equal = [x for x in arr if x == pivot]
        small = [x for x in arr if x < pivot]
        if k <= len(big):
            return quick_select(big, k)
        elif k <= len(big) + len(equal):
            return pivot
        else:
            return quick_select(small, k - len(big) - len(equal))
    return quick_select(nums, k)
```
