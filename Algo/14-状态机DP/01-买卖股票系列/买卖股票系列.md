# 买卖股票系列

> 状态机 DP 的代表题。用统一框架 dp[i][k][0/1] 解决全部 6 道股票题。

---

## 1. 统一框架

```
状态机:
  ┌──────────┐   买入    ┌──────────┐
  │  未持有   │ ────────▶ │   持有    │
  │ (可以买)  │ ◀──────── │ (可以卖)  │
  └──────────┘   卖出    └──────────┘

统一定义:
  dp[i][k][0] = 第 i 天结束, 最多还能交易 k 次, 未持股 的最大利润
  dp[i][k][1] = 第 i 天结束, 最多还能交易 k 次, 持股 的最大利润

转移:
  dp[i][k][0] = max(dp[i-1][k][0],            不操作
                    dp[i-1][k][1] + price[i])  卖出
  dp[i][k][1] = max(dp[i-1][k][1],            不操作
                    dp[i-1][k-1][0] - price[i]) 买入(消耗一次)

⚠️ "一次交易" = 买入+卖出。买入时消耗次数还是卖出时消耗
   只要统一, 结果一样。通常 买入时消耗。

各题对应:
  LC 121: k=1
  LC 122: k=∞ (不限次数)
  LC 309: k=∞ + 冷冻期
  LC 714: k=∞ + 手续费
  LC 123: k=2
  LC 188: k=任意
```

---

## 2. LC 121: 最多交易 1 次

```
k=1, 最简单的情况
  遍历时维护历史最低价, 每天算 "今天卖" 的利润

  min_price: 到目前为止的最低买入价
  max_profit: 到目前为止的最大利润
```

```python
def maxProfit(prices):
    min_price = float('inf')
    max_profit = 0
    for p in prices:
        min_price = min(min_price, p)           # ⚠️ 更新最低价
        max_profit = max(max_profit, p - min_price)  # ⚠️ 今天卖的利润
    return max_profit

# ⚠️ 不需要 dp 数组, 一遍扫描即可
# ⚠️ 初始 min_price = inf (保证第一天能更新)
# ⚠️ 初始 max_profit = 0 (不交易利润为 0)
```

## 3. LC 122: 不限交易次数

```
k = ∞, 只要有利润就做
  所有上涨段的差值之和

  prices = [7,1,5,3,6,4]
  利润 = (5-1) + (6-3) = 4 + 3 = 7
```

```python
# 贪心
def maxProfit(prices):
    profit = 0
    for i in range(1, len(prices)):
        if prices[i] > prices[i-1]:
            profit += prices[i] - prices[i-1]  # ⚠️ 每天看是否上涨
    return profit

# 状态机 DP
def maxProfit_dp(prices):
    hold = -prices[0]    # 持有
    cash = 0             # 未持有
    for i in range(1, len(prices)):
        new_hold = max(hold, cash - prices[i])   # 买入
        new_cash = max(cash, hold + prices[i])    # 卖出
        hold, cash = new_hold, new_cash
    return cash

# ⚠️ 贪心更简单, 但状态机 DP 是通用框架的基础
```

## 4. LC 309: 含冷冻期

```
卖出后, 下一天不能买入 (冷冻期 1 天)

三个状态:
  hold:     持有股票
  cash:     未持有, 可以买入 (非冷冻)
  cooldown: 未持有, 刚卖出 (冷冻, 不能买)

  ┌──────────┐
  │ cooldown │ → cash → hold → cooldown (循环)
  └──────────┘

转移:
  new_hold = max(hold, cash - price)    从 cash 买入 (不是 cooldown!)
  new_cash = max(cash, cooldown)        冷冻结束 → cash
  new_cooldown = hold + price           持有 → 卖出 → 冷冻
```

```python
def maxProfit(prices):
    hold = -prices[0]
    cash = 0
    cooldown = 0
    
    for i in range(1, len(prices)):
        new_hold = max(hold, cash - prices[i])    # ⚠️ 从 cash 买
        new_cash = max(cash, cooldown)             # ⚠️ 冷冻→非冷冻
        new_cooldown = hold + prices[i]            # ⚠️ 卖出→冷冻
        hold, cash, cooldown = new_hold, new_cash, new_cooldown
    
    return max(cash, cooldown)   # ⚠️ 最终不持股, 取两者 max

# ⚠️ 不能从 cooldown 买入! 只能从 cash 买入
# ⚠️ 最终答案 max(cash, cooldown), 不是只看 cash
# ⚠️ 必须用 new_ 临时变量, 不能直接覆盖!
#    因为三个状态的更新互相依赖同一天的旧值
```

## 5. LC 714: 含手续费

```
每次卖出需要手续费 fee
= 不限次数, 但利润要减去手续费
```

```python
def maxProfit(prices, fee):
    hold = -prices[0]
    cash = 0
    
    for i in range(1, len(prices)):
        new_hold = max(hold, cash - prices[i])
        new_cash = max(cash, hold + prices[i] - fee)  # ⚠️ 卖出扣手续费
        hold, cash = new_hold, new_cash
    
    return cash

# ⚠️ 手续费在 买入 或 卖出 时扣都可以, 只要统一
#    这里在卖出时扣: hold + prices[i] - fee
```

## 6. LC 188: 最多 k 次交易 (通用)

```
dp[j][0/1]: 最多完成 j 笔交易时的最大利润

⚠️ 优化: 当 k ≥ n//2 时, 等价于不限次数
   因为最多只能做 n//2 次完整交易 (买入+卖出各占一天)
```

```python
def maxProfit(k, prices):
    n = len(prices)
    if not prices:
        return 0
    
    # ⚠️ k 足够大时退化为无限次
    if k >= n // 2:
        return sum(max(prices[i+1]-prices[i], 0) for i in range(n-1))
    
    # dp[j][0] = 最多 j 笔, 未持股
    # dp[j][1] = 最多 j 笔, 持股
    dp = [[0, -float('inf')] for _ in range(k + 1)]
    
    for price in prices:
        for j in range(k, 0, -1):   # ⚠️ 倒序避免覆盖
            dp[j][0] = max(dp[j][0], dp[j][1] + price)    # 卖
            dp[j][1] = max(dp[j][1], dp[j-1][0] - price)  # 买
    
    return dp[k][0]

# ⚠️ dp[j][1] 初始化为 -inf (还没买入过, 利润为负无穷)
#    dp[j][0] 初始化为 0 (没交易, 利润 0)
```

## 7. LC 123: 最多 2 次交易

```
k=2 的特化版, 可以直接用 4 个变量
  buy1, sell1: 第 1 次买入/卖出的最优
  buy2, sell2: 第 2 次买入/卖出的最优
```

```python
def maxProfit(prices):
    buy1 = buy2 = -float('inf')
    sell1 = sell2 = 0
    
    for p in prices:
        buy1 = max(buy1, -p)             # ⚠️ 第1次买 (从0利润买入)
        sell1 = max(sell1, buy1 + p)      # 第1次卖
        buy2 = max(buy2, sell1 - p)       # ⚠️ 第2次买 (从sell1利润买入)
        sell2 = max(sell2, buy2 + p)      # 第2次卖
    
    return sell2

# ⚠️ 顺序: buy1 → sell1 → buy2 → sell2
#    buy2 依赖 sell1, 但因为同一行赋值，
#    即使 sell1 先更新了也没关系 (等价于更早卖出)
```
