# 最长公共子序列 (LCS)

> 两个序列的最长公共子序列，经典二维 DP。LCS 是编辑距离、序列比对等问题的核心。

---

## 1. 问题与可视化

```
问题: 给定两个字符串 s1, s2
     找它们的最长公共子序列 (子序列可以不连续)

示例:
  s1 = "abcde"
  s2 = "ace"
  LCS = "ace"，长度 = 3

  a b c d e
  ↕   ↕   ↕
  a   c   e

⚠️ 子序列 vs 子串:
   子序列: 保持相对顺序，可以跳过元素
   子串:   必须连续
```

---

## 2. 状态定义与转移

```
状态: dp[i][j] = s1 前 i 个字符 与 s2 前 j 个字符 的 LCS 长度

⚠️ 注意: dp[i][j] 中的 i 表示 s1[:i] (前 i 个字符)
   s1[i-1] 是第 i 个字符 (0 索引下标 i-1)

转移 (两种情况):
  ① s1[i-1] == s2[j-1]:  
     dp[i][j] = dp[i-1][j-1] + 1
     这个字符匹配了，LCS 长度 +1

  ② s1[i-1] != s2[j-1]:  
     dp[i][j] = max(dp[i-1][j], dp[i][j-1])
     不匹配，取"去掉 s1 末尾"或"去掉 s2 末尾"的较大值

Base case:
  dp[0][j] = 0  (s1 为空)
  dp[i][0] = 0  (s2 为空)

填表:
        ""  a  c  e
   ""    0  0  0  0     ← base case: 第 0 行全 0
   a     0  1  1  1     ← a==a → dp[0][0]+1=1
   b     0  1  1  1     ← b≠a,c,e → 取左或上的 max
   c     0  1  2  2     ← c==c → dp[1][1]+1=2
   d     0  1  2  2
   e     0  1  2  3     ← e==e → dp[3][2]+1=3 ✓

答案: dp[5][3] = 3
```

---

## 3. 代码模板

```python
def longestCommonSubsequence(text1, text2):
    m, n = len(text1), len(text2)
    # ⚠️ dp 大小 (m+1) × (n+1)，第 0 行/列是 base case
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:   # ⚠️ i-1, j-1 (0索引)
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    
    return dp[m][n]

# 时间 O(mn), 空间 O(mn)
# 可优化为 O(n) 空间 (滚动数组)

# ⚠️ 常见错误:
#   ① text1[i] 和 text2[j] → 应该是 text1[i-1] 和 text2[j-1]
#   ② dp 大小 m×n → 应该是 (m+1)×(n+1)
#   ③ 循环从 0 开始 → 应该从 1 开始
```

## 4. 空间优化

```python
def longestCommonSubsequence_opt(text1, text2):
    m, n = len(text1), len(text2)
    # ⚠️ 只保留一行 dp
    dp = [0] * (n + 1)
    
    for i in range(1, m + 1):
        prev = 0   # ⚠️ prev = dp[i-1][j-1] 的旧值
        for j in range(1, n + 1):
            temp = dp[j]   # 保存旧 dp[j] (= dp[i-1][j])
            if text1[i-1] == text2[j-1]:
                dp[j] = prev + 1
            else:
                dp[j] = max(dp[j], dp[j-1])
            prev = temp    # ⚠️ 更新 prev 为 dp[i-1][j]
    
    return dp[n]

# ⚠️ 为什么需要 prev?
#    dp[j] = dp[i-1][j-1] + 1 中
#    dp[j-1] 已经被更新为当前行的值
#    但需要的是上一行的 dp[j-1], 即 dp[i-1][j-1]
#    所以用 prev 保存旧值
```

## 5. 输出 LCS 字符串

```python
def getLCS(text1, text2):
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i-1] == text2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    
    # 回溯路径
    i, j = m, n
    result = []
    while i > 0 and j > 0:
        if text1[i-1] == text2[j-1]:
            result.append(text1[i-1])
            i -= 1
            j -= 1
        elif dp[i-1][j] >= dp[i][j-1]:
            i -= 1
        else:
            j -= 1
    
    return ''.join(reversed(result))

# ⚠️ 回溯需要完整 dp 表, 不能用空间优化版本
```

## 6. LeetCode 72: 编辑距离 ⭐

```
题目: word1 变成 word2 的最少操作数
     操作: 插入/删除/替换 一个字符

示例: word1="horse", word2="ros" → 3
     horse → rorse (替换) → ros (删除se 两次) → ros

状态: dp[i][j] = word1[:i] 变成 word2[:j] 的最少操作数

转移:
  word1[i-1] == word2[j-1]:
    dp[i][j] = dp[i-1][j-1]         (无需操作)
  
  word1[i-1] != word2[j-1]:
    dp[i][j] = min(
      dp[i-1][j] + 1,     # 删除 word1[i-1]
      dp[i][j-1] + 1,     # 插入 word2[j-1]
      dp[i-1][j-1] + 1    # 替换
    )

⚠️ Base case:
   dp[i][0] = i  (word1[:i] → "" 需要删 i 次)
   dp[0][j] = j  ("" → word2[:j] 需要插 j 次)
```

```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # ⚠️ base case: 第 0 行和第 0 列
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]   # 不操作
            else:
                dp[i][j] = min(
                    dp[i-1][j],      # 删除
                    dp[i][j-1],      # 插入
                    dp[i-1][j-1]     # 替换
                ) + 1
    
    return dp[m][n]

# 填表:
#      ""  r  o  s
# ""    0  1  2  3     ← dp[0][j] = j
# h     1  1  2  3
# o     2  2  1  2     ← o==o → dp[1][1] = 1
# r     3  2  2  2
# s     4  3  3  2     ← s==s → dp[3][2] = 2
# e     5  4  4  3     ← 答案
```

## 7. LeetCode 583: 两个字符串的删除操作

```
题目: 每步删除任一字符串中的一个字符，使两个字符串相同
     返回最少步数

转化: 保留的部分就是 LCS
     答案 = m + n - 2 * LCS长度

⚠️ 也可以直接 DP:
   dp[i][j] = word1[:i] 和 word2[:j] 变相同的最少删除次数
```

```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    lcs = longestCommonSubsequence(word1, word2)
    return m + n - 2 * lcs

# 或直接 DP:
def minDistance_dp(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        dp[i][0] = i       # 删除 word1 的前 i 个
    for j in range(n + 1):
        dp[0][j] = j       # 删除 word2 的前 j 个
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]   # 匹配, 不需要删
            else:
                dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + 1
    
    return dp[m][n]
```

## 8. LeetCode 1035: 不相交的线

```
题目: 两行数字, 相同数字之间连线, 线不能相交, 最多几条线?

示例:
  nums1 = [1,4,2]
  nums2 = [1,2,4]

  1---1
     X  (4-4 和 2-2 会交叉, 只能选一个)
  最多 2 条线

本质: 就是 LCS!
     不相交 → 保持相对顺序 → 子序列
```

```python
def maxUncrossedLines(nums1, nums2):
    # 完全等价于 LCS
    m, n = len(nums1), len(nums2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if nums1[i-1] == nums2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]
```

## 9. LCS vs 最长公共子数组

```
⚠️ 子序列 (LCS): 可以不连续
   子数组/子串:    必须连续

   LCS 转移:
     s1[i-1]==s2[j-1]: dp[i][j] = dp[i-1][j-1] + 1
     s1[i-1]!=s2[j-1]: dp[i][j] = max(dp[i-1][j], dp[i][j-1])

   最长公共子数组 (LC 718):
     s1[i-1]==s2[j-1]: dp[i][j] = dp[i-1][j-1] + 1
     s1[i-1]!=s2[j-1]: dp[i][j] = 0  ← 不匹配就断了
     答案 = max(所有 dp[i][j]), 不是 dp[m][n]!

⚠️ 子数组的答案不在 dp[m][n], 而是所有位置的最大值
```

## 本节要点速查

```
✅ dp[i][j] 表示 s1[:i] 和 s2[:j] 的 LCS
✅ 匹配: dp[i-1][j-1]+1; 不匹配: max(dp[i-1][j], dp[i][j-1])
✅ i 表示前 i 个字符，实际下标是 i-1
✅ base case: dp[0][j]=0, dp[i][0]=0
✅ 编辑距离: base case dp[i][0]=i, dp[0][j]=j
✅ 空间优化需要 prev 变量保存 dp[i-1][j-1]
✅ 子数组: 不匹配时 dp=0, 答案是全局 max
```
