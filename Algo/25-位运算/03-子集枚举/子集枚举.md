# 子集枚举

> 利用位运算高效枚举所有子集、固定大小的子集、某个集合的所有子集。

---

## 1. 枚举所有子集

```
n 个元素, 共 2^n 个子集

  mask 从 0 到 2^n - 1
  mask 的二进制表示 = 子集

  mask = 0b0000 → {}        (空集)
  mask = 0b0101 → {0, 2}
  mask = 0b1111 → {0,1,2,3} (全集)
```

```python
# LC 78: 子集
def subsets(nums):
    n = len(nums)
    result = []
    for mask in range(1 << n):             # ⚠️ 0 到 2^n - 1
        subset = []
        for i in range(n):
            if (mask >> i) & 1:            # ⚠️ 第 i 位为 1
                subset.append(nums[i])
        result.append(subset)
    return result

# ⚠️ 和回溯法等价, 但位运算更简洁
#    时间 O(n × 2^n), 空间 O(n × 2^n)
```

## 2. 枚举某个 mask 的所有子集

```
给定 mask, 枚举它的所有子集

  例: mask = 0b1010 的子集:
    1010, 1000, 0010, 0000

  技巧: sub = (sub - 1) & mask

  原理:
    sub - 1: 最低的 1 变 0, 更低的位变 1
    & mask:  只保留 mask 中有的位

    1010 → (1010-1)&1010 = 1001&1010 = 1000
    1000 → (1000-1)&1010 = 0111&1010 = 0010
    0010 → (0010-1)&1010 = 0001&1010 = 0000
    0000 → 退出循环
```

```python
def enumerate_subsets(mask):
    """枚举 mask 的所有非空子集"""
    subs = []
    sub = mask
    while sub:
        subs.append(sub)
        sub = (sub - 1) & mask             # ⚠️ 关键操作
    # subs 不含空集 0
    # 如果需要空集: subs.append(0)
    return subs

# ⚠️ 总时间: 枚举 n 位集合的所有子集的所有子集
#    Σ C(n,k) × 2^k = 3^n (二项式定理)
#    所以在状压DP中枚举子集的子集, 总复杂度 O(3^n)

# 使用示例: 状压DP中划分子集
# dp[mask] = min(dp[sub] + cost(mask ^ sub)) for sub ⊂ mask
def partition_dp(n, cost):
    dp = [float('inf')] * (1 << n)
    dp[0] = 0
    for mask in range(1, 1 << n):
        sub = mask
        while sub:
            dp[mask] = min(dp[mask], dp[mask ^ sub] + cost(sub))
            sub = (sub - 1) & mask         # ⚠️ 枚举 mask 的子集
    return dp[(1 << n) - 1]
```

## 3. 枚举大小为 k 的子集 (Gosper's Hack)

```
从 n 个元素中选 k 个, 按字典序枚举

  初始: sub = (1<<k) - 1    最小的 k 位子集
  终止: sub >= (1<<n)

  每步生成下一个:
    c = sub & (-sub)          最低的1
    r = sub + c               进位
    sub = (((r ^ sub) >> 2) // c) | r

  例: n=4, k=2
    0011 → 0101 → 0110 → 1001 → 1010 → 1100
```

```python
def gosper_hack(n, k):
    """枚举 n 个元素中选 k 个的所有子集"""
    sub = (1 << k) - 1                     # ⚠️ 初始: 最低 k 位全1
    result = []
    while sub < (1 << n):
        result.append(sub)
        c = sub & (-sub)                   # ⚠️ 最低的1
        r = sub + c                        # ⚠️ 进位
        sub = (((r ^ sub) >> 2) // c) | r  # ⚠️ 生成下一个
    return result

# ⚠️ 每步 O(1), 总共 C(n,k) 个子集
#    Gosper's Hack 比过滤更高效:
#    暴力: 枚举 2^n 个, 过滤 popcount==k 的
#    Gosper: 直接跳到下一个大小为 k 的
```

## 4. LeetCode 78: 子集

```python
# 位运算解法 (已在上面)
def subsets(nums):
    n = len(nums)
    return [
        [nums[i] for i in range(n) if (mask >> i) & 1]
        for mask in range(1 << n)
    ]

# 回溯解法 (对比)
def subsets_backtrack(nums):
    result = []
    def dfs(i, path):
        if i == len(nums):
            result.append(path[:])
            return
        dfs(i + 1, path)                   # 不选
        path.append(nums[i])
        dfs(i + 1, path)                   # 选
        path.pop()
    dfs(0, [])
    return result
```

## 5. LeetCode 2044: 统计按位或能得到最大值的子集数目

```
选若干元素, 按位 OR 尽可能大
求有多少种选法能达到最大

暴力状压: 枚举所有 2^n 子集, 计算OR, 统计最大值出现次数
n ≤ 16 → 2^16 = 65536, 可行
```

```python
def countMaxOrSubsets(nums):
    n = len(nums)
    max_or = 0
    count = 0
    
    for mask in range(1, 1 << n):          # ⚠️ 从1开始, 排除空集
        or_val = 0
        for i in range(n):
            if (mask >> i) & 1:
                or_val |= nums[i]
        if or_val > max_or:
            max_or = or_val
            count = 1
        elif or_val == max_or:
            count += 1
    
    return count

# ⚠️ 优化: 先算 max_or = OR(所有元素)
#    然后只统计 or_val == max_or 的子集数
```

## 6. 集合运算与子集判断

```python
# 集合运算
A | B    # 并集
A & B    # 交集
A ^ B    # 对称差 (在A或B但不在两者中)
A & ~B   # 差集 (A - B)
~A       # 补集

# 子集判断
def is_subset(sub, full):
    return sub & full == sub               # ⚠️ sub 是 full 的子集

# 集合大小 (popcount)
def set_size(mask):
    return bin(mask).count('1')

# 遍历集合中的所有元素
def iterate_elements(mask):
    elements = []
    while mask:
        i = (mask & -mask).bit_length() - 1  # ⚠️ 最低1的位置
        elements.append(i)
        mask &= mask - 1                     # ⚠️ 去掉最低1
    return elements

# ⚠️ Python 中 int.bit_length() 返回二进制位数
#    (mask & -mask).bit_length() - 1 = 最低1的位索引
```

## 本节要点速查

```
✅ 全子集枚举: mask 从 0 到 2^n-1
✅ mask的子集: sub = (sub-1) & mask, 总复杂度 O(3^n)
✅ 大小为k的子集: Gosper's Hack, O(1)每步
✅ 子集判断: sub & full == sub
✅ 集合大小: bin(mask).count('1')

```
