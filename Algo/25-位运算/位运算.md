# 位运算

> 位运算直接操作二进制位，速度极快。掌握常见位运算技巧可以在很多题目中实现巧妙的 O(1) 操作。

---

## 1. 位运算基础 ⭐

### 理解与可视化

```
基本运算:
  AND  &   1010 & 1100 = 1000    两位都1才1
  OR   |   1010 | 1100 = 1110    任一位1则1
  XOR  ^   1010 ^ 1100 = 0110    不同为1
  NOT  ~   ~1010 = ...0101       按位取反
  左移 <<  1010 << 1 = 10100     乘以 2
  右移 >>  1010 >> 1 = 0101      除以 2

常用技巧:
  x & (x-1)     去掉最低位的 1     1100 & 1011 = 1000
  x & (-x)      只保留最低位的 1    1100 & 0100 = 0100
  x ^ x = 0     自己异或自己 = 0
  x ^ 0 = x     异或 0 不变
  x | (1<<i)    将第 i 位置 1
  x & ~(1<<i)   将第 i 位置 0
  (x>>i) & 1    取第 i 位的值
```

### 代码模板

```python
# 位1的个数 (汉明重量)
def hamming_weight(n):
    count = 0
    while n:
        n &= n - 1  # 去掉最低位的 1
        count += 1
    return count

# 也可以用 bin(n).count('1')

# 2 的幂
def is_power_of_two(n):
    return n > 0 and n & (n - 1) == 0

# 汉明距离 (两个数不同的位数)
def hamming_distance(x, y):
    return bin(x ^ y).count('1')
```

## 2. 异或性质与经典题 ⭐⭐

### 理解与可视化

```
异或核心性质:
  a ^ a = 0       (相同抵消)
  a ^ 0 = a       (恒等)
  交换律 & 结合律

  应用:
    一组数中只有一个出现奇数次 → 全部异或, 剩下的就是答案
    两个不同的数 → 异或结果中找到一个不同位, 分组异或
```

### 代码模板

```python
# 只出现一次的数字（其余出现两次）
def single_number(nums):
    res = 0
    for x in nums:
        res ^= x
    return res

# 只出现一次的数字 III（两个数各出现一次，其余两次）
def single_number_iii(nums):
    xor_all = 0
    for x in nums:
        xor_all ^= x
    # 找到 xor_all 中任一为 1 的位（说明 a 和 b 在此位不同）
    diff_bit = xor_all & (-xor_all)
    a = b = 0
    for x in nums:
        if x & diff_bit:
            a ^= x
        else:
            b ^= x
    return [a, b]

# 只出现一次的数字 II（一个出现一次，其余出现三次）
def single_number_ii(nums):
    ones = twos = 0
    for x in nums:
        ones = (ones ^ x) & ~twos
        twos = (twos ^ x) & ~ones
    return ones

# 不用加号做加法
def get_sum(a, b):
    mask = 0xFFFFFFFF
    while b & mask:
        carry = (a & b) << 1
        a = a ^ b
        b = carry
    return a & mask if b > mask else a
```

## 3. 子集枚举

### 代码模板

```python
# 枚举 mask 的所有子集
def enumerate_subsets(mask):
    sub = mask
    while sub:
        # 处理子集 sub
        sub = (sub - 1) & mask
    # 别忘了空集 sub = 0

# 枚举大小为 k 的所有子集 (Gosper's hack)
def enumerate_k_subsets(n, k):
    sub = (1 << k) - 1
    while sub < (1 << n):
        # 处理子集 sub
        c = sub & (-sub)
        r = sub + c
        sub = (((r ^ sub) >> 2) // c) | r
```
