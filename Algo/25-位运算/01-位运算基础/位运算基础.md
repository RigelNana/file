# 位运算基础

> 位运算直接操作二进制位，速度极快，是很多巧妙 O(1) 技巧的基础。

---

## 1. 基本运算速查

```
运算名   符号   示例 (4位)                  含义
─────────────────────────────────────────────────
AND        &    1010 & 1100 = 1000    两位都1才1
OR         |    1010 | 1100 = 1110    任一位1则1
XOR        ^    1010 ^ 1100 = 0110    不同为1, 同为0
NOT        ~    ~1010 = ...0101       按位取反
左移      <<    1010 << 1 = 10100     ×2 (末尾补0)
右移      >>    1010 >> 1 = 0101      ÷2 (丢弃末位)

⚠️ Python 整数无溢出, 可以任意大
   但模拟32位时用 mask = 0xFFFFFFFF
```

## 2. 常用位操作技巧

```
操作                  表达式              示例
──────────────────────────────────────────────────
取第 i 位             (x >> i) & 1       x=1010, i=1 → 1
将第 i 位置 1         x | (1 << i)       x=1000, i=1 → 1010
将第 i 位置 0         x & ~(1 << i)      x=1010, i=1 → 1000
翻转第 i 位           x ^ (1 << i)       x=1010, i=1 → 1000
去掉最低位的1         x & (x - 1)        1100 → 1000
只保留最低位的1       x & (-x)           1100 → 0100  (lowbit)
判断是否2的幂         x & (x-1) == 0     1000 → True

⚠️ x & (x-1):
   x   = ...1000  (最低1在第3位)
   x-1 = ...0111  (第3位变0, 低位全变1)
   结果 = ...0000  (去掉了最低的1)

⚠️ x & (-x):
   -x  = ~x + 1   (补码)
   效果: 只保留最低的1, 其余全0
   这就是 lowbit, 树状数组的核心操作
```

## 3. 位1的个数 (汉明重量)

```python
# 方法1: Brian Kernighan 算法
def hammingWeight(n):
    count = 0
    while n:
        n &= n - 1                         # ⚠️ 每次去掉最低的1
        count += 1
    return count
    # O(k), k = 1的个数

# 方法2: Python 内置
def hammingWeight(n):
    return bin(n).count('1')

# ⚠️ bin(10) = '0b1010', 直接 count('1')
```

## 4. 2 的幂

```python
def isPowerOfTwo(n):
    return n > 0 and n & (n - 1) == 0
    # ⚠️ n > 0 不能省! 0 & (-1) = 0 但 0 不是2的幂
    # 2的幂: 二进制只有一个1 → 去掉后变0

# 4 的幂
def isPowerOfFour(n):
    return n > 0 and n & (n - 1) == 0 and n & 0xAAAAAAAA == 0
    # ⚠️ 4的幂: 那个1在偶数位 (0, 2, 4, ...)
    #    0xAAAAAAAA = 10101010...10 (奇数位全1)
    #    4的幂 & 0xAAAA == 0 → 确认1在偶数位
```

## 5. 汉明距离

```python
def hammingDistance(x, y):
    return bin(x ^ y).count('1')
    # ⚠️ x ^ y: 不同位变1 → 统计1的个数

# 总汉明距离 (所有数对的汉明距离之和)
# LC 477
def totalHammingDistance(nums):
    total = 0
    for i in range(32):
        ones = sum((x >> i) & 1 for x in nums)
        zeros = len(nums) - ones
        total += ones * zeros              # ⚠️ 每个1和每个0配对
    return total
    # O(32n) 而非 O(n²)
```

## 6. 颠倒二进制位

```python
def reverseBits(n):
    result = 0
    for _ in range(32):
        result = (result << 1) | (n & 1)   # ⚠️ 取末位, 放到result
        n >>= 1
    return result

# ⚠️ 循环恰好32次
#    每次取 n 的最低位, 左移放入 result
```

## 7. 缺失数字

```python
# LC 268: nums = [0,1,3], 缺失 2
def missingNumber(nums):
    n = len(nums)
    res = n                                # ⚠️ 初始化为 n
    for i, x in enumerate(nums):
        res ^= i ^ x                       # ⚠️ i 和 x 配对异或
    return res

# 原理: 0^1^2^3 ^ 0^1^3 = 2 (缺失的)
# i: 0,1,2,...,n-1 + 初始的 n → 包含 0 到 n
# x: nums 中的值
# 出现两次的抵消, 剩下的就是缺失的
```

## 本节要点速查

```
✅ x & (x-1): 去掉最低的1 → 汉明重量, 2的幂
✅ x & (-x): 保留最低的1 → lowbit, 树状数组
✅ x ^ y: 不同位变1 → 汉明距离
✅ 全部异或: 出现两次的抵消 → 找缺失/单一
✅ Python 无溢出, 32位模拟用 & 0xFFFFFFFF
✅ 颠倒: 循环32次, 取末位左移放入结果
```
