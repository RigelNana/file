## 3. 模板速查

### BFS (无权图)

```python
from collections import deque

def bfs_shortest(graph, start, end, n):
    dist = [-1] * n
    dist[start] = 0
    queue = deque([start])
    while queue:
        u = queue.popleft()
        if u == end:
            return dist[u]
        for v in graph[u]:
            if dist[v] == -1:
                dist[v] = dist[u] + 1
                queue.append(v)
    return -1
```

### Dijkstra (无负权)

```python
import heapq

def dijkstra(graph, start, n):
    dist = [float('inf')] * n
    dist[start] = 0
    heap = [(0, start)]
    while heap:
        d, u = heapq.heappop(heap)
        if d > dist[u]: continue       # ⚠️ 剪枝
        for v, w in graph[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(heap, (dist[v], v))
    return dist
```

### Bellman-Ford (有负权/限制步数)

```python
def bellman_ford(n, edges, start, max_steps=None):
    dist = [float('inf')] * n
    dist[start] = 0
    rounds = max_steps if max_steps else n - 1
    for _ in range(rounds):
        prev = dist[:]                  # ⚠️ 限制步数时必须拷贝
        for u, v, w in edges:
            if prev[u] + w < dist[v]:
                dist[v] = prev[u] + w
    return dist
```

### Floyd (多源/n小)

```python
def floyd(n, edges):
    INF = float('inf')
    dist = [[INF]*n for _ in range(n)]
    for i in range(n): dist[i][i] = 0
    for u, v, w in edges:
        dist[u][v] = min(dist[u][v], w)
    for k in range(n):                  # ⚠️ k在最外层
        for i in range(n):
            for j in range(n):
                if dist[i][k] + dist[k][j] < dist[i][j]:
                    dist[i][j] = dist[i][k] + dist[k][j]
    return dist
```

## 4. Dijkstra 变体总结

```
┌─────────────────────┬─────────────────────────────┐
│ 变体                │ 修改点                      │
├─────────────────────┼─────────────────────────────┤
│ 求最大概率          │ 初始0, 起点1, 取负, > 更新  │
│ 求路径最大差最小    │ new = max(dist[u], diff)    │
│ 求方案数            │ 加 cnt 数组, 等于时累加     │
│ 分层图              │ dist[node][state] 多维状态  │
│ 提前终止            │ 弹出终点直接返回            │
└─────────────────────┴─────────────────────────────┘
```

## 5. 常见错误清单

```
❌ Dijkstra 处理负权边 → 贪心失效
❌ Floyd 的 k 放在内层 → 答案错误
❌ Bellman-Ford 限制步数不用 prev → 多走了步
❌ 忘记处理 dist[u]==inf 的情况 → inf+w 溢出
❌ 1-indexed 节点用 0-indexed 的数组 → 越界
❌ 无向图只加了一个方向的边 → 漏路径
❌ Dijkstra 不加剪枝 d>dist[u] → TLE
❌ Floyd 忘记 dist[i][i]=0 → 自环错误
```

## 本节要点速查

```
✅ 无权/权1 → BFS O(V+E)
✅ 权0/1 → 0-1 BFS O(V+E)
✅ 无负权单源 → Dijkstra O(ElogV)
✅ 有负权 → Bellman-Ford O(VE)
✅ 限制步数 → BF k轮 + prev拷贝
✅ 多源 n≤400 → Floyd O(n³)
✅ k在最外层, dist[i][i]=0, 重边取min
✅ Dijkstra 剪枝 d>dist[u] 不可少
```
