# 最短路算法

> 最短路是图论最核心的问题之一。不同算法适用于不同场景：单源/多源、有无负权边。

---

## 1. Dijkstra 算法 ⭐⭐

### 理解与可视化

```
Dijkstra: 单源最短路（无负权边）

  核心思想: 贪心 ── 每次选距离最小的未确定节点，更新其邻居

  使用最小堆优化, 时间 O(E log V)

  例:
         1
    0 ──────▶ 1
    |         |
   4|         |2
    ▼         ▼
    2 ──────▶ 3
         3

  过程 (从 0 出发):
    初始: dist = [0, ∞, ∞, ∞]
    堆: [(0, 0)]

    弹出 (0, 0): 更新 1→1, 2→4
    堆: [(1, 1), (4, 2)]

    弹出 (1, 1): 更新 3→3
    堆: [(3, 3), (4, 2)]

    弹出 (3, 3): 无更新
    堆: [(4, 2)]

    弹出 (4, 2): 更新 3→7 (不优于3, 跳过)

    结果: dist = [0, 1, 4, 3]
```

### 代码模板

```python
import heapq
from collections import defaultdict

# Dijkstra (邻接表 + 最小堆)
def dijkstra(graph, start, n):
    dist = [float('inf')] * n
    dist[start] = 0
    heap = [(0, start)]
    while heap:
        d, u = heapq.heappop(heap)
        if d > dist[u]:     # 已找到更短路径, 跳过
            continue
        for v, w in graph[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                heapq.heappush(heap, (dist[v], v))
    return dist

# 使用示例: 网络延迟时间
def network_delay_time(times, n, k):
    graph = defaultdict(list)
    for u, v, w in times:
        graph[u].append((v, w))
    dist = dijkstra(graph, k, n + 1)
    ans = max(dist[1:])
    return ans if ans < float('inf') else -1
```

## 2. Bellman-Ford / SPFA

### 理解与可视化

```
Bellman-Ford: 单源最短路（允许负权边, 可检测负环）

  做 n-1 轮松弛（每轮遍历所有边）
  如果第 n 轮还能松弛 → 存在负环

  时间: O(VE)

SPFA: Bellman-Ford 的队列优化
  只有被更新过的节点才需要松弛邻居
  平均 O(E)，最坏 O(VE)
```

### 代码模板

```python
from collections import deque, defaultdict

# Bellman-Ford
def bellman_ford(n, edges, start):
    dist = [float('inf')] * n
    dist[start] = 0
    for _ in range(n - 1):
        for u, v, w in edges:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
    # 检测负环
    for u, v, w in edges:
        if dist[u] + w < dist[v]:
            return None  # 存在负环
    return dist

# SPFA (Bellman-Ford 队列优化)
def spfa(graph, start, n):
    dist = [float('inf')] * n
    dist[start] = 0
    in_queue = [False] * n
    count = [0] * n       # 入队次数，用于检测负环
    queue = deque([start])
    in_queue[start] = True
    while queue:
        u = queue.popleft()
        in_queue[u] = False
        for v, w in graph[u]:
            if dist[u] + w < dist[v]:
                dist[v] = dist[u] + w
                count[v] += 1
                if count[v] >= n:
                    return None  # 负环
                if not in_queue[v]:
                    queue.append(v)
                    in_queue[v] = True
    return dist
```

## 3. Floyd-Warshall 算法 ⭐

### 理解与可视化

```
Floyd: 多源最短路（求所有节点对之间的最短路）

  核心: 三层循环, 枚举中转点 k

  dp[i][j] = min(dp[i][j], dp[i][k] + dp[k][j])

  时间: O(n³)   空间: O(n²)

  适用: n 较小（≤400）的稠密图
```

### 代码模板

```python
# Floyd-Warshall
def floyd(n, edges):
    dist = [[float('inf')] * n for _ in range(n)]
    for i in range(n):
        dist[i][i] = 0
    for u, v, w in edges:
        dist[u][v] = min(dist[u][v], w)  # 处理重边

    for k in range(n):
        for i in range(n):
            for j in range(n):
                dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])
    return dist  # dist[i][j] = i→j 的最短距离
```

## 4. 算法选择总结

```
┌─────────────────────────────────────────────────────────┐
│ 算法          │ 适用场景           │ 时间复杂度         │
├─────────────────────────────────────────────────────────┤
│ BFS           │ 无权图单源最短路    │ O(V + E)          │
│ 0-1 BFS       │ 边权 0/1 的最短路   │ O(V + E)          │
│ Dijkstra      │ 非负权单源最短路    │ O(E log V)        │
│ Bellman-Ford  │ 允许负权(检负环)    │ O(VE)             │
│ SPFA          │ BF优化(平均更快)    │ O(E)~O(VE)        │
│ Floyd         │ 多源最短路          │ O(V³)             │
└─────────────────────────────────────────────────────────┘
```
