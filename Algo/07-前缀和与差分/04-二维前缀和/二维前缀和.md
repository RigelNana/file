# 二维前缀和

> 一维前缀和解决区间和，二维前缀和解决矩形区域和，核心是容斥原理。

---

## 1. 二维前缀和构建

```
给定 m×n 矩阵 mat, 构建前缀和矩阵 pre
pre[i][j] = mat[0..i-1][0..j-1] 的元素和

公式 (容斥原理):

  pre[i][j] = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + mat[i-1][j-1]

可视化:
  ┌─────────┬───────┐
  │         │       │
  │  A      │   B   │
  │         │       │
  ├─────────┼───────┤
  │  C      │ (i,j) │
  └─────────┴───────┘

  pre[i][j] = (A+B) + (A+C) - A + mat[i-1][j-1]
            = pre[i-1][j] + pre[i][j-1] - pre[i-1][j-1] + mat[i-1][j-1]
```

```python
def build_2d_prefix(mat):
    m, n = len(mat), len(mat[0])
    # ⚠️ 前缀和矩阵大小 (m+1) × (n+1), 多一行一列全 0 的哨兵
    pre = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            pre[i][j] = (pre[i-1][j] + pre[i][j-1] 
                        - pre[i-1][j-1] + mat[i-1][j-1])
    return pre

# ⚠️ pre 矩阵下标从 1 开始, mat 下标从 0 开始
#    pre[i][j] 对应 mat 的前 i 行、前 j 列
#    mat[i-1][j-1] 是原矩阵坐标
#
# ⚠️ 哨兵行列 (第0行第0列全0) 的好处:
#    不用特判 i=0 或 j=0 的边界情况
```

## 2. 矩形区域和查询

```
查询子矩阵 (r1,c1) 到 (r2,c2) 的元素和 (0-indexed)

公式 (容斥):
  sum = pre[r2+1][c2+1] - pre[r1][c2+1] - pre[r2+1][c1] + pre[r1][c1]

可视化:
  ┌──────┬────────┬──────┐
  │      │        │      │
  │  A   │   B    │      │
  │      │        │      │
  ├──────┼────────┤      │
  │  C   │ target │      │
  │      │        │      │
  ├──────┴────────┘      │
  │                      │
  └──────────────────────┘

  全部 = pre[r2+1][c2+1]       = A + B + C + target
  上方 = pre[r1][c2+1]         = A + B
  左方 = pre[r2+1][c1]         = A + C
  左上 = pre[r1][c1]           = A

  target = 全部 - 上方 - 左方 + 左上    (A被减了两次, 加回来)
```

```python
def query(pre, r1, c1, r2, c2):
    # ⚠️ r1,c1,r2,c2 是原矩阵的 0-indexed 坐标
    return (pre[r2+1][c2+1] - pre[r1][c2+1] 
            - pre[r2+1][c1] + pre[r1][c1])

# ⚠️ +1 的原因: pre 比 mat 多一行一列哨兵
#    pre[r2+1] 对应"包含第r2行"
#    pre[r1] 对应"不包含第r1行" (只到r1-1行)
#
# ⚠️ 常见错误:
#    写成 pre[r2][c2] - pre[r1-1][c2] - ...
#    当 r1=0 时 r1-1=-1 越界!
#    用 (m+1)×(n+1) 的哨兵可以避免
```

## 3. 题目实战

### LeetCode 304: 二维区域和检索

```python
class NumMatrix:
    def __init__(self, matrix):
        m, n = len(matrix), len(matrix[0])
        self.pre = [[0] * (n + 1) for _ in range(m + 1)]
        for i in range(1, m + 1):
            for j in range(1, n + 1):
                self.pre[i][j] = (self.pre[i-1][j] + self.pre[i][j-1]
                                 - self.pre[i-1][j-1] + matrix[i-1][j-1])
    
    def sumRegion(self, r1, c1, r2, c2):
        # ⚠️ 题目保证 r1<=r2, c1<=c2
        p = self.pre
        return p[r2+1][c2+1] - p[r1][c2+1] - p[r2+1][c1] + p[r1][c1]

# 预处理 O(mn), 每次查询 O(1)
```

### LeetCode 1314: 矩阵区域和

```
给定 m×n 矩阵 mat 和整数 k
answer[i][j] = 以(i,j)为中心, 大小为(2k+1)×(2k+1)窗口内的元素和

步骤:
1. 构建二维前缀和
2. 对每个 (i,j), 查询 [max(0,i-k), max(0,j-k)] 到 [min(m-1,i+k), min(n-1,j+k)]
```

```python
def matrixBlockSum(mat, k):
    m, n = len(mat), len(mat[0])
    pre = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            pre[i][j] = (pre[i-1][j] + pre[i][j-1]
                        - pre[i-1][j-1] + mat[i-1][j-1])
    
    ans = [[0] * n for _ in range(m)]
    for i in range(m):
        for j in range(n):
            r1 = max(0, i - k)
            c1 = max(0, j - k)
            r2 = min(m - 1, i + k)     # ⚠️ min(m-1, ...) 不是 min(m, ...)
            c2 = min(n - 1, j + k)
            ans[i][j] = (pre[r2+1][c2+1] - pre[r1][c2+1]
                        - pre[r2+1][c1] + pre[r1][c1])
    return ans

# ⚠️ 边界 clamp: max(0, ...) 和 min(m-1, ...)
#    确保不超出矩阵范围
```

## 4. 一维 vs 二维对比

```
┌────────────┬──────────────────┬──────────────────────────────┐
│            │ 一维             │ 二维                         │
├────────────┼──────────────────┼──────────────────────────────┤
│ 前缀和大小 │ n+1              │ (m+1)×(n+1)                  │
│ 构建公式   │ pre[i]=pre[i-1]  │ pre[i][j]=pre[i-1][j]        │
│            │   +a[i-1]        │   +pre[i][j-1]-pre[i-1][j-1] │
│            │                  │   +mat[i-1][j-1]              │
│ 查询公式   │ pre[r+1]-pre[l]  │ pre[r2+1][c2+1]-pre[r1][c2+1]│
│            │                  │  -pre[r2+1][c1]+pre[r1][c1]   │
│ 构建时间   │ O(n)             │ O(mn)                        │
│ 查询时间   │ O(1)             │ O(1)                         │
└────────────┴──────────────────┴──────────────────────────────┘

⚠️ 二维就是一维的推广, 关键是容斥原理
   一维: 减一次 (减左边)
   二维: 减两次加一次 (减上方、减左方、加左上角)
```

## 本节要点速查

```
✅ 构建: pre[i][j] = 上+左-左上+当前, 大小(m+1)×(n+1)
✅ 查询: 全部-上方-左方+左上 (容斥原理)
✅ 哨兵: 第0行第0列全0, 避免越界特判
✅ 坐标: pre下标从1开始, mat从0开始, 查询时+1
✅ 边界clamp: max(0,...)和min(m-1,...)/min(n-1,...)
```
