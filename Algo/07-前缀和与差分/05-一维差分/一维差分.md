# 一维差分

> 差分是前缀和的逆运算。前缀和 O(1) 查询、O(n) 修改；差分 O(1) 修改、O(n) 查询。

---

## 1. 差分的概念

```
原数组:   a = [a0,  a1,  a2,  a3,  ...]
差分数组: d = [a0, a1-a0, a2-a1, a3-a2, ...]

d[0] = a[0]
d[i] = a[i] - a[i-1]   (i ≥ 1)

性质: 对 d 做前缀和 = 还原 a
  a[i] = d[0] + d[1] + ... + d[i] = sum(d[:i+1])

前缀和 ↔ 差分 是互逆操作:
  a → 差分 → d → 前缀和 → a
```

---

## 2. 区间加法

```
将 a[l..r] 每个元素都 +val

暴力: O(n) 遍历 [l, r]
差分: O(1) 只修改两个位置!

操作:
  d[l]   += val   ← 从 l 开始, 后面所有元素 +val
  d[r+1] -= val   ← 从 r+1 开始, 抵消掉多加的

可视化:
  d[l] += 3          d[r+1] -= 3
    ↓                  ↓
  结果: ..., +3, +3, +3, +3, 0, 0, ...
              l   ...       r   r+1
```

```python
def range_add(d, l, r, val):
    d[l] += val
    if r + 1 < len(d):       # ⚠️ 防越界
        d[r + 1] -= val

# ⚠️ r+1 可能 == len(d), 此时不需要 -val
#    因为数组已经到头了, 不需要"抵消"
#
# ⚠️ 完整流程:
#    1. 初始化差分数组 d (全 0 或从原数组计算)
#    2. 执行若干次 range_add
#    3. 对 d 做前缀和, 还原出最终数组
```

---

## 3. 完整模板

```python
def difference_array(n, operations):
    """
    n: 数组长度
    operations: [(l, r, val), ...] 每次将 [l, r] 区间加 val
    返回最终数组
    """
    d = [0] * (n + 1)           # ⚠️ 多开一个, 避免 r+1 越界判断
    
    for l, r, val in operations:
        d[l] += val
        d[r + 1] -= val         # ⚠️ 因为多开了一个, 不用判断越界
    
    # 前缀和还原
    a = [0] * n
    a[0] = d[0]
    for i in range(1, n):
        a[i] = a[i - 1] + d[i]  # ⚠️ d 的前缀和 = 原数组
    
    return a

# ⚠️ d 的长度是 n+1 不是 n:
#    多出来的 d[n] 是"垃圾桶", 只用来接收 r+1 的减法
#    最终还原时只取 a[0..n-1]
```

## 4. 题目实战

### LeetCode 1109: 航班预订统计

```
n 个航班 (1-indexed), bookings[i] = [first, last, seats]
表示从 first 到 last 的每个航班预定了 seats 个座位
返回每个航班的总座位数

示例:
  bookings = [[1,2,10],[2,3,20],[2,5,25]]
  n = 5
  
  差分操作 (转0-indexed):
  d[0] += 10, d[2] -= 10   → [1,2] 区间 +10
  d[1] += 20, d[3] -= 20   → [2,3] 区间 +20  
  d[1] += 25, d[5] -= 25   → [2,5] 区间 +25

  d = [10, 45, -10, -20, 0, -25]
  前缀和: [10, 55, 45, 25, 25]
```

```python
def corpFlightBookings(bookings, n):
    d = [0] * (n + 1)
    for first, last, seats in bookings:
        d[first - 1] += seats          # ⚠️ 1-indexed → 0-indexed
        d[last] -= seats               # ⚠️ last-1+1 = last (0-indexed的r+1)
    
    ans = [0] * n
    ans[0] = d[0]
    for i in range(1, n):
        ans[i] = ans[i-1] + d[i]
    return ans

# ⚠️ 1-indexed 转换:
#    first (1-indexed) → first-1 (0-indexed) 是 l
#    last (1-indexed) → last-1 (0-indexed) 是 r
#    r+1 = last-1+1 = last
#    所以 d[first-1] += seats, d[last] -= seats
```

### LeetCode 1094: 拼车

```
capacity 座位的车, trips[i] = [numPassengers, from, to]
乘客在 from 上车, 在 to 下车 (到达 to 时已下车)
判断能否完成所有行程

关键: [from, to) 是左闭右开!
  乘客在 to 已经下车了

差分:
  d[from] += numPassengers
  d[to] -= numPassengers      ← 不是 d[to+1]!
```

```python
def carPooling(trips, capacity):
    d = [0] * 1001              # ⚠️ 站点最大1000
    for num, frm, to in trips:
        d[frm] += num
        d[to] -= num            # ⚠️ 左闭右开! 不用 to+1
    
    cur = 0
    for x in d:
        cur += x
        if cur > capacity:
            return False
    return True

# ⚠️ 左闭右闭 [l, r]: d[l] += val, d[r+1] -= val
#    左闭右开 [l, r): d[l] += val, d[r] -= val   ← 本题
#
# ⚠️ 判断过程: 不用真正还原整个数组
#    前缀和过程中 cur 就是当前位置的值, 直接判断即可
```

### LeetCode 370: 区间加法 (会员题)

```
长度 n 的全 0 数组, 执行 k 次操作 [startIndex, endIndex, inc]
将区间 [startIndex, endIndex] 上的每个元素加 inc

直接套差分模板:
```

```python
def getModifiedArray(length, updates):
    d = [0] * (length + 1)
    for l, r, inc in updates:
        d[l] += inc
        d[r + 1] -= inc         # ⚠️ 左闭右闭, 所以是 r+1
    
    ans = [0] * length
    ans[0] = d[0]
    for i in range(1, length):
        ans[i] = ans[i-1] + d[i]
    return ans
```

## 5. 前缀和 vs 差分

```
┌──────────┬──────────────────┬──────────────────┐
│          │ 前缀和           │ 差分             │
├──────────┼──────────────────┼──────────────────┤
│ 适用场景 │ 多次查询区间和   │ 多次修改区间     │
│ 预处理   │ O(n) 构建前缀和  │ O(n) 构建差分    │
│ 查询     │ O(1) 区间和      │ O(n) 前缀和还原  │
│ 修改     │ O(n) 暴力修改    │ O(1) 区间加      │
│ 关系     │ 差分的逆运算     │ 前缀和的逆运算   │
└──────────┴──────────────────┴──────────────────┘

⚠️ 多次查少次改 → 前缀和
   少次查多次改 → 差分
   先做完所有修改, 再查询 → 差分 (本质: 离线处理)
```

## 6. 左闭右闭 vs 左闭右开

```
⚠️ 这是差分最容易出错的地方!

左闭右闭 [l, r]:
  d[l] += val
  d[r+1] -= val

左闭右开 [l, r):
  d[l] += val
  d[r] -= val

判断方法: 看题意中 r 是否被包含
  "从 l 到 r" = 包含 r = 左闭右闭
  "从 l 上车, 在 r 下车" = 不包含 r = 左闭右开
```
