# 前缀和基础

> 前缀和：预处理一次 O(n)，此后 O(1) 查询任意区间和。子数组和问题的核心工具。

---

## 1. 什么是前缀和

### 理解与可视化

```
原数组:    nums = [2, 3, 1, 4, 5]
前缀和:    pre  = [0, 2, 5, 6, 10, 15]
                   ↑
                   ⚠️ pre[0] = 0 (哨兵)

pre[i] = nums[0] + nums[1] + ... + nums[i-1]

区间和 nums[l..r] = pre[r+1] - pre[l]

  例: nums[1..3] = 3 + 1 + 4 = 8
      pre[4] - pre[1] = 10 - 2 = 8 ✓

  ⚠️ 前缀和数组比原数组长 1 (多了 pre[0]=0)
  ⚠️ 区间 [l, r] 的和 = pre[r+1] - pre[l]
     不是 pre[r] - pre[l]!
```

### 代码模板

```python
# 构建前缀和
def build_prefix(nums):
    n = len(nums)
    pre = [0] * (n + 1)              # ⚠️ 长度 n+1
    for i in range(n):
        pre[i + 1] = pre[i] + nums[i]
    return pre

# 查询区间 [l, r] 的和
def query(pre, l, r):
    return pre[r + 1] - pre[l]       # ⚠️ r+1 不是 r

# ⚠️ 也可以用 itertools.accumulate:
from itertools import accumulate
pre = list(accumulate(nums, initial=0))  # 自带 pre[0]=0

# ⚠️ 下标关系:
#   pre[0] = 0
#   pre[1] = nums[0]
#   pre[i] = nums[0] + ... + nums[i-1]
#   sum(nums[l:r+1]) = pre[r+1] - pre[l]
```

## 2. 区域和检索 - 不可变

### LeetCode 303

```
NumArray(nums) → 初始化
sumRange(l, r) → 返回 nums[l] + ... + nums[r]
```

```python
class NumArray:
    def __init__(self, nums):
        self.pre = [0]
        for x in nums:
            self.pre.append(self.pre[-1] + x)
    
    def sumRange(self, left, right):
        return self.pre[right + 1] - self.pre[left]  # ⚠️ right+1

# ⚠️ 初始化 O(n), 每次查询 O(1)
# ⚠️ 如果不加 pre[0]=0:
#   left=0 时需要特殊处理
#   加了哨兵就统一了
```

## 3. 和为 K 的子数组

### LeetCode 560

### 理解与可视化

```
nums = [1, 1, 1], k = 2 → 输出 2

暴力: 枚举所有子数组 O(n²)
前缀和:
  pre = [0, 1, 2, 3]
  子数组 nums[l..r] 和 = k 等价于 pre[r+1] - pre[l] = k
  即 pre[l] = pre[r+1] - k

  对于每个 r, 查有多少个 l 使得 pre[l] = pre[r+1] - k
  → 枚举右端点, 哈希表查左端信息!
```

```python
from collections import defaultdict

def subarraySum(nums, k):
    cnt = defaultdict(int)
    cnt[0] = 1                        # ⚠️ 空前缀 pre[0]=0
    pre = 0
    ans = 0
    
    for x in nums:
        pre += x
        ans += cnt[pre - k]           # ⚠️ 有多少个前缀和 = pre-k
        cnt[pre] += 1                 # ⚠️ 记录当前前缀和
    
    return ans

# ⚠️ cnt[0] = 1 非常重要!
#   表示"空前缀"(什么都不选)的前缀和为 0
#   如果 pre 本身 = k, 那 pre - k = 0, cnt[0]=1 → 计数+1
#   如果忘记初始化 → 漏掉从下标0开始的子数组
#
# ⚠️ 先查后存: ans += cnt[pre-k] → cnt[pre] += 1
#    避免 l > r (子数组为空)
#
# ⚠️ 这题不能用滑窗! 因为有负数
#    有负数→和不单调→滑窗不适用
#    必须用前缀和+哈希表
```

## 4. 和可被 K 整除的子数组

### LeetCode 974

```
pre[r+1] - pre[l] ≡ 0 (mod k)
等价于 pre[r+1] ≡ pre[l] (mod k)

统计前缀和 mod k 相同的对数
```

```python
def subarraysDivByK(nums, k):
    cnt = defaultdict(int)
    cnt[0] = 1
    pre = 0
    ans = 0
    
    for x in nums:
        pre = (pre + x) % k          # ⚠️ 取模
        ans += cnt[pre]               # ⚠️ 同余的前缀和个数
        cnt[pre] += 1
    
    return ans

# ⚠️ Python 中 负数 % k 结果为非负 (和 C++/Java 不同)
#    Python: -1 % 3 = 2 ✓
#    C++:    -1 % 3 = -1 → 需要 ((pre%k)+k)%k 修正
#
# ⚠️ 和 LC560 的区别:
#    LC560: ans += cnt[pre - k]  (差恰好为 k)
#    LC974: ans += cnt[pre % k]  (差能整除 k → 余数相同)
```

## 5. 前缀和 vs 滑窗

```
┌──────────────────┬─────────────────┬──────────────────┐
│ 条件             │ 推荐方法         │ 原因              │
├──────────────────┼─────────────────┼──────────────────┤
│ 元素全正, 求最长  │ 滑窗            │ 和单调递增         │
│ 元素全正, 求最短  │ 滑窗            │ 和单调递增         │
│ 有负数, 求个数    │ 前缀和+哈希表    │ 和不单调           │
│ 有负数, 求和=k    │ 前缀和+哈希表    │ 和不单调           │
│ 静态区间查询      │ 前缀和          │ O(1) 查询         │
└──────────────────┴─────────────────┴──────────────────┘

⚠️ 最关键的区分: 有没有负数
  全正 → 滑窗 (和单调,收缩有效)
  有负 → 前缀和+哈希表 (滑窗不适用)
```

## 本节要点速查

```
✅ pre[0]=0 哨兵, pre[i]=nums[0]+...+nums[i-1]
✅ 区间[l,r]和 = pre[r+1] - pre[l]
✅ 和=k: cnt[0]=1, ans+=cnt[pre-k], 先查后存
✅ 整除k: 取模后同余即可
✅ 全正→滑窗, 有负→前缀和+哈希
✅ Python负数取模自动非负, C++需手动修正
```
