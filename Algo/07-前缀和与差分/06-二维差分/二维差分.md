# 二维差分

> 一维差分处理区间修改，二维差分处理矩形区域修改，核心还是容斥原理。

---

## 1. 二维差分的思想

```
原矩阵: a[m][n]
差分矩阵: d[m][n]

操作: 将子矩阵 (r1,c1) 到 (r2,c2) 的每个元素都 +val

暴力: O(mn) 遍历子矩阵
二维差分: O(1) 修改 4 个位置!

最终: 对 d 做二维前缀和 = 还原 a
```

---

## 2. 修改操作

```
将 (r1,c1) 到 (r2,c2) 区域 +val:

  d[r1][c1]     += val   ← 从(r1,c1)往右下全+val
  d[r1][c2+1]   -= val   ← 抵消右边多加的
  d[r2+1][c1]   -= val   ← 抵消下边多加的
  d[r2+1][c2+1] += val   ← 被减了两次, 加回来

可视化 (每个操作的影响范围):

  d[r1][c1] += val          影响: (r1,c1) 到右下角全部
  ┌─────────────────┐
  │ +val +val +val  │
  │ +val +val +val  │
  │ +val +val +val  │
  └─────────────────┘

  d[r1][c2+1] -= val       影响: 抵消右侧
  d[r2+1][c1] -= val       影响: 抵消下方
  d[r2+1][c2+1] += val     影响: 补回右下角

  最终效果:
  ┌──────┬────────┬──────┐
  │  0   │   0    │  0   │
  ├──────┼────────┼──────┤
  │  0   │ +val   │  0   │
  │  0   │ +val   │  0   │
  ├──────┼────────┼──────┤
  │  0   │   0    │  0   │
  └──────┴────────┴──────┘
         (r1,c1)→(r2,c2) 区域
```

## 3. 代码模板

```python
def diff_2d(m, n, operations):
    """
    m, n: 矩阵大小
    operations: [(r1, c1, r2, c2, val), ...]
    返回最终矩阵
    """
    # ⚠️ 差分矩阵多开一行一列, 避免越界
    d = [[0] * (n + 2) for _ in range(m + 2)]
    
    for r1, c1, r2, c2, val in operations:
        d[r1][c1]         += val
        d[r1][c2 + 1]     -= val     # ⚠️ 容斥: 减右边
        d[r2 + 1][c1]     -= val     # ⚠️ 容斥: 减下边
        d[r2 + 1][c2 + 1] += val     # ⚠️ 容斥: 加右下角
    
    # 二维前缀和还原
    a = [[0] * n for _ in range(m)]
    for i in range(m):
        for j in range(n):
            # ⚠️ 先做前缀和, 再赋值
            d[i][j] += (d[i-1][j] if i > 0 else 0) \
                     + (d[i][j-1] if j > 0 else 0) \
                     - (d[i-1][j-1] if i > 0 and j > 0 else 0)
            a[i][j] = d[i][j]
    
    return a

# ⚠️ 坐标系统: 上面模板使用 0-indexed
#    如果题目是 1-indexed, 直接用即可 (因为多开了2行2列)
#
# ⚠️ 多开 2 行 2 列 (不只是 1):
#    差分需要 r2+1 和 c2+1, 如果 r2=m-1 则 r2+1=m
#    前缀和还原又需要 i-1 和 j-1
#    所以 (m+2)×(n+2) 最安全
```

## 4. 简化的模板 (推荐)

```python
def diff_2d_simple(m, n, operations):
    """使用哨兵简化, 0-indexed"""
    # ⚠️ 差分矩阵 (m+1)×(n+1), r2+1 和 c2+1 最大到 m 和 n
    d = [[0] * (n + 1) for _ in range(m + 1)]
    
    for r1, c1, r2, c2, val in operations:
        d[r1][c1]         += val
        d[r1][c2 + 1]     -= val
        d[r2 + 1][c1]     -= val
        d[r2 + 1][c2 + 1] += val
    
    # 原地前缀和还原 (只在 d 上操作)
    for i in range(m):
        for j in range(n):
            if i > 0: d[i][j] += d[i-1][j]
            if j > 0: d[i][j] += d[i][j-1]
            if i > 0 and j > 0: d[i][j] -= d[i-1][j-1]
    
    return [row[:n] for row in d[:m]]   # ⚠️ 只取前 m×n 部分

# ⚠️ 和二维前缀和的公式完全一样:
#    d[i][j] += d[i-1][j] + d[i][j-1] - d[i-1][j-1]
#    差分 → 前缀和还原 = "正向建前缀和"
```

## 5. 题目实战

### LeetCode 2536: 子矩阵元素加 1

```
给定 n×n 全 0 矩阵和 queries[i] = [r1, c1, r2, c2]
对每个 query 将子矩阵 (r1,c1)→(r2,c2) 的元素 +1
返回最终矩阵

直接套二维差分模板:
```

```python
def rangeAddQueries(n, queries):
    d = [[0] * (n + 1) for _ in range(n + 1)]
    
    for r1, c1, r2, c2 in queries:
        d[r1][c1]         += 1
        d[r1][c2 + 1]     -= 1
        d[r2 + 1][c1]     -= 1
        d[r2 + 1][c2 + 1] += 1
    
    # 前缀和还原
    for i in range(n):
        for j in range(n):
            if i > 0: d[i][j] += d[i-1][j]
            if j > 0: d[i][j] += d[i][j-1]
            if i > 0 and j > 0: d[i][j] -= d[i-1][j-1]
    
    return [row[:n] for row in d[:n]]

# ⚠️ 时间复杂度:
#    暴力: O(q * n²)
#    二维差分: O(q + n²) ← 每次修改 O(1), 最后还原 O(n²)
```

### 实际执行过程

```
queries = [[1,1,2,2], [0,0,1,1]]
n = 3

  d[1][1] += 1        d = [[0,0,0,0],
  d[1][3] -= 1             [0,1,0,-1],
  d[3][1] -= 1             [0,0,0,0],
  d[3][3] += 1             [0,-1,0,1]]

第二个 query [0,0,1,1]:
  d[0][0] += 1        d = [[1,0,-1,0],

前缀和还原:
  i=0: d = [[1,1,0,0], ...]

```

## 6. 一维差分 vs 二维差分

```
┌────────────┬──────────────────┬──────────────────────────────┐
│            │ 一维差分         │ 二维差分                     │
├────────────┼──────────────────┼──────────────────────────────┤
│ 操作       │ d[l]+=v          │ d[r1][c1]+=v                 │
│            │ d[r+1]-=v        │ d[r1][c2+1]-=v               │
│            │                  │ d[r2+1][c1]-=v               │
│            │                  │ d[r2+1][c2+1]+=v             │
│ 修改点数   │ 2 个             │ 4 个                         │
│ 还原       │ 一维前缀和       │ 二维前缀和                   │
│ 修改复杂度 │ O(1)             │ O(1)                         │
│ 还原复杂度 │ O(n)             │ O(mn)                        │
│ 容斥       │ 加起点减终点     │ 加左上减右上减左下加右下     │
└────────────┴──────────────────┴──────────────────────────────┘

⚠️ 记忆: 一维 2 个点符号 +-
         二维 4 个点符号 + - - + (对角线同号)
```

## 7. 常见坑

```
⚠️ 1. 数组大小:
   差分矩阵至少 (m+1)×(n+1)
   因为 r2+1 最大 = m, c2+1 最大 = n

⚠️ 2. 还原顺序:
   必须从 (0,0) 到 (m-1,n-1) 按行按列顺序
   不能倒序或乱序 (前缀和依赖左边和上边)

⚠️ 3. 同号规则:
   (r1,c1) 和 (r2+1,c2+1) 同号 (+val)
   (r1,c2+1) 和 (r2+1,c1) 同号 (-val)
   对角线上的两个点符号相同

⚠️ 4. 与二维前缀和的关系:
   前缀和: 查询 O(1), 修改 O(mn)
   差分:   修改 O(1), 查询需要 O(mn) 还原
   互为逆运算
```

```
✅ 修改: 4个点 d[r1][c1]+=v, d[r1][c2+1]-=v, d[r2+1][c1]-=v, d[r2+1][c2+1]+=v

✅ 符号: 对角线同号 (左上右下+, 右上左下-)
✅ 大小: 差分矩阵至少 (m+1)×(n+1)
✅ 差分是前缀和的逆运算
```
