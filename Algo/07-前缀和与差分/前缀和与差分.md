# 前缀和与差分

> 前缀和用于快速求区间和，差分用于快速给区间加值。两者互为逆运算，是数组类题目的核心技巧。

---

## 1. 枚举技巧

### 理解与可视化

```
枚举右，维护左:
  遍历数组时，当前位置作为"右端点"，维护左边的信息

枚举中间:
  固定中间元素，分别向两侧找答案

这些技巧和前缀和/哈希表配合使用效果最好
```

### 代码模板

```python
# 两数之和（枚举右，哈希表维护左）
def two_sum(nums, target):
    seen = {}
    for i, x in enumerate(nums):
        if target - x in seen:
            return [seen[target - x], i]
        seen[x] = i
    return []
```

## 2. 前缀和基础 ⭐

### 理解与可视化

```
问题: 多次查询数组某区间 [l, r] 的和

暴力: 每次 O(n) 遍历区间求和
前缀和: 预处理 O(n)，每次查询 O(1)

原数组:      [2, 1, 5, 3, 4]
             idx: 0  1  2  3  4

前缀和数组:   [0, 2, 3, 8, 11, 15]
             idx: 0  1  2  3  4   5

  pre[i] = nums[0] + nums[1] + ... + nums[i-1]
  pre[0] = 0（空前缀）

  区间和 sum(l, r) = pre[r+1] - pre[l]

  例: sum(1,3) = nums[1]+nums[2]+nums[3] = 1+5+3 = 9
      = pre[4] - pre[1] = 11 - 2 = 9 ✓

可视化:
  pre:  0   2   3   8   11  15
        ├───┼───┼───┼───┼───┤
        0   1   2   3   4   5

  sum(1,3) = pre[4] - pre[1]
             ├───────────┤
             └── 这段就是 nums[1]+nums[2]+nums[3]
```

### 代码模板

```python
# 前缀和模板
def prefix_sum(nums):
    n = len(nums)
    pre = [0] * (n + 1)
    for i in range(n):
        pre[i + 1] = pre[i] + nums[i]
    return pre

# 查询区间 [l, r] 的和
def range_sum(pre, l, r):
    return pre[r + 1] - pre[l]

# Python 简洁写法
from itertools import accumulate

def prefix_sum_v2(nums):
    return [0] + list(accumulate(nums))
```

## 3. 前缀和与哈希表 ⭐

### 理解与可视化

```
问题: 找和为 k 的子数组个数

暴力: O(n²) 枚举所有子数组
前缀和+哈希表: O(n)

关键等式:
  sum(i, j) = pre[j+1] - pre[i] = k
  → pre[i] = pre[j+1] - k

  遍历到 j 时，查找之前有多少个 pre[i] 等于 pre[j+1]-k

可视化:
  nums:  [1, 1, 1], k=2

  前缀和:  0  1  2  3
                  ↑  当 pre=2 时，找 pre-k=0，哈希表中有 1 个 → 答案+1
                     当 pre=3 时，找 pre-k=1，哈希表中有 1 个 → 答案+1
  答案: 2（子数组 [1,1] 出现两次）
```

### 代码模板

```python
# 和为 k 的子数组个数（⭐ 高频）
from collections import defaultdict

def subarray_sum(nums, k):
    count = 0
    pre = 0
    seen = defaultdict(int)
    seen[0] = 1  # 空前缀
    for x in nums:
        pre += x
        count += seen[pre - k]
        seen[pre] += 1
    return count

# 和可被 k 整除的子数组个数
def subarrays_div_by_k(nums, k):
    count = 0
    pre = 0
    seen = defaultdict(int)
    seen[0] = 1
    for x in nums:
        pre = (pre + x) % k
        count += seen[pre]
        seen[pre] += 1
    return count

# 连续数组（0和1个数相等的最长子数组，把0看成-1）
def find_max_length(nums):
    pre = 0
    first_seen = {0: -1}
    ans = 0
    for i, x in enumerate(nums):
        pre += 1 if x == 1 else -1
        if pre in first_seen:
            ans = max(ans, i - first_seen[pre])
        else:
            first_seen[pre] = i
    return ans
```

## 4. 二维前缀和

### 理解与可视化

```
二维前缀和: 快速求矩阵中任意子矩阵的元素和

  矩阵:           二维前缀和:
  1 2 3            0  0  0  0
  4 5 6            0  1  3  6
  7 8 9            0  5  12 21
                   0  12 27 45

  pre[i][j] = 从 (0,0) 到 (i-1,j-1) 的矩形元素和

  求 (r1,c1) 到 (r2,c2) 的子矩阵和:
  = pre[r2+1][c2+1] - pre[r1][c2+1] - pre[r2+1][c1] + pre[r1][c1]

  容斥原理:
  ┌──────────┬──────┐
  │    A     │  B   │
  ├──────────┼──────┤   大矩形 - B - C + D(被减了两次)
  │    C     │目标D │
  └──────────┴──────┘
```

### 代码模板

```python
# 二维前缀和
def matrix_prefix_sum(matrix):
    m, n = len(matrix), len(matrix[0])
    pre = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m):
        for j in range(n):
            pre[i+1][j+1] = pre[i][j+1] + pre[i+1][j] - pre[i][j] + matrix[i][j]
    return pre

# 查询子矩阵 (r1,c1) 到 (r2,c2) 的和
def query(pre, r1, c1, r2, c2):
    return pre[r2+1][c2+1] - pre[r1][c2+1] - pre[r2+1][c1] + pre[r1][c1]
```

## 5. 一维差分 ⭐

### 理解与可视化

```
差分: 前缀和的逆运算
  用于快速对区间 [l, r] 整体加减一个值

问题: 对数组多次区间加操作后，求最终数组

暴力: 每次操作 O(n)
差分: 每次操作 O(1)，最后求前缀和 O(n)

差分数组: diff[i] = nums[i] - nums[i-1]

  对区间 [l, r] 加 val:
    diff[l] += val      ← 从 l 开始影响
    diff[r+1] -= val    ← 从 r+1 开始取消影响

  最后对 diff 求前缀和 → 得到最终数组

可视化:
  原数组:  [0, 0, 0, 0, 0]
  操作: 对 [1,3] 加 2

  差分:    [0, 2, 0, 0, -2]
               ↑           ↑
              +2          -2

  前缀和:  [0, 2, 2, 2, 0]  ← 最终数组 ✓
```

### 代码模板

```python
# 一维差分模板
def difference_array(n, operations):
    """
    n: 数组长度
    operations: [(l, r, val), ...] 对 [l,r] 加 val
    返回最终数组
    """
    diff = [0] * (n + 1)
    for l, r, val in operations:
        diff[l] += val
        diff[r + 1] -= val
    # 求前缀和得到最终数组
    res = [0] * n
    res[0] = diff[0]
    for i in range(1, n):
        res[i] = res[i-1] + diff[i]
    return res

# 航班预订统计
def corp_flight_bookings(bookings, n):
    diff = [0] * (n + 1)
    for first, last, seats in bookings:
        diff[first - 1] += seats
        diff[last] -= seats
    res = [0] * n
    res[0] = diff[0]
    for i in range(1, n):
        res[i] = res[i-1] + diff[i]
    return res

# 拼车（判断是否能完成所有行程）
def car_pooling(trips, capacity):
    diff = [0] * 1001
    for num, fr, to in trips:
        diff[fr] += num
        diff[to] -= num
    cur = 0
    for d in diff:
        cur += d
        if cur > capacity:
            return False
    return True
```

## 6. 二维差分

### 理解与可视化

```
二维差分: 对二维矩阵中的子矩阵整体加值

对 (r1,c1) 到 (r2,c2) 加 val:
  diff[r1][c1]     += val
  diff[r1][c2+1]   -= val
  diff[r2+1][c1]   -= val
  diff[r2+1][c2+1] += val

  和二维前缀和的容斥原理对应
```

### 代码模板

```python
# 二维差分
def diff_2d(m, n, operations):
    diff = [[0] * (n + 2) for _ in range(m + 2)]
    for r1, c1, r2, c2, val in operations:
        diff[r1][c1] += val
        diff[r1][c2+1] -= val
        diff[r2+1][c1] -= val
        diff[r2+1][c2+1] += val
    # 求二维前缀和
    for i in range(m):
        for j in range(n):
            diff[i][j] += (diff[i-1][j] if i > 0 else 0) + \
                          (diff[i][j-1] if j > 0 else 0) - \
                          (diff[i-1][j-1] if i > 0 and j > 0 else 0)
    return [row[:n] for row in diff[:m]]
```
