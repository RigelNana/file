# 单调栈与单调队列

> 单调栈/队列维护一个元素单调递增（或递减）的序列，高效解决「下一个更大/小元素」和「滑动窗口最值」问题。

---

## 1. 单调栈 ⭐⭐

### 理解与可视化

```
单调栈: 栈内元素始终保持单调递增（或递减）

典型问题: 找每个元素右边第一个更大的元素

  nums = [2, 1, 2, 4, 3]

  从右往左遍历, 维护单调递减栈:
  ┌──────────────────────────────────────┐
  │ i=4  num=3  栈: []      → push 3    │ 答案[4]=-1
  │ i=3  num=4  栈: [3]     → pop 3     │
  │                  []      → push 4    │ 答案[3]=-1
  │ i=2  num=2  栈: [4]     → 4>2,stop  │
  │                          → push 2   │ 答案[2]=4
  │ i=1  num=1  栈: [4,2]   → 2>1,stop  │
  │                          → push 1   │ 答案[1]=2
  │ i=0  num=2  栈: [4,2,1] → pop 1     │
  │              栈: [4,2]   → 2≤2,pop  │
  │              栈: [4]     → push 2   │ 答案[0]=4
  └──────────────────────────────────────┘
  结果: [4, 2, 4, -1, -1]

核心模式:
  从右往左: 找右边第一个更大/小
  从左往右: 找左边第一个更大/小
```

### 代码模板

```python
# 下一个更大元素（从右往左, 单调递减栈）
def next_greater(nums):
    n = len(nums)
    ans = [-1] * n
    stack = []
    for i in range(n - 1, -1, -1):
        while stack and stack[-1] <= nums[i]:
            stack.pop()
        if stack:
            ans[i] = stack[-1]
        stack.append(nums[i])
    return ans

# 从左往右写法（用下标更通用）
def next_greater_v2(nums):
    n = len(nums)
    ans = [-1] * n
    stack = []  # 存下标
    for i in range(n):
        while stack and nums[stack[-1]] < nums[i]:
            ans[stack.pop()] = nums[i]
        stack.append(i)
    return ans

# 每日温度（等几天到更暖的温度）
def daily_temperatures(temperatures):
    n = len(temperatures)
    ans = [0] * n
    stack = []
    for i in range(n):
        while stack and temperatures[stack[-1]] < temperatures[i]:
            j = stack.pop()
            ans[j] = i - j
        stack.append(i)
    return ans
```

## 2. 柱状图最大矩形 ⭐

### 理解与可视化

```
核心思路: 对每根柱子, 找左右两边第一个比它矮的柱子
→ 以这根柱子为高度的矩形宽度就确定了

  heights = [2, 1, 5, 6, 2, 3]

  对于 h=5 (i=2):
    左边第一个更矮: i=1 (h=1)
    右边第一个更矮: i=4 (h=2)
    宽度 = 4 - 1 - 1 = 2
    面积 = 5 * 2 = 10

  用单调递增栈一次遍历解决
```

### 代码模板

```python
# 柱状图中最大的矩形
def largest_rectangle_area(heights):
    stack = []
    max_area = 0
    heights.append(0)  # 哨兵
    for i, h in enumerate(heights):
        while stack and heights[stack[-1]] > h:
            height = heights[stack.pop()]
            width = i if not stack else i - stack[-1] - 1
            max_area = max(max_area, height * width)
        stack.append(i)
    heights.pop()  # 恢复
    return max_area

# 最大矩形 (85) —— 把每行看作柱状图
def maximal_rectangle(matrix):
    if not matrix:
        return 0
    cols = len(matrix[0])
    heights = [0] * cols
    max_area = 0
    for row in matrix:
        for j in range(cols):
            heights[j] = heights[j] + 1 if row[j] == '1' else 0
        max_area = max(max_area, largest_rectangle_area(heights[:]))
    return max_area
```

## 3. 单调队列 ⭐

### 理解与可视化

```
单调队列 (双端队列): 维护窗口内的最大/最小值

  滑动窗口最大值:
  nums = [1, 3, -1, -3, 5, 3, 6, 7], k = 3

  队列存下标, 保持对应值递减:

  窗口 [1, 3, -1]  队列: [1(3)]        最大=3
  窗口 [3, -1, -3] 队列: [1(3)]        最大=3
  窗口 [-1, -3, 5] 队列: [4(5)]        最大=5
  窗口 [-3, 5, 3]  队列: [4(5), 5(3)]  最大=5
  窗口 [5, 3, 6]   队列: [6(6)]        最大=6
  窗口 [3, 6, 7]   队列: [7(7)]        最大=7

  操作:
    1. 新元素入队前, 从尾部弹出所有 ≤ 新元素的（维护单调性）
    2. 队首超出窗口范围 → 弹出
    3. 队首就是当前窗口最大值
```

### 代码模板

```python
from collections import deque

# 滑动窗口最大值
def max_sliding_window(nums, k):
    dq = deque()  # 存下标, 对应值递减
    result = []
    for i, x in enumerate(nums):
        # 保持单调递减: 弹出 ≤ x 的
        while dq and nums[dq[-1]] <= x:
            dq.pop()
        dq.append(i)
        # 移除超出窗口的
        if dq[0] <= i - k:
            dq.popleft()
        # 窗口形成后记录
        if i >= k - 1:
            result.append(nums[dq[0]])
    return result

# 绝对差不超过限制的最长子数组
def longest_subarray(nums, limit):
    max_dq = deque()
    min_dq = deque()
    left = 0
    ans = 0
    for right, x in enumerate(nums):
        while max_dq and nums[max_dq[-1]] <= x:
            max_dq.pop()
        while min_dq and nums[min_dq[-1]] >= x:
            min_dq.pop()
        max_dq.append(right)
        min_dq.append(right)
        while nums[max_dq[0]] - nums[min_dq[0]] > limit:
            left += 1
            if max_dq[0] < left:
                max_dq.popleft()
            if min_dq[0] < left:
                min_dq.popleft()
        ans = max(ans, right - left + 1)
    return ans
```
