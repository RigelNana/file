# 单调队列

> 双端队列维护滑动窗口内的最值，O(1) 获取窗口最大/最小值。

---

## 1. 核心思想

```
单调队列 = 双端队列 (deque) + 单调性维护

  解决: 滑动窗口最大值/最小值

  与单调栈的区别:
    单调栈: 只从栈顶操作 (后进先出)
    单调队列: 两端都操作 (队尾维护单调性, 队首移除过期)

  可视化 (滑动窗口最大值, k=3):
  nums = [1, 3, -1, -3, 5, 3, 6, 7]

  ┌──────────────────────────────────────────────────┐
  │ 窗口         队列(下标→值)     最大值            │
  ├──────────────────────────────────────────────────┤
  │ i=0: [1]     [0→1]                              │
  │ i=1: [1,3]   [1→3]           3比1大,弹1          │
  │ i=2: [1,3,-1] [1→3, 2→-1]   窗口满 → max=3     │
  │ i=3: [3,-1,-3] [1→3, 3→-3]  1过期? 1≥3-3+1=1,不│
  │                               弹1➜[1→3, 3→-3]   │
  │               等等...实际:   队首1仍在窗口        │
  │                               → max=3            │
  │ i=4: [-1,-3,5] [4→5]        5最大,全弹→max=5    │
  │ i=5: [-3,5,3]  [4→5, 5→3]   → max=5            │
  │ i=6: [5,3,6]   [6→6]        6最大→max=6         │
  │ i=7: [3,6,7]   [7→7]        7最大→max=7         │
  └──────────────────────────────────────────────────┘
  结果: [3, 3, 5, 5, 6, 7]
```

## 2. LeetCode 239: 滑动窗口最大值

```python
from collections import deque

def maxSlidingWindow(nums, k):
    dq = deque()                       # ⚠️ 存下标, 对应值单调递减
    result = []
    
    for i, x in enumerate(nums):
        # 1. 保持单调递减: 弹出 ≤ x 的
        while dq and nums[dq[-1]] <= x:
            dq.pop()                   # ⚠️ 从尾部弹出
        
        dq.append(i)
        
        # 2. 移除超出窗口的队首
        if dq[0] <= i - k:            # ⚠️ 队首下标 ≤ i-k 说明过期
            dq.popleft()              # ⚠️ 从头部弹出
        
        # 3. 窗口形成后记录答案
        if i >= k - 1:                 # ⚠️ i ≥ k-1 时窗口已满
            result.append(nums[dq[0]]) # ⚠️ 队首 = 窗口最大值
    
    return result

# ⚠️ 三个操作:
#    尾部弹出: 维护单调递减 (新元素入队前)
#    头部弹出: 移除过期元素 (超出窗口)
#    队首取值: 窗口最大值
#
# ⚠️ 为什么存下标?
#    需要判断元素是否过期 (下标 ≤ i-k)
#    值无法判断过期
#
# ⚠️ <= x 弹出 vs < x 弹出:
#    ≤ x: 等于也弹出 → 后面的可以替代
#    < x: 等于保留 → 保留重复最大值
#    通常用 ≤ (去重), 不影响正确性
#
# ⚠️ 时间: O(n), 每个元素入队出队各一次
```

## 3. 滑动窗口最小值

```python
from collections import deque

def minSlidingWindow(nums, k):
    dq = deque()                       # ⚠️ 单调递增队列
    result = []
    
    for i, x in enumerate(nums):
        while dq and nums[dq[-1]] >= x:  # ⚠️ ≥ 弹出 (维护递增)
            dq.pop()
        dq.append(i)
        
        if dq[0] <= i - k:
            dq.popleft()
        
        if i >= k - 1:
            result.append(nums[dq[0]])
    
    return result

# ⚠️ 最大值 → 递减队列 → ≤ 弹出
#    最小值 → 递增队列 → ≥ 弹出
```

## 4. LeetCode 1438: 绝对差不超过限制的最长子数组

```
找最长子数组, 使得 max - min ≤ limit

需要同时维护窗口最大和最小值
→ 两个单调队列!
```

```python
from collections import deque

def longestSubarray(nums, limit):
    max_dq = deque()                   # ⚠️ 递减队列 → 最大值
    min_dq = deque()                   # ⚠️ 递增队列 → 最小值
    left = 0
    ans = 0
    
    for right, x in enumerate(nums):
        # 维护最大值队列
        while max_dq and nums[max_dq[-1]] <= x:
            max_dq.pop()
        max_dq.append(right)
        
        # 维护最小值队列
        while min_dq and nums[min_dq[-1]] >= x:
            min_dq.pop()
        min_dq.append(right)
        
        # 缩小窗口
        while nums[max_dq[0]] - nums[min_dq[0]] > limit:
            left += 1
            if max_dq[0] < left:       # ⚠️ < left (不是 <=)
                max_dq.popleft()
            if min_dq[0] < left:
                min_dq.popleft()
        
        ans = max(ans, right - left + 1)
    
    return ans

# ⚠️ 两个队列各自维护:
#    max_dq: 递减, 队首=窗口最大
#    min_dq: 递增, 队首=窗口最小
#
# ⚠️ 缩窗时判断 dq[0] < left:
#    不是 <= i-k, 而是 < left
#    因为这里不是固定窗口, left 在变
```

## 5. LeetCode 862: 和至少为K的最短子数组

```
找最短子数组, 使得子数组和 ≥ k
(可能有负数, 所以不能简单滑动窗口)

方法: 前缀和 + 单调队列
```

```python
from collections import deque

def shortestSubarray(nums, k):
    n = len(nums)
    prefix = [0] * (n + 1)
    for i in range(n):
        prefix[i+1] = prefix[i] + nums[i]
    
    dq = deque()                       # ⚠️ 存前缀和下标, 值递增
    ans = float('inf')
    
    for i in range(n + 1):
        # prefix[i] - prefix[dq[0]] ≥ k → 更新答案, 弹出队首
        while dq and prefix[i] - prefix[dq[0]] >= k:
            ans = min(ans, i - dq.popleft())
            # ⚠️ 弹出: 因为后续 i 更大, 差只会更长, 不是最短
        
        # 维护递增: 弹出尾部 ≥ prefix[i] 的
        while dq and prefix[dq[-1]] >= prefix[i]:
            dq.pop()
            # ⚠️ 弹出: prefix[dq[-1]] ≥ prefix[i]
            #    用 prefix[i] 做起点更好 (值更小, 差更大, 且下标更近)
        
        dq.append(i)
    
    return ans if ans != float('inf') else -1

# ⚠️ 前缀和单调队列:
#    队列存前缀和下标, 值递增
#    队首弹出: 找到满足条件的最短子数组
#    队尾弹出: 前缀和更大的没用 (减出来更小)
```

## 6. 总结对比

```
┌────────────────────┬──────────────┬─────────────────────┐
│                    │ 单调栈       │ 单调队列            │
├────────────────────┼──────────────┼─────────────────────┤
│ 数据结构           │ 栈           │ 双端队列            │
│ 操作端             │ 只有栈顶     │ 两端都操作          │
│ 过期处理           │ 无           │ 队首弹出过期元素    │
│ 典型问题           │ 下一个更大   │ 滑动窗口最值        │
│ 时间复杂度         │ O(n)         │ O(n)                │
└────────────────────┴──────────────┴─────────────────────┘

队列单调性选择:
  求最大值 → 递减队列 (队首最大)
  求最小值 → 递增队列 (队首最小)
```

## 本节要点速查

```
✅ 单调队列: deque, 尾部维护单调性, 头部移除过期
✅ 存下标: 判断过期 dq[0] <= i-k
✅ 最大值: 递减队列, ≤弹尾, 队首=最大
✅ 最小值: 递增队列, ≥弹尾, 队首=最小
✅ 两个队列: 同时维护最大最小
✅ 时间O(n): 每个元素入出队各一次
✅ 前缀和+单调队列: 处理含负数的子数组和问题
```
