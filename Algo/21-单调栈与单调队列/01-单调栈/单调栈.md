# 单调栈

> 维护栈内元素单调递增（或递减），高效解决「下一个更大/小元素」类问题。

---

## 1. 核心思想

```
单调栈: 栈内元素保持单调性

  当新元素要入栈时:
    弹出所有不满足单调性的元素
    然后入栈

  典型问题: 找每个元素右边第一个更大的元素

  两种写法:
    从右往左: 找右边答案, 栈维护"右边的候选"
    从左往右: 找右边答案, 栈维护"还没找到答案的下标"

  可视化 (从右往左, 单调递减栈):
  nums = [2, 1, 2, 4, 3]

  ┌──────────────────────────────────────────┐
  │ i=4  num=3  栈:[]        → push 3       │ ans[4]=-1
  │ i=3  num=4  栈:[3]       → pop 3        │
  │                           → push 4      │ ans[3]=-1
  │ i=2  num=2  栈:[4]       → 4>2, stop    │
  │                           → push 2      │ ans[2]=4
  │ i=1  num=1  栈:[4,2]     → 2>1, stop    │
  │                           → push 1      │ ans[1]=2
  │ i=0  num=2  栈:[4,2,1]   → pop 1       │
  │             栈:[4,2]      → 2≤2, pop    │
  │             栈:[4]        → push 2      │ ans[0]=4
  └──────────────────────────────────────────┘
  结果: [4, 2, 4, -1, -1]
```

---

## 2. 从右往左写法

```python
# 下一个更大元素 (从右往左, 栈存值)
def next_greater_right_to_left(nums):
    n = len(nums)
    ans = [-1] * n
    stack = []                         # ⚠️ 维护单调递减栈
    
    for i in range(n - 1, -1, -1):
        while stack and stack[-1] <= nums[i]:  # ⚠️ ≤ 弹出
            stack.pop()
        if stack:
            ans[i] = stack[-1]         # ⚠️ 栈顶就是右边第一个更大
        stack.append(nums[i])
    
    return ans

# ⚠️ while 条件: stack[-1] <= nums[i]
#    ≤ 表示"不够大, 不是答案, 弹掉"
#    如果改成 <: 找的是"≥"的元素, 不是"严格大于"
#
# ⚠️ 从右往左: 栈中是右边还没"过期"的候选
#    大于当前的留着 (可能是未来元素的答案)
#    小于等于当前的弹掉 (被当前元素"挡住"了)
```

## 3. 从左往右写法

```python
# 下一个更大元素 (从左往右, 栈存下标)
def next_greater_left_to_right(nums):
    n = len(nums)
    ans = [-1] * n
    stack = []                         # ⚠️ 存下标, 维护递减栈
    
    for i in range(n):
        while stack and nums[stack[-1]] < nums[i]:
            j = stack.pop()
            ans[j] = nums[i]           # ⚠️ nums[i] 是 j 的答案
        stack.append(i)
    
    return ans

# ⚠️ 从左往右: 栈中存"还没找到答案的下标"
#    当 nums[i] > 栈顶: 栈顶找到答案了 → pop 并赋值
#    最后栈中剩下的: 右边没有更大元素 → 保持 -1
#
# ⚠️ < vs <=:
#    < nums[i]: 严格小于 → 找"下一个严格更大"
#    <= nums[i]: 小于等于 → 找"下一个大于等于"
```

## 4. LeetCode 739: 每日温度

```
给定温度数组, 返回每天要等几天才能到更暖的温度
如果之后没有更暖的, 返回 0
```

```python
def dailyTemperatures(temperatures):
    n = len(temperatures)
    ans = [0] * n
    stack = []                         # ⚠️ 存下标
    
    for i in range(n):
        while stack and temperatures[stack[-1]] < temperatures[i]:
            j = stack.pop()
            ans[j] = i - j             # ⚠️ 等待天数 = 下标差
        stack.append(i)
    
    return ans

# ⚠️ ans[j] = i - j: 下标差 = 等待天数
#    存下标而非值, 因为需要计算距离
```

## 5. LeetCode 496 / 503: 下一个更大元素

```
496: 两个数组, nums1 是 nums2 的子集
     对 nums1 中每个元素, 找它在 nums2 中的下一个更大元素

503: 循环数组, 最后一个元素的下一个是第一个元素
```

```python
# 496: 用哈希表存 nums2 的结果
def nextGreaterElement(nums1, nums2):
    stack = []
    mapping = {}                       # ⚠️ num → 右边第一个更大
    
    for num in nums2:
        while stack and stack[-1] < num:
            mapping[stack.pop()] = num
        stack.append(num)
    
    return [mapping.get(num, -1) for num in nums1]

# 503: 循环数组 → 遍历两遍
def nextGreaterElements(nums):
    n = len(nums)
    ans = [-1] * n
    stack = []
    
    for i in range(2 * n - 1, -1, -1):     # ⚠️ 遍历 2n
        idx = i % n                         # ⚠️ 取模对应原始下标
        while stack and stack[-1] <= nums[idx]:
            stack.pop()
        if stack and i < n:                 # ⚠️ 只在第一轮记录答案
            ans[idx] = stack[-1]
        stack.append(nums[idx])
    
    return ans

# ⚠️ 循环数组技巧:
#    遍历 2n 个元素, 用 i%n 映射到原始下标
#    等价于把数组复制一份接在后面
#    只需在 i < n 时记录答案
```

## 6. 单调栈变体总结

```
┌──────────────────────┬──────────────┬───────────────┐
│ 问题                 │ 栈单调性     │ while 条件    │
├──────────────────────┼──────────────┼───────────────┤
│ 右边第一个更大       │ 递减栈       │ ≤ 弹出        │
│ 右边第一个更大或等于 │ 严格递减栈   │ < 弹出         │
│ 右边第一个更小       │ 递增栈       │ ≥ 弹出        │
│ 左边第一个更大       │ (从左往右)   │ 同上反向       │
└──────────────────────┴──────────────┴───────────────┘

⚠️ "递减栈" = 从栈底到栈顶递减
   新元素入栈前弹出所有 ≤ 它的 → 栈保持递减
   栈顶是"第一个更大"的候选
```

## 本节要点速查

```
✅ 从右往左: 栈存候选, 栈顶=答案, ≤弹出
✅ 从左往右: 栈存未解决下标, 遇到答案时pop赋值
✅ 存下标比存值更通用 (能算距离)
✅ ≤ vs <: 控制"严格大于"还是"大于等于"
✅ 循环数组: 遍历2n, i%n映射
✅ 时间O(n): 每个元素入栈出栈各一次
```
