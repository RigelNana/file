# 遍历链表

> 遍历是链表最基础的操作——从头到尾访问每个节点。所有链表算法都建立在遍历之上。

---

## 1. 基本遍历

### 理解与可视化

```
遍历就是用一个指针 cur 从 head 开始，每次走一步直到 None:

  head → [1] → [2] → [3] → [4] → None
  cur=1   ✓     处理 val=1
         cur=2  ✓     处理 val=2
                cur=3  ✓     处理 val=3
                       cur=4  ✓     处理 val=4
                              cur=None  ← 循环结束

⚠️ 关键: while cur（不是 while cur.next）
   while cur:          遍历所有节点, cur 最终为 None
   while cur.next:     遍历到倒数第二个, cur 停在最后一个节点

   什么时候用哪个?
     ● 只是读取/处理每个节点 → while cur
     ● 需要操作 cur.next（例如删除/插入）→ while cur.next
```

### 代码模板

```python
# ━━━━━━━━━━━ 遍历链表（打印所有值）━━━━━━━━━━━
def traverse(head):
    cur = head          # ⚠️ 不要写 cur = head.next（会漏掉头节点）
    while cur:          # ⚠️ 不要写 while cur.next（会漏掉最后一个）
        print(cur.val)  # 处理当前节点
        cur = cur.next  # 指针前进一步
    # 循环结束后 cur == None

# ⚠️ 易错:
#   ❌ cur = head.next   → 漏掉第一个节点
#   ❌ while cur.next    → 漏掉最后一个节点
#   ❌ 忘写 cur = cur.next → 死循环!
```

## 2. 计算链表长度

### 代码模板

```python
def get_length(head):
    length = 0
    cur = head
    while cur:
        length += 1      # ⚠️ 先计数再前进
        cur = cur.next
    return length

# ⚠️ 注意:
#   空链表 head=None → while 不进入 → return 0 ✓
#   单节点 head→None → length=1 ✓
```

## 3. 查找节点

### 理解与可视化

```
按值查找: 找到第一个 val == target 的节点

  head → [1] → [3] → [5] → [3] → None
  target = 3

  cur=1  val≠3, 继续
  cur=3  val==3, 找到! 返回这个节点

  如果 target=7:
  一直走到 None → 返回 None 表示"没找到"
```

### 代码模板

```python
# 按值查找——返回第一个匹配的节点
def find_by_value(head, target):
    cur = head
    while cur:
        if cur.val == target:
            return cur      # 找到就返回（第一个匹配）
        cur = cur.next
    return None             # 遍历完没找到

# 按索引查找——返回第 index 个节点（0-indexed）
def find_by_index(head, index):
    cur = head
    for i in range(index):
        if cur is None:
            return None     # 索引越界
        cur = cur.next
    return cur              # 可能是 None（index 刚好等于长度）

# ⚠️ 注意 0-indexed vs 1-indexed:
#   0-indexed: 第0个是 head, range(index) 走 index 步
#   1-indexed: 第1个是 head, range(index - 1) 走 index-1 步
#   LeetCode 大部分链表题不涉及索引, 按题意判断
```

## 4. 遍历并收集结果

### 代码模板

```python
# 链表转数组
def to_array(head):
    result = []
    cur = head
    while cur:
        result.append(cur.val)
        cur = cur.next
    return result

# 求链表最大值
def find_max(head):
    if not head:
        return None       # ⚠️ 空链表要特判
    max_val = head.val
    cur = head.next       # 这里可以从 head.next 开始，因为 head 已处理
    while cur:
        if cur.val > max_val:
            max_val = cur.val
        cur = cur.next
    return max_val

# 求链表倒数第 k 个节点的值
def find_kth_from_end(head, k):
    """
    ⚠️ k 是 1-indexed: 倒数第1个 = 最后一个节点
    思路: 先求长度 n, 倒数第 k 个 = 正数第 n-k+1 个 (1-indexed)
                                   = 正数第 n-k 个 (0-indexed, 走 n-k 步)
    """
    n = get_length(head)
    if k > n or k <= 0:
        return None       # ⚠️ 越界保护
    cur = head
    for _ in range(n - k):  # ⚠️ 走 n-k 步, 不是 n-k+1
        cur = cur.next
    return cur.val
```

## 5. 递归遍历

### 理解与可视化

```
递归遍历 = 把链表看成 "头节点 + 更短的链表":

  traverse(1→2→3→4→None):
    处理 1
    traverse(2→3→4→None):
      处理 2
      traverse(3→4→None):
        处理 3
        traverse(4→None):
          处理 4
          traverse(None):
            return  ← 递归终点

  前序遍历（先处理再递归）: 1, 2, 3, 4
  后序遍历（先递归再处理）: 4, 3, 2, 1  ← 自动逆序!
```

### 代码模板

```python
# 前序递归遍历 (从前到后处理)
def traverse_recursive(head):
    if not head:           # ⚠️ 递归终点: None
        return
    print(head.val)        # 先处理当前节点
    traverse_recursive(head.next)  # 再递归处理后面

# 后序递归遍历 (从后到前处理) —— 实现逆序打印
def traverse_reverse(head):
    if not head:
        return
    traverse_reverse(head.next)   # 先递归到底
    print(head.val)               # 回溯时处理 → 逆序!

# ⚠️ 递归深度:
#   Python 默认递归深度限制 ~1000
#   链表长度超过 1000 会 RecursionError
#   实际面试/比赛中, 长链表优先用迭代
#   若必须用递归: import sys; sys.setrecursionlimit(100000)
```

## 6. 同时遍历两个链表

### 理解与可视化

```
场景: 两个链表同时走, 比如合并两个有序链表

  l1: 1 → 3 → 5 → None
  l2: 2 → 4 → None

  方式1: 两个都有值时同时走
    while l1 and l2:   ← 较短的走完就停

  方式2: 任一有值就走
    while l1 or l2:    ← 较长的走完才停

  ⚠️ 区别:
    while l1 and l2:  处理重叠部分，剩余的需要额外处理
    while l1 or l2:   全部处理，但内部要判断 l1/l2 是否为 None
```

### 代码模板

```python
# 同时遍历（例: 逐位相加, LeetCode 2）
def add_two_numbers(l1, l2):
    dummy = ListNode()
    cur = dummy
    carry = 0                           # 进位
    while l1 or l2 or carry:            # ⚠️ 三个条件用 or
        val1 = l1.val if l1 else 0      # ⚠️ 检查 None
        val2 = l2.val if l2 else 0
        total = val1 + val2 + carry
        carry = total // 10             # ⚠️ 整除取进位
        cur.next = ListNode(total % 10) # ⚠️ 取余得当前位
        cur = cur.next
        if l1: l1 = l1.next             # ⚠️ 检查再前进
        if l2: l2 = l2.next
    return dummy.next

# ⚠️ 注意事项:
#   1. while 条件包含 carry → 最后一个进位不会丢失 (如 99+1=100)
#   2. l1/l2 到 None 后不再前进, 用 0 代替
#   3. % 10 取个位, // 10 取进位
```

## 7. 题目详解

### LeetCode 876. 链表的中间结点

```
题意: 返回链表中间节点。如果有两个中间节点, 返回第二个。

  [1,2,3,4,5]   → 返回 3 (中间)
  [1,2,3,4,5,6] → 返回 4 (偏右的中间)

方法: 快慢指针 (后面会专门讲, 这里先用遍历)
  1. 先求长度 n
  2. 走 n // 2 步
     n=5: n//2=2, 走2步 → 到第3个 ✓
     n=6: n//2=3, 走3步 → 到第4个 ✓
```

```python
# 方法1: 遍历两次
def middle_node(head):
    n = get_length(head)
    cur = head
    for _ in range(n // 2):    # ⚠️ 走 n//2 步, 不是 (n-1)//2
        cur = cur.next
    return cur

# 方法2: 快慢指针（一次遍历, 后续章节详解）
def middle_node_fast(head):
    slow = fast = head
    while fast and fast.next:  # ⚠️ 两个条件都要检查
        slow = slow.next
        fast = fast.next.next
    return slow
```

### LeetCode 160. 相交链表

```
题意: 两个链表在某个节点开始合并, 找到这个交叉点。

  a1 → a2 ↘
            c1 → c2 → c3 → None
  b1 → b2 → b3 ↗

方法: 双指针走完自己的走对方的, 最终在交点或 None 相遇
  指针 A: a1 → a2 → c1 → c2 → c3 → b1 → b2 → b3 → c1 ← 交点!
  指针 B: b1 → b2 → b3 → c1 → c2 → c3 → a1 → a2 → c1 ← 交点!
  
  ⚠️ 正确性: A 走了 a + c + b 步, B 走了 b + c + a 步 → 相同!
```

```python
def get_intersection_node(headA, headB):
    a, b = headA, headB
    while a != b:                    # ⚠️ 比较的是节点引用, 不是 val
        a = a.next if a else headB   # ⚠️ a 到 None 后跳到 headB
        b = b.next if b else headA   # ⚠️ b 到 None 后跳到 headA
    return a  # 交点 or None(不相交)

# ⚠️ 关键细节:
#   if a else headB  →  a 为 None 时跳到 B 的头
#   不是 if a.next else headB  →  那样会跳过 None，导致不相交时死循环!
```

## 本节要点速查

```
✅ while cur: 遍历全部节点（最后 cur=None）
✅ while cur.next: 停在最后一个节点（用于操作 cur.next）
✅ 0-indexed 走 i 步到第 i 个; 1-indexed 走 i-1 步
✅ 空链表 head=None 要特判
✅ 递归遍历有深度限制（~1000），长链表用迭代
✅ 同时遍历两链表：while l1 or l2, 内部检查 None
✅ 相交链表：走完自己走对方，步数 a+c+b == b+c+a
```
