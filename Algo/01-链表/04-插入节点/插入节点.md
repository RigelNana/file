# 插入节点

> 插入操作的核心：先连新节点的 next，再改前节点的 next。顺序反了就会断链。

---

## 1. 插入原理

### 理解与可视化

```
在节点 A 后面插入 new:

  之前: ... → [A] → [B] → ...

  正确顺序:
    step 1: new.next = A.next    先让新节点指向 B
    step 2: A.next = new         再让 A 指向新节点

  之后: ... → [A] → [new] → [B] → ...

  ❌ 错误顺序:
    step 1: A.next = new         A 指向新节点, 但 B 的引用丢了!
    step 2: new.next = A.next    A.next 现在是 new, new.next = new → 死循环!

  ⚠️ 黄金法则: 先连后断!
    先把新节点连上后面（new.next = ...）
    再把前面改指向新节点（prev.next = new）
```

## 2. 头部插入

### 代码模板

```python
# 在链表头部插入新节点
def insert_at_head(head, val):
    new_node = ListNode(val)
    new_node.next = head     # ⚠️ 先连: 新节点指向原头
    return new_node           # ⚠️ 新节点成为新的头

# ⚠️ 头部插入不需要 dummy
#   因为新节点就是新的 head
#   如果 head 是 None（空链表）也没问题: new_node.next = None ✓
```

## 3. 尾部插入

### 代码模板

```python
# 在链表尾部插入新节点
def insert_at_tail(head, val):
    new_node = ListNode(val)
    if not head:             # ⚠️ 空链表特判
        return new_node

    cur = head
    while cur.next:          # ⚠️ while cur.next 停在最后一个节点
        cur = cur.next       #    while cur 会走过头到 None
    cur.next = new_node      # 最后一个节点指向新节点
    return head

# ⚠️ 时间 O(n) —— 每次尾部插入都要从头走到尾
#   如果需要频繁尾插, 可以维护一个 tail 指针
```

```python
# 带尾指针的链表
class LinkedList:
    def __init__(self):
        self.head = None
        self.tail = None

    def append(self, val):
        new_node = ListNode(val)
        if not self.head:
            self.head = self.tail = new_node
        else:
            self.tail.next = new_node
            self.tail = new_node     # ⚠️ 更新尾指针
    # 尾插 O(1) ✓
```

## 4. 指定位置插入

### 代码模板

```python
# 在第 index 个位置插入（0-indexed, 插入后新节点是第 index 个）
def insert_at_index(head, index, val):
    dummy = ListNode(next=head)
    cur = dummy
    for i in range(index):          # ⚠️ 走 index 步到达插入位置的前一个
        if cur.next is None:
            return dummy.next       # ⚠️ index 越界, 不插入
        cur = cur.next
    
    new_node = ListNode(val)
    new_node.next = cur.next        # ⚠️ 先连
    cur.next = new_node             # ⚠️ 后断
    return dummy.next

# ⚠️ 边界分析:
#   index=0: cur=dummy, 插在 dummy 后面 = 新头 ✓
#   index=n: cur=最后一个节点, 插在末尾 ✓
#   index>n: 循环中 cur.next=None, 提前返回 ✓
```

## 5. 有序链表中插入（保持有序）

### 理解与可视化

```
  有序链表: [1] → [3] → [5] → [7] → None
  插入 val=4

  找到第一个 >= 4 的节点的前一个: cur=[3], cur.next=[5]
  
  new_node(4).next = [5]
  cur.next = new_node(4)
  
  结果: [1] → [3] → [4] → [5] → [7] → None ✓

⚠️ 边界:
  val=0 → 小于所有 → 插在头部 → 需要 dummy
  val=9 → 大于所有 → 插在尾部 → while 走到末尾自然处理
```

### 代码模板

```python
def insert_sorted(head, val):
    dummy = ListNode(next=head)
    cur = dummy
    
    # 找到第一个 cur.next.val >= val 的位置
    while cur.next and cur.next.val < val:   # ⚠️ 严格小于
        cur = cur.next
    
    new_node = ListNode(val, cur.next)       # 一行搞定: 创建并连 next
    cur.next = new_node
    return dummy.next

# ⚠️ 为什么是 < val 而不是 <= val?
#   < val:  相等时停下, 插在相等元素前面 → [3,4,4,5]
#   <= val: 相等时继续, 插在相等元素后面 → [3,4,4,5]
#   两种都可以, 视题意而定
#   如果要求稳定（保持原来相同值的顺序）, 用 <= (插在后面)
```

## 6. 题目详解

### LeetCode 707. 设计链表

```
题意: 实现一个链表类, 支持以下操作:
  get(index)         → 获取第 index 个节点的值
  addAtHead(val)     → 在头部插入
  addAtTail(val)     → 在尾部插入
  addAtIndex(index, val) → 在第 index 个节点前插入
  deleteAtIndex(index)   → 删除第 index 个节点

⚠️ index 是 0-indexed
```

```python
class MyLinkedList:
    def __init__(self):
        self.dummy = ListNode()     # 哨兵节点
        self.size = 0               # ⚠️ 维护长度, 方便边界检查

    def get(self, index):
        if index < 0 or index >= self.size:   # ⚠️ 越界检查
            return -1
        cur = self.dummy.next       # ⚠️ 从真正第0个开始
        for _ in range(index):
            cur = cur.next
        return cur.val

    def addAtHead(self, val):
        self.addAtIndex(0, val)     # 复用 addAtIndex

    def addAtTail(self, val):
        self.addAtIndex(self.size, val)   # 复用 addAtIndex

    def addAtIndex(self, index, val):
        if index > self.size:       # ⚠️ > size 不插入; == size 是尾部插入
            return
        if index < 0:
            index = 0               # ⚠️ 负数当作 0 处理
        cur = self.dummy
        for _ in range(index):      # ⚠️ 走 index 步到前驱
            cur = cur.next
        new_node = ListNode(val, cur.next)
        cur.next = new_node
        self.size += 1

    def deleteAtIndex(self, index):
        if index < 0 or index >= self.size:   # ⚠️ 越界不删
            return
        cur = self.dummy
        for _ in range(index):      # ⚠️ 走 index 步到前驱
            cur = cur.next
        cur.next = cur.next.next
        self.size -= 1

# ⚠️ 核心注意点:
#   1. dummy 永远存在, 简化头部操作
#   2. 维护 size 做越界检查
#   3. addAtIndex(size, val) = 尾部插入 → 所以条件是 > size 才不插入
#   4. 所有定位都是"走到前驱":
#      get:    从 dummy.next 走 index 步 → 到达目标节点
#      add:    从 dummy 走 index 步 → 到达前驱
#      delete: 从 dummy 走 index 步 → 到达前驱
```

### LeetCode 147. 对链表进行插入排序

```
题意: 用插入排序对链表排序

思路: 从第二个节点开始, 每次把当前节点插入到前面已排序部分的正确位置

  [4] → [2] → [1] → [3]
  
  第1步: [2] 插入到 [4] 前面 → [2] → [4]
  第2步: [1] 插入到 [2] 前面 → [1] → [2] → [4]
  第3步: [3] 插入到 [2] 和 [4] 之间 → [1] → [2] → [3] → [4]
```

```python
def insertion_sort_list(head):
    dummy = ListNode(next=head)
    last_sorted = head                    # 已排序部分的最后一个
    cur = head.next if head else None     # 待插入的节点

    while cur:
        if last_sorted.val <= cur.val:
            last_sorted = cur             # 已经有序, 直接前进
        else:
            # 从头找到插入位置
            prev = dummy
            while prev.next.val < cur.val:  # ⚠️ < 不是 <=, 保持稳定
                prev = prev.next
            # 把 cur 从原位置取出
            last_sorted.next = cur.next
            # 插入到 prev 后面
            cur.next = prev.next
            prev.next = cur
        cur = last_sorted.next            # ⚠️ 下一个待处理节点
    return dummy.next

# ⚠️ 时间 O(n²), 空间 O(1)
# ⚠️ 实际排序链表更推荐归并排序 O(n log n) → 后续章节
```

## 本节要点速查

```
✅ 插入顺序: 先连新节点的 next，再改前节点的 next（先连后断）
✅ 头部插入: new.next = head; return new
✅ 尾部插入: 走到最后 while cur.next → cur.next = new
✅ 中间插入: 走 index 步到前驱 → 标准两步操作
✅ 有序插入: while cur.next.val < val 找位置
✅ 维护 size 做越界检查（设计链表类时）
✅ addAtIndex(size, val) 是尾部插入 → index > size 才拒绝
```
