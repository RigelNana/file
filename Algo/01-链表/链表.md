# 链表

> 链表是最基础的数据结构之一，也是面试高频考点。掌握链表能帮你理解指针操作和递归思维。

---

## 1. 链表基础概念

### 理解与可视化

**什么是链表？**

想象一列火车：每节车厢（节点）里装着货物（数据），并且知道下一节车厢在哪（指针）。

```
单链表:
  ┌───────────┐    ┌───────────┐    ┌───────────┐    ┌───────────┐
  │ val: 1    │    │ val: 2    │    │ val: 3    │    │ val: 4    │
  │ next: ──────→  │ next: ──────→  │ next: ──────→  │ next: None│
  └───────────┘    └───────────┘    └───────────┘    └───────────┘
  head                                                tail

双链表:
  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
  │ prev: None  │    │ prev: ←───────── │ prev: ←──────── 
  │ val: 1      │    │ val: 2      │    │ val: 3      │
  │ next: ──────────→│ next: ──────────→│ next: None  │
  └─────────────┘    └─────────────┘    └─────────────┘

数组 vs 链表:
  数组:   [1][2][3][4][5]   ← 内存连续，随机访问 O(1)
  链表:   1→2→3→4→5        ← 内存不连续，访问 O(n)，插入删除 O(1)
```

**链表 vs 数组对比：**

| 操作 | 数组 | 链表 |
|------|------|------|
| 按索引访问 | O(1) | O(n) |
| 头部插入 | O(n) | O(1) |
| 尾部插入 | O(1)* | O(n) / O(1)** |
| 中间插入 | O(n) | O(1)*** |
| 删除 | O(n) | O(1)*** |

> *摊还 O(1)  **有尾指针时  ***已知位置时

### 代码模板

```python
# 链表节点定义
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

# 从列表创建链表（刷题辅助）
def build_list(arr):
    dummy = ListNode()
    cur = dummy
    for x in arr:
        cur.next = ListNode(x)
        cur = cur.next
    return dummy.next

# 打印链表（调试辅助）
def print_list(head):
    vals = []
    while head:
        vals.append(str(head.val))
        head = head.next
    print(" → ".join(vals))
```

## 2. 遍历链表

### 理解与可视化

遍历就是从头到尾，逐个访问每个节点。就像沿着火车逐节车厢走一遍。

```
遍历过程:
  head → [1] → [2] → [3] → [4] → None
  cur=1   ✓
         cur=2  ✓
                cur=3  ✓
                       cur=4  ✓
                              cur=None → 结束
```

### 代码模板

```python
# 遍历链表
def traverse(head):
    cur = head
    while cur:
        print(cur.val)  # 处理当前节点
        cur = cur.next

# 计算链表长度
def get_length(head):
    length = 0
    cur = head
    while cur:
        length += 1
        cur = cur.next
    return length

# 查找值为 target 的节点
def find(head, target):
    cur = head
    while cur:
        if cur.val == target:
            return cur
        cur = cur.next
    return None
```

## 3. 删除节点

### 理解与可视化

删除节点的核心：让前一个节点的 `next` 跳过要删除的节点。

```
删除值为 3 的节点:

  之前: [1] → [2] → [3] → [4] → None
                      ↑ 要删除

  之后: [1] → [2] ─────→ [4] → None
                    [3] ← 被跳过，自动被回收

关键操作:  prev.next = prev.next.next

哨兵节点技巧（dummy head）:
  当可能删除头节点时，在头部加一个虚拟节点:
  dummy → [1] → [2] → [3] → None
  这样头节点也和其他节点一样处理，最后返回 dummy.next
```

### 代码模板

```python
# 删除所有值为 val 的节点
def remove_elements(head, val):
    dummy = ListNode(next=head)  # 哨兵节点
    cur = dummy
    while cur.next:
        if cur.next.val == val:
            cur.next = cur.next.next  # 跳过
        else:
            cur = cur.next
    return dummy.next

# 删除第 n 个节点（1-indexed）
def remove_nth(head, n):
    dummy = ListNode(next=head)
    cur = dummy
    for _ in range(n - 1):
        cur = cur.next
    cur.next = cur.next.next
    return dummy.next

# 删除倒数第 n 个节点（快慢指针）
def remove_nth_from_end(head, n):
    dummy = ListNode(next=head)
    fast = slow = dummy
    for _ in range(n + 1):  # fast 先走 n+1 步
        fast = fast.next
    while fast:
        fast = fast.next
        slow = slow.next
    slow.next = slow.next.next  # 删除 slow 的下一个
    return dummy.next

# 删除排序链表中的重复元素（保留一个）
def delete_duplicates(head):
    cur = head
    while cur and cur.next:
        if cur.val == cur.next.val:
            cur.next = cur.next.next
        else:
            cur = cur.next
    return head
```

## 4. 插入节点

### 理解与可视化

```
在节点 2 后面插入 99:

  之前: [1] → [2] → [3] → None
  
  步骤:
  1. new_node.next = cur.next   // 新节点指向 3
  2. cur.next = new_node        // 节点 2 指向新节点

  之后: [1] → [2] → [99] → [3] → None

⚠️ 注意顺序！先连新节点的 next，再改前节点的 next
   如果反过来，会丢失后面的链表！
```

### 代码模板

```python
# 在链表头部插入
def insert_at_head(head, val):
    new_node = ListNode(val)
    new_node.next = head
    return new_node  # 新的头节点

# 在链表尾部插入
def insert_at_tail(head, val):
    new_node = ListNode(val)
    if not head:
        return new_node
    cur = head
    while cur.next:
        cur = cur.next
    cur.next = new_node
    return head

# 在排序链表中插入（保持有序）
def insert_sorted(head, val):
    dummy = ListNode(next=head)
    cur = dummy
    while cur.next and cur.next.val < val:
        cur = cur.next
    new_node = ListNode(val, cur.next)
    cur.next = new_node
    return dummy.next
```

## 5. 反转链表 ⭐

### 理解与可视化

反转链表是最高频的链表题，必须闭眼写出来。

```
原始: 1 → 2 → 3 → 4 → None

迭代法（逐个翻转指针方向）:

  初始:  prev=None  cur=1→2→3→4→None

  第1步: None ← 1   2→3→4→None
         prev=1  cur=2

  第2步: None ← 1 ← 2   3→4→None
                prev=2  cur=3

  第3步: None ← 1 ← 2 ← 3   4→None
                      prev=3  cur=4

  第4步: None ← 1 ← 2 ← 3 ← 4
                            prev=4  cur=None → 结束

  结果:  4 → 3 → 2 → 1 → None

每一步做三件事:
  1. 保存 next = cur.next     （先记住下一个）
  2. cur.next = prev           （翻转指针）
  3. prev, cur = cur, next     （前进一步）
```

### 代码模板

```python
# 迭代法反转（⭐ 必背）
def reverse_list(head):
    prev, cur = None, head
    while cur:
        next_node = cur.next  # 暂存下一个
        cur.next = prev       # 翻转指针
        prev = cur            # prev 前进
        cur = next_node       # cur 前进
    return prev               # prev 就是新头

# 递归法反转
def reverse_list_recursive(head):
    if not head or not head.next:
        return head
    new_head = reverse_list_recursive(head.next)
    head.next.next = head  # 后一个节点指回自己
    head.next = None       # 断开原来的指向
    return new_head

# 反转链表的前 n 个节点
def reverse_n(head, n):
    prev, cur = None, head
    for _ in range(n):
        next_node = cur.next
        cur.next = prev
        prev = cur
        cur = next_node
    head.next = cur  # 原头节点连上剩余部分
    return prev

# 反转区间 [left, right]（LeetCode 92）
def reverse_between(head, left, right):
    dummy = ListNode(next=head)
    prev = dummy
    for _ in range(left - 1):
        prev = prev.next
    cur = prev.next
    for _ in range(right - left):
        next_node = cur.next
        cur.next = next_node.next
        next_node.next = prev.next
        prev.next = next_node
    return dummy.next
```

## 6. 快慢指针 ⭐

### 理解与可视化

两个指针以不同速度移动。就像操场跑步，快的人每次跑两步，慢的人每次一步。

```
找中间节点（快指针到终点时，慢指针在中间）:

  slow: 一次走 1 步
  fast: 一次走 2 步

  1 → 2 → 3 → 4 → 5 → None
  s    
  f         

  1 → 2 → 3 → 4 → 5 → None
       s    
            f         

  1 → 2 → 3 → 4 → 5 → None
            s    
                      f → None → 停

  slow 指向 3，就是中间节点 ✓

判断链表有环:
  如果有环，快指针一定会追上慢指针（就像跑操场不停的两个人）

  1 → 2 → 3 → 4
              ↑   ↓
              6 ← 5   （3→4→5→6→3 形成环）

  快慢指针最终会在环内相遇
```

### 代码模板

```python
# 找链表中间节点（⭐ 必会）
def find_middle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    return slow  # 奇数个返回正中间，偶数个返回后半部分头

# 判断链表是否有环
def has_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    return False

# 找环的入口（LeetCode 142）
def detect_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            # 相遇后，一个从 head 出发，一个从相遇点出发
            # 两者一步一步走，再次相遇就是环入口
            p = head
            while p != slow:
                p = p.next
                slow = slow.next
            return p
    return None

# 判断链表是否为回文（快慢指针 + 反转后半）
def is_palindrome(head):
    # 1. 找中间
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    # 2. 反转后半
    prev = None
    while slow:
        next_node = slow.next
        slow.next = prev
        prev = slow
        slow = next_node
    # 3. 对比前半和后半
    while prev:
        if head.val != prev.val:
            return False
        head = head.next
        prev = prev.next
    return True
```

## 7. 双指针

### 理解与可视化

```
前后指针（不同于快慢指针，两个指针从不同位置出发）:

  例：找链表倒数第 k 个节点
  让 fast 先走 k 步，然后 fast 和 slow 一起走
  fast 到终点时，slow 就在倒数第 k 个

  k=2:
  [1] → [2] → [3] → [4] → [5] → None
  slow
              fast (先走了2步)

  [1] → [2] → [3] → [4] → [5] → None
                     slow
                                  fast → None

  slow 指向 4，就是倒数第 2 个 ✓
```

### 代码模板

```python
# 链表倒数第 k 个节点
def find_kth_from_end(head, k):
    fast = slow = head
    for _ in range(k):
        fast = fast.next
    while fast:
        fast = fast.next
        slow = slow.next
    return slow
```

## 8. 合并链表

### 理解与可视化

```
合并两个有序链表:

  l1: 1 → 3 → 5
  l2: 2 → 4 → 6

  用 dummy 节点简化操作:
  dummy → 比较 l1和l2 头部，取小的接上

  dummy → 1 → 2 → 3 → 4 → 5 → 6

  每一步:
  1. 比较 l1.val 和 l2.val
  2. 取较小的接到结果链表
  3. 被取的那个链表前进一步
  4. l1 或 l2 遍历完后，把剩余的直接接上
```

### 代码模板

```python
# 合并两个有序链表（⭐ 必会）
def merge_two_lists(l1, l2):
    dummy = cur = ListNode()
    while l1 and l2:
        if l1.val <= l2.val:
            cur.next = l1
            l1 = l1.next
        else:
            cur.next = l2
            l2 = l2.next
        cur = cur.next
    cur.next = l1 or l2  # 接上剩余部分
    return dummy.next

# 合并 K 个有序链表（分治法）
def merge_k_lists(lists):
    if not lists:
        return None
    if len(lists) == 1:
        return lists[0]
    mid = len(lists) // 2
    left = merge_k_lists(lists[:mid])
    right = merge_k_lists(lists[mid:])
    return merge_two_lists(left, right)
```

## 9. 排序链表（分治）

### 理解与可视化

```
链表排序最优解：归并排序（自顶向下分治）

  原始:    4 → 2 → 1 → 3

  分割:    4 → 2  |  1 → 3     （用快慢指针找中间，断开）

  继续分:  4 | 2  |  1 | 3

  合并:    2 → 4  |  1 → 3     （两两合并有序链表）

  合并:    1 → 2 → 3 → 4

  关键组合技: 找中点 + 合并两个有序链表
```

### 代码模板

```python
# 链表归并排序
def sort_list(head):
    if not head or not head.next:
        return head
    # 1. 找中点并断开
    slow, fast = head, head.next
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    mid = slow.next
    slow.next = None  # 断开
    # 2. 递归排序两半
    left = sort_list(head)
    right = sort_list(mid)
    # 3. 合并
    return merge_two_lists(left, right)
```

## 10. 综合应用

### 代码模板

```python
# 重排链表 L0→Ln→L1→Ln-1→...（LeetCode 143）
def reorder_list(head):
    # 1. 找中间
    slow, fast = head, head
    while fast.next and fast.next.next:
        slow = slow.next
        fast = fast.next.next
    # 2. 反转后半
    prev, cur = None, slow.next
    slow.next = None
    while cur:
        next_node = cur.next
        cur.next = prev
        prev = cur
        cur = next_node
    # 3. 交替合并
    first, second = head, prev
    while second:
        tmp1, tmp2 = first.next, second.next
        first.next = second
        second.next = tmp1
        first, second = tmp1, tmp2

# 复制带随机指针的链表（LeetCode 138）
def copy_random_list(head):
    if not head:
        return None
    # 哈希表映射 原节点 → 新节点
    old_to_new = {}
    cur = head
    while cur:
        old_to_new[cur] = ListNode(cur.val)
        cur = cur.next
    cur = head
    while cur:
        old_to_new[cur].next = old_to_new.get(cur.next)
        old_to_new[cur].random = old_to_new.get(cur.random)
        cur = cur.next
    return old_to_new[head]

# LRU 缓存（双链表 + 哈希表）（LeetCode 146）
class LRUNode:
    def __init__(self, key=0, val=0):
        self.key = key
        self.val = val
        self.prev = None
        self.next = None

class LRUCache:
    def __init__(self, capacity):
        self.cap = capacity
        self.cache = {}
        self.head = LRUNode()  # 哨兵头
        self.tail = LRUNode()  # 哨兵尾
        self.head.next = self.tail
        self.tail.prev = self.head

    def _remove(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev

    def _add_to_front(self, node):
        node.next = self.head.next
        node.prev = self.head
        self.head.next.prev = node
        self.head.next = node

    def get(self, key):
        if key not in self.cache:
            return -1
        node = self.cache[key]
        self._remove(node)
        self._add_to_front(node)
        return node.val

    def put(self, key, value):
        if key in self.cache:
            self._remove(self.cache[key])
        node = LRUNode(key, value)
        self._add_to_front(node)
        self.cache[key] = node
        if len(self.cache) > self.cap:
            lru = self.tail.prev
            self._remove(lru)
            del self.cache[lru.key]
```
