# 反转链表 ⭐

> 反转链表是链表题中的最高频考点，必须闭眼写出迭代和递归两种写法。能解决反转全部、反转区间、K 组反转等问题。

---

## 1. 全部反转（迭代法）⭐⭐

### 理解与可视化

```
原始: 1 → 2 → 3 → 4 → None

核心思想: 逐个把每个节点的 next 指针"翻转"方向

  需要三个变量:
    prev = 已翻转部分的头 (初始 None)
    cur  = 当前要处理的节点
    nxt  = cur 的下一个 (暂存, 不然翻转后就找不到了)

  ┌─── 初始状态 ──────────────────────────────┐
  │  prev=None   cur=1→2→3→4→None             │
  └───────────────────────────────────────────┘

  ┌─── 第 1 步 ──────────────────────────────┐
  │  nxt = cur.next      保存: nxt=2          │
  │  cur.next = prev     翻转: 1→None         │
  │  prev = cur          前进: prev=1          │
  │  cur = nxt           前进: cur=2           │
  │                                            │
  │  None ← 1    2→3→4→None                   │
  │        prev  cur                           │
  └───────────────────────────────────────────┘

  ┌─── 第 2 步 ──────────────────────────────┐
  │  None ← 1 ← 2    3→4→None               │
  │             prev  cur                     │
  └───────────────────────────────────────────┘

  ┌─── 第 3 步 ──────────────────────────────┐
  │  None ← 1 ← 2 ← 3    4→None             │
  │                  prev  cur                │
  └───────────────────────────────────────────┘

  ┌─── 第 4 步 ──────────────────────────────┐
  │  None ← 1 ← 2 ← 3 ← 4                   │
  │                       prev  cur=None      │
  │  cur=None → 循环结束, 返回 prev            │
  └───────────────────────────────────────────┘

  结果: 4 → 3 → 2 → 1 → None ✓
```

### 代码模板

```python
# ━━━━━━━━━━━ 迭代反转（⭐ 必须默写）━━━━━━━━━━━
def reverse_list(head):
    prev = None              # ⚠️ 初始化 None, 不是 head
    cur = head
    while cur:               # ⚠️ while cur, 不是 while cur.next
        nxt = cur.next       # 1. 暂存下一个
        cur.next = prev      # 2. 翻转指针
        prev = cur           # 3. prev 前进
        cur = nxt            # 4. cur 前进
    return prev              # ⚠️ 返回 prev（新头），不是 cur（已经是 None 了）

# ⚠️ 易错点总结:
#   ❌ prev = head       → 第一个节点的 next 不会变成 None, 形成环
#   ❌ while cur.next    → 最后一个节点不会被翻转
#   ❌ return cur        → cur 是 None, 返回了空
#   ❌ return head       → head 现在是尾部, 指向 None
#   ✓  return prev      → prev 是真正的新头

# ⚠️ 空链表:
#   head=None → while 不进入 → prev=None → return None ✓
# ⚠️ 单节点:
#   head→None → nxt=None, cur.next=None, prev=head, cur=None
#                → return head ✓
```

## 2. 全部反转（递归法）

### 理解与可视化

```
递归思路:
  reverse(1→2→3→4) 
  = 先递归反转 2→3→4 得到 4→3→2→None
  = 然后把 1 接到反转后的末尾

  关键: 反转后, 2 变成了末尾! 而 1.next 还指着 2!
  所以: head.next.next = head  (让 2 指回 1)
        head.next = None       (让 1 指向 None, 变成新尾)

  图示:
    递归前: 1 → 2 → 3 → 4 → None
    递归后:      4 → 3 → 2 → None   (2→3→4 反转完成)
            1 ──→ 2                  (1 还指着 2)
    
    head.next.next = head:
            1 ← 2                    (2 指回 1)
    head.next = None:
       None ← 1 ← 2 ← 3 ← 4       (1 指向 None)
    
    返回 new_head = 4 ✓
```

### 代码模板

```python
def reverse_list_recursive(head):
    # 递归终点
    if not head or not head.next:  # ⚠️ 空链表 or 单节点
        return head
    
    new_head = reverse_list_recursive(head.next)  # 递归反转后面
    
    head.next.next = head          # ⚠️ 后一个节点指回自己
    head.next = None               # ⚠️ 断开原来的指向（当前成为尾部）
    
    return new_head                # ⚠️ 返回递归拿到的新头（一直是原尾部）

# ⚠️ 递归深度: O(n), 链表长度超过 ~1000 可能栈溢出
# ⚠️ 面试推荐迭代法, 但递归法也必须会
```

## 3. 反转前 N 个节点

### 理解与可视化

```
只反转前 n 个, 后面保持不变

  原始: 1 → 2 → 3 → 4 → 5 → None,  n=3
  结果: 3 → 2 → 1 → 4 → 5 → None
                 ↑
                 原来的 head 现在要接上第 n+1 个节点

  和全反转的区别:
    全反转: 原 head 的 next 最终指向 None
    反转前N: 原 head 的 next 最终指向第 n+1 个节点
```

### 代码模板

```python
def reverse_n(head, n):
    prev = None
    cur = head
    for _ in range(n):           # ⚠️ 只循环 n 次, 不是 while cur
        nxt = cur.next
        cur.next = prev
        prev = cur
        cur = nxt
    head.next = cur              # ⚠️ 关键! 原头节点接上剩余部分
    return prev                  # prev 是反转部分的新头

# ⚠️ 和全反转的区别只有两点:
#   1. 循环 for _ in range(n) 而非 while cur
#   2. 最后 head.next = cur 连接后续部分
#
# ⚠️ 如果 n >= 链表长度, 等价于全反转
#   此时 cur=None, head.next=None 也正确
```

## 4. 反转区间 [left, right] ⭐⭐

### LeetCode 92

```
题意: 反转链表中第 left 到第 right 个节点（1-indexed）
  [1,2,3,4,5], left=2, right=4  → [1,4,3,2,5]

思路: 
  1. 找到第 left-1 个节点 (反转区间的前驱) → 叫它 p0
  2. 对 p0 后面的 right-left+1 个节点做"反转前N个"
  3. 连接前后
```

### 代码模板

```python
def reverse_between(head, left, right):
    dummy = ListNode(next=head)
    
    # 1. 找到反转区间的前驱 p0
    p0 = dummy
    for _ in range(left - 1):    # ⚠️ 走 left-1 步（1-indexed）
        p0 = p0.next
    
    # 2. 反转 [left, right] 的 right-left+1 个节点
    prev = None
    cur = p0.next                # 反转区间的第一个节点
    for _ in range(right - left + 1):  # ⚠️ 反转 right-left+1 个
        nxt = cur.next
        cur.next = prev
        prev = cur
        cur = nxt
    
    # 3. 连接
    p0.next.next = cur           # ⚠️ 原来的第 left 个（现在是区间尾）接后续
    p0.next = prev               # ⚠️ 前驱接反转后的头
    
    return dummy.next

# ⚠️ 关键索引:
#   left, right 是 1-indexed
#   走 left-1 步到达前驱
#   反转 right-left+1 个节点
#   
# ⚠️ 连接顺序:
#   先 p0.next.next = cur  (p0.next 此时还指向原来的第 left 个)
#   再 p0.next = prev      (改 p0.next 后，原来的 p0.next 引用就变了)
#   顺序不能反! 反了会丢失原 p0.next 的引用
```

```
可视化: [1,2,3,4,5], left=2, right=4

  dummy → [1] → [2] → [3] → [4] → [5] → None
          p0

  反转 [2,3,4]:
    prev=None, cur=[2]

    1: nxt=[3], [2]→None, prev=[2], cur=[3]
    2: nxt=[4], [3]→[2], prev=[3], cur=[4]
    3: nxt=[5], [4]→[3], prev=[4], cur=[5]

    反转后: prev=[4]→[3]→[2]→None,  cur=[5]

  连接:
    p0.next.next = cur:  [2]→None 改为 [2]→[5]
    p0.next = prev:      [1]→[2] 改为 [1]→[4]

  结果: [1] → [4] → [3] → [2] → [5] → None ✓
```

## 5. K 个一组翻转 ⭐⭐

### LeetCode 25

```
题意: 每 k 个节点一组进行翻转, 不足 k 个不翻转
  [1,2,3,4,5], k=2  → [2,1,4,3,5]
  [1,2,3,4,5], k=3  → [3,2,1,4,5]

思路:
  1. 先数 k 个, 确认够不够
  2. 够 → 对这 k 个做反转
  3. 连接前后部分
  4. 移动起点, 重复 1-3
```

### 代码模板

```python
def reverse_k_group(head, k):
    dummy = ListNode(next=head)
    p0 = dummy

    while True:
        # 1. 检查是否有 k 个节点
        check = p0
        for _ in range(k):
            check = check.next
            if not check:           # ⚠️ 不足 k 个, 不翻转, 结束
                return dummy.next
        
        # 2. 反转 k 个
        prev = None
        cur = p0.next
        for _ in range(k):
            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt
        
        # 3. 连接
        tail = p0.next              # ⚠️ 反转后, 原 p0.next 变成这组的尾
        tail.next = cur             # 尾接后续
        p0.next = prev              # 前驱接新头
        
        # 4. 移动 p0 到这组的尾部, 准备下一组
        p0 = tail                   # ⚠️ tail 就是这组反转后的最后一个

# ⚠️ 关键:
#   检查阶段: 从 p0 开始走 k 步, 如果遇到 None 就不够
#   p0 的更新: p0 = tail (反转后的尾, 下一组的前驱)
#   每组的连接和反转区间完全一样
```

## 6. 两两交换

### LeetCode 24

```
题意: 两两交换相邻节点
  [1,2,3,4] → [2,1,4,3]

就是 K 个一组翻转 k=2 的特例, 也可以单独实现:
```

```python
def swap_pairs(head):
    dummy = ListNode(next=head)
    prev = dummy
    while prev.next and prev.next.next:  # ⚠️ 需要至少两个节点
        a = prev.next                     # 第一个
        b = prev.next.next                # 第二个
        
        # 交换 a 和 b
        prev.next = b                     # 前驱指向 b
        a.next = b.next                   # a 指向 b 后面
        b.next = a                        # b 指向 a
        
        prev = a                          # ⚠️ prev 移到 a（交换后 a 在后面）
    return dummy.next

# ⚠️ 赋值顺序不能乱! 画图确认:
#   before: prev → a → b → rest
#   after:  prev → b → a → rest
```

## 本节要点速查

```
✅ 迭代反转: prev=None, while cur, 四步操作, return prev
✅ 递归反转: head.next.next=head, head.next=None
✅ 反转前N个: for range(n), 最后 head.next=cur 接后面
✅ 反转区间: 先走 left-1 步找前驱, 再反转 right-left+1 个, 最后连接
✅ K组翻转: 先检查够不够→反转→连接→移动p0→重复
✅ 易错: return prev 不是 cur; prev 初始 None 不是 head
✅ 连接: 先连尾(p0.next.next=cur) 再连头(p0.next=prev)
```
