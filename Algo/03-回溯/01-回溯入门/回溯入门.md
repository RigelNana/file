# 回溯入门：理解递归树

> 回溯法是暴力搜索的优化版本。核心思想：在递归树上做深度优先遍历，做选择→递归→撤销选择。

---

## 1. 什么是回溯

### 理解与可视化

```
回溯 = DFS + 选择 + 撤销选择

本质: 在"递归树"上遍历, 尝试所有可能的选择
     走不通就"回头"(撤销选择), 试另一条路

      []              ← 初始状态
     / | \
   [1] [2] [3]       ← 第一次选择
   /\   |
[1,2][1,3] [2,3]     ← 第二次选择
  |     |    |
[1,2,3] ... ...       ← 第三次选择

回溯的步骤:
  1. 做一个选择     → path.append(选择)
  2. 递归           → dfs(下一层)
  3. 撤销选择       → path.pop()
  
  ⚠️ 步骤 3 是关键! 不撤销就不是回溯, 只是普通递归
```

### 核心模板

```python
def backtrack(选择列表, 路径, 结果):
    if 满足结束条件:
        结果.append(路径的拷贝)       # ⚠️ 拷贝! 不是直接添加
        return
    
    for 选择 in 选择列表:
        if 选择 不合法:               # 剪枝
            continue
        路径.append(选择)             # 1. 做选择
        backtrack(剩余选择, 路径, 结果) # 2. 递归
        路径.pop()                    # 3. 撤销选择（回溯）
```

## 2. 回溯的两种视角

### 理解与可视化

```
视角一: "选或不选"（输入的视角）
  对每个元素, 考虑"选"还是"不选"
  
  元素: [1, 2, 3], 生成子集
  
  决策树:
            {}
           / \
         {1}  {}          ← 1：选 or 不选
        / \   / \
    {1,2} {1} {2} {}      ← 2：选 or 不选
    / \  / \ / \ / \
  ...  ...  ...  ...      ← 3：选 or 不选

  代码:
    def dfs(i, path):
        if i == n:
            res.append(path[:])
            return
        # 选
        path.append(nums[i])
        dfs(i + 1, path)
        path.pop()
        # 不选
        dfs(i + 1, path)


视角二: "枚举选哪个"（答案的视角）
  对答案的每个位置, 枚举放哪个元素
  
  代码:
    def dfs(i, path):
        res.append(path[:])       # ⚠️ 每个节点都是一个合法子集
        for j in range(i, n):     # 从 i 开始枚举可选元素
            path.append(nums[j])
            dfs(j + 1, path)      # ⚠️ j+1 不是 i+1
            path.pop()

  ⚠️ 两种视角生成的结果完全一样, 递归树的形状不同
  ⚠️ "选或不选" → 叶子节点才收集答案（二叉树）
  ⚠️ "枚举选哪个" → 每个节点都收集答案（多叉树）
```

## 3. 回溯的时间复杂度

```
子集:   O(2^n)   每个元素选或不选, 2^n 种组合
组合:   O(C(n,k)) 从 n 中选 k 个
排列:   O(n!)    n 个位置, 每个位置可选的越来越少
划分:   O(2^n)   类似子集, 每个间隔切或不切

⚠️ 回溯的时间复杂度通常是指数级
   但通过剪枝可以大幅减少实际搜索量
```

## 4. 常见错误

```python
# ❌ 错误 1: 忘记 path[:] 拷贝
res.append(path)           # 所有结果指向同一个 path 对象
# ✓ 正确:
res.append(path[:])        # 浅拷贝当前状态
# 或
res.append(list(path))     # 同样是拷贝

# ❌ 错误 2: 忘记撤销选择
path.append(nums[i])
dfs(i + 1)
# 缺少 path.pop()  → path 会越来越长

# ❌ 错误 3: 起始index错误
for j in range(0, n):      # 每次从 0 开始 → 有重复!
    path.append(nums[j])
    dfs(j + 1)
# ✓ 正确:
for j in range(i, n):      # 从 i 开始, 避免选之前选过的

# ❌ 错误 4: 排列和组合搞混
# 组合: [1,2] 和 [2,1] 是同一个 → 用 start 参数控制
# 排列: [1,2] 和 [2,1] 是不同的 → 用 visited 数组控制
```

## 本节要点速查

```
✅ 回溯 = DFS + 选择 + 撤销选择
✅ 两种视角: "选或不选"(二叉树) vs "枚举选哪个"(多叉树)
✅ 必须拷贝: res.append(path[:]) 不能 res.append(path)
✅ 必须回溯: path.append → dfs → path.pop
✅ 组合用 start; 排列用 visited
✅ 时间: 子集 O(2^n), 排列 O(n!), 组合 O(C(n,k))
```
