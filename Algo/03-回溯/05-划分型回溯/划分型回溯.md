# 划分型回溯

> 划分型回溯：将字符串/数组分割成若干部分，使每部分满足某个条件。

## 1. 分割回文串

### LeetCode 131

### 理解与可视化

```
输入: "aab"
输出: [["a","a","b"], ["aa","b"]]

本质: 在字符的"间隙"上决定是否切一刀
  a | a | b
  位置 0,1,2,3 (共4个端点, 3个间隙)

  每次选择: 从当前位置 i 到 j (i<=j<n) 的子串
            如果是回文 → 继续切 j+1 到末尾
            如果不是 → 剪枝

递归树:
  start=0
  ├── "a" (回文✓) → start=1
  │   ├── "a" (回文✓) → start=2
  │   │   └── "b" (回文✓) → start=3 ✓ → ["a","a","b"]
  │   └── "ab" (不是)  → 剪枝✗
  ├── "aa" (回文✓) → start=2
  │   └── "b" (回文✓) → start=3 ✓ → ["aa","b"]
  └── "aab" (不是) → 剪枝✗
```

### 代码模板

```python
def partition(s):
    res = []
    n = len(s)
    
    def dfs(start, path):
        if start == n:                # ⚠️ 切到末尾 → 完成分割
            res.append(path[:])
            return
        for end in range(start, n):   # ⚠️ end 从 start 开始, 不是 start+1
            sub = s[start:end+1]      # ⚠️ 切片 [start, end], 左闭右闭
            if sub == sub[::-1]:      # 判断回文
                path.append(sub)
                dfs(end + 1, path)    # ⚠️ 下一段从 end+1 开始
                path.pop()
    
    dfs(0, [])
    return res

# ⚠️ start == n 是终止条件（已切完所有字符）
#    不是 start >= n, 因为 end + 1 最多到 n
#
# ⚠️ 切片范围: s[start:end+1]
#    Python 切片右边是开区间
#    end 从 start 开始 → 最短子串长度为 1
#
# ⚠️ 注意和子集/组合的区别:
#    子集: 选或不选每个元素
#    分割: 选择切割点的位置
```

### 优化：预计算回文

```python
def partition(s):
    n = len(s)
    # 预计算 dp[i][j] = s[i..j] 是否是回文
    dp = [[False] * n for _ in range(n)]
    for i in range(n - 1, -1, -1):       # ⚠️ i 从大到小
        for j in range(i, n):
            if s[i] == s[j] and (j - i <= 2 or dp[i+1][j-1]):
                dp[i][j] = True
    
    res = []
    def dfs(start, path):
        if start == n:
            res.append(path[:])
            return
        for end in range(start, n):
            if dp[start][end]:            # O(1) 判断回文
                path.append(s[start:end+1])
                dfs(end + 1, path)
                path.pop()
    
    dfs(0, [])
    return res

# ⚠️ dp[i][j] 依赖 dp[i+1][j-1], 所以 i 必须从大到小遍历
# ⚠️ j - i <= 2: 长度<=3的子串只需首尾相等就是回文
#    如 "a"(j-i=0), "aa"(j-i=1), "aba"(j-i=2)
```

## 2. 复原 IP 地址

### LeetCode 93

### 理解与可视化

```
输入: "25525511135"
输出: ["255.255.11.135", "255.255.111.35"]

约束:
  - 恰好分成 4 段
  - 每段 0~255
  - 不能有前导零 ("00", "01" 不合法)
  - 每段 1~3 位数字

递归树 (简化):
  start=0, seg=0
  ├── "2"   → start=1, seg=1
  ├── "25"  → start=2, seg=1
  └── "255" → start=3, seg=1
      ├── "2"   → start=4, seg=2
      ├── "25"  → start=5, seg=2
      └── "255" → start=6, seg=2
          ├── "1"   → "11"  → ...
          ├── "11"  → "135" → ✓
          └── "111" → "35"  → ✓
```

### 代码模板

```python
def restoreIpAddresses(s):
    res = []
    n = len(s)
    
    def dfs(start, seg, path):
        if seg == 4:                      # ⚠️ 必须恰好 4 段
            if start == n:                # ⚠️ 且用完所有字符
                res.append('.'.join(path))
                return
            return                        # 4 段但没用完 → 失败
        
        # ⚠️ 剪枝: 剩余字符太多或太少
        remain = n - start
        if remain < (4 - seg) or remain > (4 - seg) * 3:
            return
        
        for length in range(1, 4):        # ⚠️ 每段 1~3 位
            if start + length > n:
                break
            sub = s[start:start+length]
            
            # ⚠️ 前导零检查: 长度>1 但以 '0' 开头
            if len(sub) > 1 and sub[0] == '0':
                break                     # ⚠️ 用 break 不是 continue!
            
            if int(sub) > 255:            # ⚠️ 超过 255
                break                     # 更长的子串只会更大, break
            
            path.append(sub)
            dfs(start + length, seg + 1, path)
            path.pop()
    
    dfs(0, 0, [])
    return res

# ⚠️ 前导零时用 break 而非 continue:
#    "01" 不合法 → "012" 也不合法, 没必要继续
#
# ⚠️ int(sub) > 255 时也用 break:
#    "256" 不合法 → "2561" 更大也不合法
#
# ⚠️ 剪枝很重要:
#    剩余 remain 个字符, 还需 (4-seg) 段
#    每段最少 1 位 → remain >= (4-seg)
#    每段最多 3 位 → remain <= (4-seg)*3
```

## 3. 划分为 k 个相等子集

### LeetCode 698

```
输入: nums = [4,3,2,3,5,2,1], k = 4
输出: true (划分为 [5],[1,4],[2,3],[2,3])

思路: 回溯尝试将每个数放入 k 个桶之一
```

```python
def canPartitionKSubsets(nums, k):
    total = sum(nums)
    if total % k != 0:
        return False
    target = total // k
    nums.sort(reverse=True)           # ⚠️ 从大到小, 加速剪枝
    
    if nums[0] > target:              # ⚠️ 最大数超过目标
        return False
    
    buckets = [0] * k
    
    def dfs(index):
        if index == len(nums):        # 所有数都放完了
            return True               # buckets 一定都 == target
        
        seen = set()                  # ⚠️ 同层去重
        for j in range(k):
            if buckets[j] + nums[index] > target:
                continue
            if buckets[j] in seen:    # ⚠️ 相同容量的桶只试一次
                continue
            seen.add(buckets[j])
            
            buckets[j] += nums[index]
            if dfs(index + 1):
                return True
            buckets[j] -= nums[index]
        
        return False
    
    return dfs(0)

# ⚠️ 排序从大到小 → 大数先放, 更早触发 > target 剪枝
# ⚠️ seen 同层去重: 两个桶当前容量相同, 放进哪个效果一样
#    避免重复搜索, 大幅提升性能
# ⚠️ 时间复杂度: O(k^n), 但剪枝后实际快很多
```

## 4. 划分型回溯 通用模式

```
划分型 vs 子集型 vs 组合型 vs 排列型:

┌────────────┬─────────────────────┬──────────────────┐
│ 类型       │ 递归含义            │ 典型题            │
├────────────┼─────────────────────┼──────────────────┤
│ 子集型     │ 每个元素选/不选      │ LC78, LC90       │
│ 组合型     │ 从 i 开始选 k 个    │ LC77, LC39       │
│ 排列型     │ 从头选未选过的       │ LC46, LC47       │
│ 划分型     │ 在间隙上切割        │ LC131, LC93      │
│ 分配型     │ 每个元素放哪个桶     │ LC698            │
└────────────┴─────────────────────┴──────────────────┘

划分型回溯模板:
  def dfs(start, path):
      if start == n:              # 切完
          res.append(path[:])
      for end in range(start, n): # 枚举切割点
          if valid(start, end):   # 检查子串/子数组
              path.append(...)
              dfs(end + 1, path)  # ⚠️ end+1, 不是 end
              path.pop()

⚠️ 注意事项:
  1. dfs(end+1) 不是 dfs(end), 否则死循环
  2. valid 检查可能涉及前导零、数值范围、回文等
  3. 如果要求恰好 k 段, 加一个 seg 参数
  4. 剩余长度的上下界可以做剪枝
```
