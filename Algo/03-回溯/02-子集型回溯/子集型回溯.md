## 1. 子集（无重复）

### LeetCode 78

### 理解与可视化

```
输入: [1, 2, 3]
输出: [[], [1], [2], [3], [1,2], [1,3], [2,3], [1,2,3]]

枚举选哪个视角（多叉树）:
           []
         / | \
       [1] [2] [3]
       /\    |
   [1,2][1,3][2,3]
     |
   [1,2,3]

  每个节点都是一个合法子集!
  递归进入时就可以收集答案

选或不选视角（二叉树）:
            []
           / \
        选1   不选1
        [1]    []
       / \    / \
    选2 不选 选2 不选
   [1,2] [1] [2]  []
   ...

  只有到叶子节点(处理完所有元素)才收集答案
```

### 代码模板

```python
# ━━━━ 方法 1: 枚举选哪个（推荐）━━━━
def subsets(nums):
    res = []
    n = len(nums)
    
    def dfs(i, path):
        res.append(path[:])           # ⚠️ 每个节点都收集! 不是只在叶子
        for j in range(i, n):         # ⚠️ 从 i 开始, 保证子集不重复
            path.append(nums[j])
            dfs(j + 1, path)          # ⚠️ j+1, 不是 i+1
            path.pop()
    
    dfs(0, [])
    return res

# ━━━━ 方法 2: 选或不选 ━━━━
def subsets_v2(nums):
    res = []
    n = len(nums)
    
    def dfs(i, path):
        if i == n:                    # ⚠️ 处理完所有元素才收集
            res.append(path[:])
            return
        # 不选 nums[i]
        dfs(i + 1, path)
        # 选 nums[i]
        path.append(nums[i])
        dfs(i + 1, path)
        path.pop()
    
    dfs(0, [])
    return res

# ⚠️ 方法 1: 更通用, 好扩展到组合、分割等问题
# ⚠️ 方法 2: 更直观, 像做决策树
# ⚠️ 两种方法结果子集相同, 但顺序可能不同
```

## 2. 子集（有重复）

### LeetCode 90

```
输入: [1, 2, 2]
输出: [[], [1], [1,2], [1,2,2], [2], [2,2]]
不能有: [2,2] 出现两次

关键: 先排序, 然后同层跳过重复元素

排序后: [1, 2, 2]

         []
        / | \
      [1] [2] [2]←跳过! 同层重复
      /\    |
  [1,2][1,2] [2,2]
    |    ↑ 这里的 2 是第二个2, 不算同层重复, 可以选
  [1,2,2]
```

```python
def subsetsWithDup(nums):
    nums.sort()                       # ⚠️ 必须先排序!
    res = []
    n = len(nums)
    
    def dfs(i, path):
        res.append(path[:])
        for j in range(i, n):
            if j > i and nums[j] == nums[j-1]:  # ⚠️ 同层去重
                continue
            path.append(nums[j])
            dfs(j + 1, path)
            path.pop()
    
    dfs(0, [])
    return res

# ⚠️ j > i 而不是 j > 0:
#   j > i: 同一层(同一个 i 的循环)中跳过重复
#   j > 0: 会连不同层的也跳过 → 错误!
#
# ⚠️ 为什么要排序?
#   排序后相同元素相邻, 才能用 nums[j]==nums[j-1] 判断
#   不排序: [1,2,2] 中两个 2 不相邻, 无法简单去重
#
# ⚠️ 去重的本质:
#   [2,2,2] 中选 2 个 2, 只有一种方式
#   排序后, 我们规定"选靠前的", 跳过靠后的相同元素
```

## 3. 幂集的位运算法

```python
# ━━━━ 位运算枚举子集（不用递归）━━━━
def subsets_bit(nums):
    n = len(nums)
    res = []
    for mask in range(1 << n):        # ⚠️ 0 到 2^n-1
        subset = []
        for i in range(n):
            if mask & (1 << i):       # 第 i 位是 1 → 选 nums[i]
                subset.append(nums[i])
        res.append(subset)
    return res

# ⚠️ mask = 0b000 → []
# ⚠️ mask = 0b001 → [nums[0]]
# ⚠️ mask = 0b010 → [nums[1]]
# ⚠️ mask = 0b011 → [nums[0], nums[1]]
# ⚠️ mask = 0b111 → [nums[0], nums[1], nums[2]]
#
# 简洁但不适用于有重复元素的情况
# 面试推荐回溯法 (更通用)
```

## 4. 字符串子集（电话号码字母组合）

### LeetCode 17

```python
def letterCombinations(digits):
    if not digits:
        return []
    phone = {
        '2': 'abc', '3': 'def', '4': 'ghi',
        '5': 'jkl', '6': 'mno', '7': 'pqrs',
        '8': 'tuv', '9': 'wxyz'
    }
    res = []
    
    def dfs(i, path):
        if i == len(digits):
            res.append("".join(path))
            return
        for ch in phone[digits[i]]:   # ⚠️ 枚举当前数字对应的所有字母
            path.append(ch)
            dfs(i + 1, path)
            path.pop()
    
    dfs(0, [])
    return res

# ⚠️ 这不是传统子集，但思路一样：
#   每一层（每个数字）枚举所有可能的选择
#   选择列表 = 当前数字对应的字母
```

## 本节要点速查

```
✅ 子集无重复: for j in range(i,n), 每个节点收集答案
✅ 子集有重复: 排序 + j>i && nums[j]==nums[j-1] 跳过
✅ 两种方法: "枚举选哪个"(推荐) vs "选或不选"
✅ j>i 不是 j>0: 同层去重, 不是跨层去重
✅ 位运算: mask & (1<<i) 判断第 i 位, 适合无重复
✅ 必须 path[:] 拷贝, 必须 pop 回溯
```
