# 搜索与剪枝

> 回溯的核心优化就是剪枝。好的剪枝能将指数级别的暴力搜索变成可接受的时间。

---

## 1. 剪枝策略总览

```
剪枝类型:

  1. 可行性剪枝 (Feasibility Pruning)
     当前状态已经不可能产生合法解 → 直接返回
     例: IP 地址段 > 255, 子集和 > target

  2. 最优性剪枝 (Optimality Pruning)
     当前状态已经不可能产生更优解 → 直接返回
     例: 当前路径已 >= 已知最短路径

  3. 对称性剪枝 (Symmetry Breaking)
     利用对称性避免重复搜索
     例: N皇后只搜一半列, 乘以 2

  4. 排序剪枝
     排序后, 如果当前元素不满足 → 后面更大的也不满足 → break
     例: 组合总和中 candidates[i] > remain → break
```

## 2. N 皇后

### LeetCode 51

### 理解与可视化

```
n=4 时的一个解:
  . Q . .     第0行: 皇后在列1
  . . . Q     第1行: 皇后在列3
  Q . . .     第2行: 皇后在列0
  . . Q .     第3行: 皇后在列2

约束: 任意两个皇后不能同行/同列/同对角线

对角线编号:
  主对角线 (\): row - col = 常数    (范围: -(n-1) 到 n-1)
  副对角线 (/): row + col = 常数    (范围: 0 到 2*(n-1))

  ⚠️ 主对角线 row-col 可能为负, 存入 set 更方便
```

### 代码模板

```python
def solveNQueens(n):
    res = []
    queens = [-1] * n                 # queens[i] = 第 i 行皇后的列号
    cols = set()                      # 已占用的列
    diag1 = set()                     # 已占用的主对角线 (row-col)
    diag2 = set()                     # 已占用的副对角线 (row+col)
    
    def dfs(row):
        if row == n:                  # ⚠️ 所有行都放完
            board = []
            for r in range(n):
                line = ['.'] * n
                line[queens[r]] = 'Q'
                board.append(''.join(line))
            res.append(board)
            return
        
        for col in range(n):
            if col in cols:
                continue
            if row - col in diag1:    # ⚠️ 主对角线冲突
                continue
            if row + col in diag2:    # ⚠️ 副对角线冲突
                continue
            
            queens[row] = col
            cols.add(col)
            diag1.add(row - col)
            diag2.add(row + col)
            
            dfs(row + 1)
            
            queens[row] = -1
            cols.remove(col)
            diag1.remove(row - col)
            diag2.remove(row + col)
    
    dfs(0)
    return res

# ⚠️ 不需要检查行冲突: 每行只放一个
# ⚠️ 对角线判断方法:
#   主对角线 \: 同对角线上 row-col 相同
#   副对角线 /: 同对角线上 row+col 相同
# ⚠️ 用 set 而非数组: row-col 可能为负, set 更方便
# ⚠️ 时间 O(n!), 空间 O(n), n=8 有 92 个解
```

## 3. 解数独

### LeetCode 37

### 理解与可视化

```
9×9 棋盘, 已填部分数字
要求: 每行/每列/每个 3×3 宫格 内 1~9 各出现一次

约束集合:
  rows[r]    = 第 r 行已有的数字集合
  cols[c]    = 第 c 列已有的数字集合
  boxes[b]   = 第 b 个宫格已有的数字集合

  ⚠️ 宫格编号: b = (r // 3) * 3 + (c // 3)
     ┌───┬───┬───┐
     │ 0 │ 1 │ 2 │
     ├───┼───┼───┤
     │ 3 │ 4 │ 5 │
     ├───┼───┼───┤
     │ 6 │ 7 │ 8 │
     └───┴───┴───┘
```

### 代码模板

```python
def solveSudoku(board):
    rows = [set() for _ in range(9)]
    cols = [set() for _ in range(9)]
    boxes = [set() for _ in range(9)]
    empty = []
    
    # 初始化已有数字
    for r in range(9):
        for c in range(9):
            if board[r][c] != '.':
                d = board[r][c]
                rows[r].add(d)
                cols[c].add(d)
                boxes[(r//3)*3 + c//3].add(d)
            else:
                empty.append((r, c))
    
    def dfs(idx):
        if idx == len(empty):         # ⚠️ 所有空格填完
            return True
        r, c = empty[idx]
        b = (r // 3) * 3 + c // 3    # ⚠️ 宫格编号公式
        
        for d in '123456789':
            if d in rows[r] or d in cols[c] or d in boxes[b]:
                continue              # 剪枝: 冲突
            
            board[r][c] = d
            rows[r].add(d)
            cols[c].add(d)
            boxes[b].add(d)
            
            if dfs(idx + 1):
                return True           # ⚠️ 只需一个解, 找到就返回
            
            board[r][c] = '.'
            rows[r].remove(d)
            cols[c].remove(d)
            boxes[b].remove(d)
        
        return False                  # ⚠️ 所有数字都不行 → 回溯
    
    dfs(0)

# ⚠️ 与 N 皇后的区别:
#   N 皇后: 收集所有解 → res.append()
#   数独:   只需一个解 → return True/False
#
# ⚠️ 宫格编号: (r//3)*3 + c//3
#   r=0~2, c=0~2 → box=0;  r=0~2, c=3~5 → box=1
#   r=3~5, c=0~2 → box=3;  ...
#
# ⚠️ 找到解立即 return True, 不做 pop/remove (保留结果)
```

## 4. 单词搜索

### LeetCode 79

```
在二维网格中找是否存在一条路径拼出给定单词
每个格子只能用一次, 可以上下左右走

输入:
  board = [["A","B","C","E"],
           ["S","F","C","S"],
           ["A","D","E","E"]]
  word = "ABCCED" → True
```

```python
def exist(board, word):
    m, n = len(board), len(board[0])
    
    def dfs(r, c, k):
        if k == len(word):            # ⚠️ 所有字符匹配完
            return True
        if r < 0 or r >= m or c < 0 or c >= n:
            return False              # 越界
        if board[r][c] != word[k]:
            return False              # 不匹配
        
        tmp = board[r][c]
        board[r][c] = '#'             # ⚠️ 原地标记已访问
        
        found = (dfs(r+1,c,k+1) or dfs(r-1,c,k+1) or
                 dfs(r,c+1,k+1) or dfs(r,c-1,k+1))
        
        board[r][c] = tmp             # ⚠️ 回溯: 恢复原值
        return found
    
    for i in range(m):
        for j in range(n):
            if dfs(i, j, 0):
                return True
    return False

# ⚠️ 原地修改 board 做标记, 比额外 visited 数组更省空间
# ⚠️ 用 '#' 标记而非 False, 因为 board 存的是字符
# ⚠️ k == len(word) 时返回 True, 不是 k == len(word)-1
#    因为 k 从 0 开始, 匹配最后一个字符后 k+1 == len(word)
# ⚠️ 四个方向用 or 连接, 任一找到即可
# ⚠️ 时间 O(m*n*3^L), L=word长度, 3=每步最多3个方向(来的方向不回)
```

## 5. 常见剪枝技巧汇总

```
┌─────────────┬────────────────────────┬─────────────────┐
│ 剪枝技巧    │ 描述                    │ 例题            │
├─────────────┼────────────────────────┼─────────────────┤
│ 排序+break  │ 排序后当前>target→break │ LC39, LC40      │
│ used去重    │ 同值同层只选一次        │ LC47, LC90      │
│ 约束集合    │ 行/列/对角线/宫格       │ LC51, LC37      │
│ 原地标记    │ 修改原数组代替visited   │ LC79            │
│ 长度剪枝    │ 剩余太多/太少→return    │ LC93            │
│ 提前return  │ 只需一个解→找到就返回   │ LC37, LC79      │
│ 同层去重    │ 同层相同容量桶只试一次  │ LC698           │
│ 预计算      │ dp预处理回文等          │ LC131           │
└─────────────┴────────────────────────┴─────────────────┘

代码模式:
  1. 排序 + break:
     if candidates[i] > remain: break

  2. 去重 + continue:
     if i > start and nums[i] == nums[i-1]: continue

  3. 约束 + continue:
     if col in cols or diag in diags: continue

  4. 找到 + return True:
     if dfs(...): return True
```

```

✅ 剪枝分4类: 可行性/最优性/对称性/排序
✅ 剪枝用 break(不需要后续) 或 continue(跳过当前)
```
