# 动态规划基础

> 动态规划 (DP) 是算法中最核心也最有挑战的部分。它通过记录子问题的解，避免重复计算，从而高效求解最优化问题。

---

## 1. DP 核心思想

### 理解与可视化

```
动态规划 vs 暴力递归:

  暴力递归: 存在大量重复计算
  动态规划: 用表格记录已计算的结果，消除重复

  例: 斐波那契 fib(5)

  暴力递归树（大量重复）:
              fib(5)
             /      \
          fib(4)   fib(3)
          /   \     /   \
       fib(3) fib(2) fib(2) fib(1)
       /  \
    fib(2) fib(1)

  DP: 从小到大计算，每个只算一次
    fib(0)=0 → fib(1)=1 → fib(2)=1 → fib(3)=2 → fib(4)=3 → fib(5)=5

DP 三步走:
  1. 定义状态:  dp[i] 表示什么？
  2. 状态转移:  dp[i] 怎么从更小的子问题推导？
  3. 初始条件:  最小的子问题的答案是什么？

两种实现方式:
  ┌─────────────────────────────────────┐
  │ 自顶向下（记忆化搜索/递归+备忘录）   │
  │   从大问题出发，递归分解             │
  │   用 @cache 或 dict 记录已算结果    │
  │                                     │
  │ 自底向上（递推/填表法）              │
  │   从最小子问题出发，逐步推到大问题   │
  │   用数组 dp[] 存储结果              │
  └─────────────────────────────────────┘

  两者等价，自底向上通常更省空间
```

## 2. 爬楼梯 ⭐

### 理解与可视化

```
问题: 每次可以爬 1 或 2 级台阶，爬到第 n 阶有多少种方法？

  n=1: 1种 (1)
  n=2: 2种 (1+1, 2)
  n=3: 3种 (1+1+1, 1+2, 2+1)
  n=4: 5种

  关键观察: 到达第 n 阶，要么从第 n-1 阶爬 1 步，要么从第 n-2 阶爬 2 步
  dp[n] = dp[n-1] + dp[n-2]  ← 就是斐波那契！

  可视化:
  阶梯:    1    2    3    4    5
  方法数:  1    2    3    5    8
                ↗ ↗   ↗ ↗   ↗ ↗
          dp[i] = dp[i-1] + dp[i-2]
```

### 代码模板

```python
# 记忆化搜索（自顶向下）
from functools import cache

def climb_stairs_memo(n):
    @cache
    def dfs(i):
        if i <= 1:
            return 1
        return dfs(i - 1) + dfs(i - 2)
    return dfs(n)

# 递推（自底向上）
def climb_stairs(n):
    if n <= 2:
        return n
    dp = [0] * (n + 1)
    dp[1], dp[2] = 1, 2
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 空间优化（只需要前两个值）
def climb_stairs_opt(n):
    if n <= 2:
        return n
    a, b = 1, 2
    for _ in range(3, n + 1):
        a, b = b, a + b
    return b
```

## 3. 打家劫舍 ⭐

### 理解与可视化

```
问题: 沿街房屋，不能偷相邻的两家，求最大金额

  房屋: [2, 7, 9, 3, 1]

  对于每间房，两种选择: 偷 or 不偷
    偷第 i 间:   金额 = nums[i] + dp[i-2]（跳过相邻的）
    不偷第 i 间: 金额 = dp[i-1]

  dp[i] = max(dp[i-1], dp[i-2] + nums[i])

  推导:
  i:    0   1   2   3   4
  num:  2   7   9   3   1
  dp:   2   7   11  11  12
            ↑   ↑
          max(2, 7)  max(7, 2+9)=11

  答案: dp[4] = 12
```

### 代码模板

```python
# 打家劫舍
def rob(nums):
    if len(nums) <= 2:
        return max(nums)
    dp = [0] * len(nums)
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])
    for i in range(2, len(nums)):
        dp[i] = max(dp[i-1], dp[i-2] + nums[i])
    return dp[-1]

# 空间优化
def rob_opt(nums):
    prev2 = prev1 = 0
    for x in nums:
        prev2, prev1 = prev1, max(prev1, prev2 + x)
    return prev1

# 打家劫舍 II（环形，首尾不能同时偷）
def rob2(nums):
    if len(nums) == 1:
        return nums[0]
    def rob_range(arr):
        prev2 = prev1 = 0
        for x in arr:
            prev2, prev1 = prev1, max(prev1, prev2 + x)
        return prev1
    return max(rob_range(nums[1:]), rob_range(nums[:-1]))
```

## 4. 最大子数组和 ⭐ (Kadane算法)

### 理解与可视化

```
问题: 找到和最大的连续子数组

  nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]

  关键: 以每个位置结尾的最大子数组和
  dp[i] = max(nums[i], dp[i-1] + nums[i])
  含义: 要么自己单独开始，要么接在前面的后面

  i:     0   1   2   3   4   5   6   7   8
  num:  -2   1  -3   4  -1   2   1  -5   4
  dp:   -2   1  -2   4   3   5   6   1   5
              ↑       ↑               ↑
         max(1,-2+1) max(4,-2+4)    max答案=6

  最大子数组: [4, -1, 2, 1]，和=6
```

### 代码模板

```python
# Kadane 算法
def max_sub_array(nums):
    dp = ans = nums[0]
    for x in nums[1:]:
        dp = max(x, dp + x)  # 单独开始 or 接上
        ans = max(ans, dp)
    return ans

# 最大子数组和（返回子数组本身）
def max_sub_array_detail(nums):
    best_sum = cur_sum = nums[0]
    best_start = best_end = cur_start = 0
    for i in range(1, len(nums)):
        if cur_sum + nums[i] < nums[i]:
            cur_sum = nums[i]
            cur_start = i
        else:
            cur_sum += nums[i]
        if cur_sum > best_sum:
            best_sum = cur_sum
            best_start = cur_start
            best_end = i
    return best_sum, nums[best_start:best_end+1]

# 环形子数组的最大和
def max_subarray_sum_circular(nums):
    # 最大子数组和 = max(普通最大子数组和, 总和 - 最小子数组和)
    max_sum = cur_max = nums[0]
    min_sum = cur_min = nums[0]
    total = nums[0]
    for x in nums[1:]:
        cur_max = max(x, cur_max + x)
        max_sum = max(max_sum, cur_max)
        cur_min = min(x, cur_min + x)
        min_sum = min(min_sum, cur_min)
        total += x
    if total == min_sum:  # 全是负数
        return max_sum
    return max(max_sum, total - min_sum)
```

## 5. 网格图 DP

### 理解与可视化

```
问题: 从左上角到右下角，只能向右或向下，有多少条路径？

  ┌───┬───┬───┐
  │ 1 │ 1 │ 1 │   dp[i][j] = dp[i-1][j] + dp[i][j-1]
  ├───┼───┼───┤   只能从上面或左边来
  │ 1 │ 2 │ 3 │
  ├───┼───┼───┤
  │ 1 │ 3 │ 6 │   答案: 6
  └───┴───┴───┘

  含障碍物的版本: 障碍物格子 dp 值为 0
```

### 代码模板

```python
# 不同路径
def unique_paths(m, n):
    dp = [[1] * n for _ in range(m)]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    return dp[m-1][n-1]

# 最小路径和
def min_path_sum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    dp[0][0] = grid[0][0]
    for i in range(1, m):
        dp[i][0] = dp[i-1][0] + grid[i][0]
    for j in range(1, n):
        dp[0][j] = dp[0][j-1] + grid[0][j]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
    return dp[m-1][n-1]

# 不同路径 II（有障碍物）
def unique_paths_with_obstacles(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0] * n for _ in range(m)]
    dp[0][0] = 1 if grid[0][0] == 0 else 0
    for i in range(m):
        for j in range(n):
            if grid[i][j] == 1:
                dp[i][j] = 0
            else:
                if i > 0:
                    dp[i][j] += dp[i-1][j]
                if j > 0:
                    dp[i][j] += dp[i][j-1]
    return dp[m-1][n-1]

# 三角形最小路径和
def minimum_total(triangle):
    n = len(triangle)
    dp = triangle[-1][:]  # 从底部开始
    for i in range(n - 2, -1, -1):
        for j in range(i + 1):
            dp[j] = min(dp[j], dp[j+1]) + triangle[i][j]
    return dp[0]
```

## 6. DP 方法论总结

```
从记忆化搜索到递推的转换:

  记忆化搜索                递推
  ┌────────────────┐       ┌─────────────────┐
  │ @cache          │       │ for i in range(): │
  │ def dfs(i):     │  →    │   dp[i] = ...    │
  │   if base: ...  │       │                   │
  │   return ...    │       │ return dp[-1]     │
  └────────────────┘       └─────────────────┘

  dfs(i) 对应 dp[i]
  dfs 的参数 → dp 的维度
  dfs 的递归调用 → dp 的状态转移
  dfs 的 base case → dp 的初始值

  空间优化: 如果 dp[i] 只依赖 dp[i-1] 和 dp[i-2]
           → 可以只用两个变量代替整个数组
```
