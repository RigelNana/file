# 打家劫舍

> 打家劫舍是"选或不选"型 DP 的代表题，变体丰富。

---

## 1. 打家劫舍 I

### LeetCode 198

```
沿街房屋, 不能偷相邻的两家, 求最大金额

nums = [2, 7, 9, 3, 1]

对于第 i 间房, 两种选择:
  偷:   dp[i] = dp[i-2] + nums[i]  (跳过第 i-1 间)
  不偷: dp[i] = dp[i-1]            (和前一间一样)

dp[i] = max(dp[i-1], dp[i-2] + nums[i])

推导:
  i:    0   1   2   3   4
  num:  2   7   9   3   1
  dp:   2   7   11  11  12
        ↑   ↑   ↑
       2  max(2,7) max(7,2+9)

答案: dp[4] = 12 (偷第0,2,4间: 2+9+1=12)
```

```python
def rob(nums):
    if len(nums) <= 2:
        return max(nums)
    dp = [0] * len(nums)
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])      # ⚠️ 前两间取较大的
    for i in range(2, len(nums)):
        dp[i] = max(dp[i-1], dp[i-2] + nums[i])
    return dp[-1]

# 空间优化
def rob_opt(nums):
    prev2 = prev1 = 0                  # ⚠️ prev2 = dp[i-2], prev1 = dp[i-1]
    for x in nums:
        prev2, prev1 = prev1, max(prev1, prev2 + x)
    return prev1

# ⚠️ dp[1] = max(nums[0], nums[1]):
#    不是 nums[1]! 前两间选最大的
#    如果 nums[0] > nums[1], 应该偷第0间
#
# ⚠️ 空间优化的初始值:
#    prev2 = prev1 = 0 (相当于 dp[-2]=0, dp[-1]=0)
#    循环中自然处理 base case
```

## 2. 打家劫舍 II (环形)

### LeetCode 213

```
首尾相邻 (环形), 第一间和最后一间不能同时偷

分解: 两次线性打家劫舍
  情况1: 不偷最后一间 → rob(nums[0..n-2])
  情况2: 不偷第一间   → rob(nums[1..n-1])
  答案 = max(情况1, 情况2)

nums = [2, 3, 2]
  不偷最后: rob([2, 3]) = 3
  不偷第一: rob([3, 2]) = 3
  答案 = 3
```

```python
def rob(nums):
    if len(nums) == 1:
        return nums[0]
    
    def rob_range(arr):
        prev2 = prev1 = 0
        for x in arr:
            prev2, prev1 = prev1, max(prev1, prev2 + x)
        return prev1
    
    return max(rob_range(nums[:-1]), rob_range(nums[1:]))
    # ⚠️ nums[:-1]: 不偷最后一间
    # ⚠️ nums[1:]:  不偷第一间

# ⚠️ 为什么这样拆分是对的?
#    第一间和最后一间至少有一个不偷
#    如果两个都不偷 → 被两种情况都包含了
#    如果偷第一间 → 一定不偷最后间 → 被情况1包含
#    如果偷最后间 → 一定不偷第一间 → 被情况2包含
#
# ⚠️ 特殊: 只有1间房时直接返回 (切片后数组为空)
```

## 3. 打家劫舍 III (树形)

### LeetCode 337

```
房屋排列成二叉树, 直接相连的不能同时偷

思路: 树形DP
  对每个节点, 返回 (偷, 不偷) 两种情况的最大值

  偷当前节点:   val + 不偷左 + 不偷右
  不偷当前节点: max(偷左, 不偷左) + max(偷右, 不偷右)
```

```python
def rob(root):
    def dfs(node):
        if not node:
            return 0, 0              # (偷, 不偷) = (0, 0)
        
        left_rob, left_not = dfs(node.left)
        right_rob, right_not = dfs(node.right)
        
        # ⚠️ 偷当前: 左右都不能偷
        rob_cur = node.val + left_not + right_not
        # ⚠️ 不偷当前: 左右各自取最大
        not_rob_cur = max(left_rob, left_not) + max(right_rob, right_not)
        
        return rob_cur, not_rob_cur
    
    return max(dfs(root))

# ⚠️ 返回元组 (偷, 不偷):
#    避免重复计算子树
#    如果用 @cache 需要把 node 作为 key (不方便)
#    直接返回两个值更高效
```

## 4. 删除并获得点数

### LeetCode 740

```
nums = [3, 4, 2]
选择 x → 获得 x 分, 同时删除所有 x-1 和 x+1

转化为打家劫舍!
  统计每个数出现的总分: sum_arr[x] = x * count(x)
  "选 x" → "偷第 x 间"
  "删 x-1 和 x+1" → "不能偷相邻的"

nums = [2,2,3,3,3,4]
  sum_arr[2] = 4, sum_arr[3] = 9, sum_arr[4] = 4
  打家劫舍: [4, 9, 4]
  答案 = max(4+4, 9) = 9
```

```python
def deleteAndEarn(nums):
    max_val = max(nums)
    earn = [0] * (max_val + 1)
    for x in nums:
        earn[x] += x                    # ⚠️ 统计每个值的总分
    
    # 打家劫舍
    prev2 = prev1 = 0
    for x in earn:
        prev2, prev1 = prev1, max(prev1, prev2 + x)
    return prev1

# ⚠️ earn[x] = x * count(x)
#    等价于 earn[x] += x (遍历nums累加)
#
# ⚠️ 转化: 值相邻 → 索引相邻 → 打家劫舍
```

## 5. 选或不选模式总结

```
"选或不选" 是最基本的 DP 模式:

  dp[i] = max(
      dp[i-1],              ← 不选第 i 个
      dp[i-2] + value[i]    ← 选第 i 个 (跳过 i-1)
  )

变体:
┌──────────────────┬────────────────────────┐
│ 题目             │ 关键转化               │
├──────────────────┼────────────────────────┤
│ 打家劫舍I        │ 直接套模板             │
│ 打家劫舍II (环形)│ 拆成两次线性           │
│ 打家劫舍III(树形)│ DFS返回(偷,不偷)       │
│ 删除并获得点数   │ 转化为打家劫舍         │
│ 不相邻元素最大和 │ 同模板                 │
└──────────────────┴────────────────────────┘

⚠️ 见到"相邻不能同时选" → 打家劫舍模式
```

## 本节要点速查

```
✅ 核心: dp[i] = max(dp[i-1], dp[i-2] + nums[i])
✅ 初始: dp[0]=nums[0], dp[1]=max(nums[0],nums[1])
✅ 环形: 拆成两次线性 (不含首 / 不含尾)
✅ 树形: DFS返回(偷,不偷)元组
✅ 转化: "值相邻不共存" → 打家劫舍
✅ 空间: prev2, prev1 两个变量即可
```
