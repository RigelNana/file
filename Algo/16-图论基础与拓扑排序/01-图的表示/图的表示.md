# 图的表示

> 图是最通用的数据结构，包含节点和边。掌握图的表示方式是图论的基础。

---

## 1. 图的基本概念

```
图 G = (V, E)
  V: 顶点集合 (vertices/nodes)
  E: 边集合 (edges)

分类:
  有向图: 边有方向 u → v
  无向图: 边无方向 u — v (等价于 u→v + v→u)
  加权图: 边有权重 u →(w) v
  DAG: 有向无环图 (Directed Acyclic Graph)

度数:
  无向图: 度 = 相邻边数
  有向图: 入度 (in-degree) = 指向该节点的边数
          出度 (out-degree) = 从该节点出发的边数
```

---

## 2. 三种表示方式

```
示例图:  0 → 1 → 3
         ↓       ↑
         2 ──────┘

1. 邻接矩阵 (Adjacency Matrix):
     0  1  2  3
  0 [0, 1, 1, 0]    适合: 稠密图 (边多)
  1 [0, 0, 0, 1]    空间: O(V²)
  2 [0, 0, 0, 1]    查边: O(1)
  3 [0, 0, 0, 0]    遍历邻居: O(V)

2. 邻接表 (Adjacency List):     ⭐ 最常用
   0 → [1, 2]       适合: 稀疏图 (边少)
   1 → [3]           空间: O(V+E)
   2 → [3]           查边: O(度)
   3 → []            遍历邻居: O(度)

3. 边列表 (Edge List):
   edges = [(0,1), (0,2), (1,3), (2,3)]
   空间: O(E)
   适合: 排序/遍历边
```

## 3. 邻接表构建模板

```python
from collections import defaultdict

# 有向图
def build_directed(n, edges):
    graph = defaultdict(list)
    for u, v in edges:
        graph[u].append(v)
    return graph

# 无向图
def build_undirected(n, edges):
    graph = defaultdict(list)
    for u, v in edges:
        graph[u].append(v)
        graph[v].append(u)           # ⚠️ 双向
    return graph

# 带权图
def build_weighted(n, edges):
    graph = defaultdict(list)
    for u, v, w in edges:
        graph[u].append((v, w))
        graph[v].append((u, w))      # ⚠️ 无向带权
    return graph

# ⚠️ defaultdict(list) vs [[] for _ in range(n)]:
#    defaultdict: 节点编号不连续时也能用
#    列表: 节点编号 0~n-1 且需要快速索引
#
# ⚠️ 也可以用列表初始化:
#    graph = [[] for _ in range(n)]
#    graph[u].append(v)
```

## 4. LeetCode 题目中的图输入

```
常见输入格式:

1. 边列表: edges = [[0,1], [0,2], [1,3]]
   → 直接遍历构建邻接表

2. 先决条件: prerequisites = [[1,0], [2,0]]
   表示 0→1, 0→2 (先修0才能学1)
   ⚠️ 注意方向! [course, prereq] = prereq → course

3. 邻接表直接给: graph = [[1,2],[3],[3],[]]
   graph[i] = 节点i的邻居列表

4. 网格图: grid[i][j]
   → 隐式图, 每个格子和上下左右相邻

5. 节点值+邻居: Node(val, neighbors)
   → 图的克隆等题目
```

```python
# 先决条件 → 邻接表 (注意方向)
def build_from_prereq(numCourses, prerequisites):
    graph = defaultdict(list)
    in_degree = [0] * numCourses
    for course, pre in prerequisites:
        graph[pre].append(course)    # ⚠️ pre → course
        in_degree[course] += 1
    return graph, in_degree

# ⚠️ [1, 0] 表示 "要学1必须先学0"
#    所以边是 0 → 1, 不是 1 → 0
```

```

```
