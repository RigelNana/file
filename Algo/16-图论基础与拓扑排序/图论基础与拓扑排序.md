# 图论基础与拓扑排序

> 图是最通用的数据结构之一。掌握图的表示、遍历和拓扑排序是处理复杂依赖关系问题的基础。

---

## 1. 图的表示

### 理解与可视化

```
图的三种常见表示:

1. 邻接矩阵 (适合稠密图)
   graph[i][j] = 1 表示 i→j 有边

     0  1  2  3
  0 [0, 1, 1, 0]
  1 [0, 0, 0, 1]
  2 [0, 0, 0, 1]
  3 [0, 0, 0, 0]

2. 邻接表 (适合稀疏图, 最常用)
   graph[i] = [j1, j2, ...]

   0 → [1, 2]
   1 → [3]
   2 → [3]
   3 → []

3. 边列表
   edges = [(0,1), (0,2), (1,3), (2,3)]

构建邻接表:
  edges → defaultdict(list)
```

### 代码模板

```python
from collections import defaultdict

# 构建邻接表
def build_graph(n, edges, directed=True):
    graph = defaultdict(list)
    for u, v in edges:
        graph[u].append(v)
        if not directed:
            graph[v].append(u)
    return graph

# 带权图
def build_weighted_graph(n, edges):
    graph = defaultdict(list)
    for u, v, w in edges:
        graph[u].append((v, w))
        graph[v].append((u, w))
    return graph
```

## 2. 图的 DFS 遍历 ⭐

### 理解与可视化

```
DFS: 深度优先，一条路走到黑再回溯

  0 → 1 → 3
  ↓
  2 → 3 (已访问跳过)

  访问顺序: 0 → 1 → 3 → 2

DFS 可以:
  - 判断连通性
  - 找到所有连通分量
  - 检测环
  - 拓扑排序（后序）
```

### 代码模板

```python
# 图的 DFS
def dfs(graph, n):
    visited = [False] * n
    def helper(node):
        visited[node] = True
        for nei in graph[node]:
            if not visited[nei]:
                helper(nei)
    # 遍历所有连通分量
    for i in range(n):
        if not visited[i]:
            helper(i)

# 检测有向图是否有环 (DFS 三色标记)
def has_cycle(graph, n):
    # 0=未访问, 1=访问中, 2=已完成
    color = [0] * n
    def dfs(node):
        color[node] = 1
        for nei in graph[node]:
            if color[nei] == 1:  # 回边 → 有环
                return True
            if color[nei] == 0 and dfs(nei):
                return True
        color[node] = 2
        return False

    return any(color[i] == 0 and dfs(i) for i in range(n))

# 求连通分量个数 (无向图)
def count_components(n, edges):
    graph = defaultdict(list)
    for u, v in edges:
        graph[u].append(v)
        graph[v].append(u)
    visited = [False] * n
    count = 0
    def dfs(node):
        visited[node] = True
        for nei in graph[node]:
            if not visited[nei]:
                dfs(nei)
    for i in range(n):
        if not visited[i]:
            dfs(i)
            count += 1
    return count
```

## 3. 图的 BFS 遍历

### 代码模板

```python
from collections import deque

# 图的 BFS（从某个节点出发的最短路径 - 无权图）
def bfs(graph, start, n):
    dist = [-1] * n
    dist[start] = 0
    queue = deque([start])
    while queue:
        node = queue.popleft()
        for nei in graph[node]:
            if dist[nei] == -1:
                dist[nei] = dist[node] + 1
                queue.append(nei)
    return dist
```

## 4. 拓扑排序 (BFS / Kahn 算法) ⭐⭐

### 理解与可视化

```
拓扑排序: 将 DAG 的节点排成线性序列，使得每条边 u→v 中 u 排在 v 前面

  例: 课程依赖关系
    0 → 1 → 3
    0 → 2 → 3

  入度:  0:0  1:1  2:1  3:2

  Kahn 算法:
  1. 将所有入度=0 的节点入队
  2. 出队 → 处理 → 将邻居入度减1
  3. 邻居入度变0 → 入队
  4. 重复直到队空

  过程:
    队列: [0]
    弹出 0, 1入度→0, 2入度→0 → 队列: [1, 2]
    弹出 1, 3入度→1             → 队列: [2]
    弹出 2, 3入度→0             → 队列: [3]
    弹出 3                      → 队列: []

  结果: [0, 1, 2, 3]

  如果最终排好的列表长度 < n → 有环!
```

### 代码模板

```python
from collections import deque, defaultdict

# BFS 拓扑排序（Kahn 算法）
def topo_sort(n, edges):
    graph = defaultdict(list)
    in_degree = [0] * n
    for u, v in edges:
        graph[u].append(v)
        in_degree[v] += 1

    queue = deque(i for i in range(n) if in_degree[i] == 0)
    order = []
    while queue:
        node = queue.popleft()
        order.append(node)
        for nei in graph[node]:
            in_degree[nei] -= 1
            if in_degree[nei] == 0:
                queue.append(nei)

    return order if len(order) == n else []  # 空=有环

# 课程表（判断能否完成所有课程）
def can_finish(num_courses, prerequisites):
    graph = defaultdict(list)
    in_degree = [0] * num_courses
    for course, pre in prerequisites:
        graph[pre].append(course)
        in_degree[course] += 1

    queue = deque(i for i in range(num_courses) if in_degree[i] == 0)
    count = 0
    while queue:
        node = queue.popleft()
        count += 1
        for nei in graph[node]:
            in_degree[nei] -= 1
            if in_degree[nei] == 0:
                queue.append(nei)
    return count == num_courses

# 课程表 II（返回拓扑排序结果）
def find_order(num_courses, prerequisites):
    graph = defaultdict(list)
    in_degree = [0] * num_courses
    for course, pre in prerequisites:
        graph[pre].append(course)
        in_degree[course] += 1

    queue = deque(i for i in range(num_courses) if in_degree[i] == 0)
    order = []
    while queue:
        node = queue.popleft()
        order.append(node)
        for nei in graph[node]:
            in_degree[nei] -= 1
            if in_degree[nei] == 0:
                queue.append(nei)
    return order if len(order) == num_courses else []
```

## 5. DFS 拓扑排序（后序反转）

### 代码模板

```python
# DFS 拓扑排序
def topo_sort_dfs(n, edges):
    graph = defaultdict(list)
    for u, v in edges:
        graph[u].append(v)

    color = [0] * n  # 0=未访问, 1=访问中, 2=完成
    order = []
    has_cycle = False

    def dfs(node):
        nonlocal has_cycle
        color[node] = 1
        for nei in graph[node]:
            if color[nei] == 1:
                has_cycle = True
                return
            if color[nei] == 0:
                dfs(nei)
                if has_cycle:
                    return
        color[node] = 2
        order.append(node)

    for i in range(n):
        if color[i] == 0:
            dfs(i)
            if has_cycle:
                return []
    order.reverse()
    return order
```

## 6. 二分图判定

### 理解与可视化

```
二分图: 所有节点可以分成两组，每条边连接不同组的节点

  判定方法: 染色法 (BFS/DFS)
    给一个节点染红色
    它的所有邻居染蓝色
    邻居的邻居染红色
    如果发现冲突 → 不是二分图

    红── 蓝
    |    |
    蓝── 红   ✓ 二分图

    红── 蓝
    |  ╲ |
    蓝── 红   ✗ 奇数环 → 不是二分图
```

### 代码模板

```python
from collections import deque

def is_bipartite(graph):
    n = len(graph)
    color = [-1] * n  # -1=未染色, 0=红, 1=蓝
    for start in range(n):
        if color[start] != -1:
            continue
        queue = deque([start])
        color[start] = 0
        while queue:
            node = queue.popleft()
            for nei in graph[node]:
                if color[nei] == -1:
                    color[nei] = 1 - color[node]
                    queue.append(nei)
                elif color[nei] == color[node]:
                    return False
    return True
```
