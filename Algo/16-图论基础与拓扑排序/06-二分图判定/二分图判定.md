# 二分图判定

> 二分图是图论中的重要概念，用染色法判定，在匹配问题中有广泛应用。

---

## 1. 什么是二分图

```
二分图: 所有节点可以分成两组 A 和 B
  每条边都连接 A 中的节点和 B 中的节点
  同组内没有边

判定方法: 2-染色
  如果能用两种颜色染色, 且相邻节点颜色不同 → 是二分图
  等价于: 不包含奇数长度的环

可视化:
  二分图:          非二分图:
  红── 蓝           红── 蓝
  |    |            |  ╲ |
  蓝── 红           蓝── 红  ← 三角形(奇数环)
```

## 2. BFS染色法

### LeetCode 785

```
给定无向图 graph, graph[i] = 节点i的邻居
判断是否是二分图
```

```python
from collections import deque

def isBipartite(graph):
    n = len(graph)
    color = [-1] * n                    # ⚠️ -1=未染色, 0=红, 1=蓝
    
    for start in range(n):              # ⚠️ 处理非连通图
        if color[start] != -1:
            continue
        
        queue = deque([start])
        color[start] = 0                # ⚠️ 起点染红
        
        while queue:
            node = queue.popleft()
            for nei in graph[node]:
                if color[nei] == -1:
                    color[nei] = 1 - color[node]  # ⚠️ 染相反颜色
                    queue.append(nei)
                elif color[nei] == color[node]:   # ⚠️ 相邻同色 = 冲突
                    return False
    
    return True

# ⚠️ 1 - color[node]: 
#    color[node]=0 → 1-0=1 (蓝)
#    color[node]=1 → 1-1=0 (红)
#    简洁的"取反"方式
#
# ⚠️ 外层 for: 非连通图可能有多个连通分量
#    每个分量都要检查
```

## 3. DFS染色法

```python
def isBipartite(graph):
    n = len(graph)
    color = [-1] * n
    
    def dfs(node, c):
        color[node] = c
        for nei in graph[node]:
            if color[nei] == -1:
                if not dfs(nei, 1 - c):     # ⚠️ 邻居染反色
                    return False
            elif color[nei] == c:            # ⚠️ 同色冲突
                return False
        return True
    
    for i in range(n):
        if color[i] == -1:
            if not dfs(i, 0):
                return False
    return True

# ⚠️ DFS 和 BFS 效果一样, 都是 O(V+E)
#    DFS 代码更简洁, BFS 不会栈溢出
```

## 4. 可能的二分法

### LeetCode 886

```
n 个人, dislikes[i] = [a, b] 表示 a 和 b 不喜欢对方
判断能否分成两组, 使得同组内没有互相讨厌的人

本质: 以 dislikes 为边建图, 判断是否为二分图
```

```python
from collections import defaultdict, deque

def possibleBipartition(n, dislikes):
    graph = defaultdict(list)
    for a, b in dislikes:
        graph[a].append(b)
        graph[b].append(a)            # ⚠️ 无向图
    
    color = [-1] * (n + 1)            # ⚠️ 1-indexed
    
    for i in range(1, n + 1):
        if color[i] != -1:
            continue
        
        queue = deque([i])
        color[i] = 0
        
        while queue:
            node = queue.popleft()
            for nei in graph[node]:
                if color[nei] == -1:
                    color[nei] = 1 - color[node]
                    queue.append(nei)
                elif color[nei] == color[node]:
                    return False
    
    return True

# ⚠️ 节点编号 1~n (1-indexed):
#    color 数组大小 n+1
#    遍历 range(1, n+1)
```

## 5. 二分图相关概念

```
二分图的应用:
  · 匹配问题 (匈牙利算法)
  · 任务分配
  · 课程/学生分组
  · 图着色

判定总结:
  ┌──────────────────────┬──────────────────────┐
  │ 是二分图             │ 不是二分图           │
  ├──────────────────────┼──────────────────────┤
  │ 能2-染色             │ 无法2-染色           │
  │ 没有奇数环           │ 存在奇数环           │
  │ 树 (一定是二分图)    │ 三角形               │
  └──────────────────────┴──────────────────────┘

⚠️ 树一定是二分图 (没有环)
⚠️ 偶数环 → 二分图
⚠️ 奇数环 → 非二分图
```

## 本节要点速查

```
✅ 二分图: 2-染色, 相邻不同色
✅ 染反色: 1 - color[node]
✅ 冲突: color[nei] == color[node] → 不是二分图
✅ 非连通图: 每个分量都要检查
✅ 奇数环 → 非二分图, 树 → 一定是二分图
```
