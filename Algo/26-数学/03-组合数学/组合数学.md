# 组合数学

> 排列、组合、杨辉三角、卡特兰数是算法题中常见的数学工具。

---

## 1. 排列与组合公式

```
排列 P(n,k) = n! / (n-k)!       从n个中选k个, 有序
组合 C(n,k) = n! / (k! × (n-k)!)  从n个中选k个, 无序

性质:
  C(n,0) = C(n,n) = 1
  C(n,k) = C(n, n-k)              对称性
  C(n,k) = C(n-1,k-1) + C(n-1,k)  杨辉三角递推
  Σ C(n,k) = 2^n                   所有子集数

⚠️ C(n,k) 中 k > n 或 k < 0 → 结果为 0
```

## 2. 组合数取模 (预处理阶乘)

```
C(n,k) mod p = n! × inv(k!) × inv((n-k)!) mod p

预处理:
  fac[i] = i! mod p           (正序递推)
  inv_fac[i] = (i!)^(-1) mod p (从 inv_fac[n] 倒推)
  
  inv_fac[n] = pow(fac[n], p-2, p)   费马小定理
  inv_fac[i] = inv_fac[i+1] × (i+1) mod p  倒推

时间: O(n) 预处理, O(1) 查询
```

```python
MOD = 10**9 + 7

def precompute(n):
    """预处理阶乘和逆元阶乘"""
    fac = [1] * (n + 1)
    for i in range(1, n + 1):
        fac[i] = fac[i-1] * i % MOD
    
    inv_fac = [1] * (n + 1)
    inv_fac[n] = pow(fac[n], MOD - 2, MOD)  # ⚠️ 费马小定理
    for i in range(n - 1, -1, -1):           # ⚠️ 倒序推导
        inv_fac[i] = inv_fac[i+1] * (i+1) % MOD
    
    return fac, inv_fac

def comb(n, k, fac, inv_fac):
    """O(1) 查询 C(n,k) mod p"""
    if k < 0 or k > n:                    # ⚠️ 边界检查
        return 0
    return fac[n] * inv_fac[k] % MOD * inv_fac[n-k] % MOD

# 使用:
# fac, inv_fac = precompute(200000)
# ans = comb(10, 3, fac, inv_fac)  → 120

# ⚠️ 预处理的 n 要足够大, 覆盖所有查询
# ⚠️ MOD 必须是质数 (费马小定理要求)
```

## 3. 杨辉三角

```
C(n,k) = C(n-1,k-1) + C(n-1,k)

        1
       1 1
      1 2 1
     1 3 3 1
    1 4 6 4 1
   1 5 10 10 5 1

每个数 = 正上方 + 左上方
第 n 行第 k 个 = C(n, k)
```

```python
# LC 118: 杨辉三角
def generate(numRows):
    tri = []
    for i in range(numRows):
        row = [1] * (i + 1)               # ⚠️ 首尾都是 1
        for j in range(1, i):             # ⚠️ j 从 1 到 i-1
            row[j] = tri[i-1][j-1] + tri[i-1][j]
        tri.append(row)
    return tri

# LC 119: 杨辉三角 II (只返回第 rowIndex 行)
def getRow(rowIndex):
    row = [1] * (rowIndex + 1)
    for i in range(1, rowIndex):
        for j in range(i, 0, -1):          # ⚠️ 倒序更新! (类似背包)
            row[j] += row[j-1]
    return row

# ⚠️ 为什么倒序?
#    正序: row[j] = row[j-1] + row[j]
#    但 row[j-1] 已经被更新了 → 用的是新值
#    倒序: row[j-1] 还没被更新 → 用的是旧值 → 正确
```

## 4. 卡特兰数

```
C_n = C(2n, n) / (n+1)

前几项: 1, 1, 2, 5, 14, 42, 132, 429, ...

经典应用:
┌───────────────────────────────────────────┐
│ n 对括号的合法排列数           → C_n       │
│ n 个节点的不同二叉搜索树数     → C_n       │
│ n+1 个数的不同乘法结合方式     → C_n       │
│ 2n 步格路不过对角线的方案数    → C_n       │
│ 出栈序列数                     → C_n       │
│ 凸多边形三角剖分数             → C_{n-2}   │
└───────────────────────────────────────────┘

递推: C_0 = 1, C_n = Σ C_i × C_{n-1-i}  (i: 0 to n-1)
```

```python
# O(n) 递推
def catalan_dp(n):
    dp = [0] * (n + 1)
    dp[0] = 1
    for i in range(1, n + 1):
        for j in range(i):
            dp[i] += dp[j] * dp[i-1-j]    # ⚠️ 左子树 j 个, 右子树 i-1-j 个
    return dp[n]

# O(n) 公式法 (取模)
def catalan_formula(n, mod=10**9+7):
    # C_n = C(2n, n) / (n+1)
    fac, inv_fac = precompute(2 * n)
    c2n_n = fac[2*n] * inv_fac[n] % mod * inv_fac[n] % mod
    return c2n_n * pow(n + 1, mod - 2, mod) % mod
    # ⚠️ 除以 (n+1) → 乘以 (n+1) 的逆元

# LC 96: 不同的二叉搜索树
def numTrees(n):
    return catalan_dp(n)
```

## 5. LeetCode 62: 不同路径

```
m×n 网格, 从左上到右下, 只能向右或向下
共 C(m+n-2, m-1) 种路径

  需要走 m-1 步向下 + n-1 步向右 = 总共 m+n-2 步
  从中选 m-1 步向下 (或 n-1 步向右)
```

```python
def uniquePaths(m, n):
    return comb(m + n - 2, m - 1)
    # 或 math.comb(m+n-2, m-1)  (Python 3.8+)

# DP 解法:
def uniquePaths_dp(m, n):
    dp = [1] * n
    for i in range(1, m):
        for j in range(1, n):
            dp[j] += dp[j-1]
    return dp[n-1]
```

## 本节要点速查

```
✅ C(n,k) = fac[n] × inv_fac[k] × inv_fac[n-k] mod p
✅ 预处理: fac 正序, inv_fac 从 n 倒推
✅ 杨辉三角: C(n,k) = C(n-1,k-1) + C(n-1,k)
✅ 单行杨辉: 倒序更新 (类似背包)
✅ 卡特兰: C_n = C(2n,n)/(n+1), 括号/BST/出栈序列
✅ 不同路径: C(m+n-2, m-1)
✅ k<0 or k>n → C(n,k) = 0
```
