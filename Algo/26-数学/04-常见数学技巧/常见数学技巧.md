## 1. LeetCode 172: 阶乘后的零

```
n! 末尾有多少个 0?

末尾的 0 来自 10 = 2 × 5
2 的因子远多于 5 → 只需统计 5 的因子数

5! = 120           → 1 个 5 → 1 个零
10! = 3628800      → 2 个 5 → 2 个零
25! = ...          → 5 + 1 = 6 个零
                     (25 = 5², 贡献 2 个 5)

公式: count = n//5 + n//25 + n//125 + ...
```

```python
def trailingZeroes(n):
    count = 0
    while n >= 5:
        n //= 5
        count += n                         # ⚠️ 累加 n//5, n//25, ...
    return count

# ⚠️ 不是只算 n//5!
#    25 贡献 2 个 5 (5×5)
#    125 贡献 3 个 5 (5×5×5)
#    所以需要反复除以 5
```

## 2. LeetCode 470: 用 Rand7 实现 Rand10

```
Rand7() 生成 [1,7] 均匀随机数
用它实现 Rand10() 生成 [1,10] 均匀随机数

方法: 拒绝采样
  两次 Rand7 → idx = (a-1)*7 + b, 范围 [1, 49]
  只取 [1, 40] (均匀 mod 10), 拒绝 [41, 49]

  ⚠️ 为什么不直接 Rand7() % 10 + 1?
     7 不能整除 10, 无法均匀映射
```

```python
def rand10():
    while True:
        row = rand7()                      # [1, 7]
        col = rand7()                      # [1, 7]
        idx = (row - 1) * 7 + col          # ⚠️ [1, 49] 均匀
        if idx <= 40:                      # ⚠️ 拒绝 41-49
            return idx % 10 + 1            # ⚠️ [1, 10] 均匀

# ⚠️ 期望调用 Rand7() 次数:
#    每次成功概率 = 40/49
#    期望循环次数 = 49/40 ≈ 1.225
#    每次循环调 2 次 → 总期望 ≈ 2.45 次

# 优化: 拒绝域 [41,49] 可以继续利用
def rand10_optimized():
    while True:
        a = rand7()
        b = rand7()
        idx = (a - 1) * 7 + b             # [1, 49]
        if idx <= 40:
            return idx % 10 + 1
        # idx ∈ [41, 49], 有 9 种 → 继续利用
        a = idx - 40                       # [1, 9]
        b = rand7()                        # [1, 7]
        idx = (a - 1) * 7 + b             # [1, 63]
        if idx <= 60:
            return idx % 10 + 1
        # idx ∈ [61, 63], 有 3 种
        a = idx - 60                       # [1, 3]
        b = rand7()                        # [1, 7]
        idx = (a - 1) * 7 + b             # [1, 21]
        if idx <= 20:
            return idx % 10 + 1
```

## 3. 约瑟夫环

```
n 个人围成一圈, 从第 1 个开始报数
每次数到 k 的人被淘汰, 从下一个继续
求最后存活者的编号

递推公式 (0-indexed):
  f(1) = 0                      只剩1人, 编号0
  f(i) = (f(i-1) + k) % i       (i 从 2 到 n)

  ⚠️ 0-indexed! 如果需要 1-indexed, 最后 +1
```

```python
def josephus(n, k):
    """返回最后存活者编号 (0-indexed)"""
    pos = 0
    for i in range(2, n + 1):             # ⚠️ i 从 2 开始
        pos = (pos + k) % i
    return pos

# 1-indexed: return josephus(n, k) + 1

# ⚠️ 递推原理:
#    i 个人时, 淘汰第 k 个人后
#    问题变成 i-1 个人, 但起点偏移了 k 位
#    f(i) = (f(i-1) + k) % i
```

## 4. LeetCode 7: 整数反转

```python
def reverse(x):
    sign = 1 if x >= 0 else -1
    x = abs(x)
    result = 0
    
    while x:
        digit = x % 10
        x //= 10
        result = result * 10 + digit
    
    result *= sign
    
    # ⚠️ 32位有符号整数范围检查
    if result < -(2**31) or result > 2**31 - 1:
        return 0
    
    return result

# ⚠️ Python 整数无溢出, 但题目要求32位范围
#    -2^31 = -2147483648
#    2^31 - 1 = 2147483647
```

## 5. LeetCode 258: 各位相加 (数字根)

```
反复将各位数字相加, 直到变成一位数

公式: digitRoot(n) = 1 + (n - 1) % 9   (n > 0)

例: 38 → 3+8 = 11 → 1+1 = 2
    1 + (38-1) % 9 = 1 + 37%9 = 1 + 1 = 2 ✓
```

```python
def addDigits(num):
    if num == 0:
        return 0
    return 1 + (num - 1) % 9              # ⚠️ O(1) 公式

# 模拟法
def addDigits_sim(num):
    while num >= 10:
        num = sum(int(d) for d in str(num))
    return num
```

## 6. 其他常用技巧

```python
# 判断回文数 (不转字符串)
def isPalindrome(x):
    if x < 0 or (x % 10 == 0 and x != 0): # ⚠️ 负数 / 末尾0
        return False
    rev = 0
    while x > rev:                         # ⚠️ 只反转一半!
        rev = rev * 10 + x % 10
        x //= 10
    return x == rev or x == rev // 10
    # ⚠️ 奇数位: x == rev // 10 (中间那位不影响)
    # 例: 12321 → x=12, rev=123, 12==123//10 ✓

# 计算 n 选 k 不取模 (Python 大整数)
import math
math.comb(10, 3)  # → 120   (Python 3.8+)

# 最大公约数求法 (多个数)
from math import gcd
from functools import reduce
def gcd_list(nums):
    return reduce(gcd, nums)
```
