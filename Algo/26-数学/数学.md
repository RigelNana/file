## 1. 质数与质因数分解

### 理解与可视化

```
质数: 大于 1 且只能被 1 和自己整除的数
  2, 3, 5, 7, 11, 13, 17, 19, 23, ...

判断质数: 试除到 √n
  只需检查 2 ~ √n 的因子

埃拉托斯特尼筛法 (Sieve):
  找出 [2, n] 中所有质数
  从 2 开始, 标记所有倍数为合数
  时间: O(n log log n)
```

### 代码模板

```python
# 判断质数
def is_prime(n):
    if n < 2:
        return False
    if n < 4:
        return True
    if n % 2 == 0 or n % 3 == 0:
        return False
    i = 5
    while i * i <= n:
        if n % i == 0 or n % (i + 2) == 0:
            return False
        i += 6
    return True

# 埃氏筛
def sieve(n):
    is_p = [True] * (n + 1)
    is_p[0] = is_p[1] = False
    for i in range(2, int(n**0.5) + 1):
        if is_p[i]:
            for j in range(i*i, n + 1, i):
                is_p[j] = False
    return [i for i in range(2, n + 1) if is_p[i]]

# 质因数分解
def factorize(n):
    factors = []
    d = 2
    while d * d <= n:
        while n % d == 0:
            factors.append(d)
            n //= d
        d += 1
    if n > 1:
        factors.append(n)
    return factors
```

## 2. GCD / LCM / 快速幂

### 代码模板

```python
import math

# GCD (最大公约数) — Python 内置
# math.gcd(a, b)

# LCM (最小公倍数)
def lcm(a, b):
    return a * b // math.gcd(a, b)

# 快速幂
def power(base, exp, mod=10**9+7):
    result = 1
    base %= mod
    while exp > 0:
        if exp & 1:
            result = result * base % mod
        base = base * base % mod
        exp >>= 1
    return result

# 模逆元 (费马小定理, mod 为质数)
def mod_inverse(a, mod=10**9+7):
    return power(a, mod - 2, mod)
```

## 3. 组合数学

### 代码模板

```python
# 组合数 C(n, k) 带取模
def comb_mod(n, k, mod=10**9+7):
    if k > n or k < 0:
        return 0
    # 预处理阶乘及逆元
    fac = [1] * (n + 1)
    for i in range(1, n + 1):
        fac[i] = fac[i-1] * i % mod
    inv_fac = [1] * (n + 1)
    inv_fac[n] = power(fac[n], mod - 2, mod)
    for i in range(n - 1, -1, -1):
        inv_fac[i] = inv_fac[i+1] * (i+1) % mod
    return fac[n] * inv_fac[k] % mod * inv_fac[n-k] % mod

# 杨辉三角生成
def generate_pascal(n):
    tri = [[1] * (i + 1) for i in range(n)]
    for i in range(2, n):
        for j in range(1, i):
            tri[i][j] = tri[i-1][j-1] + tri[i-1][j]
    return tri

# 卡特兰数 (第 n 个)
def catalan(n, mod=10**9+7):
    return comb_mod(2*n, n, mod) * power(n + 1, mod - 2, mod) % mod
```

## 4. 常见数学技巧

### 代码模板

```python
# 用 Rand7 实现 Rand10（拒绝采样）
def rand10():
    while True:
        row = rand7()
        col = rand7()
        idx = (row - 1) * 7 + col  # 1~49 均匀
        if idx <= 40:
            return idx % 10 + 1

# 约瑟夫环
def josephus(n, k):
    """n 个人, 每次数 k 个淘汰, 返回最后存活者编号 (0-indexed)"""
    pos = 0
    for i in range(2, n + 1):
        pos = (pos + k) % i
    return pos

# 阶乘后的零（统计 5 的因子数）
def trailing_zeroes(n):
    count = 0
    while n >= 5:
        n //= 5
        count += n
    return count
```
