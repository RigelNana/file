# 最长回文子序列

> 区间 DP 的入门题。dp[i][j] 表示 s[i..j] 的最长回文子序列长度，注意循环方向。

---

## 1. 问题与可视化

```
问题: 找最长的回文子序列 (可以不连续)
示例: s = "bbbab" → "bbbb"，长度=4

  b b b a b
  ↕ ↕ ↕   ↕   → 取 b,b,b,b → 回文 ✓
  
⚠️ 回文子序列 vs 回文子串:
   子序列: 可以不连续, 保持相对顺序
   子串:   必须连续

   "bbbab" 的最长回文子串 = "bbb" (长度3)
   "bbbab" 的最长回文子序列 = "bbbb" (长度4)
```

---

## 2. 状态定义与转移

```
状态: dp[i][j] = s[i..j] (闭区间) 中的最长回文子序列长度

转移 (考虑两端字符):
  s[i] == s[j]:
    dp[i][j] = dp[i+1][j-1] + 2
    两端匹配, 各贡献 1 个字符

  s[i] != s[j]:
    dp[i][j] = max(dp[i+1][j], dp[i][j-1])
    不匹配, 去掉左端 or 去掉右端, 取较大

Base case:
  dp[i][i] = 1     (单个字符是长度 1 的回文)
  dp[i][i-1] = 0   (空区间, 长度 0)

填表 (s = "bbbab"):
       b  b  b  a  b
  b  [ 1  2  3  3  4 ]
  b  [ 0  1  2  2  3 ]
  b  [ 0  0  1  1  3 ]
  a  [ 0  0  0  1  1 ]
  b  [ 0  0  0  0  1 ]

答案: dp[0][4] = 4

⚠️ 只填上三角 (i ≤ j)
   dp[i][j] 依赖 dp[i+1][j-1], dp[i+1][j], dp[i][j-1]
   → 必须先算小区间, 再算大区间
```

---

## 3. 按区间长度枚举

```python
def longestPalindromeSubseq(s):
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    
    # base case: 长度 1
    for i in range(n):
        dp[i][i] = 1
    
    # ⚠️ 按长度从 2 到 n 枚举
    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1       # ⚠️ j = i + length - 1
            if s[i] == s[j]:
                dp[i][j] = dp[i+1][j-1] + 2
            else:
                dp[i][j] = max(dp[i+1][j], dp[i][j-1])
    
    return dp[0][n-1]

# ⚠️ j = i + length - 1, 不是 i + length
#    length=2 时 j=i+1 (两个相邻字符)
#    
# ⚠️ dp[i+1][j-1] 当 length=2 时:
#    dp[i+1][i] → 这是空区间, 值为 0 ✓
#    所以 dp[i][i+1] = 0+2 = 2 (如果两端相等)
```

## 4. 按行逆序枚举

```python
def longestPalindromeSubseq_v2(s):
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    
    # ⚠️ i 从 n-1 到 0 (从下往上)
    for i in range(n - 1, -1, -1):
        dp[i][i] = 1
        # ⚠️ j 从 i+1 到 n-1 (从左往右)
        for j in range(i + 1, n):
            if s[i] == s[j]:
                dp[i][j] = dp[i+1][j-1] + 2
            else:
                dp[i][j] = max(dp[i+1][j], dp[i][j-1])
    
    return dp[0][n-1]

# ⚠️ 为什么 i 从大到小?
#    dp[i][j] 依赖 dp[i+1][...] (下面的行)
#    必须先算 i+1 行, 再算 i 行
#    所以 i 从 n-1 到 0
```

## 5. 空间优化

```python
def longestPalindromeSubseq_opt(s):
    n = len(s)
    dp = [0] * n
    
    for i in range(n - 1, -1, -1):
        dp[i] = 1
        prev = 0   # ⚠️ prev = 旧的 dp[i+1][j-1]
        for j in range(i + 1, n):
            temp = dp[j]    # 保存旧值 dp[i+1][j]
            if s[i] == s[j]:
                dp[j] = prev + 2
            else:
                dp[j] = max(dp[j], dp[j-1])
            prev = temp
    
    return dp[n-1]

# ⚠️ 和 LCS 的空间优化类似
#    prev 保存 dp[i+1][j-1] 的旧值
```

## 6. LeetCode 647: 回文子串

```
题目: 计算字符串中回文子串的个数
示例: "aaa" → 6 ("a","a","a","aa","aa","aaa")

⚠️ 这是回文子串 (连续), 不是回文子序列!

方法1: 中心扩展 (更快)
方法2: 区间 DP
  dp[i][j] = s[i..j] 是否是回文
  数 dp[i][j]=True 的个数
```

```python
# 中心扩展
def countSubstrings(s):
    n = len(s)
    count = 0
    
    def expand(l, r):
        nonlocal count
        while l >= 0 and r < n and s[l] == s[r]:
            count += 1
            l -= 1
            r += 1
    
    for i in range(n):
        expand(i, i)       # 奇数长度
        expand(i, i + 1)   # 偶数长度
    
    return count

# 区间 DP
def countSubstrings_dp(s):
    n = len(s)
    dp = [[False] * n for _ in range(n)]
    count = 0
    
    for i in range(n - 1, -1, -1):
        for j in range(i, n):
            if s[i] == s[j]:
                # ⚠️ j-i <= 2: 长度1,2,3 且两端相等 → 回文
                dp[i][j] = (j - i <= 2) or dp[i+1][j-1]
            if dp[i][j]:
                count += 1
    
    return count
```

## 7. 转化为 LCS

```
另一种思路: 最长回文子序列 = s 和 reverse(s) 的 LCS

  s = "bbbab"
  rev = "babbб" 
  LCS(s, rev) = "bbbb" → 长度 4 ✓

但直接用区间 DP 更常见，因为更容易理解和扩展
```

```python
def longestPalindromeSubseq_lcs(s):
    rev = s[::-1]
    n = len(s)
    dp = [[0] * (n + 1) for _ in range(n + 1)]
    for i in range(1, n + 1):
        for j in range(1, n + 1):
            if s[i-1] == rev[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[n][n]
```

## 本节要点速查

```
✅ dp[i][j] = s[i..j] 的最长回文子序列长度
✅ 匹配: dp[i+1][j-1]+2; 不匹配: max(dp[i+1][j], dp[i][j-1])
✅ 两种枚举: 按长度从小到大 or i 从大到小
✅ base case: dp[i][i]=1 (单字符)
✅ j-i<=2 且 s[i]==s[j] → 直接是回文 (不需要查内部)
✅ 空间优化需要 prev 保存旧值
✅ 也可转化: LPS(s) = LCS(s, reverse(s))
```
