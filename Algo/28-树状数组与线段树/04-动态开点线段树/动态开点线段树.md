# 动态开点线段树

> 值域巨大 (如 10^9) 但操作稀疏时，不预先分配所有节点，用字典按需创建。

---

## 1. 核心思想

```
普通线段树: 预分配 4n 数组, n=10^9 → 内存爆炸!

动态开点: 只在需要时创建节点
  初始: 只有根节点
  修改/查询时, 遇到不存在的节点再创建

  适用场景:
    - 值域 [1, 10^9] 但操作只有 10^5 次
    - 需要 O(log n) 的区间操作
    - 每次操作最多创建 O(log n) 个节点
    - 总节点数 ≤ O(Q × log n), Q 是操作次数

  Python 实现: 用字典 (dict) 代替数组
    tree[node] = 节点值
    lazy[node] = 懒标记
    不存在的 key → 默认值 0

  ⚠️ 也可以用类节点 (class Node), 但字典更简洁
```

## 2. 代码模板

```python
class DynamicSegTree:
    """动态开点线段树 (区间加 + 区间和)"""
    def __init__(self):
        self.tree = {}                     # ⚠️ 用字典代替数组
        self.lazy = {}
    
    def update(self, ql, qr, val, o=1, l=1, r=10**9):
        """[ql, qr] 区间加 val"""
        if ql <= l and r <= qr:            # ⚠️ 完全包含
            self.tree[o] = self.tree.get(o, 0) + val * (r - l + 1)
            self.lazy[o] = self.lazy.get(o, 0) + val
            return
        mid = (l + r) // 2
        self._push_down(o, l, r, mid)      # ⚠️ 先下传
        if ql <= mid:
            self.update(ql, qr, val, 2*o, l, mid)
        if qr > mid:
            self.update(ql, qr, val, 2*o+1, mid+1, r)
        self.tree[o] = self.tree.get(2*o, 0) + self.tree.get(2*o+1, 0)
    
    def query(self, ql, qr, o=1, l=1, r=10**9):
        """查询 [ql, qr] 区间和"""
        if ql <= l and r <= qr:
            return self.tree.get(o, 0)
        mid = (l + r) // 2
        self._push_down(o, l, r, mid)
        res = 0
        if ql <= mid:
            res += self.query(ql, qr, 2*o, l, mid)
        if qr > mid:
            res += self.query(ql, qr, 2*o+1, mid+1, r)
        return res
    
    def _push_down(self, o, l, r, mid):
        """下传懒标记"""
        lz = self.lazy.get(o, 0)
        if lz:
            # 更新左子
            self.tree[2*o] = self.tree.get(2*o, 0) + lz * (mid - l + 1)
            self.lazy[2*o] = self.lazy.get(2*o, 0) + lz
            # 更新右子
            self.tree[2*o+1] = self.tree.get(2*o+1, 0) + lz * (r - mid)
            self.lazy[2*o+1] = self.lazy.get(2*o+1, 0) + lz
            # 清除
            self.lazy[o] = 0              # ⚠️ 必须清零

# ⚠️ .get(key, 0): 不存在返回 0, 不会 KeyError
# ⚠️ 左子 2*o, 右子 2*o+1 (和普通线段树一样)
# ⚠️ 默认值域 [1, 10^9], 按题目调整
```

## 3. LeetCode 732: 我的日程安排表 III

```
多次调用 book(start, end):
  在 [start, end) 添加一个日程
  返回当前最大重叠日程数 (K重预订)

方法: 动态开点线段树
  book(start, end) → update(start, end-1, +1)
  查询整个值域的最大值
```

```python
class DynSegTreeMax:
    """动态开点: 区间加 + 区间最大值"""
    def __init__(self):
        self.tree = {}
        self.lazy = {}
    
    def update(self, ql, qr, val, o=1, l=0, r=10**9):
        if ql <= l and r <= qr:
            self.tree[o] = self.tree.get(o, 0) + val
            self.lazy[o] = self.lazy.get(o, 0) + val
            return
        mid = (l + r) // 2
        self._push_down(o)
        if ql <= mid:
            self.update(ql, qr, val, 2*o, l, mid)
        if qr > mid:
            self.update(ql, qr, val, 2*o+1, mid+1, r)
        self.tree[o] = max(self.tree.get(2*o, 0), self.tree.get(2*o+1, 0))
    
    def _push_down(self, o):
        lz = self.lazy.get(o, 0)
        if lz:
            for child in [2*o, 2*o+1]:
                self.tree[child] = self.tree.get(child, 0) + lz
                self.lazy[child] = self.lazy.get(child, 0) + lz
            self.lazy[o] = 0

class MyCalendarThree:
    def __init__(self):
        self.seg = DynSegTreeMax()
    
    def book(self, start, end):
        self.seg.update(start, end - 1, 1)  # ⚠️ [start, end) → [start, end-1]
        return self.seg.tree.get(1, 0)       # ⚠️ 根节点=全局最大值
```

## 4. 节点类实现 (替代方案)

```python
class Node:
    __slots__ = ['left', 'right', 'val', 'lazy']
    def __init__(self):
        self.left = None
        self.right = None
        self.val = 0
        self.lazy = 0

class DynSegTree2:
    def __init__(self):
        self.root = Node()
    
    def update(self, ql, qr, val, node=None, l=0, r=10**9):
        if node is None:
            node = self.root
        if ql <= l and r <= qr:
            node.val += val * (r - l + 1)
            node.lazy += val
            return
        mid = (l + r) // 2
        if not node.left:                  # ⚠️ 按需创建
            node.left = Node()
        if not node.right:
            node.right = Node()
        self._push_down(node, l, r, mid)
        if ql <= mid:
            self.update(ql, qr, val, node.left, l, mid)
        if qr > mid:
            self.update(ql, qr, val, node.right, mid+1, r)
        node.val = node.left.val + node.right.val
    
    def _push_down(self, node, l, r, mid):
        if node.lazy:
            node.left.val += node.lazy * (mid - l + 1)
            node.left.lazy += node.lazy
            node.right.val += node.lazy * (r - mid)
            node.right.lazy += node.lazy
            node.lazy = 0

# ⚠️ 节点类实现: 节点用指针连接
#    优点: 更面向对象
#    缺点: Python 中比字典略慢 (除非用 __slots__)
```

## 5. 动态开点 vs 离散化

```
┌─────────────┬──────────────────────┬──────────────────────┐
│             │ 动态开点              │ 离散化 + 普通线段树   │
├─────────────┼──────────────────────┼──────────────────────┤
│ 实现复杂度  │ 中等 (字典实现简单)   │ 中等 (需要预处理排序)  │
│ 支持在线    │ ✓ (无需预知所有操作)  │ ✗ (需要预知所有坐标)  │
│ 空间        │ O(Q log V)           │ O(Q)                 │
│ 常数        │ 字典稍慢              │ 数组更快              │
│ 适用场景    │ 在线, 流式数据        │ 离线, 预知所有操作     │
└─────────────┴──────────────────────┴──────────────────────┘

⚠️ 如果操作可以离线处理 (预知所有坐标):
   离散化 + 普通线段树 更快
⚠️ 如果是在线操作 (不预知坐标):
   动态开点线段树
```

## 本节要点速查

```
✅ 动态开点: 字典代替数组, .get(key, 0) 默认值
✅ 适用: 值域大(10^9), 操作少(10^5)
✅ 每次操作创建 O(log V) 个节点
✅ 结构与普通线段树完全一致 (2*o, 2*o+1)
✅ push_down 时也用 .get 获取子节点值
✅ 在线操作→动态开点, 离线→离散化+普通线段树
```
